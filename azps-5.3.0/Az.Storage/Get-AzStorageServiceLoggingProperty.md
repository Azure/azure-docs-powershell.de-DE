# Get-AzStorageServiceLoggingProperty

## SYNOPSIS
Ruft Protokolleigenschaften für Azure Storage Services ab.

## SYNTAX

```
Get-AzStorageServiceLoggingProperty [-ServiceType] <StorageServiceType> [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzStorageServiceLoggingProperty"** ruft Protokolleigenschaften für Azure Storage Services ab.

## BEISPIELE

### Beispiel 1: Protokollierungseigenschaften für den Blob Service erhalten
```
C:\PS>Get-AzStorageServiceLoggingProperty -ServiceType Blob
```

Dieser Befehl ruft Protokolleigenschaften für blob storage ab.

### -ServiceType
Gibt den Speicherdiensttyp an.
Dieses Cmdlet ruft die Protokolleigenschaften für den Diensttyp ab, den dieser Parameter angibt.
Die zulässigen Werte für diesen Parameter sind:
- BLOB 
- Tabelle
- Warteschlange
- Datei Der Wert von "Datei" wird derzeit nicht unterstützt.
