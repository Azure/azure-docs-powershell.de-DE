# Enable-AzStorageStaticWebsite

## SYNOPSIS
Aktivieren Sie die statische Website für das Azure Storage-Konto.

## SYNTAX

```
Enable-AzStorageStaticWebsite [[-IndexDocument] <String>] [[-ErrorDocument404Path] <String>] [-PassThru]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Enable-AzStorageStaticWebsite"** aktiviert die statische Website für das Azure Storage-Konto.

## BEISPIELE

### Beispiel 1: Aktivieren der statischen Website für das Azure -Speicherkonto
```
C:\PS>Enable-AzStorageStaticWebsite -IndexDocument $indexdoc -ErrorDocument404Path $errordoc
```

Dieser Befehl aktiviert die statische Website für das Azure Storage-Konto.
