# New-AzRecoveryServicesVault

## SYNOPSIS
Erstellt einen neuen Wiederherstellungsdienste-Tresor.

## SYNTAX

```
New-AzRecoveryServicesVault -Name <String> -ResourceGroupName <String> -Location <String> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "New-AzRecoveryServicesVault"** erstellt einen neuen Wiederherstellungsdienst-Tresor.

## BEISPIELE

### Beispiel 1
```
PS C:\> New-AzRecoveryServicesVault -Name "vaultName" -ResourceGroupName "rg" -Location "eastasia"
```

Erstellen Sie einen Wiederherstellungsdiensttresor in der Ressourcengruppe und dem angegebenen Speicherort.
