# Get-AzRecoveryServicesBackupManagementServer

## SYNOPSIS
Ruft SCDPM- und Azure Backup Management Servers ab.

## SYNTAX

```
Get-AzRecoveryServicesBackupManagementServer [[-Name] <String>] [-VaultId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzRecoveryServicesBackupManagementServer"** ruft eine Liste der Sicherungsverwaltungsserver ab, die in einem Tresor registriert sind.
Es gibt zwei Arten von Sicherungsverwaltungsservern: System Center Data Protection Manager (SCDPM) und Azure Backup Management Servers.
Sicherungsverwaltungsserver werden separat installiert, um die Sicherungs orchestrierung zu verwalten.
Legen Sie den Tresorkontext mithilfe des cmdlets Set-AzRecoveryServicesVaultContext, bevor Sie das aktuelle Cmdlet verwenden.

## BEISPIELE

### Beispiel 1: Alle Sicherungsverwaltungsserver erhalten
```
PS C:\>Get-AzRecoveryServicesBackupManagementServer
```

Dieser Befehl ruft alle Sicherungsverwaltungsserver ab, die beim Tresor registriert sind.
