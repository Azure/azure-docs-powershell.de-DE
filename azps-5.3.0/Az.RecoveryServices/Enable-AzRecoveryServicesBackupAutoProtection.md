# Enable-AzRecoveryServicesBackupAutoProtection

## SYNOPSIS
Das **Cmdlet "Enable-AzRecoveryServicesBackupAutoProtection"** richtet den automatischen Schutz von aktuellen und zukünftigen SQL-DBs innerhalb der angegebenen Instanz mit der angegebenen Richtlinie ein.

## SYNTAX

```
Enable-AzRecoveryServicesBackupAutoProtection [-InputItem] <ProtectableItemBase>
 [-BackupManagementType] <BackupManagementType> [-WorkloadType] <WorkloadType> [-Policy] <PolicyBase>
 [-PassThru] [-VaultId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Mit diesem Befehl können Benutzer automatisch alle vorhandenen ungeschützten SQL DBs und db-Dateien schützen, die später mit der angegebenen Richtlinie hinzugefügt werden.  Da die Anweisung zum Sichern aller zukünftigen Psps ist, erfolgt der Vorgang auf SQLInstance-Ebene. Der Azure-Sicherungsdienst überprüft dann automatisch geschützte Container regelmäßig nach neuen PSP-Dateien und schützt sie automatisch.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $Pol = Get-AzRecoveryServicesBackupProtectionPolicy -Name "DefaultSQLPolicy"
PS C:\> $SQLInstance = Get-AzRecoveryServicesBackupProtectableItem -workloadType MSSQL -ItemType SQLInstance -VaultId $targetVault.ID -Name "MSSQLInstance" -ServerName "TestSQLServer" 
PS C:\> Enable-AzRecoveryServicesBackupAutoProtection -InputItem $SQLInstance -BackupManagementType AzureWorkload -WorkloadType MSSQL -Policy $Pol -VaultId $targetvault.ID 
```

Das erste Cmdlet ruft ein Standardrichtlinienobjekt ab und speichert es dann in der $Pol Variable.
Das zweite Cmdlet ruft die relevante SQLInstance ab, bei der es sich um ein beschutzbares Element handelt. Der dritte Befehl richtet dann den automatischen Schutz für diese Instanz mithilfe der Richtlinie in der $Pol.

### Beispiel 2

Mit diesen Befehlen können Benutzer automatisch alle vorhandenen nicht geschützten DB-Dateien und db-Dateien schützen, die später mit der angegebenen Richtlinie hinzugefügt werden. (automatisch generiert)

```powershell <!-- Aladdin Generated Example --> 
Enable-AzRecoveryServicesBackupAutoProtection -BackupManagementType AzureVM -InputItem <ProtectableItemBase> -Policy $Pol -VaultId $vault.ID -WorkloadType AzureVM
```
