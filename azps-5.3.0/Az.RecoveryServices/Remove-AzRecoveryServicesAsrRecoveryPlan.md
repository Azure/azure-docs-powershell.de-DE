# Remove-AzRecoveryServicesAsrRecoveryPlan

## SYNOPSIS
Löscht den angegebenen ASR-Wiederherstellungsplan aus dem Recovery Services-Tresor.

## SYNTAX

### ByObject (Standard)
```
Remove-AzRecoveryServicesAsrRecoveryPlan -InputObject <ASRRecoveryPlan>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByName
```
Remove-AzRecoveryServicesAsrRecoveryPlan -Name <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzRecoveryServicesAsrRecoveryPlan"** löscht den angegebenen Wiederherstellungsplan aus dem Recovery Services-Tresor.

## BEISPIELE

### Beispiel 1
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrRecoveryPlan -RecoveryPlan $RP
```

Startet das Löschen des angegebenen Wiederherstellungsplans und gibt den ASR-Auftrag zurück, mit dem der Vorgang nachverfolgt wird.
