# Get-AzRecoveryServicesVaultProperty

## SYNOPSIS
Gibt die Eigenschaften eines Wiederherstellungsdiensttresor zurück.

## SYNTAX

```
Get-AzRecoveryServicesVaultProperty [-VaultId <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzRecoveryServicesVaultProperty"** gibt die Eigenschaften eines Tresors für Wiederherstellungsdienste zurück.

## BEISPIELE

### Beispiel 1: Erhalten der Eigenschaften eines Tresors
```
PS C:\> $vault = Get-AzRecoveryServicesVault -ResourceGroupName "resourceGroup" -Name "vaultName"
PS C:\> $props = Get-AzRecoveryServicesVaultProperty -VaultId $vault.Id
PS C:\> $encryption.encryptionProperties
```

Der erste Befehl ruft ein Tresorobjekt ab und speichert es dann in der $vault Variable.
Der zweite Befehl ruft die Tresoreigenschaften ab. Als Nächstes greifen wir auf die encryptionProperties des Tresors zu.
