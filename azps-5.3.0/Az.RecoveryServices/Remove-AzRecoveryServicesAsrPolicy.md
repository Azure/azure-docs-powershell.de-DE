# Remove-AzRecoveryServicesAsrPolicy

## SYNOPSIS
Löscht die angegebene ASR-Replikationsrichtlinie aus dem Recovery Services-Tresor.

## SYNTAX

```
Remove-AzRecoveryServicesAsrPolicy -InputObject <ASRPolicy> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzRecoveryServicesAsrPolicy"** hat die angegebene Replikationsrichtlinie aus dem Wiederherstellungsdiensttresor gelöscht.

## BEISPIELE

### Beispiel 1
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrPolicy -Policy $Policy
```

Startet das Löschen der angegebenen Replikationsrichtlinie und gibt den ASR-Auftrag zurück, mit dem der Vorgang nachverfolgt wird.
