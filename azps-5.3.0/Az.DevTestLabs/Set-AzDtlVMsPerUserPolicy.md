# Set-AzDtlVMsPerUserPolicy

## SYNOPSIS
Legt die virtuellen Computer pro Benutzerrichtlinie einer Übungseinheit in DevTest Labs fest.

## SYNTAX

### Aktivieren (Standard)
```
Set-AzDtlVMsPerUserPolicy [[-MaxVMs] <Int32>] [-Enable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Deaktivieren
```
Set-AzDtlVMsPerUserPolicy [[-MaxVMs] <Int32>] [-Disable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Set-AzDtlVMsPerUserPolicy"** legt die virtuellen Computer pro Benutzerrichtlinie einer Übungseinheit fest, mit der die maximal zulässige Anzahl virtueller Computer pro Benutzer festgelegt wird.
Das Cmdlet verwendet die angegebene Ressourcengruppe und den Namen der Übungseinheit zum Festlegen der Richtlinie.
