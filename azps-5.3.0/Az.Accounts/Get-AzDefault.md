# Get-AzDefault

## SYNOPSIS
Holen Sie sich die vom Benutzer im aktuellen Kontext festgelegten Standardwerte.

## SYNTAX

```
Get-AzDefault [-ResourceGroup] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das Get-AzDefault ruft die Ressourcengruppe ab, die der Benutzer im aktuellen Kontext als Standard festgelegt hat.

## BEISPIELE

### Beispiel 1
```
PS C:\> Get-AzDefault

Id         : /subscriptions/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx/resourceGroups/myResourceGroup
Name       : myResourceGroup
Properties : Microsoft.Azure.Management.Internal.Resources.Models.ResourceGroupProperties
Location   : eastus
ManagedBy  :
Tags       :
```

Dieser Befehl gibt die aktuellen Standardwerte zur端ck, wenn Standardwerte festgelegt sind, oder nichts, wenn kein Standardwert festgelegt ist.

### Beispiel 2
```
PS C:\> Get-AzDefault -ResourceGroup

Id         : /subscriptions/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx/resourceGroups/myResourceGroup
Name       : myResourceGroup
Properties : Microsoft.Azure.Management.Internal.Resources.Models.ResourceGroupProperties
Location   : eastus
ManagedBy  :
Tags       :
```

Dieser Befehl gibt die aktuelle Standardressourcengruppe zur端ck, wenn ein Standardsatz festgelegt ist, oder nichts, wenn kein Standardwert festgelegt ist.
