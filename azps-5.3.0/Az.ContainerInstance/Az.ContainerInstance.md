# Az.ContainerInstance Module
## Beschreibung
In diesem Thema werden Hilfethemen für die Azure Container-Instanz-Cmdlets angezeigt.

## Az.ContainerInstance-Cmdlets
### [Get-AzContainerGroup](Get-AzContainerGroup.md)
Ruft Containergruppen ab.

### [Get-AzContainerInstanceLog](Get-AzContainerInstanceLog.md)
Sie können die Protokolle einer Containerinstanz in einer Containergruppe erhalten.

### [New-AzContainerGroup](New-AzContainerGroup.md)
Erstellt eine Containergruppe.

### [Remove-AzContainerGroup](Remove-AzContainerGroup.md)
Entfernt eine Containergruppe.
