# Az.Maps-Modul
## Beschreibung
In diesem Thema finden Sie Hilfe zu den Azure Maps-Cmdlets.

## Az.Maps-Cmdlets
### [Get-AzMapsAccount](Get-AzMapsAccount.md)
Ruft das Konto ab.

### [Get-AzMapsAccountKey](Get-AzMapsAccountKey.md)
Ruft die API-Schlüssel für ein Konto ab.
Diese Schlüssel sind der Authentifizierungsmechanismus, der bei nachfolgenden Aufrufen von Azure Maps verwendet wird.

### [New-AzMapsAccount](New-AzMapsAccount.md)
Erstellt ein Azure Maps-Konto.

### [New-AzMapsAccountKey](New-AzMapsAccountKey.md)
Generiert einen Kontoschlüssel erneut.

### [Remove-AzMapsAccount](Remove-AzMapsAccount.md)
Löscht ein Azure Maps-Konto.
