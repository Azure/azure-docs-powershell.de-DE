# Get-AzDeploymentScriptLog

## SYNOPSIS
Ruft das Protokoll der Ausführung eines Bereitstellungsskripts ab.

## SYNTAX

### GetDeploymentScriptLogByName (Standard)
```
Get-AzDeploymentScriptLog [-ResourceGroupName] <String> [-Name] <String> [[-Tail] <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetDeploymentScriptLogByResourceId
```
Get-AzDeploymentScriptLog [-DeploymentScriptResourceId] <String> [[-Tail] <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetDeploymentScriptLogByInputObject
```
Get-AzDeploymentScriptLog [-DeploymentScriptObject] <PsDeploymentScript> [[-Tail] <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzDeploymentScriptLog"** ruft das Protokoll der Ausführung eines Bereitstellungsskripts ab.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Get-AzDeploymentScriptLog -Name MyDeploymentScript -ResourceGroupName DS-TestRg
```

Ruft das Protokoll eines Bereitstellungsskripts mit dem Namen "MyDeploymentScript" in der Ressourcengruppe "DS-TestRG" ab.

### Beispiel 2
```powershell
PS C:\> Get-AzDeploymentScriptLog -Name MyDeploymentScript -ResourceGroupName DS-TestRg -Tail 3
```

Ruft die letzten drei Zeilen des Protokolls eines Bereitstellungsskripts mit dem Namen "MyDeploymentScript" in der Ressourcengruppe "DS-TestRG" ab.

### Beispiel 3
```powershell
PS C:\> $ds = Get-AzDeploymentScript -Name MyDeploymentScript -ResourceGroupName DS-TestRg
PS C:\> Get-AzDeploymentScriptLog -DeploymentScriptInputObject $ds
```

Der erste Befehl ruft ein Bereitstellungsskript mit dem Namen "MyDeploymentScript" in der Ressourcengruppe "DS-TestRG" ab.
Der zweite Befehl ruft das Protokoll des angegebenen Bereitstellungsskripts ab.
