# Merge-AzReservation

## SYNOPSIS
Führt zwei `Reservation` Zellen zusammen.

## SYNTAX

### Befehlszeile (Standard)
```
Merge-AzReservation -ReservationOrderId <Guid> -ReservationId <Guid[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### PipeObject
```
Merge-AzReservation -Reservation <PSReservation[]> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Führen Sie die `Reservation` angegebenen S in einem neuen `Reservation` zusammen. Die beiden `Reservation` zusammengeführten Daten müssen über die gleichen Eigenschaften verfügen.

## BEISPIELE

### Beispiel 1
```
PS C:\> Merge-AzReservation -ReservationOrderId "1111aaaa-b1b2-c0c2-d0d2-00000fffff" -ReservationId "11111111-1111-1111-1111-1111111111","11111111-0000-0000-0000-1111111111"
```

Zusammenführen der beiden `Reservation` angegebenen Daten in einem `Reservation`
