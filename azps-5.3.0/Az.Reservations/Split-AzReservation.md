# Split-AzReservation

## SYNOPSIS
Teilen `Reservation` eines

## SYNTAX

### Befehlszeile (Standard)
```
Split-AzReservation -ReservationOrderId <Guid> -ReservationId <Guid> -Quantity <Int32[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### PipeObject
```
Split-AzReservation -Quantity <Int32[]> -Reservation <PSReservation> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Teilen Sie eine `Reservation` in zwei s mit einer `Reservation` bestimmten Mengenverteilung auf.

## BEISPIELE

### Beispiel 1
```
PS C:\> Split-AzReservation -ReservationOrderId "00000000-ffff-ffff-0000-00000fffff" -ReservationId "11111111-1111-1111-1111-1111111111" -Quantities 2,3
```

Aufteilen der `Reservation` angegebenen `Reservation` Daten in zwei s mit den entsprechenden Mengen
