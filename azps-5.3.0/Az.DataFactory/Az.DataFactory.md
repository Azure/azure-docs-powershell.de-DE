# Az.DataFactory-Modul
## Beschreibung
Azure Data Factory V2 ist die Datenintegrationsplattform, die über die Orchestrierung und Batchverarbeitung von Zeitreihenprotokolldaten von Azure Data Factory V1 hinausgeht. Ein allgemeines App-Modell unterstützt moderne Data Warehouse-Muster und Szenarien, SSIS mit Anheben und Verschieben sowie datengesteuerte SaaS-Anwendungen. Verfassen und verwalten Sie zuverlässige und sichere Workflows für die Datenintegration im maßstabsmaßstab. Verwenden Sie native ADF-Datenconnectors und Integrationslaufzeiten zum Verschieben und Transformieren von Cloud- und lokalen Daten, die unstrukturiert, halbstrukturiert und mit Hadoop, Azure Data Lake, Spark, SQL Server, Rahmen DB und vielen anderen Datenplattformen strukturiert werden können.
