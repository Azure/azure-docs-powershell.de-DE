# Remove-AzDataFactory

## SYNOPSIS
Entfernt eine Daten factory.

## SYNTAX

### ByFactoryName (Standard)
```
Remove-AzDataFactory [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Remove-AzDataFactory [-DataFactory] <PSDataFactory> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzDataFactory"** entfernt eine Datenfactory.

## BEISPIELE

### Beispiel 1: Entfernen einer Daten factory
```
PS C:\>Remove-AzDataFactory -Name "WikiADF" -ResourceGroupName "ADF"
Confirm
Are you sure you want to remove data factory 'WikiADF' in resource group 'ADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

Mit diesem Befehl wird die Daten factory namens WikiADF aus der Ressourcengruppe mit dem Namen ADF entfernt.
Dieser Befehl gibt den Wert "$True" $True.
