# Get-AzServiceFabricCluster

## SYNOPSIS
Erhalten Sie die Details der Clusterressourcen.

## SYNTAX

### BySubscription (Standard)
```
Get-AzServiceFabricCluster [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceGroup
```
Get-AzServiceFabricCluster [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByName
```
Get-AzServiceFabricCluster [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Vom **Get-AzServiceFabricCluster** werden die Clusterressourcendetails angezeigt.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Get-AzServiceFabricCluster -ResourceGroupName 'Group1' -ClusterName 'Contoso01SFCluster'
```

Mit diesem Befehl werden die Clusterressourcendetails für den Cluster "myCluster" angezeigt.
