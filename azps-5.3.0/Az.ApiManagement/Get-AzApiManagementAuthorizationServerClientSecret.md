# Get-AzApiManagementAuthorizationServerClientSecret

## SYNOPSIS
Ruft einen geheimen Serverclient f端r die API-Verwaltungsautorisierung ab.

## SYNTAX

### ContextParameterSet (Standard)
```
Get-AzApiManagementAuthorizationServerClientSecret -Context <PsApiManagementContext> [-ServerId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzApiManagementAuthorizationServerClientSecret [-ServerId <String>] -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzApiManagementAuthorizationServerClientSecret"** ruft den geheimen Clientgeheimnis des Azure API Management Authorization Servers ab.

## BEISPIELE

### Beispiel 1: Einen angegebenen Autorisierungsserverclient geheim nach ID erhalten
```
PS C:\>$ApiMgmtContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementAuthorizationServerClientSecret -Context $ApiMgmtContext -ServerId "0123456789"
```

Dieser Befehl ruft den angegebenen geheimen Schl端ssel des Autorisierungsservers ab.
