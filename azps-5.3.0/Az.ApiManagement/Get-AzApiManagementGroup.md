# Get-AzApiManagementGroup

## SYNOPSIS
Ruft alle oder bestimmte API-Verwaltungsgruppen ab.

## SYNTAX

### GetAllGroups (Standard)
```
Get-AzApiManagementGroup -Context <PsApiManagementContext> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByGroupId
```
Get-AzApiManagementGroup -Context <PsApiManagementContext> [-GroupId <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByUserId
```
Get-AzApiManagementGroup -Context <PsApiManagementContext> [-Name <String>] [-UserId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByProductId
```
Get-AzApiManagementGroup -Context <PsApiManagementContext> [-Name <String>] [-ProductId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzApiManagementGroup"** ruft alle oder bestimmte API-Verwaltungsgruppen ab.

## BEISPIELE

### Beispiel 1: Alle Gruppen erhalten
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementGroup -Context $apimContext
```

Dieser Befehl ruft alle Gruppen ab.

### Beispiel 2: Erhalten einer Gruppe nach ID
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementGroup -Context $apimContext -GroupId "0123456789"
```

Dieser Befehl ruft die Gruppen-ID mit dem Namen 0123456789 ab.

### Beispiel 3: Erhalten einer Gruppe nach Name
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementGroup -Context $apimContext -Name "Group0002"
```

Dieser Befehl ruft die Gruppe mit dem Namen "Group0002" ab.

### Beispiel 4: Alle Benutzergruppen erhalten
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementGroup -Context $apimContext -UserId "0123456789"
```

Dieser Befehl ruft alle Benutzergruppen mit der Benutzer-ID "0123456789" ab.
