# Remove-AzApiManagementPolicy

## SYNOPSIS
Entfernt die API-Verwaltungsrichtlinie aus einem angegebenen Bereich.

## SYNTAX

### RemoveTenantLevel (Standard)
```
Remove-AzApiManagementPolicy -Context <PsApiManagementContext> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveProductLevel
```
Remove-AzApiManagementPolicy -Context <PsApiManagementContext> -ProductId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveApiLevel
```
Remove-AzApiManagementPolicy -Context <PsApiManagementContext> -ApiId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveOperationLevel
```
Remove-AzApiManagementPolicy -Context <PsApiManagementContext> -ApiId <String> -OperationId <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzApiManagementPolicy"** entfernt die API-Verwaltungsrichtlinie aus dem angegebenen Bereich.

## BEISPIELE

### Beispiel 1: Entfernen der Richtlinie auf Mandantenebene
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementPolicy -Context $apimContext
```

Mit diesem Befehl wird die Richtlinie auf Mandantenebene aus der API-Verwaltung entfernt.

### Beispiel 2: Entfernen der Produktbereichsrichtlinie
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementPolicy -Context $apimContext -ProductId "0123456789"
```

Mit diesem Befehl wird die Produktbereichsrichtlinie aus der API-Verwaltung entfernt.

### Beispiel 3: Entfernen der API-Bereichsrichtlinie
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementPolicy -Context $apimContext -ApiId "9876543210"
```

Mit diesem Befehl wird die API-Bereichsrichtlinie aus der API-Verwaltung entfernt.

### Beispiel 4: Entfernen der Vorgangsbereichsrichtlinie
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementPolicy -Context $apimContext -ApiId "9876543210" -OperationId "777"
```

Mit diesem Befehl wird die Vorgangsbereichsrichtlinie aus der API-Verwaltung entfernt.
