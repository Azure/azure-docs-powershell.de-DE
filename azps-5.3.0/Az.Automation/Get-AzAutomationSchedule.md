# Get-AzAutomationSchedule

## SYNOPSIS
Ruft einen Automatisierungszeitplan ab.

## SYNTAX

### ByAll (Standard)
```
Get-AzAutomationSchedule [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzAutomationSchedule [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzAutomationSchedule"** ruft einen Zeitplan für die Azure-Automatisierung ab.

## BEISPIELE

### Beispiel 1: Erhalten eines Zeitplans
```
PS C:\>Get-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "DailySchedule08" -ResourceGroupName "ResourceGroup01"
```

Dieser Befehl ruft den Zeitplan namens "DailySchedule08" ab.
