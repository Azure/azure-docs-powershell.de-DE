# Remove-AzAutomationSoftwareUpdateConfiguration

## SYNOPSIS
Entfernt eine Updatekonfiguration für die Azure-Automatisierungssoftware.

## SYNTAX

### BySucName (Standard)
```
Remove-AzAutomationSoftwareUpdateConfiguration -Name <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### BySuc
```
Remove-AzAutomationSoftwareUpdateConfiguration -SoftwareUpdateConfiguration <SoftwareUpdateConfiguration>
 [-PassThru] [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Dieses Cmdlet hat eine Konfiguration für das Aktualisieren von Softwareupdates für die Azure-Automatisierung entfernt.

## BEISPIELE

### Beispiel 1
In diesem Beispiel wird gezeigt, wie Sie "MyUpdateConfiguration" aus einem Automatisierungskonto entfernen.

```powershell
PS C:\> Remove-AzAutomationSoftwareUpdateConfiguration -ResourceGroupName "mygroup" -AutomationAccountName "myaccount" -Name "MyUpdateConfiguration"
```
