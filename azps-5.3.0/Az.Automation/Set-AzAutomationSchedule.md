# Set-AzAutomationSchedule

## SYNOPSIS
Ändert einen Automatisierungszeitplan.

## SYNTAX

```
Set-AzAutomationSchedule [-Name] <String> [-IsEnabled <Boolean>] [-Description <String>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Set-AzAutomationSchedule"** ändert einen Zeitplan in der Azure-Automatisierung.

## BEISPIELE

### Beispiel 1: Ändern eines Zeitplans
```
PS C:\>Set-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule01" -Description "Automation Schedule" -ResourceGroupName "ResourceGroup01"
```

Mit diesem Befehl wird die Beschreibung des Zeitplans namens "Schedule01" geändert.
