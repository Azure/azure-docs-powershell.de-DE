# Publish-AzAutomationRunbook

## SYNOPSIS
Veröffentlicht ein Runbook.

## SYNTAX

```
Publish-AzAutomationRunbook [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Publish-AzAutomationRunbook** veröffentlicht ein Runbook zur Verwendung in der Produktionsumgebung der Azure-Automatisierung.

## BEISPIELE

### Beispiel 1: Veröffentlichen eines Runbooks
```
PS C:\>Publish-AzAutomationRunbook -AutomationAccountName "Contoso17" -Name "Runbk01" -ResourceGroupName "ResourceGroup01"
```

Mit diesem Befehl wird das Runbook "Runbk01" im Azure-Automatisierungskonto namens "Contoso17" veröffentlicht.
