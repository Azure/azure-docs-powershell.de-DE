# Set-AzAutomationDscNode

## SYNOPSIS
Ändert die Knotenkonfiguration, der ein DSC-Knoten zugeordnet ist.

## SYNTAX

```
Set-AzAutomationDscNode -Id <Guid> -NodeConfigurationName <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Set-AzAutomationDscNode"** ändert eine Konfiguration des Knotens "APS Desired State Configuration (DSC)".
Azure Automation speichert die KONFIGURATION von DSC-Knoten als Konfigurationsdokument für verwaltetes Objektformat (Managed Object Format, MOF).

## BEISPIELE

### Beispiel 1: Ändern der Knotenkonfigurationszuordnung
```
PS C:\>Set-AzAutomationDscNode -NodeConfigurationName "Contoso.NodeConfiguration01" -ResourceGroupName "ResourceGroup01" -Id 064a8929-c98b-25e4-80hh-111c8a6067j8
```

Dieser Befehl weist dem Knoten mit der angegebenen GUID die Knotenkonfiguration "Contoso.NodeConfiguration01" zu.
