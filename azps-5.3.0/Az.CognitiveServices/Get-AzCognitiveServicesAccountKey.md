# Get-AzCognitiveServicesAccountKey

## SYNOPSIS
Ruft die API-Schlüssel für ein Konto ab.

## SYNTAX

```
Get-AzCognitiveServicesAccountKey [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzCognitiveServicesAccountKey"** ruft die API-Schlüssel für ein bereitgestelltes Cognitive Services-Konto ab.
Ein Cognitive Services-Konto verfügt über zwei API-Schlüssel: Schlüssel1 und Schlüssel2.
Die Schlüssel ermöglichen die Interaktion mit dem Cognitive Services-Kontoendpunkt.
Verwenden New-AzCognitiveServicesAccountKey zum erneuten Generierten eines Schlüssels.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Get-AzCognitiveServicesAccountKey -ResourceGroupName cognitive-services-resource-group -name myluis

Key1                             Key2
----                             ----
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```
