# Get-AzLegacyPeering

## SYNOPSIS
Wird verwendet, um ältere Peeringressourcen in Ressourcen der Azure Resource Management (ARM) zu konvertieren. 

## SYNTAX

```
Get-AzLegacyPeering [-PeeringLocation] <String> [-Kind] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Der Befehl wird verwendet, um ältere Peeringressourcen zu sehen, die alle Von Ihnen zum Konvertieren in Azure-Ressourcenverwaltungsressourcen (ARM) verwenden.
