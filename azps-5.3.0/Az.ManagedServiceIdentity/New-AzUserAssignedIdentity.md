# New-AzUserAssignedIdentity

## SYNOPSIS
Erstellt eine neue vom Benutzer zugewiesene Identität oder aktualisiert eine vorhandene vom Benutzer zugewiesene Identität.

## SYNTAX

```
New-AzUserAssignedIdentity [-ResourceGroupName] <String> [-Name] <String> [-Location <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-Tag <Hashtable>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "New-AzUserAssignedIdentity"** erstellt eine neue vom Benutzer zugewiesene Identität. Bei Verwendung mit einer bereits vorhandenen Identität wurde die Identität aktualisiert.
Um der Identität Azure Resource Manager-Tags hinzuzufügen, verwenden Sie das Set-AzResource Cmdlet.
