# New-AzBatchAccountKey

## SYNOPSIS
Generiert einen Schlüssel eines Batchkontos erneut.

## SYNTAX

```
New-AzBatchAccountKey [-AccountName] <String> [-ResourceGroupName <String>] -KeyType <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "New-AzBatchAccountKey"** generiert den primären oder sekundären Schlüssel eines Azure Batch-Kontos erneut.
Das Cmdlet gibt ein **BatchAccountContext-Objekt** zurück, das die aktuellen **Eigenschaften "PrimaryAccountKey"** und **"SecondaryAccountKey"** enthält.

## BEISPIELE

### Beispiel 1: Erneut erstellen des Primärkontoschlüssels für ein Batchkonto
```
PS C:\>New-AzBatchAccountKey -AccountName "pfuller" -KeyType "Primary"
AccountName                  : pfuller
Location                     : westus
ResourceGroupName            : CmdletExampleRG
DedicatedCoreQuota           : 20
LowPriorityCoreQuota         : 20
PoolQuota                    : 20
ActiveJobAndJobScheduleQuota : 20
Tags                         :
TaskTenantUrl                : https://cmdletexample.westus.batch.azure.com
```

Mit diesem Befehl wird der Primärkontoschlüssel für das Batchkonto mit dem Namen "pfuller" erneut generiert.
