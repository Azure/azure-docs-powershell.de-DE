# New-AzBatchApplicationPackage

## SYNOPSIS
Erstellt ein Anwendungspaket in einem Batchkonto.

## SYNTAX

### UploadAndActivate (Standard)
```
New-AzBatchApplicationPackage [-AccountName] <String> [-ResourceGroupName] <String> [-ApplicationName] <String>
 [-ApplicationVersion] <String> [-Format] <String> -FilePath <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ActivateOnly
```
New-AzBatchApplicationPackage [-AccountName] <String> [-ResourceGroupName] <String> [-ApplicationName] <String>
 [-ApplicationVersion] <String> [-Format] <String> [-ActivateOnly] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "New-AzBatchApplicationPackage"** erstellt ein Anwendungspaket in einem Azure Batch-Konto.

## BEISPIELE

### Beispiel 1: Installieren eines Anwendungspakets in einem Batchkonto
```
PS C:\>New-AzBatchApplicationPackage -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationName "Litware" -ApplicationVersion "1.0" -FilePath "litware.1.0.zip" -Format "zip"
```

Dieser Befehl erstellt und aktiviert Version 1.0 der Litware-Anwendung und lädt den Inhalt des litware.1.0.zip als Anwendungspaketinhalt hoch.
