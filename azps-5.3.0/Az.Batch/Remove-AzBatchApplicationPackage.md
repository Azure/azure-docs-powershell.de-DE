# Remove-AzBatchApplicationPackage

## SYNOPSIS
Löscht einen Anwendungspaketdatensatz und die Binärdatei.

## SYNTAX

```
Remove-AzBatchApplicationPackage [-AccountName] <String> [-ResourceGroupName] <String>
 [-ApplicationName] <String> [-ApplicationVersion] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzBatchApplicationPackage"** löscht einen Anwendungspaketdatensatz und die Binärdatei aus einem Azure Batch-Konto.

## BEISPIELE

### Beispiel 1: Löschen eines Anwendungspakets aus einem Batchkonto
```
PS C:\>Remove-AzBatchApplicationPackage -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationName "litware" -ApplicationVersion "1.0"
```

Mit diesem Befehl wird Version 1.0 der Anwendung Litware aus dem ContosoBatchGroup-Konto gelöscht.
Der Befehl löscht sowohl den Paketdatensatz als auch das BLOB, das die Paket binäre Datei enthält.
