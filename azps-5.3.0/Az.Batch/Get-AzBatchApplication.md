# Get-AzBatchApplication

## SYNOPSIS
Ruft Informationen zur angegebenen Anwendung ab.

## SYNTAX

```
Get-AzBatchApplication [-AccountName] <String> [-ResourceGroupName] <String> [[-ApplicationName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzBatchApplication"** ruft Informationen zu einer Anwendung in einem Azure -Batchkonto ab.

## BEISPIELE

### Beispiel 1: Anzeigen der Anwendungen in einem Batchkonto
```
PS C:\>Get-AzBatchApplication -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup"
ApplicationName AllowUpdates DisplayName

------------- ------------ ----------------------------

litware       False        Litware Advanced Reticulator
```

Dieser Befehl zeigt alle Anwendungen im ContosoBatch-Konto an.
