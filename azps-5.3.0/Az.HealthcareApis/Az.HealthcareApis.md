# Az.HealthcareApis-Modul
## Beschreibung
Bietet DIE CRUD-Verwaltung für den Dienst "HealthcareApis"

## Az.HealthcareApis-Cmdlets
### [Get-AzHealthcareApisService](Get-AzHealthcareApisService.md)
Sie erhalten die Metadaten einer Dienstinstanz.

### [New-AzHealthcareApisService](New-AzHealthcareApisService.md)
Erstellt die Metadaten einer Dienstinstanz.

### [Remove-AzHealthcareApisService](Remove-AzHealthcareApisService.md)
Löscht eine Dienstinstanz.

### [Set-AzHealthcareApisService](Set-AzHealthcareApisService.md)
Aktualisiert einen vorhandenen Dienst für Gesundheitswesensapis.
