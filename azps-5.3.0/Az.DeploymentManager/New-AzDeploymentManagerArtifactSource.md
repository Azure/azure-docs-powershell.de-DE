# New-AzDeploymentManagerArtifactSource

## SYNOPSIS
Erstellt eine Artefaktquelle.

## SYNTAX

```
New-AzDeploymentManagerArtifactSource -ResourceGroupName <String> -Name <String> -Location <String>
 -SasUri <String> [-Tag <Hashtable>] [-ArtifactRoot <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "New-AzDeploymentManagerArtifactSource"** erstellt eine Artefaktquelle.
Geben Sie *"Name",* *"ResourceGroupName"* und die erforderlichen Eigenschaften an.

Sie können das zurückgegebene Objekt lokal ändern und dann die Änderungen mithilfe des cmdlets Set-AzDeploymentManagerArtifactSource Artefaktquelle anwenden.

Das Cmdlet gibt ein ArtifactSource-Objekt zurück, das eine ResourceId enthält, auf die im Cmdlet New-AzDeploymentManagerServiceTopology verwiesen werden kann, damit von diesem Speicherort aus auf Artefakte verwiesen werden kann, die für eine ServiceUnit-Ressource, die Dateien "Vorlage" und "Parameter", erforderlich sind.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> New-AzDeploymentManagerArtifactSource -ResourceGroupName ContosoResourceGroup -Name ContosoArtifactSource -Location "Central US" -SasUri "https://ContosoStorage.blob.core.windows.net/ContosoArtifacts?sasParameters"
```

Erstellt eine Artefaktquelle in der ContosoResourceGroup mit dem Namen "ContosoArtifactSource" und "USA, Mitte" als Speicherort der Ressource. Die Eigenschaft "SasUri" stellt einen Azure Storage-SAS-URI für den Speichercontainer bereit, in dem die Artefakte gespeichert sind.
