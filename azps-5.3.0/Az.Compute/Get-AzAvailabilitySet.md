# Get-AzAvailabilitySet

## SYNOPSIS
Ruft Die Verfügbarkeitssätze von Azure in einer Ressourcengruppe ab.

## SYNTAX

```
Get-AzAvailabilitySet [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzAvailabilitySet"** ruft Die Verfügbarkeitssätze von Azure in einer Ressourcengruppe ab.
Sie können den Namen eines bestimmten Verfügbarkeitssets angeben, den Sie erhalten möchten.

## BEISPIELE

