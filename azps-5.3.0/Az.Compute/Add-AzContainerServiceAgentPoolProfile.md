# Add-AzContainerServiceAgentPoolProfile

## SYNOPSIS
Fügt ein Containerdienst-Agent-Poolprofil hinzu.

## SYNTAX

```
Add-AzContainerServiceAgentPoolProfile [-ContainerService] <PSContainerService> [[-Name] <String>]
 [[-Count] <Int32>] [[-VmSize] <String>] [[-DnsPrefix] <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Add-AzContainerServiceAgentPoolProfile"** fügt einem lokalen Containerdienstdienstobjekt ein Containerdienst-Agent-Poolprofil hinzu.

## BEISPIELE

### Beispiel 1: Hinzufügen eines Profils
```
PS C:\> Add-AzContainerServiceAgentPoolProfile -Name "AgentPool01" -VmSize "Standard_A1" -DnsPrefix "APResourceGroup17"
```

Dieser Befehl fügt dem lokalen Containerdienstdienstobjekt ein Containerdienst-Agent-Poolprofil hinzu.
