# Get-AzVmssSku

## SYNOPSIS
Ruft die verfügbaren SKUs für die VMSS ab.

## SYNTAX

```
Get-AzVmssSku [-ResourceGroupName] <String> [-VMScaleSetName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzVmssSku"** ruft die verfügbaren SKUs für den Virtual Machine Scale Set (VMSS) ab.

## BEISPIELE

### Beispiel 1: Alle verfügbaren SKUs von der VMSS erhalten
```
PS C:\> Get-AzVmssSku -ResourceGroupName "ContosoGroup" -VMScaleSetName "ContosoVMSS"
```

Dieser Befehl ruft alle verfügbaren SKUs aus der VMSS namens ContosoVMSS ab, die zur Ressourcengruppe "ContosoGroup" gehört.
