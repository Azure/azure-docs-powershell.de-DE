# Start-AzVM

## SYNOPSIS
Startet einen virtuellen Azure-Computer.

## SYNTAX

### ResourceGroupNameParameterSetName (Standard)
```
Start-AzVM [-Name] <String> [-NoWait] [-ResourceGroupName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSetName
```
Start-AzVM [-NoWait] [-Id] <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Start-AzVM-Cmdlet** startet einen virtuellen Azure-Computer.

## BEISPIELE

### Beispiel 1: Starten eines virtuellen Computers
```
PS C:\> Start-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Dieser Befehl startet den virtuellen Computer "VirtualMachine07" in ResourceGroup11.
