# Remove-AzVmssDiagnosticsExtension

## SYNOPSIS
Entfernt eine Diagnoseerweiterung von der VMSS.

## SYNTAX

```
Remove-AzVmssDiagnosticsExtension [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzVmssDiagnosticsExtension"** entfernt eine Diagnoseerweiterung aus dem Virtual Machine Scale Set (VMSS).

## BEISPIELE

### Beispiel 1: Entfernen einer Diagnoseerweiterung von vmSS
```
PS C:\> Remove-AzVmssDiagnosticsExtension -VirtualMachineScaleSet $VMSS -Name $extName
```

Mit diesem Befehl wird die Diagnoseerweiterung von vmSS entfernt.
