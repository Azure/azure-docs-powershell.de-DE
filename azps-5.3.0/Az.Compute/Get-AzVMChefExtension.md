# Get-AzVMChefExtension

## SYNOPSIS
Ruft Informationen über eine Chef-Erweiterung ab.

## SYNTAX

### Linux
```
Get-AzVMChefExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>] [-Status] [-Linux]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Windows
```
Get-AzVMChefExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>] [-Status] [-Windows]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzVM VirtualExtension"** ruft Informationen über eine auf einem virtuellen Computer installierte "Chef"-Erweiterung ab.

## BEISPIELE

### Beispiel 1: Details der Erweiterung "Chef" für einen virtuellen Windows-Computer
```
PS C:\> Get-AzVMChefExtension -ResourceGroupName "ResourceGroup001" -VMName "WindowsVM001" -Windows
```

Dieser Befehl ruft die "Chef"-Erweiterung von einem virtuellen Windows-Computer namens "WindowsVM001" ab, der zur Ressourcengruppe "ResourceGroup001" gehört.

### Beispiel 2: Details der Erweiterung "Chef" für einen virtuellen Linux-Computer
```
PS C:\> Get-AzVMChefExtension -ResourceGroupName "ResourceGroup002" -VMName "LinuxVM001" -Linux
```

Dieser Befehl ruft die "Chef"-Erweiterung von einem virtuellen Linux-Computer namens LinuxVM001 ab, der zur Ressourcengruppe "ResourceGroup002" gehört.
