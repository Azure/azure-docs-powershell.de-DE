# Get-AzVMExtensionImageType

## SYNOPSIS
Ruft den Typ einer Azure-Erweiterung ab.

## SYNTAX

```
Get-AzVMExtensionImageType -Location <String> -PublisherName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzVMExtensionImageType"** ruft den Typ einer Azure-Erweiterung ab.

## BEISPIELE

### Beispiel 1: Erhalten eines Erweiterungsbildtyps
```
PS C:\> Get-AzVMExtensionImageType -Location "Central US" -PublisherName "Fabrikam"
```

Dieser Befehl ruft den Erweiterungsbildtyp für den angegebenen Herausgeber und Speicherort ab.
