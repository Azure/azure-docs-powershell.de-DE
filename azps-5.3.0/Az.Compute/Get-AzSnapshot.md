# Get-AzSnapshot

## SYNOPSIS
Ruft die Eigenschaften einer Momentaufnahme ab.

## SYNTAX

```
Get-AzSnapshot [[-ResourceGroupName] <String>] [[-SnapshotName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzSnapshot-Cmdlet** ruft die Eigenschaften einer Momentaufnahme ab.
