# Remove-AzVMDiagnosticsExtension

## SYNOPSIS
Entfernt die Diagnoseerweiterung von einem virtuellen Computer.

## SYNTAX

```
Remove-AzVMDiagnosticsExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>] [-NoWait]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzVMDiagnosticsExtension"** entfernt eine Azure-Diagnoseerweiterung von einem virtuellen Computer.
Sie müssen die Ausgabe dieses Cmdlets an das Update-AzVM übergeben, um die Änderungen zu implementieren.

## BEISPIELE

### Beispiel 1: Entfernen der Diagnoseerweiterung von einem virtuellen Computer
```
PS C:\> Remove-AzVMDiagnosticsExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM22" | Update-AzVM
```

Mit diesem Befehl wird die Diagnoseerweiterung von einem virtuellen Computer namens "ContosoVM22" entfernt.
Der Befehl übergibt das Ergebnis mithilfe des Pipelineoperators Update-AzVM an das Update-AzVM Cmdlet.
Dieser Befehl aktualisiert den virtuellen Computer.
