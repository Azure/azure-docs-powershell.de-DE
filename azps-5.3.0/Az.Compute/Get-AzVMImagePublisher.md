# Get-AzVMImagePublisher

## SYNOPSIS
Ruft die HERAUSGEBER von VMImage ab.

## SYNTAX

```

```

## BESCHREIBUNG
Das **Cmdlet "Get-AzVMImagePublisher"** ruft die VMImage-Herausgeber ab.
