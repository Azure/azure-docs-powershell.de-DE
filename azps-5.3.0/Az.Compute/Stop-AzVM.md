# Stop-AzVM

## SYNOPSIS
Beendet einen virtuellen Azure-Computer.

## SYNTAX

### ResourceGroupNameParameterSetName (Standard)
```
Stop-AzVM [-Name] <String> [-Force] [-StayProvisioned] [-NoWait] [-SkipShutdown] [-ResourceGroupName] <String>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSetName
```
Stop-AzVM [-Force] [-StayProvisioned] [-NoWait] [-SkipShutdown] [-Id] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Stop-AzVM"** beendet einen virtuellen Azure-Computer.

## BEISPIELE

### Beispiel 1: Beenden eines virtuellen Computers
```
PS C:\> Stop-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Dieser Befehl stoppt den virtuellen Computer mit dem Namen "VirtualMachine07" in ResourceGroup11.
