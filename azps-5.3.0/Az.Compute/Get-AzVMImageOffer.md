# Get-AzVMImageOffer

## SYNOPSIS
Ruft die Angebote von VMImage ab.

## SYNTAX

```
Get-AzVMImageOffer -Location <String> -PublisherName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzVMImageOffer"** ruft die "VMImage"-Angebotstypen ab.

## BEISPIELE

### Beispiel 1: Erhalten von Angebotstypen f端r einen Herausgeber
```
PS C:\> Get-AzVMImageOffer -Location "Central US" -PublisherName "Fabrikam"
```

Dieser Befehl ruft die Angebotstypen f端r den angegebenen Herausgeber in der Region "USA, Mitte" ab.
