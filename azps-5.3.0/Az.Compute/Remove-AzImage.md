# Remove-AzImage

## SYNOPSIS
Entfernt ein Bild.

## SYNTAX

```
Remove-AzImage [-ResourceGroupName] <String> [-ImageName] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzImage"** entfernt ein Bild.

## BEISPIELE

### Beispiel 1
```
PS C:\> Remove-AzImage -ResourceGroupName 'ResourceGroup01' -ImageName 'Image01' -Force;
```

Mit diesem Befehl wird das Bild mit dem Namen "Image01" in der Ressourcengruppe "ResourceGroup01" entfernt.
