## BESCHREIBUNG
Das **Cmdlet "Set-AzSynapseIntegrationRuntime"** aktualisiert eine Integrationslaufzeit mit bestimmten Parametern.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Set-AzSynapseIntegrationRuntime -WorkspaceName ContosoWorkspace -Name 'test-selfhost-ir' -Description 'New description'
```

Das Cmdlet aktualisiert die Beschreibung der Integrationslaufzeit namens "test-selfhost-ir".

### Beispiel 2
```powershell
PS C:\> Set-AzSynapseIntegrationRuntime -WorkspaceName ContosoWorkspace -Name 'test-selfhost-ir' `
                                        -SharedIntegrationRuntimeResourceId '/subscriptions/b3ee3a7f-7614-4644-ad07-afa832620b4b/resourceGroups/rg-test-dfv2/providers/Microsoft.DataFactory/factories/test-df-eu2/integrationruntimes/test-selfhost-ir' -Type "SelfHosted"
```

Das Cmdlet fügt den Arbeitsbereich zur Verwendung der Laufzeit für die gemeinsame Integration hinzu. Bei Verwendung `-SharedIntegrationRuntimeResourceId` von `-Type` Parametern muss auch das eingeschlossen werden. Beachten Sie, dass dem Arbeitsbereich die Berechtigung zur Verwendung der Integrationslaufzeit erteilt werden muss, bevor das Cmdlet ausgeführt wird.
