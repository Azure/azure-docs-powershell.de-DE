# Disable-AzSynapseSqlAdvancedDataSecurity

## SYNOPSIS
Deaktiviert die erweiterte Datensicherheit in einem Arbeitsbereich.

## SYNTAX

### DisableByNameParameterSet (Standard)
```
Disable-AzSynapseSqlAdvancedDataSecurity [-ResourceGroupName <String>] -WorkspaceName <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DisableByInputObjectParameterSet
```
Disable-AzSynapseSqlAdvancedDataSecurity -InputObject <PSSynapseWorkspace> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DisableByResourceIdParameterSet
```
Disable-AzSynapseSqlAdvancedDataSecurity -ResourceId <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Disable-AzSynapseSqlAdvancedDataSecurity"** deaktiviert Die erweiterte Datensicherheit in einem Arbeitsbereich.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Disable-AzSynapseSqlAdvancedDataSecurity -WorkspaceName ContosoWorkspace
```

Mit diesem Befehl wird Advanced Data Security im Arbeitsbereich "ContosoWorkspace" deaktiviert.

### Beispiel 1
```powershell
PS C:\> Get-AzSynapseWorkspace -Name ContosoWorkspace | Disable-AzSynapseSqlAdvancedDataSecurity
```

Mit diesem Befehl wird Advanced Data Security im Arbeitsbereich mit dem Namen "ContosoWorkspace" über die Pipeline deaktiviert.
