# Az.DataBoxEdge-Modul
## Beschreibung
Modul zum Erstellen eines Geräts und Ausführen von Vorgängen auf DataBoxEdge-Geräten

### [Get-AzDataBoxEdgeBandwidthSchedule](Get-AzDataBoxEdgeBandwidthSchedule.md)
