# Remove-AzDataLakeAnalyticsComputePolicy

## SYNOPSIS
Entfernt eine angegebene Azure Data Lake Analytics-Rechenrichtlinie.

## SYNTAX

```
Remove-AzDataLakeAnalyticsComputePolicy [-ResourceGroupName <String>] [-Account] <String> [-Name] <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Die **Remove-AzDataLakeAnalyticsComputePolicy entfernt** eine angegebene Azure Data Lake Analytics-Rechenrichtlinie.

## BEISPIELE

### Beispiel 1: Entfernen einer Berechnungsrichtlinie
```
PS C:\>Remove-AzDataLakeAnalyticsComputePolicy -Account "contosoadla" -Name myPolicy
```

Mit diesem Befehl wird die angegebene Berechnungsrichtlinie mit dem Namen "myPolicy" im Konto "contosoadla" entfernt.
