# Update-AzBlockchainMember

## SYNOPSIS
Aktualisieren Sie ein mitglied.

## SYNTAX

### UpdateExpanded (Standard)
```
Update-AzBlockchainMember -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-ConsortiumManagementAccountPassword <SecureString>] [-FirewallRule <IFirewallRule[]>]
 [-Password <SecureString>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### UpdateViaIdentityExpanded
```
Update-AzBlockchainMember -InputObject <IBlockchainIdentity>
 [-ConsortiumManagementAccountPassword <SecureString>] [-FirewallRule <IFirewallRule[]>]
 [-Password <SecureString>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## BESCHREIBUNG
Aktualisieren Sie ein mitglied.

## BEISPIELE

### Beispiel 1: Aktualisieren eines mitglieds
```powershell
PS C:\> $passwd2 = 'strongMemberAccountPassword@2' | ConvertTo-SecureString -AsPlainText -Force
PS C:\> Update-AzBlockchainMember -Name dolauli002 -ResourceGroupName testgroup -Password $passwd2

Location Name       Type
-------- ----       ----
eastus   dolauli002 Microsoft.Blockchain/blockchainMembers
```

Mit diesem Befehl wird ein ben-Member aktualisiert.

### Beispiel 2: Aktualisieren eines mitglieds
```powershell
PS C:\> $tag = @{'againupdate'='password'}
PS C:\> $member = Get-AzBlockchainMember -Name $env.blockchainMember -ResourceGroupName $env.resourceGroup
PS C:\> Update-AzBlockchainMember -InputObject $member -Tag $tag

Location Name       Type
-------- ----       ----
eastus   dolauli002 Microsoft.Blockchain/blockchainMembers
```

Mit diesem Befehl wird ein ben-Member aktualisiert.

FIREWALLRULE <IFirewallRule[]>: Ruft die Firewallregeln ab oder legt sie fest.
  - `[EndIPAddress <String>]`: Ruft die End-IP-Adresse des Firewallregelbereichs ab oder legt diese fest.
  - `[RuleName <String>]`: Ruft den Namen der Firewallregeln ab oder legt den Namen fest.
  - `[StartIPAddress <String>]`: Ruft die Start-IP-Adresse des Firewallregelbereichs ab oder legt diese fest.

INPUTOBJECT <IBlockchainIdentity> : Identity Parameter
  - `[BlockchainMemberName <String>]`: Name eines Mitgliedes.
  - `[Id <String>]`: Ressourcenidentitätspfad
  - `[Location <String>]`: Name des Standorts.
  - `[OperationId <String>]`: Vorgangs-ID.
  - `[ResourceGroupName <String>]`: Der Name der Ressourcengruppe, die die Ressource enthält. Sie können diesen Wert über die Azure Resource Manager-API oder das Portal abrufen.
  - `[SubscriptionId <String>]`: Ruft die Abonnement-ID ab, die das Microsoft Azure-Abonnement eindeutig identifiziert. Die Abonnement-ID ist Teil des URI für jeden Dienstaufruf.
  - `[TransactionNodeName <String>]`: Name des Transaktionsknotens.
