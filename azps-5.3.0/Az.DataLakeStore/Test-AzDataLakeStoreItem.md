# Test-AzDataLakeStoreItem

## SYNOPSIS
Überprüft das Vorhandensein einer Datei oder eines Ordners im Data Lake Store.

## SYNTAX

```
Test-AzDataLakeStoreItem [-Account] <String> [-Path] <DataLakeStorePathInstance> [[-PathType] <PathType>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Test-AzDataLakeStoreItem"** überprüft das Vorhandensein einer Datei oder eines Ordners im Data Lake Store.

## BEISPIELE

### Beispiel 1: Testen einer Datei
```powershell
PS C:\>Test-AzDataLakeStoreItem -AccountName "ContosoADL" -Path "/MyFiles/Test.csv"
```

Mit diesem Befehl wird überprüft, Test.csv Dateidatei im Konto "ContosoADL" vorhanden ist.

### Beispiel 2

Überprüft das Vorhandensein einer Datei oder eines Ordners im Data Lake Store. (automatisch generiert)

<!-- Aladdin Generated Example -->
```powershell
Test-AzDataLakeStoreItem -Account 'ContosoADL' -Path '/MyFiles/Test.csv' -PathType Any
```
