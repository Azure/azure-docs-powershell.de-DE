# Remove-AzDataLakeStoreVirtualNetworkRule

## SYNOPSIS
Entfernt die angegebene Virtuelle Netzwerkregel im angegebenen Data Lake Store.

## SYNTAX

```
Remove-AzDataLakeStoreVirtualNetworkRule [-Account] <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzDataLakeStoreVirtualNetworkRule"** entfernt die angegebene virtuelle Netzwerkregel im angegebenen Data Lake Store.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Remove-AzDataLakeStoreVirtualNetworkRule -Account "dls" -Name "myVNET"
```

Entfernt die Virtuelle Netzwerkregel "myVNET" aus dem Konto "dls".
