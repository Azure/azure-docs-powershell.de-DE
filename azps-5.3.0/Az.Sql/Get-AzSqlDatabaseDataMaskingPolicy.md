# Get-AzSqlDatabaseDataMaskingPolicy

## SYNOPSIS
Ruft die Datenmaskierungsrichtlinie für eine Datenbank ab.

## SYNTAX

```
Get-AzSqlDatabaseDataMaskingPolicy [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzSqlDatabaseDataMaskingPolicy"** ruft die Datenmaskierungsrichtlinie einer Azure SQL ab.
Um dieses Cmdlet zu verwenden, verwenden Sie die Parameter *"ResourceGroupName",* *"ServerName"* und *"DatabaseName",* um die Datenbank zu identifizieren.
Dieses Cmdlet wird auch vom Dienst SQL Server Stretch Database in Azure unterstützt.

## BEISPIELE

### Beispiel 1: Erhalten der Datenmaskierungsrichtlinie für eine Azure SQL Datenbank
```
PS C:\>Get-AzSqlDatabaseDataMaskingPolicy -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
DatabaseName      : Database01
ResourceGroupName : ResourceGroup01
ServerName        : Server01
DataMaskingState  : Enabled
PrivilegedUsers  :
```

Dieser Befehl ruft die Datenformatierungsrichtlinie aus datenbankdatenbank01 in der Ressourcengruppe "ResourceGroup01" auf Server Server01 ab.
