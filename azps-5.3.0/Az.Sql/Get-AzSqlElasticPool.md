# Get-AzSqlElasticPool

## SYNOPSIS
Ruft In-Pool-Pool und deren Eigenschaftswerte in einer Azure SQL ab.

## SYNTAX

```
Get-AzSqlElasticPool [[-ElasticPoolName] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzSqlElasticPool"** ruft Pool und ihre Eigenschaftswerte ab.
Geben Sie den Namen eines vorhandenen Poolpools an, um die Eigenschaftswerte nur für den Pool zu sehen.
