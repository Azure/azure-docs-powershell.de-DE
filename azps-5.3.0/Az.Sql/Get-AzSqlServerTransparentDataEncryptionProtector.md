# Get-AzSqlServerTransparentDataEncryptionProtector

## SYNOPSIS
Ruft die Transparente Datenverschlüsselung (TRANSPARENT Data Encryption, TDE) ab

## SYNTAX

```
Get-AzSqlServerTransparentDataEncryptionProtector [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das Get-AzSqlServerTransparentDataEncryptionProtector cmdlet ruft Informationen über das TDE-Gerät ab.

## BEISPIELE

### Beispiel 1: Verwenden der Transparent Data Encryption (TDE)
```
PS C:\> Get-AzSqlServerTransparentDataEncryptionProtector -ServerName 'ContosoServer' -ResourceGroup 'ContosoResourceGroup'
```

Dieser Befehl ruft das TDE-Element für den Server mit dem Namen "ContosoServer" in der Ressourcengruppe "ContosoResourceGroup" ab.
ResourceGroupName ServerName Type ServerKeyVaultKeyName
----------------- ----------                   ---- ---------------------
ContosoResourceGroup ContosoServer ServiceManaged ServiceManaged
