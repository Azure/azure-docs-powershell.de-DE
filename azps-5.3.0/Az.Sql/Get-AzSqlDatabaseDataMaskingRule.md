# Get-AzSqlDatabaseDataMaskingRule

## SYNOPSIS
Ruft die Datenmaskierungsregeln aus einer Datenbank ab.

## SYNTAX

```
Get-AzSqlDatabaseDataMaskingRule [-SchemaName <String>] [-TableName <String>] [-ColumnName <String>]
 [-ServerName] <String> [-DatabaseName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzSqlDatabaseDataMaskingRule"** ruft entweder eine bestimmte Datenmaskierungsregel oder alle Datenmaskierungsregeln für eine Azure SQL ab.
Um das Cmdlet zu verwenden, verwenden Sie die Parameter *"ResourceGroupName",* *"ServerName"* und *"DatabaseName",* um die Datenbank zu identifizieren, und den Parameter *"RuleId",* um anzugeben, welche Regel dieses Cmdlet zurückgibt.
Wenn Sie *"RuleId" nicht bereitstellen,* werden alle Datenmaskierungsregeln für diese Azure SQL zurückgegeben.
Dieses Cmdlet wird auch vom Dienst SQL Server Stretch Database in Azure unterstützt.
