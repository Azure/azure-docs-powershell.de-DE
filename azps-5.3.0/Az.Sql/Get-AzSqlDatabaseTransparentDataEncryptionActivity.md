# Get-AzSqlDatabaseTransparentDataEncryptionActivity

## SYNOPSIS
Ruft den Fortschritt einer TDE-Überprüfung einer Datenbank ab.

## SYNTAX

```
Get-AzSqlDatabaseTransparentDataEncryptionActivity [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzSqlDatabaseTransparentDataEncryptionActivity"** ruft den Fortschritt einer Transparent Data Encryption (TDE)-Überprüfung einer Azure SQL ab.
Wenn kein Verschlüsselungsspanne ausgeführt wird, gibt dieses Cmdlet eine leere Liste zurück.
Weitere Informationen finden Sie unter "Transparente Datenverschlüsselung mit Azure SQL-Datenbank" https://msdn.microsoft.com/library/dn948096 https://msdn.microsoft.com/library/dn948096) (in der Microsoft Developer Network Library).
Dieses Cmdlet wird auch vom Dienst SQL Server Stretch Database in Azure unterstützt.

## BEISPIELE

### Beispiel 1: Get TDE activity for a database
```
PS C:\>Get-AzSqlDatabaseTransparentDataEncryptionActivity -ServerName "server01" -ResourceGroupName "resourcegroup01" -DatabaseName "database01"
ResourceGroupName : resourcegroup01
ServerName        : server01
DatabaseName      : database01
Status            : Encrypting
PercentComplete   : 3.662109
```

Dieser Befehl ruft die "TDE-Aktivität" für die Datenbank mit dem Namen "database01" auf dem Server "server01" ab.
