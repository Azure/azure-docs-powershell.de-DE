# Get-AzSqlServerDisasterRecoveryConfigurationActivity

## SYNOPSIS
Ruft Die Aktivität für eine Datenbankserversystemwiederherstellungskonfiguration ab.

## SYNTAX

```
Get-AzSqlServerDisasterRecoveryConfigurationActivity [-ServerName] <String>
 -ServerDisasterRecoveryConfigurationName <String> [-OperationId <Guid>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzSqlServerDisasterRecoveryConfigurationActivity"** ruft Aktivitäten für eine SQL Wiederherstellungskonfiguration des Datenbankserversystems ab.
