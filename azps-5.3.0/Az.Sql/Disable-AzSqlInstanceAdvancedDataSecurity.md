# Disable-AzSqlInstanceAdvancedDataSecurity

## SYNOPSIS
Deaktiviert Advanced Data Security für eine verwaltete Instanz.

## SYNTAX

```
Disable-AzSqlInstanceAdvancedDataSecurity [-InputObject <AzureSqlManagedInstanceModel>] -InstanceName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Disable-AzSqlInstanceAdvancedDataSecurity** deaktiviert Advanced Data Security für eine verwaltete Instanz.

## BEISPIELE

### Beispiel 1: Deaktivieren der verwalteten Instanz Advanced Data Security
```powershell
PS C:\>  Disable-AzSqlInstanceAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```

### Beispiel 2: Deaktivieren der erweiterten Datensicherheit des Servers aus einer verwalteten Instanzressource
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Disable-AzSqlInstanceAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```
