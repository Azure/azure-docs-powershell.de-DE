# Invoke-AzSqlInstanceFailover

## SYNOPSIS
Failover einer Azure SQL verwaltete Instanz.

## SYNTAX

```
Invoke-AzSqlInstanceFailover [-Name] <String> [-AsJob] [-PassThru] [-Force] [-ReadableSecondary]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Invoke-AzSqlInstanceFailover** über ein Azure SQL Verwaltete Instanz.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Invoke-AzSqlInstanceFailover -ResourceGroupName "ResourceGroup01" -Name "ManagedInstance01"
```

Mit diesem Befehl wird ein Failover für die primäre Replikat der Instanz mit dem Namen "ManagedInstance01" ausgeführt.

### Beispiel 2
```powershell
PS C:\> Invoke-AzSqlInstanceFailover -ResourceGroupName "ResourceGroup01" -Name "ManagedInstance01" -ReadableSecondary
```

Mit diesem Befehl wird ein Failover des lesbaren sekundären Replikats der verwalteten Instanz "ManagedInstance01" ausgeführt.
