# New-AzCosmosDBSqlUniqueKeyPolicy

## SYNOPSIS
Erstellt ein neues SqlUniqueKeyPolicy-Objekt aus".

## SYNTAX

```
New-AzCosmosDBSqlUniqueKeyPolicy -UniqueKey <PSSqlUniqueKey[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "New-AzCosmosDBSqlUniqueKeyPolicy"** erstellt ein neues Objekt vom Typ "PSSqlUniqueKeyPolicy".

## BEISPIELE

### Beispiel 1
```powershell
PS C:\>New-AzCosmosDBSqlUniqueKeyPolicy -UniqueKey {psUniqueKey1, psUniqueKey2}

UniqueKey
---------
{Microsoft.Azure.Commands.CosmosDB.Models.PSSqlUniqueKey, Microsoft.Azure.Commands.CosmosDB.Models.PSSqlUniqueKey}
```
