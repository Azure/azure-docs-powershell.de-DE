# New-AzCosmosDBGremlinUniqueKeyPolicy

## SYNOPSIS
Erstellt ein neues "DessDB UniqueKeyPolicy"-Objekt.

## SYNTAX

```
New-AzCosmosDBGremlinUniqueKeyPolicy -UniqueKey <PSSqlUniqueKey[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "New-AzCosmosDBGremlinUniqueKeyPolicy"** erstellt ein neues Objekt vom Typ "PSUniqueKeyPolicy".

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $uk = New-AzCosmosDBGremlinUniqueKey -Path "abc"

 New-AzCosmosDBGremlinUniqueKeyPolicy -UniqueKey $uk,$uk
```
