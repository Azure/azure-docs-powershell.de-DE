# Test-AzStreamAnalyticsFunction

## SYNOPSIS
Überprüft, ob Stream Analytics eine Verbindung mit einer Funktion herstellen kann.

## SYNTAX

```
Test-AzStreamAnalyticsFunction [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Mit **dem Cmdlet "Test-AzStreamAnalyticsFunction"** wird überprüft, ob Azure Stream Analytics eine Verbindung mit einer Funktion herstellen kann.

## BEISPIELE

### Beispiel 1: Testen einer Datenstromanalysefunktion
```
PS C:\>Test-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -Name "ScoreTweet"
```

Mit diesem Befehl wird der Verbindungsstatus der Funktion "ScoreTweet" im Auftrag "StreamJob22" getestet.
