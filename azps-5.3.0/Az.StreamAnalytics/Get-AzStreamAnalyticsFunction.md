# Get-AzStreamAnalyticsFunction

## SYNOPSIS
Ruft Funktionen in einem Streamanalyseauftrag ab.

## SYNTAX

```
Get-AzStreamAnalyticsFunction [-JobName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzStreamAnalyticsFunction"** ruft eine Liste der Funktionen ab, die in einem Azure Stream Analytics-Auftrag definiert sind, oder Informationen zu einer bestimmten Funktion.

## BEISPIELE

### Beispiel 1: Alle Streamanalysefunktionen erhalten
```
PS C:\>Get-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22"
```

Mit diesem Befehl werden die Funktionen für den Auftrag "StreamJob22" definiert.

### Beispiel 2: Erhalten einer bestimmten Streamanalysefunktion
```
PS C:\>Get-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -Name "ScoreTweet"
```

Dieser Befehl ruft Informationen über die Funktion mit dem Namen ScoreTweet ab, die für den Auftrag "StreamJob22" definiert wurde.
