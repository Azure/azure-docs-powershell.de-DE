# New-AzFrontDoorWafManagedRuleOverrideObject

## SYNOPSIS
Objekt zum Außerkraftsetzungsobjekt für verwaltete Regeln erstellen

## SYNTAX

```
New-AzFrontDoorWafManagedRuleOverrideObject -RuleId <String> [-Action <String>] [-Disabled]
 [-Exclusion <PSManagedRuleExclusion[]>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Erstellen Sie ein PSAzureManagedRuleOverride-Objekt für verwaltete WAF-Regelgruppen, die die Objekterstellung überschreiben.

## BEISPIELE

### Beispiel 1
Erstellen Sie ein Überschreibungsobjekt für verwaltete Regeln für Regel 942250 (in der SQLI-Gruppe).

```powershell
PS C:\> New-AzFrontDoorWafManagedRuleOverrideObject -RuleId "942250" -Action Log

RuleId EnabledState Action
------ ------------ ------
942250      Enabled    Log
```
