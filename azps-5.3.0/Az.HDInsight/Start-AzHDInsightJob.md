# Start-AzHDInsightJob

## SYNOPSIS
Startet einen definierten Azure -HDInsight-Auftrag für einen angegebenen Cluster.

## SYNTAX

```
Start-AzHDInsightJob [-ClusterName] <String> [-JobDefinition] <AzureHDInsightJobDefinition>
 [-HttpCredential] <PSCredential> [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Start-AzHDInsightJob"** startet einen definierten Azure HDInsight-Auftrag für einen angegebenen Cluster.
Dies kann ein MapReduce-Auftrag, ein Streaming-MapReduce-Auftrag, eine Struktur oder ein Job im Job sein.

## BEISPIELE

### Beispiel 1: Starten eines Auftrags für den angegebenen Cluster
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential

# Hive job details
PS C:\> $statusFolder = "tempStatusFolder/"
PS C:\> $query = "SHOW TABLES"

PS C:\> New-AzHDInsightHiveJobDefinition -StatusFolder $statusFolder `
            -Query $query `
        | Start-AzHDInsightJob `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

Mit diesem Befehl wird ein Auftrag für den Cluster namens "Ihr-Hadoop-001" gestartet.
