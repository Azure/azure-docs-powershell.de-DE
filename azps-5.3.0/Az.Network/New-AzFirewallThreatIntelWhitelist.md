# New-AzFirewallThreatIntelWhitelist

## SYNOPSIS
Erstellen einer neuen Threat Intelligence-Whitelist für die Azure Firewall

## SYNTAX

```
New-AzFirewallThreatIntelWhitelist [-FQDN <String[]>] [-IpAddress <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "New-AzFirewallThreatIntelWhitelist"** erstellt ein intel whitelist-Threat-Objekt, das beim Erstellen oder Festlegen einer Azure Firewall verwendet werden kann.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> New-AzFirewallThreatIntelWhitelist -IpAddress @("2.2.2.2", "3.3.3.3") -FQDN @("bing.com", "yammer.com")
```

In diesem Beispiel wird eine Intel Whitelist für Bedrohungen erstellt, die eine FQDN-Whitelist mit zwei Einträgen und eine Ip-Adress-Whitelist mit zwei Einträgen enthält.
