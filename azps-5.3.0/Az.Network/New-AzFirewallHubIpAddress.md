# New-AzFirewallHubIpAddress

## SYNOPSIS
Ip-Adressen, die der Firewall auf dem virtuellen Hub zugeordnet sind

## SYNTAX

```
New-AzFirewallHubIpAddress [-PrivateIPAddress <String>] [-PublicIPs <PSAzureFirewallHubPublicIpAddresses>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Ip-Adressen, die der Firewall auf dem virtuellen Hub zugeordnet sind. Dabei kann es sich um öffentliche und private Adressen handelt.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $fwpips = New-AzFirewallHubPublicIpAddress -Count 2
PS C:\> New-AzFirewallHubIpAddress -PublicIPs $fwpips
```

In diesem Beispiel wird ein Hub-Ip-Adressobjekt mit zwei öffentlichen IPs erstellt. Das Objekt "HubIPAddress" ist mit der Firewall auf dem virtuellen Hub verbunden.
