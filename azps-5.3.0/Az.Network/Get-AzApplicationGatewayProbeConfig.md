# Get-AzApplicationGatewayProbeConfig

## SYNOPSIS
Ruft eine vorhandene Integritätstestkonfiguration von einem Anwendungsgateway ab.

## SYNTAX

```
Get-AzApplicationGatewayProbeConfig [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das Get-AzApplicationGatewayProbeConfig cmdlet ruft eine vorhandene Integritätstestkonfiguration von einem Anwendungsgateway ab.

## BEISPIELE

### Beispiel 1: Erhalten einer vorhandenen Probe von einem Anwendungsgateway
```
PS C:\>Get-AzApplicationGatewayProbeConfig -ApplicationGateway Gateway -Name "Probe02"
```

Dieser Befehl ruft den Integritätstest namens "Probe02" vom Anwendungsgateway namens "Gateway" ab.
