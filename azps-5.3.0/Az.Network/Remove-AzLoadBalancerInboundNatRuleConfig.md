# Remove-AzLoadBalancerInboundNatRuleConfig

## SYNOPSIS
Entfernt eine eingehende NAT-Regelkonfiguration aus einem Lastenausgleich.

## SYNTAX

```
Remove-AzLoadBalancerInboundNatRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzLoadBalancerInboundNatRuleConfig"** entfernt eine Nat (Eingehende Netzwerkadressenübersetzung)-Regelkonfiguration aus einem Azure Load Balancer.

## BEISPIELE

### 1: Löschen einer eingehenden NAT-Regel aus einem Azure Load Balancer
```
$loadbalancer = Get-AzLoadBalancer -Name mylb -ResourceGroupName myrg

 Remove-AzLoadBalancerInboundNatRuleConfig -Name "myinboundnatrule" -LoadBalancer $loadbalancer
```

Der erste Befehl lädt einen bereits vorhandenen Lastensaldo namens "mylb" und speichert ihn im Variablen- $load Balancer. Mit dem zweiten Befehl wird die diesem Lastenausgleich zugeordnete eingehende NAT-Regel entfernt.
