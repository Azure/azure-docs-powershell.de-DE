# Remove-AzVpnGateway

## SYNOPSIS
Das Remove-AzVpnGateway entfernt ein Azure VPN-Gateway. Dies ist ein Gateway speziell für die softwaredefinierte Konnektivität des virtuellen Azure-WAN.

## SYNTAX

### ByVpnGatewayName (Standard)
```
Remove-AzVpnGateway -ResourceGroupName <String> -Name <String> [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVpnGatewayObject
```
Remove-AzVpnGateway -InputObject <PSVpnGateway> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVpnGatewayResourceId
```
Remove-AzVpnGateway -ResourceId <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das Remove-AzVpnGateway entfernt ein Azure VPN-Gateway. Dies ist ein Gateway speziell für die softwaredefinierte Konnektivität des virtuellen Azure-WAN.

## BEISPIELE

### Beispiel 1

```powershell
PS C:\> New-AzResourceGroup -Location "West US" -Name "testRG"
PS C:\> $virtualWan = New-AzVirtualWan -ResourceGroupName testRG -Name myVirtualWAN -Location "West US"
PS C:\> $virtualHub = New-AzVirtualHub -VirtualWan $virtualWan -ResourceGroupName "testRG" -Name "westushub" -AddressPrefix "10.0.0.1/24"
PS C:\> New-AzVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" -VirtualHubId $virtualHub.Id -BGPPeeringWeight 10 -VpnGatewayScaleUnit 2
PS C:\> Remove-AzVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" -Passthru
```

In diesem Beispiel wird eine Ressourcengruppe, ein virtuelles WAN, ein virtueller Hub und ein skalierbares VPN-Gateway in den USA erstellt, die dann sofort gelöscht werden. Verwenden Sie das Kennzeichen "-Force", um die Eingabeaufforderung beim Löschen des virtuellen Gateways zu unterdrücken.
Dadurch werden das VpnGateway und alle daran angefügten VpnConnections gelöscht.

### Beispiel 2

```powershell
PS C:\> New-AzResourceGroup -Location "West US" -Name "testRG"
PS C:\> $virtualWan = New-AzVirtualWan -ResourceGroupName testRG -Name myVirtualWAN -Location "West US"
PS C:\> $virtualHub = New-AzVirtualHub -VirtualWan $virtualWan -ResourceGroupName "testRG" -Name "westushub" -AddressPrefix "10.0.0.1/24"
PS C:\> New-AzVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" -VirtualHubId $virtualHub.Id -BGPPeeringWeight 10 -VpnGatewayScaleUnit 2
PS C:\> Get-AzVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" | Remove-AzVpnGateway-Passthru
```

In diesem Beispiel wird eine Ressourcengruppe, ein virtuelles WAN, ein virtueller Hub und ein skalierbares VPN-Gateway in den USA erstellt, die dann sofort gelöscht werden. Dieser Löschvorgang erfolgt mithilfe der Powershell-Piping, die das vom Befehl "Get-AzVpnGateway" zurückgegebene Get-AzVpnGateway verwendet.
Verwenden Sie das Kennzeichen "-Force", um die Eingabeaufforderung beim Löschen des virtuellen Gateways zu unterdrücken.
Dadurch werden das VpnGateway und alle daran angefügten VpnConnections gelöscht.
