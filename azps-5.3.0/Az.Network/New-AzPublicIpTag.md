# New-AzPublicIpTag

## SYNOPSIS
Erstellt ein IP-Tag.

## SYNTAX

```
New-AzPublicIpTag -IpTagType <String> -Tag <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "New-AzPublicIpTag"** erstellt ein IP-Tag.

## BEISPIELE

### Beispiel 1: Erstellen eines neuen IP-Tags
```powershell
$ipTag = New-AzPublicIpTag -IpTagType $ipTagType -Tag $tag
```

Mit diesem Befehl wird ein neues IP-Tag mit dem Tagtyp wie "FirstPartyUsage" und einem Tag wie "/Sql" erstellt. Dies wird bei der Erstellung von publicIpAddress mit diesen speziellen Tags für die Zuweisung verwendet.
