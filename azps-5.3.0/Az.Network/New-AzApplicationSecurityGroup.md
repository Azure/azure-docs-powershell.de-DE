# New-AzApplicationSecurityGroup

## SYNOPSIS
Erstellt eine Anwendungssicherheitsgruppe.

## SYNTAX

```
New-AzApplicationSecurityGroup -ResourceGroupName <String> -Name <String> -Location <String> [-Tag <Hashtable>]
 [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "New-AzApplicationSecurityGroup"** erstellt eine Anwendungssicherheitsgruppe.

## BEISPIELE

### Beispiel 1
```
PS C:\> New-AzApplicationSecurityGroup -ResourceGroupName "MyResourceGroup" -Name "MyApplicationSecurityGroup" -Location "West US"
```

In diesem Beispiel wird eine Anwendungssicherheitsgruppe ohne Zuordnungen erstellt. Nachdem sie erstellt wurde, können die IP-Konfigurationen in der Netzwerkschnittstelle in die Gruppe eingeschlossen werden. Sicherheitsregeln können sich auch als Quellen oder Ziele auf die Gruppe beziehen.
