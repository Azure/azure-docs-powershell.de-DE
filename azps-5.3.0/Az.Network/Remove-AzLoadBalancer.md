# Remove-AzLoadBalancer

## SYNOPSIS
Entfernt einen Lastenausgleich.

## SYNTAX

```
Remove-AzLoadBalancer -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzLoadBalancer"** entfernt einen Azure Load Balancer.

## BEISPIELE

### Beispiel 1: Entfernen eines Lastenausgleichs
```
PS C:\>Remove-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
```

Mit diesem Befehl wird ein Lastenausgleich mit dem Namen "MyLoadBalancer" in der Ressourcengruppe "MyResourceGroup" gelöscht.
