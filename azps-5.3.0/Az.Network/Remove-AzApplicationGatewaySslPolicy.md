# Remove-AzApplicationGatewaySslPolicy

## SYNOPSIS
Entfernt eine SSL-Richtlinie von einem Azure-Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewaySslPolicy -ApplicationGateway <PSApplicationGateway> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das Remove-AzApplicationGatewaySslPolicy entfernt die "SSL-Richtlinie" aus einem Azure-Anwendungsgateway.

## BEISPIELE

### Beispiel 1: Entfernen einer "SSL"-Richtlinie von einem Anwendungsgateway
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGW = Remove-AzApplicationGatewaySslPolicy -ApplicationGateway $AppGW
```

Mit diesem Befehl wird die "SSL-Richtlinie" aus dem Anwendungsgateway namens ApplicationGateway01 entfernt.
