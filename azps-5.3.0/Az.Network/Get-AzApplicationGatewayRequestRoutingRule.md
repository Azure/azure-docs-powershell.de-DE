# Get-AzApplicationGatewayRequestRoutingRule

## SYNOPSIS
Ruft die Anforderungsroutingregel eines Anwendungsgateways ab.

## SYNTAX

```
Get-AzApplicationGatewayRequestRoutingRule [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzApplicationGatewayRequestRoutingRule"** ruft die Anforderungsroutingregel eines Anwendungsgateways ab.

## BEISPIELE

### Beispiel 1: Anfordern einer bestimmten Anforderungsroutingregel
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Rule = Get-AzApplicationGatewayRequestRoutingRule -"Rule01" -ApplicationGateway $AppGW
```

Der erste Befehl ruft das Anwendungsgateway namens ApplicationGateway01 ab und speichert das Ergebnis in der Variablen namens $AppGW.
Der zweite Befehl ruft die Anforderungsroutingregel mit dem Namen "Rule01" vom Anwendungsgateway ab, das in der Variablen namens "$AppGW.

### Beispiel 2: Anfordern einer Liste der Routingregeln für Anfragen
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Rules = Get-AzApplicationGatewayRequestRoutingRule -ApplicationGateway $AppGW
```

Der erste Befehl ruft das Anwendungsgateway namens ApplicationGateway01 ab und speichert das Ergebnis in der Variablen namens $AppGW.
Der zweite Befehl ruft eine Liste der Anforderungsroutingregeln aus dem Anwendungsgateway ab, das in der Variablen namens $AppGW.
