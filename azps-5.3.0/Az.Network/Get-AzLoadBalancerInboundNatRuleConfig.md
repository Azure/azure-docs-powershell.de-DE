# Get-AzLoadBalancerInboundNatRuleConfig

## SYNOPSIS
Ruft eine eingehende NAT-Regelkonfiguration für einen Lastenausgleich ab.

## SYNTAX

```
Get-AzLoadBalancerInboundNatRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzLoadBalancerInboundNatRuleConfig"** ruft eine oder mehrere REGELN für die eingehende Netzwerkadressenübersetzung (Nat) in einem Azure Load Balancer ab.

## BEISPIELE

### Beispiel 1: Erhalten einer eingehenden NAT-Regelkonfiguration
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerInboundNatRuleConfig -Name "MyInboundNatRule1" -LoadBalancer $slb
```

Der erste Befehl ruft den Lastenausgleich namens "MyLoadBalancer" ab und speichert ihn in der Variablen $slb.
Der zweite Befehl ruft die zugeordnete NAT-Regel namens "MyInboundNatRule1" aus dem Lastenausgleich in $slb.
