# Remove-AzPublicIpPrefix

## SYNOPSIS
Entfernt ein öffentliches IP-Präfix.

## SYNTAX

### RemoveByNameParameterSet (Standard)
```
Remove-AzPublicIpPrefix -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByResourceIdParameterSet
```
Remove-AzPublicIpPrefix -ResourceId <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByInputObjectParameterSet
```
Remove-AzPublicIpPrefix -InputObject <PSPublicIpPrefix> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzPublicIpPrefix"** entfernt ein öffentliches Azure-IP-Präfix, solange keine öffentlichen IP-Adressen zugewiesen sind.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Remove-AzPublicIpPrefix -Name $prefixName -ResourceGroupName $rgName
```

Entfernt das öffentliche IP-Präfix mit name $prefixName aus der Ressourcengruppe $rgName
