# New-AzRedisCacheScheduleEntry

## SYNOPSIS
Erstellt einen Zeitplaneintrag.

## SYNTAX

```
New-AzRedisCacheScheduleEntry -DayOfWeek <String> -StartHourUtc <Int32> [-MaintenanceWindow <TimeSpan>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "New-AzRedisCacheScheduleEntry"** erstellt ein **PSScheduleEntry-Objekt.**
Azure Redis Cache patch schedule cmdlets, such as the New-AzRedisCachePatchSchedule cmdlet, require schedule entry objects.

## BEISPIELE

### Beispiel 1: Erstellen eines Zeitplaneintrags für Wochenenden
```
PS C:\>New-AzRedisCacheScheduleEntry -DayOfWeek "Weekend" -StartHourUtc 2 -MaintenanceWindow "06:00:00"
```

Mit diesem Befehl wird ein **"PSScheduleEntry"-Objekt** erstellt, das einen Wochenendplan mit der angegebenen Startzeit und dem angegebenen Fenster darstellt.
