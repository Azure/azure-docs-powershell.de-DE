# Export-AzAnalysisServicesInstanceLog

## SYNOPSIS
Exportiert ein Protokoll aus einer Instanz des Analysis Services-Servers in der aktuell in der Umgebung angemeldeten Umgebung, wie im Befehl Add-AzAnalysisServicesAccount angegeben.

## SYNTAX

```
Export-AzAnalysisServicesInstanceLog -OutputPath <String> [-Force] [-Instance] <String> [-PassThru] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das Export-AzAnalysisServicesInstance exportiert das Protokoll aus einer Instanz des Azure Analysis Services-Servers in eine Datei.

## BEISPIELE

### Beispiel 1
```
PS C:\>Export-AzAnalysisServicesInstanceLog -Instance testserver -OutputPath C:\path\to\log\testserver.log
```

Mit diesem Befehl wird das Protokoll vom Server "testserver" in der im Befehl Add-AzAnalysisServicesAccount angegebenen Umgebung exportiert und in der in OutputPath "C:\path\to\log\testserver.log" angegebenen Datei gespeichert.
