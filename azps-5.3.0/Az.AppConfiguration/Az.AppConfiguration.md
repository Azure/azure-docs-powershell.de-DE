# Az.AppConfiguration Module
## Beschreibung
Microsoft Azure PowerShell: Cmdlets für die AppConfiguration

## Az.AppConfiguration-Cmdlets
### [Get-AzAppConfigurationStore](Get-AzAppConfigurationStore.md)
Laden Sie Konfigurationsspeicher für Apps herunter, oder listen Sie sie auf.

### [Get-AzAppConfigurationStoreKey](Get-AzAppConfigurationStoreKey.md)
Listet den Zugriffsschlüssel für den angegebenen Konfigurationsspeicher auf.

### [New-AzAppConfigurationStore](New-AzAppConfigurationStore.md)
Erstellt einen Konfigurationsspeicher mit den angegebenen Parametern.

### [New-AzAppConfigurationStoreKey](New-AzAppConfigurationStoreKey.md)
Generiert einen Zugriffsschlüssel für den angegebenen Konfigurationsspeicher erneut.

### [Remove-AzAppConfigurationStore](Remove-AzAppConfigurationStore.md)
Löscht einen Konfigurationsspeicher.

### [Test-AzAppConfigurationStoreNameAvailability](Test-AzAppConfigurationStoreNameAvailability.md)
Überprüft, ob der Name des Konfigurationsspeichers zur Verwendung verfügbar ist.

### [Update-AzAppConfigurationStore](Update-AzAppConfigurationStore.md)
Aktualisiert einen Konfigurationsspeicher mit den angegebenen Parametern.
