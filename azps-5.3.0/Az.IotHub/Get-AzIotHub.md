# Get-AzIotHub

## SYNOPSIS
Ruft Informationen zu den IotHubs in einem Abonnement ab.

## SYNTAX

### ListIotHubsByResourceGroup (Standard)
```
Get-AzIotHub [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetIotHubByName
```
Get-AzIotHub [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Ruft Informationen zu den IotHubs in einem Abonnement ab.
Sie können alle Instanzen von IotHub in einem Abonnement anzeigen oder die Ergebnisse nach einer Ressourcengruppe oder einem bestimmten Namen für IotHub filtern.

## BEISPIELE

### Beispiel 1
```
PS C:\> Get-AzIotHub
```

Ruft alle IotHubs im Abonnement ab.

### Beispiel 2
```
PS C:\> Get-AzIotHub -ResourceGroupName "myresourcegroup"
```

Ruft alle IotHubs im Abonnement ab, die zur Ressourcengruppe "myresourcegroup" gehören.

### Beispiel 3
```
PS C:\> Get-AzIotHub -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Ruft Informationen zum IotHub namens "myiothub" ab.
