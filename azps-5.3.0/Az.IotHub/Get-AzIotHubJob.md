# Get-AzIotHubJob

## SYNOPSIS
Ruft die Informationen zu einem IotHub-Auftrag ab.

## SYNTAX

```
Get-AzIotHubJob [-ResourceGroupName] <String> [-Name] <String> [[-JobId] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Ruft die Informationen zu einem IotHub-Auftrag ab.
Ein IotHub-Auftrag wird erstellt, wenn ein Import- oder Exportvorgang mithilfe der Befehle New-AzIotHubExportDevices oder New-AzIotHubImportDevices wird.
Sie können entweder alle Aufträge auflisten oder die Aufträge nach der Auftrags-ID filtern.

## BEISPIELE

### Beispiel 1: Auflisten aller Aufträge
```
PS C:\> Get-AzIotHubJob -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Ruft alle Aufträge für IotHub namens "myiothub" ab.

### Beispiel 2: Erhalten eines bestimmten Auftrags
```
PS C:\> Get-AzIotHubJob -ResourceGroupName "myresourcegroup" -Name "myiothub" -JobId 3630fc31-4caa-43e8-a232-ea0577221cb2
```

Ruft Informationen zu dem Auftrag mit dem Bezeichner "3630fc31-4caa-43e8-a232-ea0577221cb2" für IotHub namens "myiothub" ab.
