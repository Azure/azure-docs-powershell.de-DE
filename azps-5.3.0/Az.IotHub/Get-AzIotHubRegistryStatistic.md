# Get-AzIotHubRegistryStatistic

## SYNOPSIS
Ruft die Registrierungsstatistik für einen IotHub ab.

## SYNTAX

```
Get-AzIotHubRegistryStatistic [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Ruft die Registrierungsstatistik für einen IotHub ab.
Hier finden Sie Informationen zur Gesamtzahl der aktivierten und deaktivierten Geräte in einem IotHub.

## BEISPIELE

### Beispiel 1: "RegistryStatistics"
```
PS C:\> Get-AzIotHubRegistryStatistic -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Ruft die Registrierungsstatistik für IotHub mit dem Namen "myiothub" ab.
