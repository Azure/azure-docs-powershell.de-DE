# Update-AzScheduledQueryRule

## SYNOPSIS
Aktualisiert eine Protokollbenachrichtigungsregel

## SYNTAX

### ByRuleName (Standard)
```
Update-AzScheduledQueryRule -Name <String> -ResourceGroupName <String> -Enabled <Boolean>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Update-AzScheduledQueryRule -InputObject <PSScheduledQueryRuleResource> -Enabled <Boolean>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Update-AzScheduledQueryRule -ResourceId <String> -Enabled <Boolean> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Aktualisiert eine Protokollwarnungsregel, und das Aktualisieren nur der Eigenschaft "Enabled" wird von diesem Befehl unterstützt.
Informationen zum Aktualisieren anderer Eigenschaften finden Sie unter ["Set-AzScheduledQueryRule".](https://docs.microsoft.com/en-us/powershell/module/az.monitor/set-azscheduledqueryrule)
