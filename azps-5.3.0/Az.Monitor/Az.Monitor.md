# Az.Monitor-Modul
## Beschreibung
In diesem Thema werden Hilfethemen für die Azure Insights-Cmdlets angezeigt.

## Az.Monitor-Cmdlets
### [Add-AzAutoscaleSetting](Add-AzAutoscaleSetting.md)
Erstellt eine Einstellung für die Automatische Skala.

### [Add-AzLogProfile](Add-AzLogProfile.md)
Erstellt ein neues Aktivitätsprotokollprofil. Dieses Profil wird verwendet, um das Aktivitätsprotokoll entweder in einem Azure -Speicherkonto zu archivieren oder es in einen Azure-Ereignishub im selben Abonnement zu streamen. 

### [Add-AzMetricAlertRule](Add-AzMetricAlertRule.md)
Fügt eine metrisch-basierte Warnungsregel hinzu oder aktualisiert diese.

### [Add-AzMetricAlertRuleV2](Add-AzMetricAlertRuleV2.md)
Fügt eine auf V2 basierende (nicht klassische) metrische Warnungsregel hinzu oder aktualisiert diese.

### [Add-AzWebtestAlertRule](Add-AzWebtestAlertRule.md)
Fügt eine Webtestwarnungsregel hinzu oder aktualisiert diese.

### [Disable-AzActivityLogAlert](Disable-AzActivityLogAlert.md)
Deaktiviert eine Aktivitätsprotokollbenachrichtigung und legt deren Tags fest.

### [Enable-AzActivityLogAlert](Enable-AzActivityLogAlert.md)
Aktiviert eine Aktivitätsprotokollbenachrichtigung und legt deren Tags fest.

### [Get-AzActionGroup](Get-AzActionGroup.md)
Ruft Aktionsgruppe(n) ab.

### [Get-AzActivityLog](Get-AzActivityLog.md)
Rufen Sie Aktivitätsprotokollereignisse ab.

### [Get-AzActivityLogAlert](Get-AzActivityLogAlert.md)
Ruft mindestens eine Aktivitätsprotokollbenachrichtigungsressourcen ab.

### [Get-AzAlertHistory](Get-AzAlertHistory.md)
Ruft den Verlauf von Benachrichtigungen ab.

### [Get-AzAlertRule](Get-AzAlertRule.md)
Ruft Benachrichtigungsregeln ab.

### [Get-AzAutoscaleHistory](Get-AzAutoscaleHistory.md)
Ruft den Verlauf der Automatischen Skala ab.

### [Get-AzAutoscaleSetting](Get-AzAutoscaleSetting.md)
Ruft die Einstellungen für die automatische Skalierung ab.

### [Get-AzDiagnosticSetting](Get-AzDiagnosticSetting.md)
Ruft die protokollierten Kategorien und Zeitkörner ab.

### [Get-AzDiagnosticSettingCategory](Get-AzDiagnosticSettingCategory.md)
Erhalten oder Auflisten der unterstützten Diagnoseeinstellungskategorie für die Azure-Ressource.

### [Get-AzInsightsPrivateLinkScope](Get-AzInsightsPrivateLinkScope.md)

### [Get-AzInsightsPrivateLinkScopedResource](Get-AzInsightsPrivateLinkScopedResource.md)

### [Get-AzMetric](Get-AzMetric.md)

### [Get-AzMetricAlertRuleV2](Get-AzMetricAlertRuleV2.md)

### [Get-AzMetricDefinition](Get-AzMetricDefinition.md)

### [Get-AzScheduledQueryRule](Get-AzScheduledQueryRule.md)
