# Get-AzStorageAccountNetworkRuleSet

## Synopsis
Abrufen der NetWorkRule-Eigenschaft eines speicherkontos

## Syntax

```
Get-AzStorageAccountNetworkRuleSet [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzStorageAccountNetworkRuleSet** " Ruft die NetworkRule-Eigenschaft eines speicherkontos ab.

## Beispiele

### Beispiel 1: Abrufen der NetworkRule-Eigenschaft eines angegebenen speicherkontos
```
PS C:\> Get-AzStorageAccountNetworkRuleSet  -ResourceGroupName "rg1" -AccountName "mystorageaccount"
```

Dieser Befehl ruft die NetworkRule-Eigenschaft eines angegebenen speicherkontos ab.
