# Get-AzIotHubRegistryStatistic

## Synopsis
Ruft die RegistryStatistics für eine IotHub.

## Syntax

```
Get-AzIotHubRegistryStatistic [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Ruft die RegistryStatistics für eine IotHub.
Damit erhalten Sie Informationen über die Anzahl der insgesamt aktivierten und deaktivierten Geräte in einem IotHub.

## Beispiele

### Beispiel 1 Abrufen des RegistryStatistics
```
PS C:\> Get-AzIotHubRegistryStatistic -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Ruft die RegistryStatistics für die IotHub mit dem Namen "myiothub" ab.
