# Revoke-AzSnapshotAccess

## Synopsis
Widerruft einen Zugriff auf einen Schnappschuss.

## Syntax

```
Revoke-AzSnapshotAccess [-ResourceGroupName] <String> [-SnapshotName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das **REVOKE-AzSnapshotAccess-** Cmdlet widerruft einen Zugriff auf einen Schnappschuss.

## Beispiele

### Beispiel 1
```
PS C:\> Revoke-AzDiskAccess -ResourceGroupName 'ResourceGroup01' -SnapshotName 'Snapshot01'
```

Widerrufen des Zugriffs auf den Snapshot mit dem Namen "Snapshot01" in der Ressourcengruppe "ResourceGroup01"
