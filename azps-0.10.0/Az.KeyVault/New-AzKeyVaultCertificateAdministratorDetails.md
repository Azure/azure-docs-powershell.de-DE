# New-AzKeyVaultCertificateAdministratorDetails

## Synopsis
Erstellt ein Zertifikat Administrator Details-Objekt im Arbeitsspeicher.

## Syntax

```
New-AzKeyVaultCertificateAdministratorDetails [-FirstName <String>] [-LastName <String>]
 [-EmailAddress <String>] [-PhoneNumber <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **New-AzKeyVaultCertificateAdministratorDetails** wird ein speicherresidentes Zertifikat-Administrator Details-Objekt erstellt.

## Beispiele

### Beispiel 1: Erstellen eines Zertifikats Administrator Details-Objekts
```
PS C:\>$AdminDetails = New-AzKeyVaultCertificateAdministratorDetails -FirstName "Patti" -LastName "Fuller" -EmailAddress "patti.fuller@contoso.com" -PhoneNumber "5553334444"
```

Mit diesem Befehl wird ein in-Memory-Zertifikat Administrator Details-Objekt erstellt und dann in der $AdminDetails-Variablen gespeichert.
