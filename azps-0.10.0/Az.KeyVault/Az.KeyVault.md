# AZ. keyvault-Modul
## Beschreibung
In diesem Thema werden Hilfethemen zu den Azure Key Vault-Cmdlets angezeigt.

## AZ. keyvault-Cmdlets
### [Add-AzKeyVaultCertificate](Add-AzKeyVaultCertificate.md)
Fügt ein Zertifikat zu einem schlüsseltresor hinzu.

### [Add-AzKeyVaultCertificateContact](Add-AzKeyVaultCertificateContact.md)
Fügt einen Kontakt für Zertifikat Benachrichtigungen hinzu.

### [Add-AzKeyVaultKey](Add-AzKeyVaultKey.md)
Erstellt einen Schlüssel in einem schlüsseltresor oder importiert einen Schlüssel in einen schlüsseltresor.

### [Add-AzKeyVaultManagedStorageAccount](Add-AzKeyVaultManagedStorageAccount.md)
Fügt dem angegebenen schlüsseltresor ein vorhandenes Azure Storage-Konto hinzu, dessen Schlüssel vom Schlüssel-Vault-Dienst verwaltet werden.

### [Backup-AzKeyVaultKey](Backup-AzKeyVaultKey.md)
Sie können einen Schlüssel in einem schlüsseltresor sichern.

### [Backup-AzKeyVaultSecret](Backup-AzKeyVaultSecret.md)
Sichern eines Geheimnisses in einem schlüsseltresor

### [Get-AzKeyVaultCertificate](Get-AzKeyVaultCertificate.md)
Ruft ein Zertifikat aus einem schlüsseltresor ab.

### [Get-AzKeyVaultCertificateContact](Get-AzKeyVaultCertificateContact.md)
Ruft Kontakte ab, die für Zertifikat Benachrichtigungen für einen schlüsseltresor registriert sind.

### [Get-AzKeyVaultCertificateIssuer](Get-AzKeyVaultCertificateIssuer.md)
Ruft einen Zertifikataussteller für einen schlüsseltresor ab.

### [Get-AzKeyVaultCertificateOperation](Get-AzKeyVaultCertificateOperation.md)
Ruft den Status eines Zertifikat Vorgangs ab.

### [Get-AzKeyVaultCertificatePolicy](Get-AzKeyVaultCertificatePolicy.md)
Ruft die Richtlinie für ein Zertifikat in einem schlüsseltresor ab.

### [Get-AzKeyVaultKey](Get-AzKeyVaultKey.md)
Ruft schlüsseltresor Schlüssel ab.

### [Get-AzKeyVaultManagedStorageAccount](Get-AzKeyVaultManagedStorageAccount.md)
Ruft wichtige Vault Managed Azure-Speicherkonten ab.

### [Get-AzKeyVaultManagedStorageSasDefinition](Get-AzKeyVaultManagedStorageSasDefinition.md)
Ruft Schlüsselspeicher-SAS-Definitionen für verwalteten Speicher ab.

### [Get-AzKeyVaultSecret](Get-AzKeyVaultSecret.md)
Ruft die Geheimnisse in einem schlüsseltresor ab.

### [Get-AzKeyVault](Get-AzKeyVault.md)
Ruft Schlüsseldepots ab.

### [Importieren-AzKeyVaultCertificate](Import-AzKeyVaultCertificate.md)
Importiert ein Zertifikat in einen schlüsseltresor.

### [Neu – AzKeyVaultCertificateAdministratorDetails](New-AzKeyVaultCertificateAdministratorDetails.md)
Erstellt ein Zertifikat Administrator Details-Objekt im Arbeitsspeicher.

### [Neu – AzKeyVaultCertificateOrganizationDetails](New-AzKeyVaultCertificateOrganizationDetails.md)
Erstellt ein Objekt im Arbeitsspeicher Zertifikat Organisation Details.

### [Neu – AzKeyVaultCertificatePolicy](New-AzKeyVaultCertificatePolicy.md)
Erstellt ein Zertifikatrichtlinien Objekt im Arbeitsspeicher.

### [Neu – AzKeyVault](New-AzKeyVault.md)
Erstellt einen schlüsseltresor.

### [Remove-AzKeyVaultCertificate](Remove-AzKeyVaultCertificate.md)
Entfernt ein Zertifikat aus einem schlüsseltresor.

### [Remove-AzKeyVaultCertificateContact](Remove-AzKeyVaultCertificateContact.md)
Löscht einen Kontakt, der für Zertifikat Benachrichtigungen aus einem schlüsseltresor registriert ist.

### [Remove-AzKeyVaultCertificateIssuer](Remove-AzKeyVaultCertificateIssuer.md)
Löscht einen Zertifikataussteller aus einem schlüsseltresor.

### [Remove-AzKeyVaultCertificateOperation](Remove-AzKeyVaultCertificateOperation.md)
Löscht einen Zertifikat Vorgang aus einem schlüsseltresor.

### [Remove-AzKeyVaultKey](Remove-AzKeyVaultKey.md)
Löscht einen Schlüssel in einem schlüsseltresor.

### [Remove-AzKeyVaultManagedStorageAccount](Remove-AzKeyVaultManagedStorageAccount.md)
Entfernt ein zentrales Vault Managed Azure-Speicherkonto und alle zugehörigen SAS-Definitionen.

### [Remove-AzKeyVaultManagedStorageSasDefinition](Remove-AzKeyVaultManagedStorageSasDefinition.md)
Entfernt eine Schlüsselspeicher-SAS-Definition für verwalteten Azure-Speicher.

### [Remove-AzKeyVaultSecret](Remove-AzKeyVaultSecret.md)
Löscht einen geheimen Schlüssel in einem schlüsseltresor.

### [Remove-AzKeyVault](Remove-AzKeyVault.md)
Löscht einen schlüsseltresor.

### [Remove-AzKeyVaultAccessPolicy](Remove-AzKeyVaultAccessPolicy.md)
Entfernt alle Berechtigungen für einen Benutzer oder eine Anwendung aus einem schlüsseltresor.

### [Wiederherstellen – AzKeyVaultKey](Restore-AzKeyVaultKey.md)
Erstellt einen Schlüssel in einem schlüsseltresor aus einem gesicherten Schlüssel.

### [Wiederherstellen – AzKeyVaultSecret](Restore-AzKeyVaultSecret.md)
Erstellt in einem schlüsseltresor einen geheimen Schlüssel aus einem gesicherten Secret-Schlüssel.

### [Satz-AzKeyVaultCertificateAttribute](Set-AzKeyVaultCertificateAttribute.md)
Ändert bearbeitbare Attribute eines Zertifikats.

### [Satz-AzKeyVaultCertificateIssuer](Set-AzKeyVaultCertificateIssuer.md)
Legt einen Zertifikataussteller in einem schlüsseltresor fest.

### [Satz-AzKeyVaultCertificatePolicy](Set-AzKeyVaultCertificatePolicy.md)
Erstellt oder aktualisiert die Richtlinie für ein Zertifikat in einem schlüsseltresor.

### [Satz-AzKeyVaultKeyAttribute](Set-AzKeyVaultKeyAttribute.md)
Aktualisiert die Attribute eines Schlüssels in einem schlüsseltresor.

### [Satz-AzKeyVaultManagedStorageSasDefinition](Set-AzKeyVaultManagedStorageSasDefinition.md)
Legt eine freigegebene zugriffssignatur Definition (SAS) mit Schlüsselspeicher für ein gegebenes, zentrales Vault Managed Azure-Speicherkonto fest.

### [Satz-AzKeyVaultSecret](Set-AzKeyVaultSecret.md)
Erstellt oder aktualisiert ein Kennwort in einem schlüsseltresor.

### [Satz-AzKeyVaultSecretAttribute](Set-AzKeyVaultSecretAttribute.md)
Aktualisiert Attribute eines Geheimnisses in einem schlüsseltresor.

### [Satz-AzKeyVaultAccessPolicy](Set-AzKeyVaultAccessPolicy.md)
Gewährt oder ändert vorhandene Berechtigungen für einen Benutzer, eine Anwendung oder eine Sicherheitsgruppe, um Vorgänge mit einem schlüsseltresor durchzuführen.

### [Stopp-AzKeyVaultCertificateOperation](Stop-AzKeyVaultCertificateOperation.md)
Bricht eine Zertifikat Operation in Key Vault ab.

### [Undo-AzKeyVaultCertificateRemoval](Undo-AzKeyVaultCertificateRemoval.md)
Stellt ein gelöschtes Zertifikat in einem schlüsseltresor in einen aktiven Zustand ein.

### [Undo-AzKeyVaultKeyRemoval](Undo-AzKeyVaultKeyRemoval.md)
Stellt einen gelöschten Schlüssel in einem schlüsseltresor in einen aktiven Zustand.

### [Undo-AzKeyVaultSecretRemoval](Undo-AzKeyVaultSecretRemoval.md)
Stellt ein gelöschtes Geheimnis in einem schlüsseltresor in einen aktiven Zustand.

### [Undo-AzKeyVaultRemoval](Undo-AzKeyVaultRemoval.md)
Stellt einen gelöschten schlüsseltresor in einen aktiven Zustand.

### [Update-AzKeyVaultManagedStorageAccount](Update-AzKeyVaultManagedStorageAccount.md)
Aktualisieren Sie die bearbeitbaren Attribute eines von Vault verwalteten Azure-speicherkontos.

### [Update-AzKeyVaultManagedStorageAccountKey](Update-AzKeyVaultManagedStorageAccountKey.md)
Regeneriert den angegebenen Schlüssel des verwalteten Azure-speicherkontos für Schlüsselspeicher.
