# Stop-AzKeyVaultCertificateOperation

## Synopsis
Bricht eine Zertifikat Operation in Key Vault ab.

## Syntax

```
Stop-AzKeyVaultCertificateOperation [-VaultName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet " **Stop-AzKeyVaultCertificateOperation** " wird ein Zertifikat Vorgang im Azure Key Vault-Dienst abgebrochen.
