# Remove-AzPublicIpAddress

## Synopsis
Entfernt eine öffentliche IP-Adresse.

## Syntax

```
Remove-AzPublicIpAddress -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzPublicIpAddress** entfernt eine Azure Public-IP-Adresse.

## Beispiele

### 1: Entfernen einer öffentlichen IP-Adressen Ressource
```
Remove-AzPublicIpAddress -Name $publicIpName -ResourceGroupName $rgName
```

Dieser Befehl entfernt die öffentliche IP-Adressen Ressource mit dem Namen $publicIpName in der Ressourcengruppe $rgName.
