# Set-AzVirtualNetworkGatewayConnectionSharedKey

## Synopsis
Konfiguriert den freigegebenen Schlüssel der virtuellen Netzwerkgateway-Verbindung.

## Syntax

```
Set-AzVirtualNetworkGatewayConnectionSharedKey -Name <String> -ResourceGroupName <String> -Value <String>
 [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **festlegen-AzVirtualNetworkGatewayConnectionSharedKey** " konfiguriert den freigegebenen Schlüssel der virtuellen Netzwerkgateway-Verbindung.
