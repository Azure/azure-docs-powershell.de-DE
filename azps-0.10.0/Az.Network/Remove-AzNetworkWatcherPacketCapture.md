# Remove-AzNetworkWatcherPacketCapture

## Synopsis
Entfernt eine Paket Erfassungs Ressource.

## Syntax

### SetByResource (Standard)
```
Remove-AzNetworkWatcherPacketCapture -NetworkWatcher <PSNetworkWatcher> -PacketCaptureName <String>
 [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByName
```
Remove-AzNetworkWatcherPacketCapture -NetworkWatcherName <String> -ResourceGroupName <String>
 -PacketCaptureName <String> [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Beschreibung
Die Remove-AzNetworkWatcherPacketCapture entfernt eine Sammlungs Ressource. Es wird empfohlen, Stop-AzNetworkWatcherPacketCapture aufzurufen, bevor Sie Remove-AzNetworkWatcherPacketCapture aufrufen. Wenn die Paket Aufnahmesitzung ausgeführt wird, wenn Remove-AzNetworkWatcherPacketCapture aufgerufen wird, wird die Paketerfassung möglicherweise nicht gespeichert. Wenn die Sitzung vor dem Entfernen angehalten wird, wird die Cap-Datei, die die Aufnahmedaten enthält, nicht entfernt. 

## Beispiele

### ---Beispiel 1: Entfernen einer Paket Aufnahmesitzung--
```
Remove-AzNetworkWatcherPacketCapture -NetworkWatcher $networkWatcher -PacketCaptureName "PacketCaptureTest"
```

In diesem Beispiel wird eine vorhandene Paket erfassungssitzung mit dem Namen "PacketCaptureTest" entfernt.
