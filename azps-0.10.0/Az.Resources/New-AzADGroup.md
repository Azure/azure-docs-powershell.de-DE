# New-AzADGroup

## Synopsis
Erstellt eine neue Active Directory-Gruppe.

## Syntax

```
New-AzADGroup -DisplayName <String> -MailNickname <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Erstellt eine neue Active Directory-Gruppe.

## Beispiele

### Beispiel 1 – Erstellen einer neuen Anzeigengruppe

```
PS C:\> New-AzADGroup -DisplayName "MyGroupDisplayName" -MailNickname "myemail@domain.com"
```

Erstellt eine neue Anzeigengruppe mit dem Namen "MyGroupDisplayName" und dem e-Mail-Spitznamen " myemail@domain.com ".
