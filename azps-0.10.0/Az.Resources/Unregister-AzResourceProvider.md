# Unregister-AzResourceProvider

## Synopsis
Hebt die Registrierung eines Ressourcenanbieters auf.

## Syntax

```
Unregister-AzResourceProvider -ProviderNamespace <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Unregister-AzResourceProvider** " hebt die Registrierung eines Azure-Ressourcenanbieters auf.
