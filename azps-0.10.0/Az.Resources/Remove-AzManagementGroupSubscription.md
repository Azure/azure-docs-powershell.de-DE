# Remove-AzManagementGroupSubscription

## Synopsis
Entfernt ein Abonnement aus einer Verwaltungsgruppe.

## Syntax

```
Remove-AzManagementGroupSubscription [-GroupName] <String> [-SubscriptionId] <Guid> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzManagementGroupSubscription** entfernt ein Abonnement aus einer Verwaltungsgruppe.

## Beispiele

### Beispiel 1 – Entfernen des Abonnements aus einer Verwaltungsgruppe
```
PS C:\> Remove-AzManagementGroupSubscription -GroupName "TestGroup" -SubscriptionId 2120692d-35c3-44c8-81f5-631fa7351726
```
