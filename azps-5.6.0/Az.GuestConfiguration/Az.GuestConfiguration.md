# Az.GuestConfiguration Module
## Beschreibung
Modul mit Cmdlets, mit denen Benutzer den Compliancezustand ihrer virtuellen Computer in Azure basierend auf den Ergebnissen der Azure-Richtlinie verstehen können.

## Az.GuestConfiguration-Cmdlets
### [Get-AzVMGuestPolicyStatus](Get-AzVMGuestPolicyStatus.md)
Ruft gastkonfigurationsrichtlinienstatus (detailliert) für eine Initiative vom Typ "Gastkonfiguration" ab, die einer VM zugewiesen ist.
Eine Initiative ist eine Richtlinie des Definitionstyps "Initiative".

### [Get-AzVMGuestPolicyStatusHistory](Get-AzVMGuestPolicyStatusHistory.md)
Ruft den Compliancestatus der Gastkonfigurationsrichtlinie für eine Initiative vom Typ "Gastkonfiguration" ab, die einer VM zugewiesen ist.
Eine Initiative ist eine Richtlinie des Definitionstyps "Initiative".
