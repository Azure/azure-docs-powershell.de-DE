# Az.ConnectedKubernetes-Modul
## Beschreibung
Microsoft Azure PowerShell: ConnectedKubernetes-Cmdlets

## Az.ConnectedKubernetes-Cmdlets
### [Get-AzConnectedKubernetes](Get-AzConnectedKubernetes.md)
Gibt die Eigenschaften des angegebenen verbundenen Clusters zurück, einschließlich Name, Identität, Eigenschaften und zusätzlichen Clusterdetails.

### [New-AzConnectedKubernetes](New-AzConnectedKubernetes.md)
API zum Registrieren eines neuen K8s-Clusters und damit zum Erstellen einer nachverfolgten Ressource in ARM

### [Remove-AzConnectedKubernetes](Remove-AzConnectedKubernetes.md)
Löschen Sie einen verbundenen Cluster, und entfernen Sie die nachverfolgte Ressource in Azure Resource Manager (ARM).

### [Update-AzConnectedKubernetes](Update-AzConnectedKubernetes.md)
API, um bestimmte Eigenschaften der verbundenen Clusterressource zu aktualisieren.
