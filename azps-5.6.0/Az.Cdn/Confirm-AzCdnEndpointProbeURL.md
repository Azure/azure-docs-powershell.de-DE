# Confirm-AzCdnEndpointProbeURL

## SYNOPSIS
Überprüft eine Prüfpunkt-URL.

## SYNTAX

```
Confirm-AzCdnEndpointProbeURL -ProbeUrl <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Confirm-AzCdnEndpointProbeURL** bestätigt, ob die bereitgestellte Prüfpunkt-URL für die dynamische Websitebeschleunigung verwendet werden kann.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Confirm-AzCdnEndpointProbeURL -ProbeUrl "http://www.bing.com/images"
IsValid: true
ErrorCode: None
Message:
```

Überprüft die Prüfpunkt-URL " http://www.bing.com/images " "
