# Get-AzCdnEndpointNameAvailability

## SYNOPSIS
Ruft den Verfügbarkeitsstatus des CDN-Endpunkts ab.

## SYNTAX

```
Get-AzCdnEndpointNameAvailability -EndpointName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Get-AzCdnEndpointNameAvailability** erhält den Verfügbarkeitsstatus des Azure Content Delivery Network (CDN)-Endpunkts.
