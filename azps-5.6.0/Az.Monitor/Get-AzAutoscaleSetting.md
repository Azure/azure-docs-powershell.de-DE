# Get-AzAutoscaleSetting

## SYNOPSIS
Ruft Einstellungen für die automatische Skalierung ab.

## SYNTAX

```
Get-AzAutoscaleSetting -ResourceGroupName <String> [-Name <String>] [-DetailedOutput]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzAutoscaleSetting-Cmdlet** ruft alle Autoscale-Einstellungen ab, die einer Ressourcengruppe oder einer angegebenen Einstellung für die automatische Skalierung zugeordnet sind.
