# Get-AzAutoscaleHistory

## SYNOPSIS
Ruft den Verlauf der automatischen Skalierung ab.

## SYNTAX

```
Get-AzAutoscaleHistory [-ResourceId <String>] [-StartTime <DateTime>] [-EndTime <DateTime>] [-Status <String>]
 [-Caller <String>] [-DetailedOutput] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzAutoscaleHistory-Cmdlet** ruft den Verlauf von Ereignissen ab, die mit einer Einstellung für die automatische Skalierung verknüpft sind.

## BEISPIELE

### Beispiel 1: Alle ereignisse, die einem Abonnement zugeordnet sind
```
PS C:\>Get-AzAutoscaleHistory -StartTime 2015-02-09T18:35:00 -EndTime 2015-02-09T18:40:00 -DetailedOutput
```

Dieser Befehl ruft alle autoscalebezogenen Ereignisse ab, die dem aktuellen Abonnement zugeordnet sind.
