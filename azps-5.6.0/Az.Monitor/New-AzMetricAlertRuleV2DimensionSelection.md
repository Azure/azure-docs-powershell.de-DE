# New-AzMetricAlertRuleV2DimensionSelection

## SYNOPSIS
Erstellt ein lokales Dimensionsauswahlobjekt, das zum Erstellen eines Metrikwarnkriteriums verwendet werden kann.

## SYNTAX

```
New-AzMetricAlertRuleV2DimensionSelection -DimensionName <String> -ValuesToInclude <String[]>
 [-ValuesToExclude <String[]>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet New-AzMetricAlertRuleV2DimensionSelection** erstellt ein lokales Dimensionsauswahlobjekt, das beim Erstellen von Metrikwarnkriterien mithilfe des [Cmdlets New-AzMetricAlertRuleV2Criteria](https://docs.microsoft.com/powershell/module/az.monitor/new-azmetricalertrulev2criteria) helfen soll.

## BEISPIELE

### Beispiel 1: Erstellen eines Bemaßungsauswahlobjekts

```powershell
PS C:\> New-AzMetricAlertRuleV2DimensionSelection -DimensionName LUN -ValuesToInclude 1,3,4

Dimension IncludeValues ExcludeValues
--------- ------------- -------------
LUN       {1, 3, 4}
```

Mit diesem Befehl wird ein Dimensionsauswahlobjekt erstellt, das angibt, dass Werte für {1,3,4} Dimension "LUN" ausgewählt werden müssen.
