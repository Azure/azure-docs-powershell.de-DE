# Get-AzDiagnosticSetting

## SYNOPSIS
Ruft die protokollierten Kategorien und Zeitkörner ab.

## SYNTAX

```
Get-AzDiagnosticSetting [-ResourceId] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzDiagnosticSetting-Cmdlet** ruft die Kategorien und Zeitkörner ab, die für eine Ressource protokolliert werden.
Eine Zeitkörnung ist das Aggregationsintervall einer Metrik.

## BEISPIELE

### Beispiel 1: Anzeigen des Status der Protokollierungskategorien und Zeitkörner
```
PS C:\>Get-AzDiagnosticSetting -ResourceId "/subscriptions/1a66ce04-b633-4a0b-b2bc-a912ec8986a6/ResourceGroups/ContosoRG/providers/microsoft.keyvault/KeyVaults/ContosoKeyVault"
StorageAccountId   : /subscriptions/1a66ce04-b633-4a0b-b2bc-a912ec8986a6/ResourceGroups/ContosoRG/providers/microsoft.storage/accounts/ContosoStorageAccount
StorageAccountName : ContosoStorageAccount001
Metrics

Logs
Enabled  : True
Category : AuditEvent
```

Dieser Befehl ruft die Kategorien und Zeitkörner ab, die für einen Azure Key Vault mit einer *ResourceId* von /subscriptions/1a66ce04-b633-4a0b-b2bc-a912ec8986a6/ResourceGroups/ContosoRG/providers/microsoft.keyvault/KeyVaults/ContosoKeyVault protokolliert werden.
