# Restore-AzDataLakeStoreDeletedItem

## SYNOPSIS
Wiederherstellen einer gelöschten Datei oder eines gelöschten Ordners in Azure Data Lake

## SYNTAX

### Standard (Standard)
```
Restore-AzDataLakeStoreDeletedItem [-Account] <String> [-Path] <String> [-Destination] <String>
 [-Type] <String> [-RestoreAction <String>] [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### InputObject
```
Restore-AzDataLakeStoreDeletedItem [-Account] <String> [-DeletedItem] <DataLakeStoreDeletedItem>
 [-RestoreAction <String>] [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Mit **dem Cmdlet Restore-AzDataLakeStoreDeletedItem** wird eine gelöschte Datei oder ein gelöschter Ordner im Data Lake Store wiederhergestellt. Erfordert den Pfad des gelöschten Elements im Papierkorb, der von Get-AzDataLakeStoreDeletedItem zurückgegeben wird.
Vorsicht: Das Rückgängig machen von Dateien ist ein Vorgang am besten. Es gibt keine Garantien dafür, dass eine Datei nach dem Löschen wiederhergestellt werden kann. Die Verwendung dieser API wird über whitelisting aktiviert. Wenn Ihr ADL-Konto nicht auf der Whitelist aufgeführt ist, wird mit dieser Api die Ausnahme Nicht implementiert ausgelöst. Weitere Informationen und Unterstützung erhalten Sie vom Microsoft-Support.

## BEISPIELE

### Beispiel 1: Wiederherstellen einer Datei aus dem Data Lake Store mithilfe der Option "-force"
```
PS > Restore-AzDataLakeStoreDeletedItem -Account ml1ptrashtest -Path 927e8fb1-a287-4353-b50e-3b4a39ae4088 -Destination adl://ml1ptrashtest.azuredatalake.com/test0/file_1230 -Type "file" -Force
PS >
```
