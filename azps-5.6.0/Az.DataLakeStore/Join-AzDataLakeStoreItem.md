# Join-AzDataLakeStoreItem

## SYNOPSIS
Schließt eine oder mehrere Dateien an, um eine Datei im Data Lake Store zu erstellen.

## SYNTAX

```
Join-AzDataLakeStoreItem [-Account] <String> [-Paths] <DataLakeStorePathInstance[]>
 [-Destination] <DataLakeStorePathInstance> [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Join-AzDataLakeStoreItem-Cmdlet** schließt eine oder mehrere Dateien zusammen, um eine Datei im Data Lake Store zu erstellen.

## BEISPIELE

### Beispiel 1: Verbinden von zwei Elementen
```
PS C:\>Join-AzDataLakeStoreItem -AccountName "ContosoADL" -Paths "/MyFiles/File01.txt","/MyFiles/File02.txt" -Destination "/MyFiles/CombinedFile.txt"
```

Mit diesem Befehl werden File01.txt und File02.txt zum Erstellen der Datei CombinedFile.txt.
