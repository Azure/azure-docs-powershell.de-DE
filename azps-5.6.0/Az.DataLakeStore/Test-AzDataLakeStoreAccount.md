# Test-AzDataLakeStoreAccount

## SYNOPSIS
Testet das Vorhandensein eines Data Lake Store-Kontos.

## SYNTAX

```
Test-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Test-AzDataLakeStoreAccount-Cmdlet** testet das Vorhandensein eines Data Lake Store-Kontos.

## BEISPIELE

### Beispiel 1: Testen eines Kontos
```
PS C:\>Test-AzDataLakeStoreAccount -Name "ContosoADL"
```

Mit diesem Befehl wird überprüft, ob das Konto mit dem Namen ContosoADL vorhanden ist.
