# Az.DataLakeStore-Modul
## Beschreibung
Die Themen in diesem Abschnitt dokumentieren die Azure PowerShell-Cmdlets für Azure Data Lake Store im Azure Resource Manager (ARM)-Framework. Die Cmdlets sind im Microsoft.Azure.Commands.DataLakeStore-Namespace vorhanden. Diese Cmdlets funktionieren nur mit Azure Data Lake Storage Gen1-Konten.

## Az.DataLakeStore-Cmdlets
### [Add-AzDataLakeStoreFirewallRule](Add-AzDataLakeStoreFirewallRule.md)
Fügt dem angegebenen Data Lake Store-Konto eine Firewallregel hinzu.

### [Add-AzDataLakeStoreItemContent](Add-AzDataLakeStoreItemContent.md)
Fügt einem Element in einem Data Lake Store Inhalt hinzu.

### [Add-AzDataLakeStoreTrustedIdProvider](Add-AzDataLakeStoreTrustedIdProvider.md)
Fügt dem angegebenen Data Lake Store-Konto einen vertrauenswürdigen Identitätsanbieter hinzu.

### [Add-AzDataLakeStoreVirtualNetworkRule](Add-AzDataLakeStoreVirtualNetworkRule.md)
Fügt dem angegebenen Data Lake Store-Konto eine Virtuelle Netzwerkregel hinzu.

### [Enable-AzDataLakeStoreKeyVault](Enable-AzDataLakeStoreKeyVault.md)
Versucht, einen vom Benutzer verwalteten Schlüsseltresor für die Verschlüsselung des angegebenen Data Lake Store-Kontos zu aktivieren.

### [Export-AzDataLakeStoreChildItemProperty](Export-AzDataLakeStoreChildItemProperty.md)
Exportiert die Eigenschaften (Datenträgernutzung und Acl) für die gesamte Struktur aus dem angegebenen Pfad in einen Ausgabepfad.

### [Export-AzDataLakeStoreItem](Export-AzDataLakeStoreItem.md)
Lädt eine Datei aus dem Data Lake Store herunter.

### [Get-AzDataLakeStoreAccount](Get-AzDataLakeStoreAccount.md)
Ruft Details zu einem Data Lake Store-Konto ab.

### [Get-AzDataLakeStoreChildItem](Get-AzDataLakeStoreChildItem.md)
Ruft die Liste der Elemente in einem Ordner im Data Lake Store ab.

### [Get-AzDataLakeStoreChildItemSummary](Get-AzDataLakeStoreChildItemSummary.md)
Ruft die Zusammenfassung der Gesamtgröße, der Dateien und Verzeichnisse ab, die im angegebenen Pfad enthalten sind.

### [Get-AzDataLakeStoreFirewallRule](Get-AzDataLakeStoreFirewallRule.md)
Ruft die angegebenen Firewallregeln im angegebenen Data Lake Store ab.
Wenn keine Firewallregel angegeben ist, werden alle Firewallregeln für das Konto aufgeführt.

### [Get-AzDataLakeStoreItem](Get-AzDataLakeStoreItem.md)
Ruft die Details einer Datei oder eines Ordners im Data Lake Store ab.

### [Get-AzDataLakeStoreItemAclEntry](Get-AzDataLakeStoreItemAclEntry.md)
Ruft einen Eintrag in der ACL einer Datei oder eines Ordners im Data Lake Store ab.

### [Get-AzDataLakeStoreItemContent](Get-AzDataLakeStoreItemContent.md)
Ruft den Inhalt einer Datei im Data Lake Store ab.

### [Get-AzDataLakeStoreItemOwner](Get-AzDataLakeStoreItemOwner.md)
Ruft den Besitzer einer Datei oder eines Ordners im Data Lake Store ab.

### [Get-AzDataLakeStoreItemPermission](Get-AzDataLakeStoreItemPermission.md)
Ruft die Berechtigung oktal einer Datei oder eines Ordners im Data Lake Store ab.

### [Get-AzDataLakeStoreTrustedIdProvider](Get-AzDataLakeStoreTrustedIdProvider.md)
Ruft den angegebenen vertrauenswürdigen Identitätsanbieter im angegebenen Data Lake Store ab.
Wenn kein Anbieter angegeben ist, werden alle Anbieter für das Konto aufgeführt.

### [Get-AzDataLakeStoreVirtualNetworkRule](Get-AzDataLakeStoreVirtualNetworkRule.md)
Ruft die angegebenen Regeln für virtuelle Netzwerke im angegebenen Data Lake Store ab.
Wenn keine Regel für virtuelle Netzwerke angegeben ist, werden alle Regeln für das virtuelle Netzwerk für das Konto aufgeführt.

### [Import-AzDataLakeStoreItem](Import-AzDataLakeStoreItem.md)
Lädt eine lokale Datei oder ein lokales Verzeichnis in einen Data Lake Store hoch.

### [Join-AzDataLakeStoreItem](Join-AzDataLakeStoreItem.md)
Schließt eine oder mehrere Dateien an, um eine Datei im Data Lake Store zu erstellen.

### [Move-AzDataLakeStoreItem](Move-AzDataLakeStoreItem.md)
Verschiebt oder benennt eine Datei oder einen Ordner im Data Lake Store um.

### [New-AzDataLakeStoreAccount](New-AzDataLakeStoreAccount.md)
Erstellt ein neues Data Lake Store-Konto.

### [New-AzDataLakeStoreItem](New-AzDataLakeStoreItem.md)
Erstellt eine neue Datei oder einen neuen Ordner im Data Lake Store.

### [Remove-AzDataLakeStoreAccount](Remove-AzDataLakeStoreAccount.md)
Löscht ein Data Lake Store-Konto dauerhaft.

### [Remove-AzDataLakeStoreFirewallRule](Remove-AzDataLakeStoreFirewallRule.md)
Entfernt die angegebene Firewallregel im angegebenen Data Lake Store.

### [Remove-AzDataLakeStoreItem](Remove-AzDataLakeStoreItem.md)
Löscht eine Datei oder einen Ordner im Data Lake Store.

### [Remove-AzDataLakeStoreItemAcl](Remove-AzDataLakeStoreItemAcl.md)
Entfernt die ACL einer Datei oder eines Ordners im Data Lake Store.

### [Remove-AzDataLakeStoreItemAclEntry](Remove-AzDataLakeStoreItemAclEntry.md)
Entfernt einen Eintrag aus der ACL einer Datei oder eines Ordners im Data Lake Store.

### [Remove-AzDataLakeStoreTrustedIdProvider](Remove-AzDataLakeStoreTrustedIdProvider.md)
Entfernt den angegebenen vertrauenswürdigen Identitätsanbieter im angegebenen Data Lake Store.

### [Remove-AzDataLakeStoreVirtualNetworkRule](Remove-AzDataLakeStoreVirtualNetworkRule.md)
Entfernt die angegebene Regel für das virtuelle Netzwerk im angegebenen Data Lake Store.

### [Set-AzDataLakeStoreAccount](Set-AzDataLakeStoreAccount.md)
Ändert ein Data Lake Store-Konto.

### [Set-AzDataLakeStoreFirewallRule](Set-AzDataLakeStoreFirewallRule.md)
Ändert die angegebene Firewallregel im angegebenen Data Lake Store.

### [Set-AzDataLakeStoreItemAcl](Set-AzDataLakeStoreItemAcl.md)
Ändert die ACL einer Datei oder eines Ordners im Data Lake Store.

### [Set-AzDataLakeStoreItemAclEntry](Set-AzDataLakeStoreItemAclEntry.md)
Ändert einen Eintrag in der ACL einer Datei oder eines Ordners im Data Lake Store.

### [Set-AzDataLakeStoreItemExpiry](Set-AzDataLakeStoreItemExpiry.md)
Legt die Ablaufzeit für eine Datei in einem Azure Data Lake Store-Konto fest oder entfernt sie.

### [Set-AzDataLakeStoreItemOwner](Set-AzDataLakeStoreItemOwner.md)
Ändert den Besitzer einer Datei oder eines Ordners im Data Lake Store.

### [Set-AzDataLakeStoreItemPermission](Set-AzDataLakeStoreItemPermission.md)
Ändert die Oktalberechtigung einer Datei oder eines Ordners im Data Lake Store.

### [Set-AzDataLakeStoreTrustedIdProvider](Set-AzDataLakeStoreTrustedIdProvider.md)
Ändert den angegebenen vertrauenswürdigen Identitätsanbieter im angegebenen Data Lake Store.

### [Set-AzDataLakeStoreVirtualNetworkRule](Set-AzDataLakeStoreVirtualNetworkRule.md)
Ändert die angegebene Virtuelle Netzwerkregel im angegebenen Data Lake Store.

### [Test-AzDataLakeStoreAccount](Test-AzDataLakeStoreAccount.md)
Testet das Vorhandensein eines Data Lake Store-Kontos.

### [Test-AzDataLakeStoreItem](Test-AzDataLakeStoreItem.md)
Testet das Vorhandensein einer Datei oder eines Ordners im Data Lake Store.
