# Get-AzPeeringServiceCountry

## SYNOPSIS
Listet die Länder auf, die für den Peeringdienst verfügbar sind.

## SYNTAX

```
Get-AzPeeringServiceCountry [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Listet die Länder auf, die für den Peeringdienst verfügbar sind.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Get-AzPeeringServiceCountry
```

Liste der Länder, die für peering service verfügbar sind.
