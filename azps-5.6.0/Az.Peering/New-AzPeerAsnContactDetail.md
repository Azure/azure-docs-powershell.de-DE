# New-AzPeerAsnContactDetail

## SYNOPSIS
Erstellt ein In-Memory-Kontakt-Detail für PeerAsn. 

## SYNTAX

```
New-AzPeerAsnContactDetail -Role <String> -Email <String> [-Phone <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Erstellen Sie ein PeerAsn-Kontaktdetails im Arbeitsspeicher.

## BEISPIELE

### Erstellen von Kontaktdetails und Hinzufügen zu PeerAsn
```powershell
PS C:\> $nocContact = New-AzPeerAsnContactDetail -Role Noc -Email "noc@contoso.com" -Phone "+1 (887) 888-8088"
PS C:\> $customerContact = New-AzPeerAsnContactDetail -Role Noc -Email "noc@contoso.com" -Phone "+1 (887) 888-8088"
PS C:\> New-AzPeerAsn -Name $name -PeerName "Contoso Networks Limited" -PeerAsn 65000 -ContactDetail $nocContact,$customerContact
```

Rolle und E-Mail sind erforderlich. Telefon ist optional. Telefon unterstützt +-() oder Leerzeichen. Ein PeerAsn muss mindestens ein Kontaktdetail vom Typ "Noc" enthalten.
