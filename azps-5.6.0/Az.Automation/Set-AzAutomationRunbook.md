# Set-AzAutomationRunbook

## SYNOPSIS
Ändert ein Runbook.

## SYNTAX

```
Set-AzAutomationRunbook [-Name] <String> [-Description <String>] [-Tag <IDictionary>] [-LogProgress <Boolean>]
 [-LogVerbose <Boolean>] [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Set-AzAutomationRunbook** ändert die Konfiguration eines Azure Automation-Runbooks in APS.

## BEISPIELE

### Beispiel 1: Aktivieren der ausführlichen Protokollierung für ein Runbook
```
PS C:\>Set-AzAutomationRunbook -AutomationAccountName "Contoso17" -Name "Runbook02" -LogVerbose $True -ResourceGroupName "ResourceGroup01"
```

Dieser Befehl ermöglicht eine ausführliche Protokollierung für die Aufträge des angegebenen Runbooks im Azure Automation-Konto namens Contoso17.
