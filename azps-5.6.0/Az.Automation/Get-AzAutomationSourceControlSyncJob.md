# Get-AzAutomationSourceControlSyncJob

## SYNOPSIS
Ruft Azure Automation-Quellsteuerelementsynchronisierungsaufträge ab.

## SYNTAX

```
Get-AzAutomationSourceControlSyncJob -SourceControlName <String> [-JobId <Guid>] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das Get-AzAutomationSourceControlSyncJob-Cmdlet ruft Azure Automation-Quellsteuerelementsynchronisierungsaufträge ab. Wenn Sie einen bestimmten Synchronisierungsauftrag für das Quellsteuerelement erhalten möchten, geben Sie dessen ID an.

## BEISPIELE

### Beispiel 1
Dieser Befehl ruft alle Synchronisierungsaufträge des Automatisierungsquellensteuerelements für das Quellsteuerelement VSTSNative ab.


```powershell
PS C:\> Get-AzAutomationSourceControlSyncJob -ResourceGroupName "rg1" `
                                                  -AutomationAccountName "devAccount" `
                                                  -Name "VSTSNative"

SourceControlSyncJobId               SyncType Status StartTime           EndTime
----------------------               -------- ------ ---------           -------
08d6d266-27b6-463c-beea-bc48a67ace15 FullSync Failed 08/15/2018 09:17 AM 08/15/2018 09:18 AM
b566d564-878a-4641-8c44-25bf7850531e FullSync Failed 08/15/2018 09:09 AM 08/15/2018 09:10 AM
```

### Beispiel 2
Dieser Befehl ruft den Synchronisierungsauftrag für das Quellsteuerelement mit der ID 08d6d266-27b6-463c-beea-bc48a67ace15 für das Quellsteuerelement VSTSNative ab. 


```powershell

                                                  -Name "VSTSNative" `
```
