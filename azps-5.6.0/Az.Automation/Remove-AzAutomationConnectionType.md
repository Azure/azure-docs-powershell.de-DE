# Remove-AzAutomationConnectionType

## SYNOPSIS
Entfernt einen Automatisierungsverbindungstyp.

## SYNTAX

```
Remove-AzAutomationConnectionType [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzAutomationConnectionType** entfernt einen Verbindungstyp aus Azure Automation.
Alle Verbindungen, die dem verbindungstyp zugeordnet sind, den Sie löschen, werden unbrauchbar.
Entfernen Sie sie, es sei denn, Sie erstellen einen neuen Verbindungstyp, der die folgenden Kriterien erfüllt: 
- Der Typ hat denselben Namen wie der ursprüngliche Verbindungstyp. 
- Der Typ hat die gleichen Felddefinitionen wie der ursprüngliche Verbindungstyp.
Sie kann weitere Felder enthalten.

## BEISPIELE

### Beispiel 1: Entfernen eines Verbindungstyps
```
PS C:\>Remove-AzAutomationConnectionType -AutomationAccountName "Contoso17" -Name "ContosoConnectionType" -ResourceGroupName "ResourceGroup01"
```

Mit diesem Befehl wird der Verbindungstyp ContosoConnectionType im Automatisierungskonto namens Contoso17 entfernt.
