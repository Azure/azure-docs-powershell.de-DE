# Remove-AzAutomationSourceControl

## SYNOPSIS
Entfernt ein Azure Automation-Quellsteuerelement.

## SYNTAX

```
Remove-AzAutomationSourceControl [-Name] <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das Remove-AzAutomationSourceControl-Cmdlet entfernt ein Quellsteuerelement aus Azure Automation.

## BEISPIELE

### Beispiel 1
Mit diesem Befehl wird das Automatisierungsquellensteuerelement mit dem Namen VSTSNative im Konto mit dem Namen devAccount entfernt.
Dieser Befehl gibt den Parameter *Kraft* an. Daher werden Sie nicht zur Bestätigung aufgefordert.

```powershell
PS C:\> Remove-AzAutomationSourceControl -ResourceGroupName "rg1" `
                                              -AutomationAccountName "devAccount" `
                                              -Name "VSTSNative" `
                                              -Force
```
