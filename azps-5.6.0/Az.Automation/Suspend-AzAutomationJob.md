# Suspend-AzAutomationJob

## SYNOPSIS
Setzt einen Automatisierungsauftrag aus.

## SYNTAX

```
Suspend-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Suspend-AzAutomationJob** setzt einen Azure-Automatisierungsauftrag aus.
Geben Sie einen ausgeführten Automatisierungsauftrag an.
Zum Fortsetzen eines angehaltenen Auftrags verwenden Sie das cmdlet Resume-AzAutomationJob.

## BEISPIELE

### Beispiel 1: Anhalten eines Auftrags
```
PS C:\>Suspend-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Mit diesem Befehl wird der Auftrag angehalten, der die angegebene ID hat.
