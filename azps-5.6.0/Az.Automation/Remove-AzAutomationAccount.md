# Remove-AzAutomationAccount

## SYNOPSIS
Entfernt ein Automatisierungskonto.

## SYNTAX

```
Remove-AzAutomationAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzAutomationAccount** entfernt ein Azure Automation-Konto aus einer Ressourcengruppe.
Weitere Informationen zu Automatisierungskonten finden Sie im New-AzAutomationAccount Cmdlet.

## BEISPIELE

### Beispiel 1: Entfernen eines Automatisierungskontos
```
PS C:\>Remove-AzAutomationAccount -Name "ContosoAutomationAccount" -Force -ResourceGroupName "ResourceGroup01"
```

Mit diesem Befehl wird ein Automatisierungskonto namens ContosoAutomationAccount entfernt, ohne zur Benutzerüberprüfung aufgefordert zu werden.
