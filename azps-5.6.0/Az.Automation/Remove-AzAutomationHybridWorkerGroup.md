# Remove-AzAutomationHybridWorkerGroup

## SYNOPSIS
Entfernt die Hybridarbeitergruppe aus der Automatisierung.

## SYNTAX

```
Remove-AzAutomationHybridWorkerGroup [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das Remove-AzAutomationHybridWorkerGroup cmdlet entfernt eine Hybridarbeitergruppe aus der Automatisierung.

## BEISPIELE

### Beispiel 1
Mit diesem Befehl wird ein Hybridarbeiter namens entfernt.

```powershell
PS C:\> Remove-AzAutomationHybridWorkerGroup -ResourceGroupName "rg1" `
                                                  -AutomationAccountName "devAccount" `
                                                  -Name "GroupName" `
                                                  -Force
```
