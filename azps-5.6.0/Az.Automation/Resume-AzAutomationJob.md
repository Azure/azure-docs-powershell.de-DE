# Resume-AzAutomationJob

## SYNOPSIS
Setzt einen angehaltenen Automatisierungsauftrag fort.

## SYNTAX

```
Resume-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Resume-AzAutomationJob** setzt einen angehaltenen Azure-Automatisierungsauftrag fort.
Geben Sie den angehaltenen Auftrag an.
Zum Anhalten eines Auftrags verwenden Sie das Suspend-AzAutomationJob Cmdlet.

## BEISPIELE

### Beispiel 1: Fortsetzen eines angehaltenen Auftrags
```
PS C:\>Resume-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Mit diesem Befehl wird der Auftrag fortgesetzt, der die angegebene ID hat.
