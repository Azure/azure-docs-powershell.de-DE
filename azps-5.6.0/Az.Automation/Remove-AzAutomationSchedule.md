# Remove-AzAutomationSchedule

## SYNOPSIS
Löscht einen Automatisierungszeitplan.

## SYNTAX

```
Remove-AzAutomationSchedule [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzAutomationSchedule** löscht einen Zeitplan aus Azure Automation.

## BEISPIELE

### Beispiel 1: Entfernen eines Zeitplans
```
PS C:\>Remove-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule01" -ResourceGroupName "ResourceGroup01"
```

Mit diesem Befehl wird der Zeitplan "Schedule01" im Automatisierungskonto Contoso17 in der Ressourcengruppe ResourceGroup01 gelöscht.
