# Remove-AzAutomationConnection

## SYNOPSIS
Entfernt eine Automatisierungsverbindung.

## SYNTAX

```
Remove-AzAutomationConnection [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzAutomationConnection** entfernt eine Verbindung aus Azure Automation.

## BEISPIELE

### Beispiel 1: Entfernen einer Verbindung
```
PS C:\>Remove-AzAutomationConnection -AutomationAccountName "Contoso17" -Name "ContosoConnection" -ResourceGroupName "ResourceGroup01"
```

Mit diesem Befehl wird ein Zertifikat namens ContosoConnection im Automatisierungskonto namens Contoso17 entfernt.
