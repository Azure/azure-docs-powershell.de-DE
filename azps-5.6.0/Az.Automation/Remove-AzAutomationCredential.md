# Remove-AzAutomationCredential

## SYNOPSIS
Entfernt die Anmeldeinformationen für die Automatisierung.

## SYNTAX

```
Remove-AzAutomationCredential [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzAutomationCredential** entfernt eine Anmeldeinformationen aus Azure Automation.

## BEISPIELE

### Beispiel 1: Entfernen von Anmeldeinformationen
```
PS C:\>Remove-AzAutomationCredential -AutomationAccountName "Contoso17" -Name "ContosoCredential" -ResourceGroupName "ResourceGroup01"
```

Mit diesem Befehl werden die Anmeldeinformationen contosoCredential im Azure Automation-Konto mit dem Namen Contoso17 entfernt.
