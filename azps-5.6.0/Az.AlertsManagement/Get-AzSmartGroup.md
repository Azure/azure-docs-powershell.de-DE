# Get-AzSmartGroup

## SYNOPSIS
Ruft Informationen zu intelligenten Gruppen ab

## SYNTAX

### SmartGroupsListByFilter (Standard)
```
Get-AzSmartGroup [-SortBy <String>] [-SortOrder <String>] [-TimeRange <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SmartGroupById
```
Get-AzSmartGroup -SmartGroupId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
**Das Get-AzSmartGroup-Cmdlet** ruft Informationen zu intelligenten Gruppen ab.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Get-AzSmartGroup -TimeRange "1h"
```

Listen Sie alle intelligenten Gruppen auf, die in der letzten 1 Stunde gebildet wurden. Verwenden Format-List, um die vollständigen Details jeder intelligenten Gruppe in der Liste zu erhalten.

### Beispiel 2
```powershell
PS C:\> Get-AzSmartGroup -SmartGroupId "afbf1b3a-0a6c-4f19-9c9b-644ccd7b1529" | Format-List
```

Get Smart Group details by Id (GUID) or Resource Id (Complete ARM Id)
