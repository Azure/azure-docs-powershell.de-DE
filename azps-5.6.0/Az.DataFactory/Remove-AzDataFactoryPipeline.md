# Remove-AzDataFactoryPipeline

## SYNOPSIS
Entfernt eine Pipeline aus Azure Data Factory.

## SYNTAX

### ByFactoryName (Standard)
```
Remove-AzDataFactoryPipeline [-Force] [-Name] <String> [-DataFactoryName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByFactoryObject
```
Remove-AzDataFactoryPipeline [-Force] [-Name] <String> [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzDataFactoryPipeline** entfernt eine Pipeline aus Azure Data Factory.

## BEISPIELE

### Beispiel 1: Entfernen einer Pipeline
```
PS C:\>Remove-AzDataFactoryPipeline -ResourceGroupName "ADF" -Name "DPWikisample" -DataFactoryName "WikiADF"
Confirm
Are you sure you want to remove pipeline 'DPWikisample' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

Dieses Cmdlet entfernt die Pipeline mit dem Namen DPWikisample aus der Daten factory mit dem Namen WikiADF.
Der Befehl gibt einen Wert von $True.
