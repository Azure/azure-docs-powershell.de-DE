# New-AzDataFactory

## SYNOPSIS
Erstellt eine Daten factory.

## SYNTAX

```
New-AzDataFactory [-Name] <String> [-Location] <String> [[-Tag] <Hashtable>] [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet New-AzDataFactory** erstellt eine Datenfactory mit dem angegebenen Ressourcengruppennamen und -speicherort.
Führen Sie diese Vorgänge in der folgenden Reihenfolge aus: 
- Erstellen Sie eine Daten factory. 
- Erstellen sie verknüpfte Dienste. 
- Erstellen Sie Datasets. 
- Erstellen Sie eine Pipeline.

## BEISPIELE

### Beispiel 1: Erstellen einer Daten factory
```
PS C:\>New-AzDataFactory -ResourceGroupName "ADF" -Name "WikiADF" -Location "WestUS"
DataFactoryName   : WikiADF
ResourceGroupName : ADF
Location          : WestUS
Tags              : {}
Properties        : Microsoft.WindowsAzure.Commands.Utilities.PSDataFactoryConfiguration
```

Mit diesem Befehl wird eine Datenfabrik namens WikiADF in der Ressourcengruppe mit dem Namen ADF am WestUS-Speicherort erstellt.
