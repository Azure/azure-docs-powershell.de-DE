# Remove-AzWebAppSlot

## SYNOPSIS

## SYNTAX

### S1
```
Remove-AzWebAppSlot [-Force] [-AsJob] [-ResourceGroupName] <String> [-Name] <String> [-Slot] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Remove-AzWebAppSlot [-Force] [-AsJob] [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzWebAppSlot** entfernt einen Azure Web App-Slot, der die Ressourcengruppe und den Web App-Namen bereitgestellt hat.
Dieses Cmdlet entfernt standardmäßig auch alle Slots und Metriken.

## BEISPIELE

### Beispiel 1: Entfernen eines Web App-Slot
```
PS C:\>Remove-AzWebAppSlot -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite" -Slot "Slot001"
```

Mit diesem Befehl wird der Slot mit dem Namen Slot001 entfernt, der der Web App ContosoSite zugeordnet ist und zur Ressourcengruppe "Default-Web-WestUS" gehört.
