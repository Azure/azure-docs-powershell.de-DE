# Import-AzWebAppKeyVaultCertificate

## SYNOPSIS
Importieren Sie ein SSL-Zertifikat aus dem Schlüsseltresor in eine Web-App.

## SYNTAX

```
Import-AzWebAppKeyVaultCertificate [-KeyVaultName] <String> [-CertName] <String> [-ResourceGroupName] <String>
 [-WebAppName] <String> [-Slot <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Import-AzWebAppKeyVaultCertificate** importiert ein SSL-Zertifikat aus dem Schlüsseltresor in eine Web-App.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Import-AzWebAppKeyVaultCertificate -ResourceGroupName "Default-Web-WestUS" -Name "ContosoWebApp" 
-KeyVaultName "ContosoKeyVault" -CertName "ContosoCertname"
```

Mit diesem Befehl wird ein SSL-Zertifikat aus dem Schlüsseltresor in eine Web-App importiert.

### Beispiel 2
```powershell
PS C:\> Import-AzWebAppKeyVaultCertificate -ResourceGroupName "Default-Web-WestUS" -Name "ContosoWebApp" 
-KeyVaultName  '/subscriptions/[sub id]/resourceGroups/[rg]/providers/Microsoft.KeyVault/vaults/[vault name]' 
-CertName "ContosoCertname"
```

Mit diesem Befehl Importieren Eines SSL-Zertifikats aus dem Schlüsseltresor in eine Web App mithilfe der Ressourcen-ID (in der Regel, wenn sich der Schlüsseltresor in einem anderen Abonnement befindet).
