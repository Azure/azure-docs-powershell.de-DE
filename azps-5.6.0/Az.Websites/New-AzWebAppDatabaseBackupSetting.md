# New-AzWebAppDatabaseBackupSetting

## SYNOPSIS

## SYNTAX

```
New-AzWebAppDatabaseBackupSetting [-Name] <String> [-DatabaseType] <String> [-ConnectionString] <String>
 [[-ConnectionStringName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet New-AzWebAppDatabaseBackupSetting** erstellt eine neue Azure Web App-Sicherungseinstellung.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> New-AzWebAppDatabaseBackupSetting -ResourceGroupName "Default-Web-WestUS" -Name "ContosoWebApp" -ConnectionString "MyConnectionString" -DatabaseType "SqlAzure"
```

Erstellt eine Datenbanksicherungseinstellung (Verbindungszeichenfolge) vom Typ SqlAzure f체r die angegebene App ContosoWebApp, die sich innerhalb der Ressourcengruppe Default-Web-WestUS befindet.

### Beispiel 2

Das New-AzWebAppDatabaseBackupSetting-Cmdlet erstellt eine neue Azure Web App-Sicherungseinstellung. (automatisch generiert)

```powershell <!-- Aladdin Generated Example --> 
New-AzWebAppDatabaseBackupSetting -ConnectionString 'MyConnectionString' -ConnectionStringName <String> -DatabaseType 'SqlAzure' -Name 'ContosoWebApp'
```
