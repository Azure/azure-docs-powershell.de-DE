# Set-AzWebAppSlotConfigName

## SYNOPSIS
Festlegen von Web App Slot Config-Namen

## SYNTAX

### S1
```
Set-AzWebAppSlotConfigName [[-AppSettingNames] <String[]>] [[-ConnectionStringNames] <String[]>]
 [-RemoveAllAppSettingNames] [-RemoveAllConnectionStringNames] [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### S2
```
Set-AzWebAppSlotConfigName [[-AppSettingNames] <String[]>] [[-ConnectionStringNames] <String[]>]
 [-RemoveAllAppSettingNames] [-RemoveAllConnectionStringNames] [-WebApp] <PSSite>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Set-AzWebAppSlotConfigName** kennzeichnet App-Einstellungen und Verbindungszeichenfolgen als Sloteinstellungen.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Set-AzWebAppSlotConfigName -ResourceGroupName "Default-Web-WestUS" -Name "ContosoWebApp" -RemoveAllAppSettingNames -RemoveAllConnectionStringNames
```

Mit diesem Befehl werden alle App-Einstellungen und Verbindungszeichenfolgen für Web App ContosoWebApp entfernt, die der Ressourcengruppe Default-Web-WestUS zugeordnet sind.

### Beispiel 2

Legen Sie Web App Slot Config-Namen ein. (automatisch generiert)

```powershell <!-- Aladdin Generated Example --> 
Set-AzWebAppSlotConfigName -AppSettingNames <String[]> -Name 'ContosoWebApp' -ResourceGroupName 'Default-Web-WestUS'
```
