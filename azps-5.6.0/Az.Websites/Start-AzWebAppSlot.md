# Start-AzWebAppSlot

## SYNOPSIS
Startet einen Azure Web App-Slot.

## SYNTAX

### S1
```
Start-AzWebAppSlot [-ResourceGroupName] <String> [-Name] <String> [-Slot] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### S2
```
Start-AzWebAppSlot [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Start-AzWebAppSlot** startet einen Azure Web App-Slot.

## BEISPIELE

### Beispiel 1
```
PS C:\>Start-AzWebAppSlot -ResourceGroupName "Default-Web-WestUS" -Name "ContosoWebApp" -Slot "Slot001"
```

Mit diesem Befehl wird der Slot mit dem Namen Slot001 gestartet, der sich auf die Web App namens ContosoWebApp bezieht, die zur Ressourcengruppe "Default-Web-WestUS" gehört.
