# Start-AzWebApp

## SYNOPSIS
Startet eine Azure Web App.

## SYNTAX

### S1
```
Start-AzWebApp [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### S2
```
Start-AzWebApp [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Start-AzWebApp** startet eine Azure Web App.

## BEISPIELE

### Beispiel 1: Starten einer Web App
```
PS C:\>Start-AzWebApp -ResourceGroupName "Default-Web-WestUS" -Name "ContosoWebApp"
```

Mit diesem Befehl wird die Web App namens ContosoWebApp gestartet, die zur Ressourcengruppe "Default-Web-WestUS" gehört.
