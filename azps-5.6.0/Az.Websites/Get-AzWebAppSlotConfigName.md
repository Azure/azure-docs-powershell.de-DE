# Get-AzWebAppSlotConfigName

## SYNOPSIS
Liste der Web App Slot Config-Namen

## SYNTAX

### S1
```
Get-AzWebAppSlotConfigName [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### S2
```
Get-AzWebAppSlotConfigName [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzWebAppSlotConfigName-Cmdlet** ruft die Liste der App-Einstellungs- und Verbindungszeichenfolgennamen ab, die derzeit als Sloteinstellungen gekennzeichnet sind.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\>Get-AzWebAppSlotConfigName -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Dieser Befehl ruft App-Einstellungen und Verbindungszeichenfolgen ab, die sich auf die Web App mit dem Namen ContosoSite bezieht, die der Ressourcengruppe Default-Web-WestUS zugeordnet ist.
