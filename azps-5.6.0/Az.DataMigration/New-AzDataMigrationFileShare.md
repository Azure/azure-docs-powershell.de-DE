# New-AzDataMigrationFileShare

## SYNOPSIS
Erstellt das FileShare-Objekt für den Azure Database Migration Service, der die lokale Netzwerkfreigabe angibt, in die die Quelldatenbanksicherungen ausgeführt werden sollen.

## SYNTAX

```
New-AzDataMigrationFileShare -Path <String> -Credential <PSCredential>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das New-AzDataMigrationFileShare-Cmdlet erstellt das FileShare-Objekt, das die lokale Netzwerkfreigabe angibt, in die der Azure Database Migration Service Quelldatenbanksicherungen ausführen kann. Das Dienstkonto, auf dem SQL Server Instanz ausgeführt wird, muss über Schreibberechtigungen für diese Netzwerkfreigabe verfügen.

## BEISPIELE

### Beispiel 1
```
PS C:\> New-AzDmsFileShare -Path $fileSharePath -Credential $fileShareCred

```
