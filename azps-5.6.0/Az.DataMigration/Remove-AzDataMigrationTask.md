# Remove-AzDataMigrationTask

## SYNOPSIS
Entfernt eine Azure-Datenbankmigrationsdienstaufgabe aus Azure.

## SYNTAX

### ComponentNameParameterSet (Standard)
```
Remove-AzDataMigrationTask -ResourceGroupName <String> -ServiceName <String> -ProjectName <String>
 -Name <String> [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ComponentObjectParameterSet
```
Remove-AzDataMigrationTask [-InputObject] <PSProjectTask> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzDataMigrationTask [-ResourceId] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das Remove-AzDataMigrationTask-Cmdlet entfernt eine Azure-Datenbankmigrationsdienstaufgabe aus Azure.

## BEISPIELE

### Beispiel 1
```
PS C:\> Remove-AzDataMigrationTask -TaskName TestTask -ProjectName myTestProject -ServiceName MyTestService
 -ResourceGroupName MyResourceGroup
```

Im vorherigen Beispiel wird eine Azure-Datenbankmigrationsdienstaufgabe namens TestTask aus Azure entfernt, die auf dem Parameter "Vorgangsname" basiert.

### Beispiel 2
```
PS C:\> Remove-AzDataMigrationTask -InputObject $TestTask
```

Im vorherigen Beispiel wird eine Azure-Datenbankmigrationsdienstaufgabe entfernt, die auf dem übergebenen PSProjectTask-Objekt basiert.
