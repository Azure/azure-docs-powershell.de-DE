# Get-AzHDInsightCluster

## SYNOPSIS
Ruft alle Azure HDInsight-Cluster ab und auf, die dem aktuellen Abonnement oder einer angegebenen Ressourcengruppe zugeordnet sind, oder ruft einen bestimmten Cluster ab.

## SYNTAX

```
Get-AzHDInsightCluster [[-ResourceGroupName] <String>] [[-ClusterName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Im **Cmdlet Get-AzHDInsightCluster** werden die Azure HDInsight-Dienstcluster für das aktuelle Abonnement aufgeführt.
Verwenden Sie den *Parameter ClusterName,* um Details für einen bestimmten Cluster anzuzeigen.

## BEISPIELE

### Beispiel 1: Auflisten aller Azure HDInsight-Cluster
```
PS C:\>Get-AzHDInsightCluster
```

Mit diesem Befehl werden alle Azure HDInsight-Cluster aufgeführt.
