# Use-AzHDInsightCluster

## SYNOPSIS
Wählt einen Cluster aus, der mit dem cmdlet Invoke-RmAzureHDInsightHiveJob werden soll.

## SYNTAX

```
Use-AzHDInsightCluster [-ClusterName] <String> [-HttpCredential] <PSCredential> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Use-AzHDInsightCluster-Cmdlet** wählt den Azure HDInsight-Cluster für das Invoke-AzHDInsightHiveJob-Cmdlet aus, das zum Übermitteln von Hive-Aufträgen verwendet werden soll.

## BEISPIELE

### Beispiel 1: Auswählen eines Clusters für die Übermittlung von Hive-Abfragen
```
PS C:\># Cluster info
PS C:\>$clusterName = "your-hadoop-001"
PS C:\>$clusterCreds = Get-Credential

PS C:\>Use-AzHDInsightCluster `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

Mit diesem Befehl wird ein Cluster für eine Hive-Abfrageübermittlung ausgewählt.
