# Get-AzHDInsightHost

## SYNOPSIS
Listet die Hosts des HDInsight-Clusters auf.

## SYNTAX

### SetByNameParameterSet (Standard)
```
Get-AzHDInsightHost [[-ResourceGroupName] <String>] [-ClusterName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SetByResourceIdParameterSet
```
Get-AzHDInsightHost [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SetByInputObjectParameterSet
```
Get-AzHDInsightHost [-InputObject] <AzureHDInsightCluster> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzHDInsightHost-Cmdlet** listet die Hosts des HDInsight-Clusters auf.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> Get-AzHDInsightHost -ClusterName $clusterName
```

Dieser Befehl listet die Hosts des Clusters mit dem Clusternamen auf.

### Beispiel 2
```powershell
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $cluster=Get-AzHDInsightCluster -ClusterName $clusterName
PS C:\> $cluster | Get-AzHDInsightHost
```

Dieser Befehl listet die Hosts des Clusters mit Pipeline auf.
