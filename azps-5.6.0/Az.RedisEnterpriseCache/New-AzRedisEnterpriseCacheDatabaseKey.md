# New-AzRedisEnterpriseCacheDatabaseKey

## SYNOPSIS
Regeneriert die Zugriffstasten der RedisEnterprise-Datenbank.

## SYNTAX

```
New-AzRedisEnterpriseCacheDatabaseKey -ClusterName <String> -ResourceGroupName <String>
 -KeyType <AccessKeyType> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## BESCHREIBUNG
Regeneriert die Zugriffstasten der RedisEnterprise-Datenbank.

## BEISPIELE

### Beispiel 1: Neuerieren des primären Zugriffsschlüssels
```powershell
PS C:\> New-AzRedisEnterpriseCacheDatabaseKey -Name "MyCache" -ResourceGroupName "MyGroup" -KeyType "Primary"

PrimaryKey                                   SecondaryKey
----------                                   ------------
new-primary-key                              secondary-key

```

Dieser Befehl regeneriert den primären geheimen Zugriffsschlüssel, der zum Authentifizieren von Verbindungen mit der Datenbank des Redis Enterprise Cache namens MyCache verwendet wird.

### Beispiel 2: Neuerieren des sekundären Zugriffsschlüssels
```powershell
PS C:\> New-AzRedisEnterpriseCacheDatabaseKey -Name "MyCache" -ResourceGroupName "MyGroup" -KeyType "Secondary"

PrimaryKey                                   SecondaryKey
----------                                   ------------
primary-key                                  new-secondary-key

```

Mit diesem Befehl wird der sekundäre geheime Zugriffsschlüssel neu generiert, der zum Authentifizieren von Verbindungen mit der Datenbank des Redis Enterprise Cache namens MyCache verwendet wird.
