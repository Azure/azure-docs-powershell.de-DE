# Set-AzCloudServiceUpdateDomain

## SYNOPSIS
Aktualisiert die Rolleninstanzen in der angegebenen Updatedomäne.

## SYNTAX

```
Set-AzCloudServiceUpdateDomain -CloudServiceName <String> -ResourceGroupName <String> -UpdateDomain <Int32>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## BESCHREIBUNG
Aktualisiert die Rolleninstanzen in der angegebenen Updatedomäne.

## BEISPIELE

### Beispiel 1: Aktualisieren der Rolleninstanz in der Updatedomäne
```powershell
PS C:\> Set-AzCloudServiceUpdateDomain -CloudServiceName "ContosoCS" -ResourceGroupName "ContosOrg" -UpdateDomain 0
```

Mit diesem Befehl werden Rolleninstanzen in Updatedomäne 0 des Clouddiensts namens ContosoCS aktualisiert, der zur Ressourcengruppe "ContosOrg" gehört.
