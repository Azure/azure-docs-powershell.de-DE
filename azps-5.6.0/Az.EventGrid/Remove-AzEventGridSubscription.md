# Remove-AzEventGridSubscription

## SYNOPSIS

## BESCHREIBUNG
Entfernt ein Azure Event Grid-Ereignisabonnement für ein Azure Event Grid-Thema, eine Ressource, ein Azure-Abonnement oder eine Ressourcengruppe.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Remove-AzEventGridSubscription -ResourceGroup MyResourceGroup -TopicName Topic1 -EventSubscriptionName EventSubscription1
```

Entfernt das Ereignisabonnement \` EventSubscription1 zu einem Azure Event Grid \` \` Topic1 in \` der Ressourcengruppe \` MyResourceGroupName. \`

### Beispiel 2
```powershell
PS C:\> Remove-AzEventGridSubscription -ResourceGroupName MyResourceGroupName -EventSubscriptionName EventSubscription1
```

Entfernt das Ereignisabonnement \` EventSubscription1 \` zu einer Ressourcengruppe \` MyResourceGroupName. \`

### Beispiel 3
```powershell
PS C:\> Remove-AzEventGridSubscription -EventSubscriptionName EventSubscription1
```

Entfernt das Ereignisabonnement \` EventSubscription1 \` in das Standardmäßige Azure-Abonnement.

### Beispiel 4
```powershell
PS C:\> Get-AzResource -ResourceId "/subscriptions/$subscriptionId/resourceGroups/$resourceGroupName/providers/Microsoft.EventHub/namespaces/$namespaceName" | Remove-AzEventGridSubscription -EventSubscriptionName EventSubscription1
```

Entfernt das Ereignisabonnement \` EventSubscription1 \` in einen Event Hub-Namespace.

### Beispiel 5
```powershell
PS C:\> Get-AzEventGridTopic -ResourceGroup MyResourceGroup -TopicName Topic1 | Remove-AzEventGridSubscription -EventSubscriptionName EventSubscription1
```

Entfernt das Ereignisabonnement \` EventSubscription1 \` in ein Ereignisrasterthema.
