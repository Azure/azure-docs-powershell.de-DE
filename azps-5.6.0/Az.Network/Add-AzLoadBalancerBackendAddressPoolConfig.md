# Add-AzLoadBalancerBackendAddressPoolConfig

## SYNOPSIS
Fügt einem Load Balancer eine Konfiguration des Back-End-Adresspools hinzu.

## SYNTAX

```
Add-AzLoadBalancerBackendAddressPoolConfig -LoadBalancer <PSLoadBalancer> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Add-AzLoadBalancerBackend-Cmdlet** fügt einem Azure Load Balancer einen Back-End-Adresspool hinzu.

## BEISPIELE

### Beispiel 1: Hinzufügen einer Back-End-Adresspoolkonfiguration zu einem Load Balancer
```powershell
PS C:\>Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "myrg" | Add-AzLoadBalancerBackendAddressPoolConfig -Name "BackendAddressPool02" | Set-AzLoadBalancer
```

Dieser Befehl ruft den Load Balancer namens MyLoadBalancer ab, fügt den Back-End-Adresspool mit dem Namen BackAddressPool02 zu MyLoadBalancer hinzu und verwendet dann das **Set-AzLoadBalancer-Cmdlet,** um MyLoadBalancer zu aktualisieren.
