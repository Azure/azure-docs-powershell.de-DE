# Remove-AzPublicIpAddress

## SYNOPSIS
Entfernt eine öffentliche IP-Adresse.

## SYNTAX

```
Remove-AzPublicIpAddress -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzPublicIpAddress** entfernt eine öffentliche Azure-IP-Adresse.

## BEISPIELE

### 1: Entfernen einer öffentlichen IP-Adressressource
```
Remove-AzPublicIpAddress -Name $publicIpName -ResourceGroupName $rgName
```

Mit diesem Befehl wird die öffentliche IP-Adressressource $publicIpName in der Ressourcengruppe $rgName.
