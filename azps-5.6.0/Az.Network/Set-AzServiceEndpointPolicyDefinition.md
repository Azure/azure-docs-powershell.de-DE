# Set-AzServiceEndpointPolicyDefinition

## SYNOPSIS
Aktualisiert eine Richtliniendefinition für Dienstendpunkt.

## SYNTAX

```
Set-AzServiceEndpointPolicyDefinition -Name <String> -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-Description <String>] [-ServiceResource <String[]>] [-Service <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Set-AzServiceEndpointPolicyDefinition** erstellt eine Richtliniendefinition für Dienstendpunkt.

## BEISPIELE

### Beispiel 1: Aktualisiert die Definition einer Dienstendpunktrichtliniendefinition in einer Dienstendpunktrichtlinie
```
$serviceEndpointPolicy = Set-AzServiceEndpointPolicyDefinition -Name "Policydef1" -ServiceEndpointPolicy $serviceEndpointPolicy
```

Mit diesem Befehl wird eine Dienstendpunktrichtliniendefinition mit dem Namen "Policydef1" in der Vom Objektobjekt definierten Dienstendpunktrichtlinie $ServiceEndpointPolicy.
