# Remove-AzExpressRoutePort

## SYNOPSIS
Entfernt einen ExpressRoutePort.

## SYNTAX

### ResourceNameParameterSet (Standard)
```
Remove-AzExpressRoutePort -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzExpressRoutePort -InputObject <PSExpressRoutePort> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzExpressRoutePort -ResourceId <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzExpressRoutePort** entfernt einen ExpressRoutePort.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Remove-AzExpressRoutePort -Name $PortName -ResourceGroupName $rg
```

Entfernt $PortName ExpressRoutePort-Ressource in $rg Ressourcengruppe in Ihrem Abonnement.

### Beispiel 2
```powershell
PS C:\> Remove-AzExpressRoutePort -InputObject $erPort
```

Entfernt die ExpressRoutePort-Ressource in InputObject.

### Beispiel 3
```powershell
PS C:\> Remove-AzExpressRoutePort -Name $ResourceId $id
```

Entfernt die ExpressRoutePort-Ressource mit ResourceId-$id.
