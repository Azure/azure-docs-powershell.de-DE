# New-AzApplicationGatewayFirewallCustomRule

## SYNOPSIS
Erstellt eine neue benutzerdefinierte Regel für die Firewallrichtlinie für das Anwendungsgateway.

## SYNTAX

```
New-AzApplicationGatewayFirewallCustomRule -Name <String> -Priority <Int32> -RuleType <String>
 -MatchCondition <PSApplicationGatewayFirewallCondition[]> -Action <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Die **New-AzApplicationGatewayFirewallCustomRule** erstellt eine benutzerdefinierte Regel für die Firewallrichtlinie.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $customRule = New-AzApplicationGatewayFirewallCustomRule -Name example-rule -Priority 1 -RuleType MatchRule -MatchCondition $condtion -Action Allow
```

Der Befehl erstellt eine neue benutzerdefinierte Regel mit dem Namen der Beispielregel, Priorität 1, und der Regeltyp ist MatchRule mit Bedingung, die in der Bedingungsvariable definiert ist, die Aktion wird zulässig. Die neue benutzerdefinierte Regel "Übereinstimmung" wird in $customRule.
