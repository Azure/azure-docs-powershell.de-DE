# Remove-AzApplicationGatewayRedirectConfiguration

## SYNOPSIS
Entfernt eine Umleitungskonfiguration aus einem vorhandenen Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewayRedirectConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzApplicationGatewayRedirectConfiguration** entfernt eine Umleitungskonfiguration aus einem vorhandenen Anwendungsgateway.

## BEISPIELE

### Beispiel 1
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\>$AppGw = Remove-AzApplicationGatewayRedirectConfiguration -ApplicationGateway $AppGw -Name "Redirect01"
```

Der erste Befehl ruft ein Anwendungsgateway ab und speichert es in der $AppGw Variablen.
Mit dem zweiten Befehl wird die Umleitungskonfiguration mit dem Namen Redirect01 aus dem anwendungsgateway entfernt, das in $AppGw.
