# Get-AzVirtualNetworkTap

## SYNOPSIS
Ruft einen virtuellen Netzwerktipp ab

## SYNTAX

### ListParameterSet (Standard)
```
Get-AzVirtualNetworkTap [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzVirtualNetworkTap -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzVirtualNetworkTap-Cmdlet** ruft einen Azure Virtual Network Tap oder eine Liste der virtuellen Azure-Netzwerktipps in einer Ressourcengruppe ab.

## BEISPIELE

### Beispiel 1: Tippen auf ein virtuelles Netzwerk
```
PS C:\> Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
```

Dieser Befehl ruft einen VirtualNetwork-Tippverweis für bestimmte "VirtualTap1" in "ResourceGroup1" ab.

### Beispiel 2: Verwenden der Filterung zum Anzapfen aller virtuellen Netzwerktipps
```
PS C:\> Get-AzVirtualNetworkTap -Name "VirtualTap*"
```

Dieser Befehl ruft alle VirtualNetwork-Tippverweise ab, die mit "VirtualTap" beginnen.
