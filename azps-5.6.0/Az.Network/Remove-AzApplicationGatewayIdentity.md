# Remove-AzApplicationGatewayIdentity

## SYNOPSIS
Entfernt eine Identität aus einem Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewayIdentity -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
**Das Cmdlet Remove-AzApplicationGatewayIdentity** entfernt die Identität aus einem Anwendungsgateway.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $appgw = Remove-AzApplicationGatewayIdentity -ApplicationGateway $appgw
PS C:\> $updatedgateway = Set-AzApplicationGateway -ApplicationGateway $appgw
```

In diesem Beispiel wird die Identität aus einem vorhandenen Anwendungsgateway entfernt.
Hinweis: Wenn das Gateway auf einen Schlüsselvaultschlüssel verweist, ist es auch wichtig, diese Sslzertifikatverweise entlang dieses Vorgangs zu entfernen.
