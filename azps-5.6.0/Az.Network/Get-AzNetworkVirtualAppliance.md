# Get-AzNetworkVirtualAppliance

## SYNOPSIS
Virtuelle Netzwerkgeräte erhalten oder auflisten.

## SYNTAX

### ResourceNameParameterSet (Standard)
```
Get-AzNetworkVirtualAppliance [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzNetworkVirtualAppliance -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Die Get-AzNetworkVirtualAppliance -Befehle ruft Ressourcen der virtuellen Netzwerkgeräte ab oder listet sie auf.
