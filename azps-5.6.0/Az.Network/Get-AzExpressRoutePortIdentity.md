# Get-AzExpressRoutePortIdentity

## SYNOPSIS
Holen Sie sich die Identität, die einem ExpressRoutePort zugewiesen ist.

## SYNTAX

```
Get-AzExpressRoutePortIdentity -ExpressRoutePort <PSExpressRoutePort>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzExpressRoutePortIdentity-Cmdlet** ruft identitäten ab, die einem lokalen Azure ExpressRoutePort-Objekt zugewiesen werden.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $exrPort = Get-AzExpressRoutePort -Name $exrPortName -ResourceGroupName $resgpName
PS C:\> $identity = Get-AzExpressRoutePortIdentity -ExpressRoutePort $exrPort
```
