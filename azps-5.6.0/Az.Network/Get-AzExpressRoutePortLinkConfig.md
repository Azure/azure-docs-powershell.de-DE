# Get-AzExpressRoutePortLinkConfig

## SYNOPSIS
Ruft eine ExpressRoutePort-Linkkonfiguration ab.

## SYNTAX

### ResourceNameParameterSet (Standard)
```
Get-AzExpressRoutePortLinkConfig -ExpressRoutePort <PSExpressRoutePort> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzExpressRoutePortLinkConfig -ResourceId <String> -ExpressRoutePort <PSExpressRoutePort>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzExpressRoutePortLinkConfig-Cmdlet** ruft die Konfiguration eines Links eines ExpressRoutePort ab.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Get-AzExpressRoutePortLinkConfig -ExpressRoutePort $erport -Name Link1
```

Ruft die Link1-Konfiguration von ExpressRoutePort $erport

### Beispiel 2
```powershell
PS C:\> Get-AzExpressRoutePortLinkConfig -ExpressRoutePort $erport -ResourceId $id
```

Ruft die Konfiguration des Links mit ResourceId-$id in ExpressRoutePort $erport
