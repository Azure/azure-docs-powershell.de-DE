# Set-AzApplicationGatewaySku

## SYNOPSIS
Ändert die SKU eines Anwendungsgateways.

## SYNTAX

```
Set-AzApplicationGatewaySku -ApplicationGateway <PSApplicationGateway> -Name <String> -Tier <String>
 [-Capacity <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Set-AzApplicationGatewaySku** ändert die Bestandshaltungseinheit (Stock Keeping Unit, SKU) eines Anwendungsgateways.

## BEISPIELE

### Beispiel 1: Aktualisieren der Anwendungsgateway-SKU
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewaySku -ApplicationGateway $AppGw -Name "Standard_Small" -Tier "Standard" -Capacity 2
```

Der erste Befehl ruft das Anwendungsgateway mit dem Namen ApplicationGateway01 ab, das zur Ressourcengruppe "ResourceGroup01" gehört, und speichert es in der $AppGw Variablen.
Der zweite Befehl aktualisiert die SKU des Anwendungsgateways.
