# Remove-AzApplicationGatewaySslPolicy

## SYNOPSIS
Entfernt eine SSL-Richtlinie aus einem Azure-Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewaySslPolicy -ApplicationGateway <PSApplicationGateway> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das Remove-AzApplicationGatewaySslPolicy-Cmdlet entfernt die SSL-Richtlinie aus einem Azure-Anwendungsgateway.

## BEISPIELE

### Beispiel 1: Entfernen einer SSL-Richtlinie aus einem Anwendungsgateway
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGW = Remove-AzApplicationGatewaySslPolicy -ApplicationGateway $AppGW
```

Mit diesem Befehl wird die SSL-Richtlinie aus dem Anwendungsgateway mit dem Namen ApplicationGateway01 entfernt.
