# Remove-AzVirtualNetworkGatewayConnection

## SYNOPSIS
Löscht eine Verbindung mit einem virtuellen Netzwerkgateway

## SYNTAX

```
Remove-AzVirtualNetworkGatewayConnection -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Die Virtual Network Gateway Connection ist das Objekt, das den IPsec-Tunnel (Site-to-Site oder Vnet-to-Vnet) darstellt, der mit Ihrem Virtuellen Netzwerkgateway in Azure verbunden ist.
Das **Cmdlet Remove-AzVirtualNetworkGatewayConnection** löscht das Objekt Ihrer Verbindung basierend auf Name und Ressourcengruppenname.

## BEISPIELE

### Beispiel 1: Löschen einer Virtuellen Netzwerkgatewayverbindung
```powershell
Remove-AzVirtualNetworkGatewayConnection -Name myTunnel -ResourceGroupName myRG
```

Löscht das Objekt der Virtual Network Gateway Connection mit dem Namen "myTunnel" innerhalb der Ressourcengruppe "myRG"
