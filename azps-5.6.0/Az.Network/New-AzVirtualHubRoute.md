# New-AzVirtualHubRoute

## SYNOPSIS
Erstellt ein Azure Virtual Hub Route-Objekt.

## SYNTAX

```
New-AzVirtualHubRoute -AddressPrefix <String[]> -NextHopIpAddress <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Erstellt ein Azure Virtual Hub Route-Objekt.

## BEISPIELE

### Beispiel 1

```powershell
PS C:\> $route1 = New-AzVirtualHubRoute -AddressPrefix @("10.0.0.0/16", "11.0.0.0/16") -NextHopIpAddress "12.0.0.5"

AddressPrefixes            NextHopIpAddress
---------------            ----------------
{10.0.0.0/16, 11.0.0.0/16} 12.0.0.5
```

Mit dem oben genannten Objekt wird ein virtuelles Hubroutenobjekt erstellt, das in die Routetabelle des virtuellen Hubs aufgenommen werden kann.

Die virtuelle Hubroute ist ein Speicherobjekt, das zum Erstellen eines VirtualHubRouteTable-Objekts verwendet werden kann.
