# Get-AzNetworkUsage

## SYNOPSIS
Liste der Netzwerknutzungen für ein Abonnement

## SYNTAX

```
Get-AzNetworkUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das Get-AzNetworkUsage-Cmdlet erhält Grenzwerte und die aktuelle Verwendung für Netzwerkressourcen.

## BEISPIELE

