# New-AzFirewallPublicIpAddress

## SYNOPSIS
Dies ist der Platzhalter für die Ip-Adresse, die für multi pip in der Azure-Firewall verwendet werden kann.

## SYNTAX

```
New-AzFirewallPublicIpAddress [-Address <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Dies ist der Platzhalter für die Ip-Adresse, die für multi pip in der Azure-Firewall verwendet werden kann.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $publicIp = New-AzFirewallPublicIpAddress -Address 20.2.3.4
```

$publicIp wird der Platzhalter für die IP-Adresse 20.2.3.4
