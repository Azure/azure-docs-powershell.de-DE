# Set-AzApplicationGatewaySslProfile

## SYNOPSIS
Aktualisiert das Ssl-Profil für ein Anwendungsgateway.

## SYNTAX

```
Set-AzApplicationGatewaySslProfile -ApplicationGateway <PSApplicationGateway> -Name <String>
 [-SslPolicy <PSApplicationGatewaySslPolicy>]
 [-ClientAuthConfiguration <PSApplicationGatewayClientAuthConfiguration>]
 [-TrustedClientCertificates <PSApplicationGatewayTrustedClientCertificate[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Set-AzApplicationGatewaySslProfile** aktualisiert das Ssl-Profil für ein Azure-Anwendungsgateway.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewaySslProfile -ApplicationGateway $AppGw -Name "Profile01" -ClientAuthConfiguration $newclientconfig
```

Der erste Befehl ruft das Anwendungsgateway mit dem Namen ApplicationGateway01 ab, das zur Ressourcengruppe "ResourceGroup01" gehört, und speichert es in der $AppGw Variablen. Der zweite Befehl aktualisiert das SSL-Profil des Anwendungsgateways in der $AppGw-Variable, um die Clientauth-Konfiguration auf den neuen Wert zu aktualisieren, der in der App $newclientconfig.
