# Remove-AzApplicationGatewayTrustedClientCertificate

## SYNOPSIS
Entfernt das Zertifikatketteobjekt der vertrauenswürdigen Client-Zertifizierungsstelle aus einem Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewayTrustedClientCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzApplicationGatewayTrustedClientCertificate** entfernt das Zertifikatketteobjekt der vertrauenswürdigen Client-CA aus einem Anwendungsgateway.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name "TrustedClientCertificate01"
```

Der erste Befehl ruft ein Anwendungsgateway ab und speichert es in der $gw Variablen. Mit dem zweiten Befehl wird die Zertifikatkette der vertrauenswürdigen Client-Zertifizierungsstelle mit dem Namen "TrustedClientCertificate01" aus dem In-App-Gateway $gw.
