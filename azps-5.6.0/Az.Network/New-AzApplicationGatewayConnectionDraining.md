# New-AzApplicationGatewayConnectionDraining

## SYNOPSIS
Erstellt eine neue Verbindungsablaufkonfiguration für Back-End-HTTP-Einstellungen.

## SYNTAX

```
New-AzApplicationGatewayConnectionDraining -Enabled <Boolean> -DrainTimeoutInSec <Int32>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet New-AzApplicationGatewayConnectionDraining** erstellt eine neue Verbindungsablaufkonfiguration für Back-End-HTTP-Einstellungen.

## BEISPIELE

### Beispiel 1
```
PS C:\> $connectionDraining = New-AzApplicationGatewayConnectionDraining -Enabled $True -DrainTimeoutInSec 42
```

Mit dem Befehl wird eine neue Konfiguration für die Verbindungsentwässerung erstellt, deren Einstellung Auf True und DrainTimeoutInSec auf 42 Sekunden festgelegt ist, und speichert sie in $connectionDraining.
