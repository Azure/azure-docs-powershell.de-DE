# Remove-AzApplicationGatewaySslCertificate

## SYNOPSIS
Entfernt ein SSL-Zertifikat aus einem Azure-Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewaySslCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzApplicationGatewaySslCertificate** entfernt ein SSL-Zertifikat (Secure Sockets Layer) aus einem Azure-Anwendungsgateway.

## BEISPIELE

### Beispiel 1: Entfernen eines SSL-Zertifikats aus einem Anwendungsgateway
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewaySslCertificate -ApplicationGateway $AppGW -Name "Cert02"
```

Der erste Befehl ruft das Anwendungsgateway mit dem Namen ApplicationGateway01 ab und speichert das Ergebnis in der Variablen namens $AppGW.
Mit dem zweiten Befehl wird das #A0 mit dem Namen Cert02 aus dem Anwendungsgateway entfernt, das in der $AppGW gespeichert ist.
