# New-AzApplicationGatewayFirewallExclusionConfig

## SYNOPSIS
Erstellt eine neue Ausschlussregelliste für Anwendungsgateway waf

## SYNTAX

```
New-AzApplicationGatewayFirewallExclusionConfig -Variable <String> -Operator <String> -Selector <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet New-AzApplicationGatewayFirewallExclusionConfig** führt eine neue Ausschlussregelliste für das Anwendungsgateway waf aus.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $exclusion1 = New-AzApplicationGatewayFirewallExclusionConfig -Variable "RequestHeaderNames" -Operator "StartsWith" -Selector "xyz"
```

Mit diesem Befehl wird eine neue Ausschlussregel erstellt, in der die Konfiguration für die Variable "RequestHeaderNames" und der Operator "StartsWith" und "Selector" mit dem Namen xyz aufgeführt wird. Die Konfiguration der Ausschlussliste wird in $exclusion 1 gespeichert.
