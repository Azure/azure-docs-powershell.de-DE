# Get-AzRouteTable

## SYNOPSIS
Ruft Routentabellen ab.

## SYNTAX

### NoExpand (Standard)
```
Get-AzRouteTable [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Erweitern
```
Get-AzRouteTable -ResourceGroupName <String> -Name <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzRouteTable-Cmdlet** ruft Azure-Routentabellen ab.
Sie können eine einzelne Routentabelle oder alle Routentabellen in einer Ressourcengruppe oder in Ihrem Abonnement erhalten.
