# Get-AzApplicationGatewayBackendHttpSetting

## SYNOPSIS
Ruft die Back-End-HTTP-Einstellungen eines Anwendungsgateways ab.

## SYNTAX

```
Get-AzApplicationGatewayBackendHttpSetting [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das Get-AzApplicationGatewayBackendHttpSetting-Cmdlet ruft die Back-End-HTTP-Einstellungen eines Anwendungsgateways ab.

## BEISPIELE

### Beispiel 1: Erhalten von Back-End-HTTP-Einstellungen nach Name
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Settings  = Get-AzApplicationGatewayBackendHttpSetting -Name "Settings01" -ApplicationGateway $AppGw
```

Der erste Befehl ruft das Anwendungsgateway mit dem Namen ApplicationGateway01 in der Ressourcengruppe "ResourceGroup01" ab und speichert es in der $AppGw Variablen. Der zweite Befehl ruft die HTTP-Einstellungen mit dem Namen Settings01 für $AppGw ab und speichert die Einstellungen in $Settings Variablen.

### Beispiel 2: Erhalten einer Sammlung von Back-End-HTTP-Einstellungen
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $SettingsList  = Get-AzApplicationGatewayBackendHttpSetting -ApplicationGateway $AppGw
```

Der erste Befehl ruft das Anwendungsgateway mit dem Namen ApplicationGateway01 in der Ressourcengruppe "ResourceGroup01" ab und speichert es in der $AppGw Variablen. Der zweite Befehl ruft die Sammlung der HTTP-Einstellungen für $AppGw ab und speichert die Einstellungen in der $SettingsList Variablen.
