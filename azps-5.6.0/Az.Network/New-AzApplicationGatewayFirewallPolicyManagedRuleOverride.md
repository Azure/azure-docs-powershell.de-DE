# New-AzApplicationGatewayFirewallPolicyManagedRuleOverride

## SYNOPSIS
Erstellt einen managedRuleOverride-Eintrag für den Eintrag RuleGroupOverrideGroup.

## SYNTAX

```
New-AzApplicationGatewayFirewallPolicyManagedRuleOverride -RuleId <String> [-State <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Die **New-AzApplicationGatewayFirewallPolicyManagedRuleOverride** erstellt einen ruleOverride-Eintrag.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> ruleOverrideEntry = New-AzApplicationGatewayFirewallPolicyManagedRuleOverride -RuleId $ruleId -State Disabled
```

Erstellt eine ruleOverride Entry with RuleId as $ruleId und State as Disabled.
