# Get-AzApplicationGatewayProbeConfig

## SYNOPSIS
Ruft eine vorhandene Integritätstestkonfiguration aus einem Anwendungsgateway ab.

## SYNTAX

```
Get-AzApplicationGatewayProbeConfig [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das Get-AzApplicationGatewayProbeConfig-Cmdlet ruft eine vorhandene Integritätstestkonfiguration aus einem Anwendungsgateway ab.

## BEISPIELE

### Beispiel 1: Erhalten eines vorhandenen Prüfpunkts aus einem Anwendungsgateway
```
PS C:\>Get-AzApplicationGatewayProbeConfig -ApplicationGateway Gateway -Name "Probe02"
```

Dieser Befehl ruft den Integritätstest mit dem Namen Probe02 aus dem Anwendungsgateway mit dem Namen Gateway ab.
