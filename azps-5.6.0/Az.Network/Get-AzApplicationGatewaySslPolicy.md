# Get-AzApplicationGatewaySslPolicy

## SYNOPSIS
Ruft die SSL-Richtlinie eines Anwendungsgateways ab.

## SYNTAX

```
Get-AzApplicationGatewaySslPolicy -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Get-AzApplicationGatewaySslPolicy** ruft die SSL-Richtlinie eines Anwendungsgateways ab.

## BEISPIELE

### 1:
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $sslpolicy = Get-AzApplicationGatewaySslPolicy -ApplicationGateway $AppGW
```

Der erste Befehl ruft das Application Gateway mit dem Namen ApplicationGateway01 ab und speichert das Ergebnis in der Variablen namens $AppGW.
Der zweite Befehl ruft die SSL-Richtlinie aus dem Anwendungsgateway ab, das in der Variablen namens $AppGW.
