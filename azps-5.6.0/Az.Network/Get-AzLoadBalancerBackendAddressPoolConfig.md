# Get-AzLoadBalancerBackendAddressPoolConfig

## SYNOPSIS
Ruft eine Back-End-Adresspoolkonfiguration für einen Load Balancer ab.

## SYNTAX

```
Get-AzLoadBalancerBackendAddressPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzLoadBalancerBackendAddressPoolConfig-Cmdlet** ruft einen einzelnen Back-End-Adresspool oder eine Liste der Back-End-Adresspools in einem Load Balancer ab.

## BEISPIELE

### Beispiel 1: Erhalten des Back-End-Adresspools
```
PS C:\>$loadbalancer = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerBackendAddressPoolConfig -Name "BackendAddressPool02" -LoadBalancer $loadbalancer
```

Der erste Befehl ruft einen vorhandenen Load Balancer mit dem Namen MyLoadBalancer in der Ressourcengruppe mit dem Namen MyResourceGroup ab und speichert ihn dann in der $loadbalancer Variablen.
Der zweite Befehl ruft die zugeordnete Back-End-Adresspoolkonfiguration mit dem Namen Back-EndAddressPool02 für den Load Balancer in $loadbalancer.
