# Get-AzApplicationGatewayRequestRoutingRule

## SYNOPSIS
Ruft die Anforderungsroutingregel eines Anwendungsgateways ab.

## SYNTAX

```
Get-AzApplicationGatewayRequestRoutingRule [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Get-AzApplicationGatewayRequestRoutingRule** ruft die Anforderungsroutingregel eines Anwendungsgateways ab.

## BEISPIELE

### Beispiel 1: Anfordern einer bestimmten Anforderungsroutingregel
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Rule = Get-AzApplicationGatewayRequestRoutingRule -"Rule01" -ApplicationGateway $AppGW
```

Der erste Befehl ruft das Application Gateway mit dem Namen ApplicationGateway01 ab und speichert das Ergebnis in der Variablen namens $AppGW.
Der zweite Befehl ruft die Anforderungsroutingregel mit dem Namen Regel01 aus dem Anwendungsgateway ab, das in der Variablen namens $AppGW.

### Beispiel 2: Anfordern einer Liste mit Anforderungsroutingregeln
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Rules = Get-AzApplicationGatewayRequestRoutingRule -ApplicationGateway $AppGW
```

Der erste Befehl ruft das Application Gateway mit dem Namen ApplicationGateway01 ab und speichert das Ergebnis in der Variablen namens $AppGW.
Der zweite Befehl ruft eine Liste der Anforderungsroutingregeln aus dem Anwendungsgateway ab, die in der Variablen namens $AppGW.
