# New-AzApplicationGatewayAutoscaleConfiguration

## SYNOPSIS
Erstellt eine Konfiguration für die automatische Skalierung für das Anwendungsgateway.

## SYNTAX

```
New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity <Int32> [-MaxCapacity <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet New-AzApplicationGatewayAutoscaleConfiguration** erstellt die Konfiguration automatischer Skalierung für ein Azure-Anwendungsgateway.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $autoscaleConfig = New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity 3
PS C:\> $gw = New-AzApplicationGateway -Name $appgwName -ResourceGroupName $rgname ..  -AutoscaleConfiguration $autoscaleConfig
```

Mit dem ersten Befehl wird eine Konfiguration für die automatische Skalierung mit einer Mindestkapazität von 3 erstellt.
Mit dem zweiten Befehl wird ein Anwendungsgateway mit der Konfiguration der automatischen Skalierung erstellt.
