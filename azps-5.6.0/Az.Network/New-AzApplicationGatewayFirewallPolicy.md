# New-AzApplicationGatewayFirewallPolicy

## SYNOPSIS
Erstellt eine Anwendungsgatewayfirewallrichtlinie.

## SYNTAX

```
New-AzApplicationGatewayFirewallPolicy -Name <String> -ResourceGroupName <String> -Location <String>
 [-CustomRule <PSApplicationGatewayFirewallCustomRule[]>]
 [-PolicySetting <PSApplicationGatewayFirewallPolicySettings>]
 [-ManagedRule <PSApplicationGatewayFirewallPolicyManagedRules>] [-Tag <Hashtable>] [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet New-AzApplicationGatewayFirewallPolicy** erstellt eine Firewallrichtlinie für das Anwendungsgateway.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $firewallPolicy = New-AzApplicationGatewayFirewallPolicy -Name wafResource1 -ResourceGroupName "rg1"  -Location  "westus" -CustomRule $customRule
```

Mit diesem Befehl wird eine neue Azure-Anwendungsgatewayfirewallrichtlinie mit dem Namen "wafResource1" in der Ressourcengruppe "rg1" am Speicherort "westus" mit benutzerdefinierten Regeln erstellt, die in der Variablen "$customRule" definiert sind.
