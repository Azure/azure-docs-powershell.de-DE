# Set-AzVirtualNetworkTap

## SYNOPSIS
Aktualisiert die Anzapfung eines virtuellen Netzwerks.

## SYNTAX

```
Set-AzVirtualNetworkTap -VirtualNetworkTap <PSVirtualNetworkTap> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Set-AzVirtualNetworkTap** aktualisiert einen virtuellen Netzwerktipp.

## BEISPIELE

### Beispiel 1: Konfigurieren eines virtuellen Netzwerktippens
```
PS C:\>$vTap = Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
PS C:\>$vTap.DestinationNetworkInterfaceIPConfiguration = $newDestinationNic.IpConfigurations[0]
PS C:\>Set-AzVirtualNetworkTap -VirtualNetworkTap $vTap
```

Der Befehl aktualisiert die Ziel-IpConfiguration und aktualisiert den Tippen auf das virtuelle Netzwerk.
Wenn tippende Konfigurationen auf sie verweisen, wird nicht der ganze Quelldatenverkehr nach dem Update an die neue Ziel-IP-Konfiguration gespiegelt.
