# Get-AzVirtualNetworkGatewayVpnClientConnectionHealth

## SYNOPSIS 
Hier finden Sie die Liste der Integrität der Vpn-Clientverbindung eines virtuellen Azure-Netzwerkgateways für pro VPN-Clientverbindung.

## SYNTAX

```
Get-AzVirtualNetworkGatewayVpnClientConnectionHealth -ResourceName <String> -ResourceGroupName <String> -InputObject <PSVirtualNetworkGateway> -ResourceId <ResourceId>
 [-AsJob] [<CommonParameters>]
```

## BESCHREIBUNG
Listen Sie den Integritätszustand aller verbundenen Vpn-Client-Verbindungen auf. Dies umfasst: vpnConnectionId vpnConnectionDuration vpnConnectionTime publicIpAddress privateIpAddress vpnUserName maxBandwidth egressPacketsTransferred egressBytesTransferred ingressPacketsTransferred ingressBytesTransferred maxPacketsPerSecond
