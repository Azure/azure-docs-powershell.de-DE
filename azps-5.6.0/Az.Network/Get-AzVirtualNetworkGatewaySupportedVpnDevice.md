# Get-AzVirtualNetworkGatewaySupportedVpnDevice

## SYNOPSIS
Dieses Befehlslet gibt eine Liste der unterstützten VPN-Gerätemarken, -Modelle und -Firmwareversionen zurück.

## SYNTAX

```
Get-AzVirtualNetworkGatewaySupportedVpnDevice -Name <String> -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Dieses Befehlslet gibt eine Liste der unterstützten VPN-Gerätemarken, -Modelle und -Firmwareversionen zurück.

## BEISPIELE

### Beispiel 1
```
PS C:\> Get-AzVirtualNetworkGatewaySupportedVpnDevice -ResourceGroupName TestRG -Name TestGateway 
<?xml version="1.0" encoding="utf-8"?>
<RpVpnDeviceList version="1.0">
  <Vendor name="Cisco-Test">
    <DeviceFamily name="IOS-Test">
       <FirmwareVersion name="20" />
    </DeviceFamily>
  </Vendor>
</RpVpnDeviceList>
```

Gibt eine Liste der unterstützten VPN-Gerätemarken, -Modelle und -Firmwareversionen zurück:
<?xml version="1.0" encoding="utf-8"?>
<RpVpnDeviceList version="1.0">
  <Vendor name="Cisco-Test">
    <DeviceFamily name="IOS-Test">
       <FirmwareVersion name="20" />
    </DeviceFamily>
  </Vendor>
</RpVpnDeviceList>
