# Get-AzApplicationGatewayFirewallPolicy

## SYNOPSIS
Ruft eine Anwendungsgatewayfirewallrichtlinie ab.

## SYNTAX

```
Get-AzApplicationGatewayFirewallPolicy [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzApplicationGatewayFirewallPolicy-Cmdlet** ruft eine Anwendungsgateway-Firewallrichtlinie ab.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $AppGwFirewallPolicy = Get-AzApplicationGatewayFirewallPolicy -Name "FirewallPolicy1" -ResourceGroupName "ResourceGroup01"
```

Dieser Befehl ruft die Firewallrichtlinie für das Anwendungsgateway mit dem Namen FirewallPolicy1 ab, die zur Ressourcengruppe "ResourceGroup01" gehört, und speichert sie in der $AppGwFirewallPolicy-Variable.
