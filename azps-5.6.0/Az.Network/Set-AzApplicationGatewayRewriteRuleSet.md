# Set-AzApplicationGatewayRewriteRuleSet

## SYNOPSIS
Ändert einen Neuschreibregelsatz für ein Anwendungsgateway.

## SYNTAX

```
Set-AzApplicationGatewayRewriteRuleSet -ApplicationGateway <PSApplicationGateway> -Name <String>
 -RewriteRule <System.Collections.Generic.List`1[Microsoft.Azure.Commands.Network.Models.PSApplicationGatewayRewriteRule]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Set-AzApplicationGatewayRewriteRuleSet** ändert eine Anforderungsroutingregel.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewayRewriteRuleSet -ApplicationGateway $AppGw -Name "ruleset1" -RewriteRule $rule
```

Der erste Befehl ruft das Anwendungsgateway mit dem Namen ApplicationGateway01 ab und speichert es in der $AppGw Variablen.
Mit dem zweiten Befehl wird der Satz der Neuschreibregel für das Anwendungsgateway so geändert, dass die in der Variablen "$rule" angegebenen Neuschreibregeln verwendet werden.
