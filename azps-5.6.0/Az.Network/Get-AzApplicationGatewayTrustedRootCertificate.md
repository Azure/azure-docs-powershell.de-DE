# Get-AzApplicationGatewayTrustedRootCertificate

## SYNOPSIS
Ruft das vertrauenswürdige Stammzertifikat mit einem bestimmten Namen aus dem Anwendungsgateway ab.

## SYNTAX

```
Get-AzApplicationGatewayTrustedRootCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzApplicationGatewayTrustedRootCertificate-Cmdlet** ruft vertrauenswürdiges Stammzertifikat mit einem bestimmten Namen aus dem Anwendungsgateway ab.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $trustedRootCert = Get-AzApplicationGatewayTrustedRootCertificate -ApplicationGateway $gw -Name $certName --CertificateFile ".\rootCA.cer"
```

Der erste Befehl ruft das Anwendungsgateway ab und speichert es in $gw Variablen.
Der zweite Befehl ruft das vertrauenswürdige Stammzertifikat mit einem angegebenen Namen aus dem Anwendungsgateway ab.
