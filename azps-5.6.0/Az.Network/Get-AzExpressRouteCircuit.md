# Get-AzExpressRouteCircuit

## SYNOPSIS
Ruft einen Azure ExpressRoute-Schaltkreis aus Azure ab.

## SYNTAX

```
Get-AzExpressRouteCircuit [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzExpressRouteCircuit-Cmdlet** wird verwendet, um ein ExpressRoute-Schaltkreisobjekt aus Ihrem Abonnement abzurufen. Das zurückgegebene Schaltkreisobjekt kann als Eingabe für andere Cmdlets verwendet werden, die auf ExpressRoute-Schaltkreisen funktionieren.
