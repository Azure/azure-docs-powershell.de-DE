# Remove-AzApplicationGatewayProbeConfig

## SYNOPSIS
Entfernt einen Integritätstest aus einem vorhandenen Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewayProbeConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das Remove-AzApplicationGatewayProbeConfig-Cmdlet entfernt einen Heath-Sonden aus einem vorhandenen Anwendungsgateway.

## BEISPIELE

### Beispiel 1: Entfernen eines Integritätstestes aus einem vorhandenen Anwendungsgateway
```
PS C:\>$Gateway = Remove-AzApplicationGatewayProbeConfig -ApplicationGateway Gateway -Name "Probe04"
```

Mit diesem Befehl wird der Integritätstest mit dem Namen Probe04 aus dem Anwendungsgateway mit dem Namen Gateway entfernt.
