# Get-AzLoadBalancerInboundNatPoolConfig

## SYNOPSIS
Ruft eine oder mehrere eingehende NAT-Poolkonfigurationen aus einem Load Balancer ab.

## SYNTAX

```
Get-AzLoadBalancerInboundNatPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzLoadBalancerInboundNatPoolConfig-Cmdlet** ruft eine oder mehrere eingehende NAT-Poolkonfigurationen aus einem Load Balancer ab.

## BEISPIELE

### 1: Get
```
PS C:\> $slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> $slb | Get-AzLoadBalancerInboundNatPoolConfig -Name myInboundNatPool
```
