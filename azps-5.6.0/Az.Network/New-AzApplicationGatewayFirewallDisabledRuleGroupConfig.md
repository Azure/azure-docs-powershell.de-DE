# New-AzApplicationGatewayFirewallDisabledRuleGroupConfig

## SYNOPSIS
Erstellt eine neue deaktivierte Regelgruppenkonfiguration.

## SYNTAX

```
New-AzApplicationGatewayFirewallDisabledRuleGroupConfig -RuleGroupName <String> [-Rules <Int32[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet New-AzApplicationGatewayFirewallDisabledRuleGroupConfig** erstellt eine neue deaktivierte Regelgruppenkonfiguration.

## BEISPIELE

### Beispiel 1
```
PS C:\> $disabledRuleGroup1 = New-AzApplicationGatewayFirewallDisabledRuleGroupConfig -RuleGroupName "REQUEST-942-APPLICATION-ATTACK-SQLI" -Rules 942130,942140
```

Der Befehl erstellt eine neue deaktivierte Regelgruppenkonfiguration f체r die Regelgruppe mit dem Namen "REQUEST-942-APPLICATION-ATTACK-SQLI", bei der Regel 942130 und Regel 942140 deaktiviert werden. Die konfiguration der neuen deaktivierten Regelgruppe wird in $disabledRuleGroup 1 gespeichert.
