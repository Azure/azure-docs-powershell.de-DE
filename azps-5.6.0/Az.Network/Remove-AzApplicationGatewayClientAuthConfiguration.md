# Remove-AzApplicationGatewayClientAuthConfiguration

## SYNOPSIS
Entfernt die Clientauthentifizierungskonfiguration eines SSL-Profilobjekts.

## SYNTAX

```
Remove-AzApplicationGatewayClientAuthConfiguration -SslProfile <PSApplicationGatewaySslProfile>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzApplicationGatewayClientAuthConfiguration** entfernt die Clientauthentifizierungskonfiguration eines SSL-Profilobjekts.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $profile  = Get-AzApplicationGatewaySslProfile -Name "Profile01" -ApplicationGateway $AppGw
PS C:\> Remove-AzApplicationGatewayClientAuthConfiguration -SslProfile $profile
```

Der erste Befehl ruft das Anwendungsgateway mit dem Namen ApplicationGateway01 in der Ressourcengruppe "ResourceGroup01" ab und speichert es in der $AppGw Variablen. Der zweite Befehl ruft das SSL-Profil mit dem Namen Profile01 f체r $AppGw ab und speichert es in der $profile Variablen. Mit dem letzten Befehl wird die Clientauthentifizierungskonfiguration des ssl-Profils entfernt, das in $profile.
