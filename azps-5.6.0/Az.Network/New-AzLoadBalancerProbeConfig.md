# New-AzLoadBalancerProbeConfig

## SYNOPSIS
Erstellt eine Prüfpunktkonfiguration für einen Load Balancer.

## SYNTAX

```
New-AzLoadBalancerProbeConfig -Name <String> [-Protocol <String>] -Port <Int32> -IntervalInSeconds <Int32>
 -ProbeCount <Int32> [-RequestPath <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet New-AzLoadBalancerProbeConfig** erstellt eine Prüfpunktkonfiguration für einen Azure Load Balancer.

## BEISPIELE

### Beispiel 1: Erstellen einer Testkonfiguration
```powershell
PS C:\>New-AzLoadBalancerProbeConfig -Name "MyProbe" -Protocol "http" -Port 80 -IntervalInSeconds 15 -ProbeCount 15
```

Mit diesem Befehl wird mithilfe des HTTP-Protokolls eine Testkonfiguration namens MyProbe erstellt.
Der neue Prüfpunkt wird eine Verbindung mit einem Lastenausgleichsdienst auf Port 80 herstellen.

### Beispiel 2

Erstellt eine Prüfpunktkonfiguration für einen Load Balancer. (automatisch generiert)

<!-- Aladdin Generated Example -->
```powershell
New-AzLoadBalancerProbeConfig -IntervalInSeconds 15 -Name 'MyProbe' -Port 80 -ProbeCount 15 -Protocol 'http' -RequestPath 'healthcheck.aspx'
```
