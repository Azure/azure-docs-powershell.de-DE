# Remove-AzApplicationGatewaySslProfile

## SYNOPSIS
Entfernt das Ssl-Profil aus einem Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewaySslProfile -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzApplicationGatewaySslProfile** entfernt das SSL-Profil aus einem Anwendungsgateway.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewaySslProfile -ApplicationGateway $AppGw -Name "SslProfile01"
```

Der erste Befehl ruft ein Anwendungsgateway mit dem Namen ApplicationGateway01 ab, das zur Ressourcengruppe "ResourceGroup01" gehört und es in der Variablen $AppGw speichert. Mit dem zweiten Befehl wird das ssl-Profil mit dem Namen SslProfile01 aus dem anwendungsgateway entfernt, das in $AppGw.
