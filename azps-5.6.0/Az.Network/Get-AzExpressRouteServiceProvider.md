# Get-AzExpressRouteServiceProvider

## SYNOPSIS
Ruft eine Liste der ExpressRoute-Dienstanbieter und deren Attribute ab.

## SYNTAX

```
Get-AzExpressRouteServiceProvider [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzExpressRouteServiceProvider-Cmdlet** ruft eine Liste von ExpressRoute-Dienstanbietern und deren Attributen ab. Attribute umfassen Standort- und Bandbreitenoptionen.

## BEISPIELE

### Beispiel 1: Erhalten einer Liste von Dienstanbietern mit Standorten im "Silicon Valley"
```
Get-AzExpressRouteServiceProvider |
   Where-Object PeeringLocations -Contains "Silicon Valley" |
   Select-Object Name
```
