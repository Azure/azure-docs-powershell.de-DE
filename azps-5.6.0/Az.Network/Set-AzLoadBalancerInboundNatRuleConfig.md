# Set-AzLoadBalancerInboundNatRuleConfig

## SYNOPSIS
Legt eine eingehende NAT-Regelkonfiguration für einen Load Balancer fest.

## SYNTAX

### SetByResource (Standard)
```
Set-AzLoadBalancerInboundNatRuleConfig -LoadBalancer <PSLoadBalancer> -Name <String> [-Protocol <String>]
 [-FrontendPort <Int32>] [-BackendPort <Int32>] [-IdleTimeoutInMinutes <Int32>] [-EnableFloatingIP]
 [-EnableTcpReset] [-FrontendIpConfiguration <PSFrontendIPConfiguration>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByResourceId
```
Set-AzLoadBalancerInboundNatRuleConfig -LoadBalancer <PSLoadBalancer> -Name <String> [-Protocol <String>]
 [-FrontendPort <Int32>] [-BackendPort <Int32>] [-IdleTimeoutInMinutes <Int32>] [-EnableFloatingIP]
 [-EnableTcpReset] [-FrontendIpConfigurationId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Set-AzLoadBalancerInboundNatRuleConfig** legt eine Nat-Regelkonfiguration (Inbound Network Address Translation) für einen Azure Load Balancer fest.

## BEISPIELE

### Beispiel 1: Ändern der Konfiguration der eingehenden NAT-Regel für einen Load Balancer
```powershell
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> $slb | Add-AzLoadBalancerInboundNatRuleConfig -Name "NewNatRule" -FrontendIPConfiguration $slb.FrontendIpConfigurations[0] -Protocol "Tcp" -FrontendPort 3350 -BackendPort 3350 -EnableFloatingIP
PS C:\> $slb | Set-AzLoadBalancerInboundNatRuleConfig -Name "NewNatRule" -FrontendIPConfiguration $slb.FrontendIpConfigurations[0] -Protocol "Tcp" -FrontendPort 3350 -BackendPort 3350
```

Der erste Befehl ruft den Load Balancer mit dem Namen MyLoadBalancer ab und speichert ihn dann in der $slb Variablen.
Der zweite Befehl verwendet den Pipelineoperator, um den Lastenausgleich in $slb an Add-AzLoadBalancerInboundNatRuleConfig zu übergeben, wodurch ihm eine eingehende NAT-Regelkonfiguration hinzugefügt wird.
Der dritte Befehl übergibt den Load Balancer an **Set-AzLoadBalancerInboundNatRuleConfig**, mit dem die eingehende NAT-Regelkonfiguration gespeichert und aktualisiert wird.
Beachten Sie, dass die Regelkonfiguration festgelegt wurde, ohne unverankerte IP zu aktivieren, die durch den vorherigen Befehl aktiviert wurde.
