# Remove-AzRouteFilter

## SYNOPSIS
Entfernt einen Routenfilter.

## SYNTAX

```
Remove-AzRouteFilter -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzRouteFilter** entfernt einen Routenfilter.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Remove-AzRouteFilter -Name "RouteFilter01" -ResourceGroupName "ResourceGroup01"
```

Der Befehl entfernt den Routenfilter mit dem Namen RouteFilter01 in der Ressourcengruppe "ResourceGroup01".
