# Get-AzLoadBalancerFrontendIpConfig

## SYNOPSIS
Ruft eine Front-End-IP-Konfiguration in einem Load Balancer ab.

## SYNTAX

```
Get-AzLoadBalancerFrontendIpConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzLoadBalancerFrontendIpConfig-Cmdlet** ruft eine Front-End-IP-Konfiguration oder eine Liste der Front-End-IP-Konfigurationen in einem Load Balancer ab.

## BEISPIELE

### Beispiel 1: Erhalten einer Front-End-IP-Konfiguration in einem Load Balancer
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerFrontendIpConfig -Name "MyFrontEnd" -LoadBalancer $slb
```

Der erste Befehl ruft den Load Balancer mit dem Namen MyLoadBalancer ab und speichert ihn dann in der Variablen $slb.
Der zweite Befehl ruft die front-end-IP-Konfiguration ab, die diesem Load Balancer zugeordnet ist.
