# New-AzApplicationGatewayCustomError

## SYNOPSIS
Erstellt einen benutzerdefinierten Fehler mit http-Statuscode und benutzerdefinierter Fehlerseiten-URL 

## SYNTAX

```
New-AzApplicationGatewayCustomError -StatusCode <String> -CustomErrorPageUrl <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet New-AzApplicationGatewayCustomError** erstellt einen benutzerdefinierten Fehler.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $customError403Url = "https://mycustomerrorpages.blob.core.windows.net/errorpages/403-another.htm"
PS C:\> $ce = New-AzApplicationGatewayCustomError -StatusCode HttpStatus403 -CustomErrorPageUrl $customError403Url
```

Mit diesem Befehl wird der benutzerdefinierte Fehler von http-Statuscode 403 erstellt.
