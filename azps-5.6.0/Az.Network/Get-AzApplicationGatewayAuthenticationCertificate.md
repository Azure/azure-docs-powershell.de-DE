# Get-AzApplicationGatewayAuthenticationCertificate

## SYNOPSIS
Ruft ein Authentifizierungszertifikat für ein Anwendungsgateway ab.

## SYNTAX

```
Get-AzApplicationGatewayAuthenticationCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Get-AzApplicationGatewayAuthenticationCertificate** erhält ein Authentifizierungszertifikat für ein Azure-Anwendungsgateway.

## BEISPIELE

### Beispiel 1: Erhalten eines angegebenen Authentifizierungszertifikats
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $cert = Get-AzApplicationGatewayAuthenticationCertificate -Name "cert01" -ApplicationGateway $appgw
```

Der erste Befehl ruft das Anwendungsgateway mit dem Namen "appGwName" ab und speichert es in der $appgw-Variable.
Der zweite Befehl ruft das Authentifizierungszertifikat mit dem Namen cert01 ab und speichert es in der $cert-Variable.
