# Add-AzRouteFilterRuleConfig

## SYNOPSIS
Fügt einem Routenfilter eine Routenfilterregel hinzu.

## SYNTAX

```
Add-AzRouteFilterRuleConfig -RouteFilter <PSRouteFilter> [-Force] -Name <String> -Access <String>
 -RouteFilterRuleType <String> -CommunityList <String[]> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das Add-AzRouteFilterRuleConfig-Cmdlet fügt einem Azure-Routenfilter eine Routenfilterregel hinzu.

## BEISPIELE

### Beispiel 1: Hinzufügen einer Routenfilterregel zu einem Routenfilter
```
PS C:\>$RouteFilter = Get-AzRouteFilter -ResourceGroupName "ResourceGroup11" -Name "routefilter01"
                      PS C:\> Add-AzRouteFilterRuleConfig -Name "rule13" -Access Allow -RouteFilterRuleType Community -RouteFilter $RouteFilter
```

Der erste Befehl ruft einen Routenfilter mit dem Namen routefilter01 mithilfe des cmdlets Get-AzRouteFilter ab.
Der Befehl speichert den Filter in der $RouteFilter Variablen.
