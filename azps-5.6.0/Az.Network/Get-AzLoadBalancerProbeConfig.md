# Get-AzLoadBalancerProbeConfig

## SYNOPSIS
Ruft eine Testkonfiguration für einen Lastenausgleich ab.

## SYNTAX

```
Get-AzLoadBalancerProbeConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzLoadBalancerProbeConfig-Cmdlet** ruft eine oder mehrere Prüfpunktkonfigurationen für einen Load Balancer ab.

## BEISPIELE

### Beispiel 1: So wird die Sondenkonfiguration eines Lastenausgleichs
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerProbeConfig -Name "MyProbe" -LoadBalancer $slb
```

Der erste Befehl ruft den Load Balancer mit dem Namen MyLoadBalancer ab und speichert ihn dann in der Variablen $slb.
Der zweite Befehl ruft die zugehörige Sondenkonfiguration mit dem Namen MyProbe aus dem Load Balancer in $slb.
