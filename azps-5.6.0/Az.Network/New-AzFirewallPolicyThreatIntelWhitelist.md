# New-AzFirewallPolicyThreatIntelWhitelist

## SYNOPSIS
Erstellen einer neuen Whitelist für threat intelligence für Azure Firewall Policy

## SYNTAX

```
New-AzFirewallPolicyThreatIntelWhitelist [-FQDN <String[]>] [-IpAddress <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet New-AzFirewallPolicyThreatIntelWhitelist** erstellt ein Intel Whitelist-Objekt für Bedrohungen, das beim Erstellen oder Festlegen einer Azure-Firewallrichtlinie verwendet werden kann.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> New-AzFirewallPolicyThreatIntelWhitelist -IpAddress 23.46.72.91,192.79.236.79 -FQDN microsoft.com
```

In diesem Beispiel wird eine Intel-Whitelist für Bedrohungen erstellt, die eine FQDN-Whitelist mit einem Eintrag und eine Ip-Adress-Whitelist mit zwei Einträgen enthält.
