# Get-AzLoadBalancerInboundNatRuleConfig

## SYNOPSIS
Ruft eine eingehende NAT-Regelkonfiguration f체r einen Load Balancer ab.

## SYNTAX

```
Get-AzLoadBalancerInboundNatRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzLoadBalancerInboundNatRuleConfig-Cmdlet** ruft eine oder mehrere NAT-Regeln (Inbound Network Address Translation) in einem Azure Load Balancer ab.

## BEISPIELE

### Beispiel 1: Erhalten einer eingehenden NAT-Regelkonfiguration
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerInboundNatRuleConfig -Name "MyInboundNatRule1" -LoadBalancer $slb
```

Der erste Befehl ruft den Load Balancer mit dem Namen MyLoadBalancer ab und speichert ihn in der Variablen $slb.
Der zweite Befehl ruft die zugeordnete NAT-Regel mit dem Namen MyInboundNatRule1 aus dem Load Balancer in $slb.
