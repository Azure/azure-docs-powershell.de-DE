# Set-AzNetworkProfile

## SYNOPSIS
Aktualisiert ein Netzwerkprofil.

## SYNTAX

```
Set-AzNetworkProfile -NetworkProfile <PSNetworkProfile> [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Set-AzPublicIpPrefix** aktualisiert ein Netzwerkprofil.

## BEISPIELE

### Beispiel 1
```powershell
$networkProfile = Get-AzNetworkProfile -Name np1 -ResourceGroupName rg1

$networkProfile.Tags = "TestTag"

$networkProfile.ContainerNetworkInterfaceConfigurations = New-AzNetworkProfileContainerNicConfig -Name cnicconfig1

$networkProfile | Set-AzNetworkProfile
```

Der erste Befehl ruft ein vorhandenes Netzwerkprofil ab. Der zweite Befehl aktualisiert ein Tag, und der dritte Befehl fügt eine Netzwerkschnittstellenkonfiguration im Netzwerkprofil hinzu. Mit dem vierten Befehl wird das Netzwerkprofil aktualisiert.
