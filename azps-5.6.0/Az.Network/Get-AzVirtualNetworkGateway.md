# Get-AzVirtualNetworkGateway

## SYNOPSIS
Ruft ein Virtuelles Netzwerkgateway ab

## SYNTAX

```
Get-AzVirtualNetworkGateway [-Name <String>] -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das Virtual Network Gateway ist das Objekt, das Ihr Gateway in Azure darstellt.
Das **Get-AzVirtualNetworkGateway-Cmdlet** gibt das Objekt Ihres Gateways in Azure basierend auf Name und Ressourcengruppenname zurück.

## BEISPIELE

### Beispiel 1: Erhalten eines Virtuellen Netzwerkgateways
```powershell
Get-AzVirtualNetworkGateway -Name myGateway1 -ResourceGroupName myRG
```

Gibt das Objekt des Virtuellen Netzwerkgateways mit dem Namen "myGateway1" innerhalb der Ressourcengruppe "myRG" zurück.

### Beispiel 2: Erhalten eines Virtuellen Netzwerkgateways
```powershell
Get-AzVirtualNetworkGateway -Name myGateway* -ResourceGroupName myRG
```

Gibt alle Virtuellen Netzwerkgateways zurück, die mit "myGateway" innerhalb der Ressourcengruppe "myRG" beginnen.
