# Set-AzApplicationGatewayBackendAddressPool

## SYNOPSIS
Aktualisiert einen Back-End-Adresspool für ein Anwendungsgateway.

## SYNTAX

```
Set-AzApplicationGatewayBackendAddressPool -ApplicationGateway <PSApplicationGateway> -Name <String>
 [-BackendIPAddresses <String[]>] [-BackendFqdns <String[]>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Set-AzApplicationGatewayBackendAddressPool** aktualisiert einen Back-End-Adresspool für ein Azure-Anwendungsgateway.
Back-End-Adressen können als IP-Adressen, vollqualifizierte Domänennamen (FQDN) oder IP-Konfigurations-IDs angegeben werden.

## BEISPIELE

### Beispiel 1: Festlegen eines Back-End-Adresspools mithilfe von FQDNs
```
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewayBackendAddressPool -ApplicationGateway $AppGw -Name "Pool02" -BackendFqdns "contoso1.com", "contoso2.com"
```

Der erste Befehl ruft das Anwendungsgateway mit dem Namen ApplicationGateway01 in der Ressourcengruppe "ResourceGroup01" ab und speichert es in der $AppGw Variablen.
Der zweite Befehl aktualisiert den Back-End-Adresspool des Anwendungsgateways in $AppGw mithilfe von FQDNs.

### Beispiel 2: Festlegen eines Back-End-Adresspools mithilfe von Back-End-Server-IP-Adressen
```
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewayBackendAddressPool -ApplicationGateway $AppGw -Name "Pool02" -BackendIPAddresses "10.10.10.10", "10.10.10.11"
```

Der erste Befehl ruft das Anwendungsgateway mit dem Namen ApplicationGateway01 in der Ressourcengruppe "ResourceGroup01" ab und speichert es in der $AppGw Variablen.
Der zweite Befehl aktualisiert den Back-End-Adresspool des Anwendungsgateways in $AppGw mithilfe von IP-Adressen.
