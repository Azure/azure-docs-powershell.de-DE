# Remove-AzApplicationGatewayAuthenticationCertificate

## SYNOPSIS
Entfernt ein Authentifizierungszertifikat aus einem Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewayAuthenticationCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzApplicationGatewayAuthenticationCertificate** entfernt ein Authentifizierungszertifikat aus einem Azure-Anwendungsgateway.

## BEISPIELE

### Beispiel 1: Entfernen eines Authentifizierungszertifikats aus einem Anwendungsgateway
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $appgw = Remove-AzApplicationGatewayAuthenticationCertificate -ApplicationGateway $appgw -Name "cert01"
PS C:\> $appgw = Set-AzApplicationGateway -ApplicationGateway $appgw
```

Der erste Befehl ruft das Anwendungsgateway mit dem Namen "appGwName" ab und speichert das Ergebnis in der $appgw Variablen.
Mit dem zweiten Befehl wird das Authentifizierungszertifikat "cert01" aus dem Anwendungsgateway entfernt.
Mit dem dritten Befehl wird das Anwendungsgateway aktualisiert.
