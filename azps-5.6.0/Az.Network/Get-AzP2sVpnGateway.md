# Get-AzP2sVpnGateway

## SYNOPSIS
Ruft ein vorhandenes P2SVpnGateway unter VirtualHub ab.

## SYNTAX

### ListBySubscriptionId (Standard)
```
Get-AzP2sVpnGateway [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ListByResourceGroupName
```
Get-AzP2sVpnGateway [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Mit **dem Get-AzP2sVpnGateway-Cmdlet** können Sie ein vorhandenes P2SVpnGateway unter VirtualHub abrufen.
