# Set-AzExpressRoutePortIdentity

## SYNOPSIS
Aktualisiert eine identität, die einem ExpressRoutePort zugewiesen ist.

## SYNTAX

```
Set-AzExpressRoutePortIdentity -ExpressRoutePort <PSExpressRoutePort> -UserAssignedIdentityId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Set-AzExpressRoutePortIdentity** aktualisiert ein lokales Azure ExpressRoutePort-Objekt. Verwenden **Sie Set-AzExpressRoutePort,** um ihn ExpressRoutePort zuzuordnen.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\>$exrport = Get-AzExpressRoutePort -Name $portName -ResourceGroupName $rgName
PS C:\>$identity = New-AzUserAssignedIdentity -Name $identityName -ResourceGroupName $rgName -Location $location
PS C:\>$exrPortIdentity = Set-AzExpressRoutePortIdentity -UserAssignedIdentity $identity.Id -ExpressRoutePort $exrPort
PS C:\>$updatedExrPort = Set-AzExpressRoutePort -ExpressRoutePort $exrPort
```
