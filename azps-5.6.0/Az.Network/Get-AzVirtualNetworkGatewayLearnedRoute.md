# Get-AzVirtualNetworkGatewayLearnedRoute

## SYNOPSIS
Listet routen auf, die von einem virtuellen Azure-Netzwerkgateway gelernt wurden

## SYNTAX

```
Get-AzVirtualNetworkGatewayLearnedRoute -VirtualNetworkGatewayName <String> -ResourceGroupName <String>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Enumerates routes learned by a Azure virtual network gateway from various sources. Dazu gehören Routen, die über BGP gelernt wurden, sowie statische Routen. 

## BEISPIELE

### Beispiel 1
```
PS C:\> Get-AzVirtualNetworkGatewayLearnedRoute -ResourceGroupName resourceGroup -VirtualNetworkGatewayname gatewayName

AsPath       :
LocalAddress : 10.1.0.254
Network      : 10.1.0.0/16
NextHop      :
Origin       : Network
SourcePeer   : 10.1.0.254
Weight       : 32768

AsPath       :
LocalAddress : 10.1.0.254
Network      : 10.0.0.254/32
NextHop      :
Origin       : Network
SourcePeer   : 10.1.0.254
Weight       : 32768

AsPath       : 65515
LocalAddress : 10.1.0.254
Network      : 10.0.0.0/16
NextHop      : 10.0.0.254
Origin       : EBgp
SourcePeer   : 10.0.0.254
Weight       : 32768
```

Ruft für das azure virtual network gateway named gatewayname in resource group resourceGroup Routen ab, die dem Azure-Gateway bekannt sind. Das virtuelle Azure-Netzwerkgateway verfügt in diesem Fall über zwei statische Routen (10.1.0.0/16 und 10.0.0.254/32) sowie eine Route, die über BGP (10.0.0.0/16) gelernt wurde.
