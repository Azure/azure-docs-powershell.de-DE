# Remove-AzApplicationGatewayBackendHttpSetting

## SYNOPSIS
Entfernt die Back-End-HTTP-Einstellungen aus einem Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewayBackendHttpSetting -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das Remove-AzApplicationGatewayBackendHttpSetting-Cmdlet entfernt die Back-End-Einstellungen des Hypertext Transfer Protocol (HTTP) aus einem Azure-Anwendungsgateway.

## BEISPIELE

### Beispiel 1: Entfernen von Back-End-HTTP-Einstellungen aus einem Anwendungsgateway
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayBackendHttpSetting -ApplicationGateway $AppGw -Name "BackEndSetting02"
```

Der erste Befehl ruft ein Anwendungsgateway mit dem Namen ApplicationGateway01 ab, das zur Ressourcengruppe "ResourceGroup01" gehört und es in der Variablen $AppGw speichert.
Mit dem zweiten Befehl wird die Back-End-HTTP-Einstellung mit dem Namen BackEndSetting02 aus dem anwendungsgateway entfernt, das in $AppGw.
