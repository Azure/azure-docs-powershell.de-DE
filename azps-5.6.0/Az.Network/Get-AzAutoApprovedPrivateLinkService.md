# Get-AzAutoApprovedPrivateLinkService

## SYNOPSIS
Ruft ein Array privater Linkdienst-ID ab, das mit einem privaten Endpunkt verknüpft werden kann, der automatisch genehmigt wurde.

## SYNTAX

```
Get-AzAutoApprovedPrivateLinkService -Location <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Der **Get-AzAutoApprovedPrivateLinkService** ruft ein Array privater Linkdienst-ID ab, das mit einem privaten Endpunkt verknüpft werden kann, der automatisch genehmigt wurde.

## BEISPIELE

### Beispiel
```
Get-AzAutoApprovedPrivateLinkService -Location westus -ResourceGroupName TestResourceGroup
```

In diesem Beispiel wird ein Array privater Linkdienst-ID zurückgeben, das mit einem privaten Endpunkt verknüpft werden kann, bei dem die automatische Genehmigung genehmigt wurde.
