# Remove-AzApplicationSecurityGroup

## SYNOPSIS
Entfernt eine Anwendungssicherheitsgruppe.

## SYNTAX

```
Remove-AzApplicationSecurityGroup -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzApplicationSecurityGroup** entfernt eine Anwendungssicherheitsgruppe.

## BEISPIELE

### Beispiel 1
```
PS C:\>Remove-AzApplicationSecurityGroup -Name "MyApplicationSecurityGrouo" -ResourceGroupName "MyResourceGroup"
```

Mit diesem Befehl wird eine Anwendungssicherheitsgruppe mit dem Namen MyApplicationSecurityGrouo in der Ressourcengruppe "MyResourceGroup" gelöscht.
