# Clear-AzContext

## SYNOPSIS
Entfernen Sie alle Azure-Anmeldeinformationen, Konto- und Abonnementinformationen.

## SYNTAX

```
Clear-AzContext [-PassThru] [-Force] [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Entfernen Sie alle Azure-Anmeldeinformationen, Konto- und Abonnementinformationen.

## BEISPIELE

### Beispiel 1: Löschen des globalen Kontexts
```powershell
PS C:\> Clear-AzContext -Scope CurrentUser
```

Entfernen Sie alle Konto-, Abonnement- und Anmeldeinformationen für eine beliebige Powershellsitzung.
