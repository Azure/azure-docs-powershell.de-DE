# Get-AzEnvironment

## SYNOPSIS
Holen Sie sich Endpunkte und Metadaten für eine Instanz von Azure-Diensten.

## SYNTAX

```
Get-AzEnvironment [[-Name] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das Get-AzEnvironment-Cmdlet ruft Endpunkte und Metadaten für eine Instanz von Azure-Diensten ab.

## BEISPIELE

### Beispiel 1: Abrufen der AzureCloud-Umgebung
```
PS C:\> Get-AzEnvironment AzureCloud

Name       Resource Manager Url          ActiveDirectory Authority
----       --------------------          -------------------------
AzureCloud https://management.azure.com/ https://login.microsoftonline.com/
```

In diesem Beispiel wird gezeigt, wie Sie die Endpunkte und Metadaten für die AzureCloud-Umgebung (Standardumgebung) erhalten.
