# Get-AzIotHubValidSku

## SYNOPSIS
Ruft alle gültigen Skus ab, zu der dieser IotHub überwechseln kann.

## SYNTAX

```
Get-AzIotHubValidSku [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Ruft alle gültigen Skus ab, zu der dieser IotHub überwechseln kann.
Ein IotHub kann nicht zwischen dem kostenlosen und dem kostenpflichtigen Skus wechseln und umgekehrt. Wenn Sie dies erreichen möchten, müssen Sie den Iothub löschen und neu erstellen.

## BEISPIELE

### Beispiel 1 : Gültiges Skus
```
PS C:\> Get-AzIotHubValidSku -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Ruft eine Liste aller Skus für den IotHub mit dem Namen "myiothub" ab.
