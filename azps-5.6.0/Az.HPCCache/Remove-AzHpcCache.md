# Remove-AzHpcCache

## SYNOPSIS
Entfernt einen HPC-Cache.

## SYNTAX

```
Remove-AzHpcCache -ResourceGroupName <String> -Name <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzHpcCache** entfernt einen Azure HPC-Cache.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Remove-AzHpcCache -ResourceGroupName testRG -CacheName testCache
```
