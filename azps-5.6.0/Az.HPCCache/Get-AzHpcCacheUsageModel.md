# Get-AzHpcCacheUsageModel

## SYNOPSIS
Ruft alle usageModels für DAS NFS-Speicherziel ab.

## SYNTAX

```
Get-AzHpcCacheUsageModel [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzHpcCacheUsageModel-Cmdlet** gibt eine Liste der Verwendungsmodelle für DAS NFS-Speicherziel zurück.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Get-AzHpcCacheUsageModel
```
