# Get-AzHpcCacheSku

## SYNOPSIS
Ruft alle im Abonnement verfügbaren SKUs ab.

## SYNTAX

```
Get-AzHpcCacheSku [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzHpcCacheSku-Cmdlet** gibt eine Liste der skUs zurück, die im Abonnement verfügbar sind.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Get-AzHpcCacheSku
```
