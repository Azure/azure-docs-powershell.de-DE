# Remove-AzServiceFabricClusterCertificate

## SYNOPSIS
Entfernen Sie ein Clusterzertifikat aus der Verwendung für die Clustersicherheit.

## SYNTAX

```
Remove-AzServiceFabricClusterCertificate -Thumbprint <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Verwenden **Sie Remove-AzServiceFabricClusterCertificate,** um ein Clusterzertifikat aus dem Cluster zu entfernen, solange ein weiteres gültiges Zertifikat vorhanden ist, das bereits im Cluster verwendet wird.

## BEISPIELE

### Beispiel 1
```
PS C:\> Remove-AzServiceFabricClusterCertificate -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -Thumbprint '5F3660C715EBBDA31DB1FFDCF508302348DE8E7A
```

Mit diesem Befehl wird das Zertifikat mit dem Fingerabdruck 5F3660C715EBBDA31DB1FFDCF508302348DE8E7A für die Clustersicherheit entfernt.
