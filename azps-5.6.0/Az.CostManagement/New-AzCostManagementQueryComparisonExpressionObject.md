# New-AzCostManagementQueryComparisonExpressionObject

## SYNOPSIS
Erstellen eines Speicherobjekts für QueryComparisonExpression

## SYNTAX

```
New-AzCostManagementQueryComparisonExpressionObject -Name <String> -Operator <OperatorType> -Value <String[]>
 [<CommonParameters>]
```

## BESCHREIBUNG
Erstellen eines Speicherobjekts für QueryComparisonExpression

## BEISPIELE

### Beispiel 1: Erstellen eines Vergleichsausdruckobjekts der Abfrage für den Kostenverwaltungsexport
```powershell
PS C:\> New-AzCostManagementQueryComparisonExpressionObject -Name 'ResourceLocation' -Value @('East US', 'West Europe')

Name             Operator Value
----             -------- -----
ResourceLocation In       {East US, West Europe}
```

Mit diesem Befehl wird ein Vergleichsausdruckobjekt der Abfrage für den Kostenverwaltungsexport erstellt.
