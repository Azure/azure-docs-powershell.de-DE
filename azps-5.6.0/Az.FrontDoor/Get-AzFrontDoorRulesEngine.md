# Get-AzFrontDoorRulesEngine

## SYNOPSIS
Konfigurationen des Regelmoduls.

## SYNTAX

```
Get-AzFrontDoorRulesEngine -ResourceGroupName <String> -FrontDoorName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzFrontDoorRulesEngine-Cmdlet** ruft eine bestimmte Konfiguration des Regelmoduls ab oder ruft alle Konfigurationen des Regelmoduls ab, die mit einer Front Door verknüpft sind. 

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Get-AzFrontDoorRulesEngine -ResourceGroupName $resourceGroupName -FrontDoorName $frontDoorName -Name rulesengine3

Name         RulesEngineRules
----         ----------------
rulesEngine3 {rules1}
```

Holen Sie sich eine bestimmte Konfiguration des Regelmoduls.

### Beispiel 2

```powershell
PS C:\> Get-AzFrontDoorRulesEngine -ResourceGroupName $resourceGroupName -FrontDoorName $frontDoorName

Name         RulesEngineRules
----         ----------------
rulesEngine1 {Rule1}
rulesEngine2 {Rule1}
rulesEngine3 {rules1}
```

Holen Sie sich alle Konfigurationen des Regelmoduls in einer Eingangstür.

### Beispiel 3

```powershell
PS C:\> Get-AzFrontDoorRulesEngine -ResourceGroupName $resourceGroupName -FrontDoorName $frontDoorName -Name nonexistent
Get-AzFrontDoorRulesEngine : Rules Engine with name 'nonexistent' in Front Door 'frontDoorName' is not found.
```
