# New-AzFrontDoorWafManagedRuleOverrideObject

## SYNOPSIS
Erstellen eines Überschreibungsobjekts für verwaltete Regeln

## SYNTAX

```
New-AzFrontDoorWafManagedRuleOverrideObject -RuleId <String> [-Action <String>] [-Disabled]
 [-Exclusion <PSManagedRuleExclusion[]>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Erstellen Sie PSAzureManagedRuleOverride-Objekt für verwaltete WAF-Regelgruppenüberschreiben der Objekterstellung.

## BEISPIELE

### Beispiel 1
Erstellen Sie ein verwaltetes Regelüberschreibungsobjekt für Regel 942250 (das sich in der SQLI-Gruppe befindet).

```powershell
PS C:\> New-AzFrontDoorWafManagedRuleOverrideObject -RuleId "942250" -Action Log

RuleId EnabledState Action
------ ------------ ------
942250      Enabled    Log
```
