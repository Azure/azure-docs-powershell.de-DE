# Get-AzEnrollmentAccount

## SYNOPSIS
Sie können Registrierungskonten erhalten.

## SYNTAX

### Liste (Standard)
```
Get-AzEnrollmentAccount [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Single
```
Get-AzEnrollmentAccount [-ObjectId] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzEnrollmentAccount-Cmdlet** ruft Registrierungskonten ab.

## BEISPIELE

### Beispiel 1
```
PS C:\> Get-AzEnrollmentAccount

ObjectId                             PrincipalName
--------                             -------------
dbd8453d-071f-4fb4-8e01-c99f5b067649 jason@contoso.onmicrosoft.com
7ff524ac-a0de-4402-876f-934ccee3b601 carol@contoso.onmicrosoft.com
```

Alle verfügbaren Registrierungskonten erhalten.

### Beispiel 2
```
PS C:\> Get-AzEnrollmentAccount -ObjectId dbd8453d-071f-4fb4-8e01-c99f5b067649

ObjectId                             PrincipalName
--------                             -------------
dbd8453d-071f-4fb4-8e01-c99f5b067649 jason@contoso.onmicrosoft.com
```

Holen Sie sich das Registrierungskonto mit der angegebenen Objekt-ID.
