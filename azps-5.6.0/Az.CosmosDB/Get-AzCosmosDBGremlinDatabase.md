# Get-AzCosmosDBGremlinDatabase

## SYNOPSIS
Ruft die CosmosDB-Gremlin-Datenbank ab.

## SYNTAX

### ByNameParameterSet (Standard)
```
Get-AzCosmosDBGremlinDatabase -ResourceGroupName <String> -AccountName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Get-AzCosmosDBGremlinDatabase [-Name <String>] -ParentObject <PSDatabaseAccountGetResults>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzCosmosDBGremlinDatabase-Cmdlet** erhält die CosmosDB-Gremlin-Datenbank.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Get-AzCosmosDBGremlinDatabase -ResourceGroupName {rgName} -AccountName {accountName} -Name {databaseName}

Name    Id   Resource
{name}  {id} Microsoft.Azure.Commands.CosmosDB.Models.PSGremlinDatabaseGetPropertiesResource
```

Ressourcenobjekt enthält _rid, _ts, _etag.
