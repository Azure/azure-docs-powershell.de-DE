# New-AzCosmosDBSqlUniqueKey

## SYNOPSIS
Erstellt ein neues CosmosDB Sql UniqueKey-Objekt.

## SYNTAX

```
New-AzCosmosDBSqlUniqueKey -Path <String[]> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet New-AzCosmosDBSqlUniqueKey** erstellt ein neues Objekt vom Typ PSUniqueKey.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> New-AzCosmosDBSqlUniqueKey -Path {path}

Path
----
{path}
```
