# New-AzCosmosDBCassandraColumn

## SYNOPSIS
Erstellt eine neue Spalte "CosmosDB-Kassandra".

## SYNTAX

```
New-AzCosmosDBCassandraColumn -Name <String> -Type <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Die **New-AzCosmosDBCassandraColumn** erstellt eine neue Spalte "CosmosDB-Kassandra".

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> New-AzCosmosDBCassandraColumn -Name "name" -Type int

Name Type
---- ----
name int
```
