# Get-AzCosmosDBSqlContainer

## SYNOPSIS
Ruft den CosmosDB Sql Container ab.

## SYNTAX

### ByNameParameterSet (Standard)
```
Get-AzCosmosDBSqlContainer -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] -AccountName <String> -DatabaseName <String> [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Get-AzCosmosDBSqlContainer [-Name <String>] -ParentObject <PSSqlDatabaseGetResults>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzCosmosDBSqlContainer-Cmdlet** ruft die Liste aller vorhandenen CosmosDB-Sql-Container für einen bestimmten ResourceGroupName, AccountName und DatabaseName ab und ruft einen einzelnen CosmosDB Sql Container für einen bestimmten ResourceGroupName, AccountName, DatabaseName und ContainerName ab.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Get-AzCosmosDBSqlContainer -AccountName {accountName} -ResourceGroupName {resourceGroupName} -DatabaseName {databaseName}

Name                     : {containerName1}
Id                       : Id
Resource                 : Microsoft.Azure.Commands.CosmosDB.Models.PSSqlContainerGetPropertiesResource
```
