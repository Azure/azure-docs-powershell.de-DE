# New-AzCosmosDBCassandraClusterKey

## SYNOPSIS
Erstellt einen neuen CosmosDB-Kassandra-Clusterschlüssel.

## SYNTAX

```
New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Der **New-AzCosmosDBCassandraClusterKey** erstellt einen neuen CosmosDB-Kassandra-Clusterschlüssel.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String>

Name   OrderBy
----   -------
{name}  Asc
```
