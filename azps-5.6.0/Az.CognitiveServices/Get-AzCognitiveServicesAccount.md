# Get-AzCognitiveServicesAccount

## SYNOPSIS
Ruft ein Konto ab.

## SYNTAX

### ResourceGroupParameterSet
```
Get-AzCognitiveServicesAccount [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### AccountNameParameterSet
```
Get-AzCognitiveServicesAccount [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzCognitiveServicesAccount-Cmdlet** ruft die bereitgestellten Cognitive Services-Konten in der Ressourcengruppe ab, die durch den *Parameter ResourceGroupName angegeben* wird.
Wenn Sie den Parameter *ResourceGroupName* nicht angeben, ruft dieses Cmdlet alle Cognitive Services-Konten f체r das aktuelle Abonnement ab.
