# Get-AzDataShareAccount

## SYNOPSIS
Ruft Informationen zu DataShare-Konten ab

## SYNTAX

### ByFieldsParameterSet (Standard)
```
Get-AzDataShareAccount [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceGroupParameterSet
```
Get-AzDataShareAccount -ResourceGroupName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Get-AzDataShareAccount -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzDataShareAccount-Cmdlet** ruft Informationen zu Datenfreigabekonten in einer Azure-Abonnement-/Ressourcengruppe ab.
Wenn Sie den Namen eines Kontos angeben, erh채lt dieses Cmdlet Informationen zu diesem datshare-Konto.
Wenn Sie keinen Namen angeben, ruft dieses Cmdlet Informationen zu allen Datenfreigabekonten in einer Azure-Abonnement-/Ressourcengruppe ab.
