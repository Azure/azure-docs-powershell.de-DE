# Get-AzReservationHistory

## SYNOPSIS
Erhalten `Reservation` Sie den Überarbeitungsverlauf.

## SYNTAX

### Befehlszeile (Standard)
```
Get-AzReservationHistory -ReservationOrderId <Guid> -ReservationId <Guid>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### PipeObject
```
Get-AzReservationHistory -Reservation <PSReservation> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Liste aller Überarbeitungen für `Reservation` .

## BEISPIELE

### Beispiel 1
```
PS C:\> Get-AzReservationHistory -ReservationOrderId "1111aaaa-b1b2-c0c2-d0d2-00000fffff" -ReservationId "00000000-ffff-ffff-0000-00000fffff"
```

Erhalten des Überarbeitungsverlaufs der bestimmten Reservierung
