# Save-AzApiManagementTenantGitConfiguration

## SYNOPSIS
Speichert Änderungen, indem ein Commit für die aktuelle Konfiguration erstellt wird.

## SYNTAX

```
Save-AzApiManagementTenantGitConfiguration -Context <PsApiManagementContext> -Branch <String> [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Save-AzApiManagementTenantGitConfiguration** speichert die Änderungen, indem ein Commit erstellt wird, der die aktuelle Konfigurationsmomentaufnahme in einer Verzweigung im Repository enthält.

## BEISPIELE

### Beispiel 1: Speichern von Änderungen an der Konfiguration
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Save-AzApiManagementTenantGitConfiguration -Context $apimContext -Branch 'master' -PassThru
```

Mit diesem Befehl werden die Änderungen gespeichert, indem ein Commit mit der aktuellen Konfigurationsmomentaufnahme für die angegebene Verzweigung im Repository erstellt wird.
