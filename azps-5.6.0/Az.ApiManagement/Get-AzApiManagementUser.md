# Get-AzApiManagementUser

## SYNOPSIS
Ruft einen Benutzer oder Benutzer ab.

## SYNTAX

### GeAllUsers (Standard)
```
Get-AzApiManagementUser -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByUserId
```
Get-AzApiManagementUser -Context <PsApiManagementContext> [-UserId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByUser
```
Get-AzApiManagementUser -Context <PsApiManagementContext> [-FirstName <String>] [-LastName <String>]
 [-State <PsApiManagementUserState>] [-Email <String>] [-GroupId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzApiManagementUser-Cmdlet** ruft einen angegebenen Benutzer oder alle Benutzer ab, wenn kein Benutzer angegeben ist.

## BEISPIELE

### Beispiel 1: Alle Benutzer erhalten
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementUser -Context $apimContext
```

Dieser Befehl ruft alle Benutzer ab.

### Beispiel 2: Benutzer nach ID erhalten
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementUser -Context $apimContext -UserId "0123456789"
```

Dieser Befehl ruft einen Benutzer nach ID ab.

### Beispiel 3: Benutzer nach Nachnamen holen
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementUser -Context $apimContext -LastName "Fuller"
```

Dieser Befehl ruft Benutzer ab, die einen angegebenen Nachnamen haben, "Fuller".

### Beispiel 4: Benutzer per E-Mail-Adresse erhalten
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementUser -Context $apimContext -Email "user@contoso.com"
```

Dieser Befehl ruft den Benutzer ab, der über die angegebene E-Mail-Adresse verfügt.

### Beispiel 5: Alle Benutzer innerhalb einer Gruppe erhalten
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementUser -Context $apimContext -GroupId "0001"
```

Dieser Befehl ruft alle Benutzer innerhalb der angegebenen Gruppe ab.
