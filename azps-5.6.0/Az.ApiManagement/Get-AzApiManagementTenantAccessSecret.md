# Get-AzApiManagementTenantAccessSecret

## SYNOPSIS
Ruft die Zugriffskonfigurationsschlüssel für einen Mandanten ab.

## SYNTAX

```
Get-AzApiManagementTenantAccessSecret -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Ruft die Zugriffskonfigurationsschlüssel für einen Mandanten ab.

## BEISPIELE

### Beispiel 1: Erhalten von Konfigurationsschlüsseln für den Mandantenzugriff
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantAccessSecret -Context $apimContext
```

Dieser Befehl ruft die Konfigurationsschlüssel für den Mandantenzugriff für den angegebenen Kontext ab.
