# Remove-AzApiManagementAuthorizationServer

## SYNOPSIS
Entfernt einen Autorisierungsserver.

## SYNTAX

```
Remove-AzApiManagementAuthorizationServer -Context <PsApiManagementContext> -ServerId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzApiManagementAuthorizationServer** entfernt einen Azure API Management-Autorisierungsserver.

## BEISPIELE

### Beispiel 1: Entfernen eines Autorisierungsservers
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementAuthorizationServer -Context $ApiMgmtContext -ServerId "authserverid" -Force
```

Mit diesem Befehl wird der angegebene API-Verwaltungsautorisierungsserver entfernt.
Da der *Parameter Erzwingen* angegeben ist, ist keine Bestätigung erforderlich.
