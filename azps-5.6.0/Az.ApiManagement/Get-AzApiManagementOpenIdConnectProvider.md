# Get-AzApiManagementOpenIdConnectProvider

## SYNOPSIS
Ruft OpenID Connect-Anbieter ab.

## SYNTAX

### GetAllOpenIdConnectProviders (Standard)
```
Get-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByOpenIdConnectProviderId
```
Get-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext> [-OpenIdConnectProviderId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByName
```
Get-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzApiManagementOpenIdConnectProvider-Cmdlet** ruft OpenID Connect-Anbieter in Azure API Management ab.
ClientSecret wird nicht in ergebnisdetails einbezogen. Verwenden Sie **Get-AzApiManagementOpenIdConnectProviderClientSecret,** um den Geheimtipp des Clients zu erhalten.

## BEISPIELE

### Beispiel 1: Alle Anbieter erhalten
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOpenIdConnectProvider -Context $apimContext
```

Dieser Befehl ruft alle OpenID Connect-Anbieter für den angegebenen Kontext ab.

### Beispiel 2: Erstellen eines Anbieters mithilfe einer ID
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOpenIdConnectProvider -Context $apimContext -OpenIdConnectProviderId "OICProvider01"
```

Dieser Befehl ruft den Anbieter ab, der über die ID OICProvider01 verfügt.

### Beispiel 3: Erhalten eines Anbieters mithilfe eines Namens
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOpenIdConnectProvider -Context $apimContext -Name "Contoso OpenID Connect Provider"
```

Dieser Befehl ruft den Anbieter namens Contoso OpenID Connect Provider ab.
