# Remove-AzApiManagementLogger

## SYNOPSIS
Entfernt einen API-Verwaltungsprotokollger.

## SYNTAX

```
Remove-AzApiManagementLogger -Context <PsApiManagementContext> -LoggerId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzApiManagementLogger** entfernt einen Azure API **Management Logger**.

## BEISPIELE

### Beispiel 1: Entfernen eines Loggers
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementLogger -Context $apimContext -LoggerId "Logger123" -Force
```

Mit diesem Befehl wird ein Logger entfernt, der über die ID Logger123 verfügt.
