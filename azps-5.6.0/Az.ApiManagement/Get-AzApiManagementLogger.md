# Get-AzApiManagementLogger

## SYNOPSIS
Ruft API-Verwaltungsprotokollierungsobjekte ab.

## SYNTAX

### GetAllLoggers (Standard)
```
Get-AzApiManagementLogger -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByLoggerId
```
Get-AzApiManagementLogger -Context <PsApiManagementContext> -LoggerId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzApiManagementLogger-Cmdlet** ruft einen Azure API-Verwaltungslogger oder alle **Logger** ab.

## BEISPIELE

### Beispiel 1: Alle Protokollierungsdateien erhalten
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementLogger -Context $apimContext
```

Dieser Befehl ruft alle Logger für den angegebenen Kontext ab.

### Beispiel 2: Erhalten eines bestimmten Loggers
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementLogger -Context $apimContext -LoggerId "Logger123"
```

Mit diesem Befehl wird ein Logger entfernt, der über die ID Logger123 verfügt.
