# Get-AzApiManagementGatewayKey

## SYNOPSIS
Ruft Schlüssel des vorhandenen Gateways ab

## SYNTAX

```
Get-AzApiManagementGatewayKey -Context <PsApiManagementContext> -GatewayId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzApiManagementGatewayKey-Cmdlet** ruft Schlüssel des vorhandenen Gateways ab.

## BEISPIELE

### Beispiel 2: Erhalten eines Gateways nach ID
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementGatewayKey -Context $apimContext -GatewayId "0123456789"
```

Dieser Befehl ruft die Schlüssel für ein "0123456789"-Gateway ab.
