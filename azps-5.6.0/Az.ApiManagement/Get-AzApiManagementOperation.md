# Get-AzApiManagementOperation

## SYNOPSIS
Ruft eine Liste oder einen angegebenen API-Vorgang ab.

## SYNTAX

### GetAllApiOperations (Standard)
```
Get-AzApiManagementOperation -Context <PsApiManagementContext> -ApiId <String> [-ApiRevision <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetById
```
Get-AzApiManagementOperation -Context <PsApiManagementContext> -ApiId <String> [-ApiRevision <String>]
 -OperationId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Die **Get-AzApiManagementOperation** ruft eine Liste oder einen angegebenen API-Vorgang ab.

## BEISPIELE

### Beispiel 1: Abrufen aller API-Verwaltungsvorgänge
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOperation -Context $apimContext -ApiId $APIId
```

Dieser Befehl ruft alle API-Verwaltungsvorgänge ab.

### Beispiel 2: Abrufen eines API-Verwaltungsvorgangs nach Vorgangs-ID
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOperation -Context $apimContext -ApiId $APIId -OperationId "Operation003"
```

Dieser Befehl ruft einen API-Verwaltungsvorgang nach Vorgangs-ID mit dem Namen Operation0003 ab.
