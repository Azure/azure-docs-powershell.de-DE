# Get-AzApiManagementTenantGitAccess

## SYNOPSIS
Ruft die Git-Zugriffskonfiguration für einen Mandanten ab.

## SYNTAX

```
Get-AzApiManagementTenantGitAccess -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Get-AzApiManagementTenantGitAccess** ruft die Git-Zugriffskonfiguration für einen Mandanten ab.
Schlüssel werden nicht in ergebnisdetails einbezogen. Verwenden Sie **Get-AzApiManagementTenantGitAccessSecret,** um den Geheimtipp des Clients zu erhalten.

## BEISPIELE

### Beispiel 1: Konfiguration des Mandantenzugriffs
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantGitAccess -Context $apimContext
```
