# Get-AzApiManagementUserSsoUrl

## SYNOPSIS
Generiert eine SSO-URL für einen Benutzer.

## SYNTAX

```
Get-AzApiManagementUserSsoUrl -Context <PsApiManagementContext> -UserId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Get-AzApiManagementUserSsoUrl** generiert eine SSO-URL (Single Sign-On) für einen Benutzer.

## BEISPIELE

### Beispiel 1: Erhalten der SSO-URL eines Benutzers
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementUserSsoUrl -Context $apimContext -UserId "0123456789"
```

Dieser Befehl ruft die SSO-URL eines Benutzers ab.
