# Set-AzOperationalInsightsDataSource

## SYNOPSIS
Aktualisiert eine Datenquelle.

## SYNTAX

```
Set-AzOperationalInsightsDataSource [-DataSource] <PSDataSource> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Set-AzOperationalInsightsDataSource aktualisiert** eine Datenquelle.

## BEISPIELE
