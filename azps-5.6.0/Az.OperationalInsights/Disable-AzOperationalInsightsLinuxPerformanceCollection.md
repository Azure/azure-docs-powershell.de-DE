# Disable-AzOperationalInsightsLinuxPerformanceCollection

## SYNOPSIS
Beendet die Sammlung von Leistungsindikatoren von Linux-Computern.

## SYNTAX

### ByWorkspaceName (Standard)
```
Disable-AzOperationalInsightsLinuxPerformanceCollection [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByWorkspaceObject
```
Disable-AzOperationalInsightsLinuxPerformanceCollection [-Workspace] <PSWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Disable-AzOperationalInsightsLinuxPerformanceCollection** beendet die Sammlung von Leistungsindikatoren von verbundenen Linux-Computern in einem Arbeitsbereich.
