# Move-AzResource

## SYNOPSIS
Verschiebt eine Ressource in eine andere Ressourcengruppe oder ein anderes Abonnement.

## SYNTAX

```
Move-AzResource -DestinationResourceGroupName <String> [-DestinationSubscriptionId <Guid>]
 -ResourceId <String[]> [-Force] [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Move-AzResource** verschiebt vorhandene Ressourcen in eine andere Ressourcengruppe.
Diese Ressourcengruppe kann in einem anderen Abonnement enthalten sein.

## BEISPIELE

### Beispiel 1: Verschieben einer Ressource in eine Ressourcengruppe
```
PS C:\>$Resource = Get-AzResource -ResourceType "Microsoft.ClassicCompute/storageAccounts" -ResourceName "ContosoStorageAccount"
PS C:\> Move-AzResource -ResourceId $Resource.ResourceId -DestinationResourceGroupName "ResourceGroup14"
```

Der erste Befehl ruft eine Ressource mit dem Namen ContosoStorageAccount mithilfe des cmdlets Get-AzResource ab und speichert diese Ressource dann in der $Resource Variablen.
Der zweite Befehl verschiebt diese Ressource in die Ressourcengruppe "ResourceGroup14".
Der Befehl identifiziert die zu verschiebende Ressource mithilfe der **ResourceId-Eigenschaft** $Resource.
