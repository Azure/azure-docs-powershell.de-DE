# Remove-AzResourceGroupDeployment

## SYNOPSIS
Entfernt eine Ressourcengruppenbereitstellung und alle zugehörigen Vorgänge.

## SYNTAX

### RemoveByResourceGroupName (Standard)
```
Remove-AzResourceGroupDeployment [-ResourceGroupName] <String> [-Name] <String> [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByResourceGroupDeploymentId
```
Remove-AzResourceGroupDeployment -Id <String> [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG

Das **Cmdlet Remove-AzResourceGroupDeployment** entfernt eine Azure-Ressourcengruppenbereitstellung und alle zugehörigen Vorgänge.

## BEISPIELE

### Beispiel 1: Entfernt eine Ressourcengruppenbereitstellung mit ResourceId

```powershell
PS C:\>Remove-AzResourceGroupDeployment -ResourceId /subscriptions/{subId}/resourceGroups/testGroup/providers/Microsoft.Resources/deployments/testDeployment1

True
```

Mit diesem Befehl wird eine Ressourcengruppenbereitstellung mit der vollqualifizierten Ressourcen-ID der Bereitstellung entfernt.
Das erfolgreiche Entfernen gibt "true" zurück.

### Beispiel 2: Entfernt eine Ressourcengruppenbereitstellung mit ResourceGroupName und ResourceName

```powershell
PS C:\>Remove-AzResourceGroupDeployment -ResourceGroupName testGroup -Name testDeployment1

True
```

Mit diesem Befehl wird eine Ressourcengruppenbereitstellung mit dem bereitgestellten ResourceGroupName und ResourceName entfernt.
Das erfolgreiche Entfernen gibt "true" zurück.
