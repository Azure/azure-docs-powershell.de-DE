# Remove-AzADUser

## SYNOPSIS
Löscht einen Active Directory-Benutzer.

## SYNTAX

### UPNOrObjectIdParameterSet (Standard)
```
Remove-AzADUser -UPNOrObjectId <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### UPNParameterSet
```
Remove-AzADUser -UserPrincipalName <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ObjectIdParameterSet
```
Remove-AzADUser -ObjectId <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### DisplayNameParameterSet
```
Remove-AzADUser -DisplayName <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzADUser -InputObject <PSADUser> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Löscht einen Active Directory-Benutzer (Arbeits-/Schulkonto, auch bekannt als Organisations-ID).

## BEISPIELE

### Beispiel 1: Entfernen eines Benutzers nach Dem Prinzipalnamen

```powershell
PS C:\> Remove-AzADUser -UserPrincipalName foo@domain.com
```

Entfernt den Benutzer mit dem Benutzernamen " foo@domain.com " aus dem Mandanten.

### Beispiel 2: Entfernen eines Benutzers nach Objekt-ID

```powershell
PS C:\> Remove-AzADUser -ObjectId 7a9582cf-88c4-4319-842b-7a5d60967a69
```

Entfernt den Benutzer mit der Objekt-ID '7a9582cf-88c4-4319-842b-7a5d60967a69' aus dem Mandanten.

### Beispiel 3: Entfernen eines Benutzers durch Piping

```powershell
PS C:\> Get-AzADUser -ObjectId 7a9582cf-88c4-4319-842b-7a5d60967a69 | Remove-AzADUser
```

Ruft den Benutzer mit der Objekt-ID '7a9582cf-88c4-4319-842b-7a5d60967a69' ab und gibt diese an das Remove-AzADUser-Cmdlet weiter, um den Benutzer aus dem Mandanten zu entfernen.
