# Unregister-AzProviderFeature

## SYNOPSIS
Aufheben der Registrierung eines Azure-Anbieterfeatures in Ihrem Konto

## SYNTAX

```
Unregister-AzProviderFeature -FeatureName <String> -ProviderNamespace <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Mit **dem Cmdlet "Unregister-AzProviderFeature"** wird die Registrierung eines Azure-Anbieterfeatures in Ihrem Konto aufgehoben.

## BEISPIELE

### Beispiel 1: Aufheben der Registrierung eines Features
```
PS C:\>Unregister-AzProviderFeature -FeatureName AllowApplicationSecurityGroups -ProviderNamespace Microsoft.Network
```

Dadurch wird die Registrierung des Features AllowApplicationSecurityGroups für Microsoft.Network von Ihrem Konto aufgehoben.
