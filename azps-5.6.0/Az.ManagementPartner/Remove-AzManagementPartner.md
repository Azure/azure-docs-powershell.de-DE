# Remove-AzManagementPartner

## SYNOPSIS
Löschen Sie die Microsoft Partner Network(MPN)-ID des aktuellen authentifizierten Benutzers oder Dienstprinzipals.

## SYNTAX

```
Remove-AzManagementPartner [-PartnerId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Löschen Sie die Microsoft Partner Network(MPN)-ID des aktuellen authentifizierten Benutzers oder Dienstprinzipals.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\>Remove-AzManagementPartner -PartnerId 123457 -PassThru
true
```

Entfernen eines Verwaltungspartners
