# Set-AzSynapsePipeline

## SYNOPSIS
Erstellt eine Pipeline im Arbeitsbereich.

## SYNTAX

### SetByName (Standard)
```
Set-AzSynapsePipeline -WorkspaceName <String> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByObject
```
Set-AzSynapsePipeline -WorkspaceObject <PSSynapseWorkspace> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Set-AzSynapsePipeline** erstellt eine Pipeline im Arbeitsbereich.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Set-AzSynapsePipeline -WorkspaceName ContosoWorkspace -Name ContosoPipeline -DefinitionFile "C:\pipeline.json"
```

Mit diesem Befehl wird eine Pipeline namens ContosoPipeline im Arbeitsbereich namens ContosoWorkspace erstellt.
Der Befehl basiert auf Informationen in pipeline.jsDatei.
Diese Datei enthält Informationen zu Aktivitäten.

### Beispiel 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Set-AzSynapsePipeline -Name ContosoPipeline -DefinitionFile "C:\pipeline.json"
```

Mit diesem Befehl wird eine Pipeline mit dem Namen ContosoPipeline im Arbeitsbereich namens ContosoWorkspace über pipeline erstellt.
Der Befehl basiert auf Informationen in pipeline.jsDatei.
Diese Datei enthält Informationen zu Aktivitäten.
