# Test-AzSynapseWorkspace

## SYNOPSIS
Überprüft, ob ein Synapse Analytics-Arbeitsbereich vorhanden ist.

## SYNTAX

```
Test-AzSynapseWorkspace [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Test-AzSynapseWorkspace** überprüft, ob ein Synapse Analytics-Arbeitsbereich vorhanden ist.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Test-AzSynapseWorkspace -Name ContosoWorkspace
```

Mit diesem Befehl wird überprüft, ob ein Synapse Analytics-Arbeitsbereich vorhanden ist.
