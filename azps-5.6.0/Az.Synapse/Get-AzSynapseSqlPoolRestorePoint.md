# Get-AzSynapseSqlPoolRestorePoint

## SYNOPSIS
Ruft die eindeutigen Wiederherstellungspunkte ab, aus denen ein Synapse Analytics-SQL wiederhergestellt werden kann.

## SYNTAX

### GetByNameParameterSet (Standard)
```
Get-AzSynapseSqlPoolRestorePoint [-ResourceGroupName <String>] -WorkspaceName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByParentObjectParameterSet
```
Get-AzSynapseSqlPoolRestorePoint -Name <String> -WorkspaceObject <PSSynapseWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByInputObjectParameterSet
```
Get-AzSynapseSqlPoolRestorePoint -InputObject <PSSynapseSqlPool> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzSynapseSqlPoolRestorePoint -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzSynapseSqlPoolRestorePoint-Cmdlet** ruft die eindeutigen Wiederherstellungspunkte ab, aus der ein Azure Synapse Analytics SQL-Pool wiederhergestellt werden kann.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Get-AzSynapseSqlPoolRestorePoint -WorkspaceName -WorkspaceName ContosoWorkspace -Name ContosoSqlPool
```

Dieser Befehl gibt alle verfügbaren Wiederherstellungspunkte für den Azure Synapse Analytics SQL ContosoSqlPool zurück.
