# Remove-AzContainerRegistryTag

## SYNOPSIS
Enttag ACR-Tag.

## SYNTAX

```
Remove-AzContainerRegistryTag -RepositoryName <String> -Name <String> -RegistryName <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Enttag ACR-Tag.
Um dieses Cmdlet zu verwenden, führen Sie `Update-AzContainerRegistryRepository -RegistryName XXX -Repository XXX -DeleteEnable $true`
zuerst.

## BEISPIELE

### Beispiel 1
```powershell
Remove-AzContainerRegistryTag -RegistryName registry -RepositoryName alpine -Name latest
True
```

Enttag alpine:tag under registry.
