# Restart-AzSpringCloudAppDeployment

## SYNOPSIS
Starten Sie die Bereitstellung neu.

## SYNTAX

### Neustart (Standard)
```
Restart-AzSpringCloudAppDeployment -AppName <String> -Name <String> -ResourceGroupName <String>
 -ServiceName <String> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

### RestartViaIdentity
```
Restart-AzSpringCloudAppDeployment -InputObject <ISpringCloudIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## BESCHREIBUNG
Starten Sie die Bereitstellung neu.

## BEISPIELE

### Beispiel 1: Starten Sie den Spring Cloud Service nach Name neu.
```powershell
PS C:\> Restart-AzSpringCloudAppDeployment -ResourceGroupName spring-cloud-rg -ServiceName spring-cloud-service -AppName gateway -DeploymentName default
```

Starten Sie den Spring Cloud Service nach Name neu.

### Beispiel 2: Starten Sie den Quellwolkendienst von pipe neu.
```powershell
PS C:\> Get-AzSpringCloudAppDeployment -ResourceGroupName spring-cloud-rg -ServiceName spring-cloud-service -AppName gateway -DeploymentName default | Restart-AzSpringCloud
```

Starten Sie den Spring Cloud Service von pipe neu.
