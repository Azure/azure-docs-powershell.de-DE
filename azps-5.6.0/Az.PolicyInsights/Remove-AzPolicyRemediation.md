# Remove-AzPolicyRemediation

## SYNOPSIS
Löscht eine Richtliniensanierung.

## SYNTAX

### ByName (Standard)
```
Remove-AzPolicyRemediation -Name <String> [-Scope <String>] [-ManagementGroupName <String>]
 [-ResourceGroupName <String>] [-AllowStop] [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Remove-AzPolicyRemediation -ResourceId <String> [-AllowStop] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzPolicyRemediation -InputObject <PSRemediation> [-AllowStop] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzPolicyRemediation** löscht eine Richtliniensanierung.

## BEISPIELE

### Beispiel 1: Löschen einer Richtliniensanierung im Ressourcengruppenbereich
```
PS C:\> Remove-AzPolicyRemediation -ResourceGroupName "myRG" -Name "remediation1"
```

Mit diesem Befehl wird die Behebung mit dem Namen "Behebung1" in der Ressourcengruppe "myRG" gelöscht.

### Beispiel 2: Löschen einer Behebung einer Verwaltungsgruppe über Piping
```
PS C:\> $remediation = Get-AzPolicyRemediation -ManagementGroupName "mg1" -Name "remediation1"
PS C:\> $remediation | Remove-AzPolicyRemediation -Confirm
```

Mit diesem Befehl wird die Behebung mit dem Namen "Behebung1" aus der Verwaltungsgruppe "mg1" gelöscht. Vor dem Löschen der Ressource wird eine Bestätigungsaufforderung angezeigt.

### Beispiel 3: Abbrechen und Löschen einer Richtliniensanierung
```
PS C:\> Remove-AzPolicyRemediation -ResourceGroupName "myRG" -Name "remediation1" -AllowStop
```

Mit diesem Befehl wird die Behebung mit dem Namen "Behebung1" in der Ressourcengruppe "myRG" gelöscht. Wenn die Behebung gerade ausgeführt wird, wird sie abgebrochen, bevor sie gelöscht wird.
