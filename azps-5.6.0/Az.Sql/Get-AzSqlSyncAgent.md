# Get-AzSqlSyncAgent

## SYNOPSIS
Gibt Informationen zu Azure SQL Synchronisierungs-Agents zurück.

## SYNTAX

```
Get-AzSqlSyncAgent [[-Name] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzSqlSyncAgent-Cmdlet** gibt Informationen zu einem oder mehreren Azure SQL Synchronisierungs-Agents zurück.
Geben Sie den Namen eines Synchronisierungs-Agents an, um Informationen nur für diesen Synchronisierungs-Agent eins zu sehen.
