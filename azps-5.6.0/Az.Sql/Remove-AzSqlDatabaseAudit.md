# Remove-AzSqlDatabaseAudit

## SYNOPSIS
Entfernt die Überwachungseinstellungen einer Azure SQL Datenbank.

## SYNTAX

### DatabaseParameterSet (Standard)
```
Remove-AzSqlDatabaseAudit [-ResourceGroupName] <String> [-ServerName] <String> [-DatabaseName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DatabaseObjectParameterSet
```
Remove-AzSqlDatabaseAudit -DatabaseObject <AzureSqlDatabaseModel> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzSqlDatabaseAudit** entfernt die Überwachungseinstellungen einer Azure SQL Datenbank.
Um das Cmdlet zu verwenden, verwenden Sie die *Parameter ResourceGroupName,* *ServerName* und *DatabaseName,* um die Datenbank zu identifizieren.

## BEISPIELE

### Beispiel 1: Entfernen der Überwachungseinstellungen einer Azure SQL Datenbank
```
PS C:\>Remove-AzSqlDatabaseAudit -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

### Beispiel 2: Entfernen der Überwachungseinstellungen einer Azure SQL Pipeline
```
PS C:\> Get-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01" | Remove-AzSqlDatabaseAudit
```
