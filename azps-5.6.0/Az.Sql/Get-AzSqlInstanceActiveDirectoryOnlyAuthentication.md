# Get-AzSqlInstanceActiveDirectoryOnlyAuthentication

## SYNOPSIS
Ruft nur azure AD-Authentifizierung für eine bestimmte verwaltete SQL ab.

## SYNTAX

### UseResourceGroupAndInstanceNameParameterSet (Standard)
```
Get-AzSqlInstanceActiveDirectoryOnlyAuthentication [-ResourceGroupName] <String> [-InstanceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### UseInputObjectParameterSet
```
Get-AzSqlInstanceActiveDirectoryOnlyAuthentication -InputObject <AzureSqlManagedInstanceModel>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### UserResourceIdParameterSet
```
Get-AzSqlInstanceActiveDirectoryOnlyAuthentication [-ResourceId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzSqlInstanceActiveDirectoryOnlyAuthentication-Cmdlet** ruft azure Active Directory (Azure AD) nur die Authentifizierungsanforderung für eine verwaltete AzureSQL-Instanz im aktuellen Abonnement ab.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\>Get-AzSqlInstanceActiveDirectoryOnlyAuthentication -ResourceGroupName "ResourceGroup01" -InstanceName "ManagedInstance01"
```

Dieser Befehl ruft azure active Directory (Azure AD) nur die Authentifizierungsanforderung für eine verwaltete AzureSQL-Instanz mit dem Namen ManagedInstance01 ab, die einer Ressourcengruppe mit dem Namen ResourceGroup01 zugeordnet ist.
