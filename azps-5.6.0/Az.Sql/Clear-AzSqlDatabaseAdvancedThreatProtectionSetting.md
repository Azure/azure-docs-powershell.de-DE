# Clear-AzSqlDatabaseAdvancedThreatProtectionSetting

## SYNOPSIS
Entfernt die erweiterten Bedrohungsschutzeinstellungen aus einer Datenbank.

## SYNTAX

```
Clear-AzSqlDatabaseAdvancedThreatProtectionSetting [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Clear-AzSqlDatabaseAdvancedThreatProtectionSetting-Cmdlet** entfernt die erweiterten Bedrohungsschutzeinstellungen aus einer AzureAzure-SQL Datenbank.
Um dieses Cmdlet zu verwenden, geben Sie die *Parameter ResourceGroupName* und *ServerName* an, um die Datenbank zu identifizieren, aus der dieses Cmdlet die Einstellungen entfernt.

## BEISPIELE

### Beispiel 1: Entfernen erweiterter Bedrohungsschutzeinstellungen für eine Datenbank
```
PS C:\>Clear-AzSqlDatabaseAdvancedThreatProtectionSetting -ResourceGroupName "ResourceGroup11" -ServerName "Server01" -DatabaseName "Database01"
```

Mit diesem Befehl werden die erweiterten Einstellungen für den Bedrohungsschutz aus einer Datenbank mit dem Namen Datenbank01 auf dem Server mit dem Namen Server01 entfernt.
