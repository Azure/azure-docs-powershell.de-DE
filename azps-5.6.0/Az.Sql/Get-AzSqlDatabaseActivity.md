# Get-AzSqlDatabaseActivity

## SYNOPSIS
Ruft den Status von Datenbankvorgängen ab.

## SYNTAX

```
Get-AzSqlDatabaseActivity [-ServerName] <String> [-ElasticPoolName <String>] -DatabaseName <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzSqlDatabaseActivity-Cmdlet** erhält den Status von Datenbankvorgängen in Azure SQL Database.

## BEISPIELE

### Beispiel 1: Status für alle SQL Datenbankinstanzen
```
PS C:\>Get-AzSqlDatabaseActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Dieser Befehl gibt den Vorgangsstatus aller SQL Datenbankinstanzen in einem Pool mit dem Namen "ElasticPool01" zurück.

### Beispiel 2: Status für alle SQL Datenbankvorgänge
```
PS C:\>Get-AzSqlDatabaseActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Dieser Befehl gibt den Status aller SQL Datenbankvorgänge in einer Datenbank zurück.
