# New-AzSqlServerCommunicationLink

## SYNOPSIS
Erstellt eine Kommunikationsverbindung für flexible Datenbanktransaktionen zwischen zwei SQL Datenbankservern.

## SYNTAX

```
New-AzSqlServerCommunicationLink -LinkName <String> -PartnerServer <String> [-AsJob] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet New-AzSqlServerCommunicationLink** erstellt eine Kommunikationsverbindung für flexible Datenbanktransaktionen zwischen zwei logischen Servern in Azure SQL Database.
Flexible Datenbanktransaktionen können Datenbanken auf einem der gekoppelten Server umfassen.
Sie können mehrere Links auf einem Server erstellen.
Daher können sich transaktionen mit flexiblen Datenbanken über eine größere Anzahl von Servern erstreckt haben.

## BEISPIELE

### Beispiel 1: Erstellen eines Kommunikationslinks
```
PS C:\>New-AzSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17" -LinkName "Link01" -PartnerServer "ContosoServer02"
```

Mit diesem Befehl wird ein Link mit dem Namen Link01 zwischen ContosoServer17 und ContosoServer02 erstellt.
