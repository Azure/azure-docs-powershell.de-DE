# Set-AzSqlServerRecommendedActionState

## SYNOPSIS
Aktualisiert den Status einer Azure-SQL Server empfohlenen Aktion.

## SYNTAX

```
Set-AzSqlServerRecommendedActionState -RecommendedActionName <String> -State <RecommendedActionState>
 -ServerName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Set-AzSqlServerRecommendedActionState** aktualisiert den Status einer Azure-SQL Server empfohlene Aktion.
Dieses Cmdlet wendet die empfohlene Aktion basierend auf dem neuen Zustand an, wiederherstellen oder verwirft sie.
