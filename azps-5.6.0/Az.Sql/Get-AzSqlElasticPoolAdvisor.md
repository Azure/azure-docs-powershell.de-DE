# Get-AzSqlElasticPoolAdvisor

## SYNOPSIS
Ruft einen oder mehrere Berater für einen Azure SQL-Pool ab.

## SYNTAX

```
Get-AzSqlElasticPoolAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -ElasticPoolName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzSqlElasticPoolAdvisor-Cmdlet** erhält einen oder mehrere Azure SQL -Berater für den Flexiblen Pool für einen Azure SQL-Pool.
