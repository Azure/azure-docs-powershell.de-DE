# Get-AzSqlElasticPoolDatabase

## SYNOPSIS
Ruft flexible Datenbanken in einem Pool für flexible Datenbanken und deren Eigenschaftswerte ab.

## SYNTAX

```
Get-AzSqlElasticPoolDatabase [-ElasticPoolName] <String> [-DatabaseName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzSqlElasticPoolDatabase-Cmdlet** ruft flexible Datenbanken in einem Pool mit flexiblen Bändern und deren Eigenschaftswerten ab.
Sie können den Namen einer flexiblen Datenbank in Azure SQL-Datenbank angeben, um die Eigenschaftswerte nur für diese Datenbank zu sehen.

## BEISPIELE

### Beispiel 1: Alle Datenbanken in einem Pool mit einem Flexiblen Pool erhalten
```
PS C:\> Get-AzSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Dieser Befehl ruft alle Datenbanken in einem Pool mit dem Namen "ElasticPool01" ab.

### Beispiel 2: Holen Sie sich alle Datenbanken in einem Pool mit einem Flexiblen Pool mithilfe von Filtern.
```
PS C:\> Get-AzSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01" -DatabaseName "Database*"
```

Dieser Befehl ruft alle Datenbanken in einem Pool mit dem Namen "ElasticPool01" ab, der mit "Datenbank" beginnt.
