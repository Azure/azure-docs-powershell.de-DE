# Invoke-AzSqlInstanceFailover

## SYNOPSIS
Failover einer Azure SQL verwaltete Instanz.

## SYNTAX

```
Invoke-AzSqlInstanceFailover [-Name] <String> [-AsJob] [-PassThru] [-Force] [-ReadableSecondary]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Invoke-AzSqlInstanceFailover** failovert eine Azure SQL verwaltete Instanz.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Invoke-AzSqlInstanceFailover -ResourceGroupName "ResourceGroup01" -Name "ManagedInstance01"
```

Dieser Befehl über failovert das primäre Replikat der Instanz mit dem Namen "ManagedInstance01".

### Beispiel 2
```powershell
PS C:\> Invoke-AzSqlInstanceFailover -ResourceGroupName "ResourceGroup01" -Name "ManagedInstance01" -ReadableSecondary
```

Mit diesem Befehl wird das lesbare sekundäre Replikat der verwalteten Instanz "ManagedInstance01" failovern.
