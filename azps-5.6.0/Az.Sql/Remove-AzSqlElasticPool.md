# Remove-AzSqlElasticPool

## SYNOPSIS
Löscht einen pool für flexible Datenbanken.

## SYNTAX

```
Remove-AzSqlElasticPool [-ElasticPoolName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzSqlElasticPool** löscht einen Azure SQL Database-Pool.

## BEISPIELE

### Beispiel 1: Löschen eines Pool für einen Flexiblen Pool
```
PS C:\>Remove-AzSqlElasticPool -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Mit diesem Befehl wird ein Pool mit dem Namen "ElasticPool01" gelöscht.
