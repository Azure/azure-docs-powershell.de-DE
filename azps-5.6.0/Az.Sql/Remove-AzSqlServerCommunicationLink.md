# Remove-AzSqlServerCommunicationLink

## SYNOPSIS
Löscht eine Kommunikationsverbindung für flexible Datenbanktransaktionen zwischen zwei Servern.

## SYNTAX

```
Remove-AzSqlServerCommunicationLink [-LinkName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzSqlServerCommunicationLink** löscht einen Server-zu-Server-Kommunikationslink für flexible Datenbanktransaktionen in Azure SQL Database.

## BEISPIELE

### Beispiel 1: Löschen eines Kommunikationslinks
```
PS C:\>Remove-AzSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17" -LinkName "Link01"
```

Mit diesem Befehl wird ein Server-zu-Server-Kommunikationslink namens Link01 auf ContosoServer17 gelöscht.
