# Get-AzSqlDatabaseBackupLongTermRetentionPolicy

## SYNOPSIS
Ruft eine datenbank langfristige Aufbewahrungsrichtlinie ab.

## SYNTAX

```
Get-AzSqlDatabaseBackupLongTermRetentionPolicy [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzSqlDatabaseBackupLongTermRetentionPolicy-Cmdlet** ruft die langfristige Aufbewahrungsrichtlinie ab, die für diese Datenbank registriert ist.
Die Richtlinie ist eine Azure Backup-Ressource, die zum Definieren der Sicherungsspeicherrichtlinie verwendet wird.

## BEISPIELE

### Beispiel 1: Holen Sie sich die aktuelle Version der langfristigen Aufbewahrungsrichtlinie.
```powershell
PS C:\> Get-AzSqlDatabaseBackupLongTermRetentionPolicy -ResourceGroupName resourcegroup01 -ServerName server01 -DatabaseName database01


ResourceGroupName                      : resourcegroup01
ServerName                             : server01
DatabaseName                           : database01
WeeklyRetention                        : P2W
MonthlyRetention                       : PT0S
YearlyRetention                        : PT0S
WeekOfYear                             : 0
State                                  :
RecoveryServicesBackupPolicyResourceId :
Location                               :
```

Dieser Befehl ruft die aktuelle Version der langfristigen Aufbewahrungsrichtlinie für Datenbank ab01
