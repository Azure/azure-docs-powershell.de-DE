# Get-AzSqlServerTransparentDataEncryptionProtector

## SYNOPSIS
Ruft den Schutzschutz für transparente Datenverschlüsselung (Transparent Data Encryption, TDE) ab.

## SYNTAX

```
Get-AzSqlServerTransparentDataEncryptionProtector [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das Get-AzSqlServerTransparentDataEncryptionProtector-Cmdlet erhält Informationen über den TDE-Protektor.

## BEISPIELE

### Beispiel 1: Schützen der Transparent Data Encryption (TDE)
```
PS C:\> Get-AzSqlServerTransparentDataEncryptionProtector -ServerName 'ContosoServer' -ResourceGroup 'ContosoResourceGroup'
```

Dieser Befehl ruft den TDE-Protektor für den Server namens ContosoServer in der Ressourcengruppe "ContosoResourceGroup" ab.
ResourceGroupName ServerName Type ServerKeyVaultKeyName
----------------- ----------                   ---- ---------------------
ContosoResourceGroup ContosoServer ServiceManaged ServiceManaged
