# New-AzSqlSyncAgentKey

## SYNOPSIS
Erstellt einen Azure SQL-Agent-Schlüssel.

## SYNTAX

```
New-AzSqlSyncAgentKey [-ServerName] <String> [-SyncAgentName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet New-AzSqlSyncAgentKey** erstellt einen Azure SQL-Agent-Schlüssel.

## BEISPIELE

### Beispiel 1: Erstellen eines Synchronisierungs-Agent-Schlüssels für einen Azure SQL Synchronisierungs-Agent.
```
PS C:\> New-AzSqlSyncAgentKey -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -SyncAgentName "SyncAgent01"
SyncAgentKey                  : Key
```

Mit diesem Befehl wird ein Synchronisierungs-Agent-Schlüssel für einen Azure SQL Synchronisierungs-Agent erstellt.
