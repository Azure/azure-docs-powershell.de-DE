# Get-AzSqlSyncGroup

## SYNOPSIS
Gibt Informationen zu Azure SQL Datenbanksynchronisierungsgruppen zurück.

## SYNTAX

```
Get-AzSqlSyncGroup [[-Name] <String>] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzSqlSyncGroup-Cmdlet** gibt Informationen zu einer oder mehreren Azure SQL Datenbanksynchronisierungsgruppen zurück.
Geben Sie den Namen einer Synchronisierungsgruppe an, um Informationen nur für diese Synchronisierungsgruppe zu sehen.
