# Remove-AzSqlServerDnsAlias

## SYNOPSIS
Entfernt Azure SQL Server DNS-Alias.

## SYNTAX

### Entfernen eines Server-Dns-Alias aus cmdlet-Eingabeparametern
```
Remove-AzSqlServerDnsAlias -Name <String> -ServerName <String> [-ResourceGroupName] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Entfernen eines Server-Dns-Alias aus der AzureSqlServerDnsAliasModel-Instanzdefinition
```
Remove-AzSqlServerDnsAlias -InputObject <AzureSqlServerDnsAliasModel> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Entfernen eines Server-Dns-Alias aus einer Azure-Ressourcen-ID
```
Remove-AzSqlServerDnsAlias -ResourceId <String> [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Mit diesen Befehlen wird azure SQL Server DNS-Alias vom Server entfernt, und der Server bleibt erhalten.

## BEISPIELE

### Beispiel 1
```
PS C:\> Remove-AzSqlServerDnsAlias -DnsAliasName aliasName -ServerName serverName -ResourceGroupName rg
```

Entfernt Azure SQL Server-DNS-Alias mit dem NamenaliasName vom Server mit dem Namenservername
