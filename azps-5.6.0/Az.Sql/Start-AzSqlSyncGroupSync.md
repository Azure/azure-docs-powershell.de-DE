# Start-AzSqlSyncGroupSync

## SYNOPSIS
Startet die Synchronisierung einer Synchronisierungsgruppe.

## SYNTAX

```
Start-AzSqlSyncGroupSync [-SyncGroupName] <String> [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Start-AzSqlSyncGroupSync** startet eine Synchronisierungsgruppesynchronisierung.

## BEISPIELE

### Beispiel 1
```
PS C:\> Start-AzSqlSyncGroupSync -SyncGroupName mysg [-ServerName] mysrv [-DatabaseName] mydb [-ResourceGroupName] myrg
```

Mit diesem Befehl wird eine Synchronisierungsrunde für die Synchronisierungsgruppe mysg gestartet.
