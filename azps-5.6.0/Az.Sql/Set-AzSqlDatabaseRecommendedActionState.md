# Set-AzSqlDatabaseRecommendedActionState

## SYNOPSIS
Aktualisiert den Status einer empfohlenen Azure SQL-Datenbank.

## SYNTAX

```
Set-AzSqlDatabaseRecommendedActionState -RecommendedActionName <String> -State <RecommendedActionState>
 -ServerName <String> -DatabaseName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Set-AzSqlDatabaseRecommendedActionState** aktualisiert den Zustand einer empfohlenen Azure SQL-Datenbankaktion.
Dadurch kann eine empfohlene Aktion basierend auf dem neuen Zustand angewendet, zurückgesetzt oder verworfen werden.
