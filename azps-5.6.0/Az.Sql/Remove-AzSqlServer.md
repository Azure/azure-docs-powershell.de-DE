# Remove-AzSqlServer

## SYNOPSIS
Entfernt einen Azure SQL-Datenbankserver.

## SYNTAX

```
Remove-AzSqlServer [-ServerName] <String> [-Force] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzSqlServer** entfernt einen Azure SQL Datenbankserver.
Der Löschvorgang ist asynchron und kann einige Zeit dauern. Überprüfen Sie daher, ob der Vorgang abgeschlossen ist, bevor Sie weitere Vorgänge ausführen, die vom vollständigen Löschen des Servers abhängig sind.
Beispielsweise müssen Sie einen neuen Server erstellen, der denselben Namen verwendet.

## BEISPIELE

### Beispiel 1: Entfernen eines Servers
```
PS C:\>Remove-AzSqlServer -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

Mit diesem Befehl wird der Azure SQL-Datenbankserver mit dem Namen Server01 entfernt.
