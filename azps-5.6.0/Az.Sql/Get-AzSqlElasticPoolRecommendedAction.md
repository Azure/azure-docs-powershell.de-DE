# Get-AzSqlElasticPoolRecommendedAction

## SYNOPSIS
Ruft eine oder mehrere empfohlene Aktionen für einen Azure SQL -Poolratgeber ab.

## SYNTAX

```
Get-AzSqlElasticPoolRecommendedAction [-RecommendedActionName <String>] -ServerName <String>
 -ElasticPoolName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzSqlElasticPoolRecommendedAction-Cmdlet** ruft eine oder mehrere empfohlene Aktionen für einen Azure SQL -Berater für den Flexiblen Pool ab.
