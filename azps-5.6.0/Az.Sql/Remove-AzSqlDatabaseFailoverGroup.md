# Remove-AzSqlDatabaseFailoverGroup

## SYNOPSIS
Entfernt eine Azure SQL-Failovergruppe.

## SYNTAX

```
Remove-AzSqlDatabaseFailoverGroup [-ServerName] <String> [-FailoverGroupName] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Mit diesem Befehl wird die Failovergruppe mit dem angegebenen Namen entfernt, damit alle Datenbanken und Replikationsbeziehungen intakt bleiben. Der Listenerendpunkt wird von DNS nicht registriert.
Der primäre Server der Failovergruppe sollte zum Ausführen des Befehls verwendet werden.

## BEISPIELE

### Beispiel 1
```
PS C:\> Remove-AzSqlDatabaseFailoverGroup -ResourceGroupName rg -ServerName primaryserver -FailoverGroupName fg
```

Entfernen sie eine Failovergruppe.
