# Remove-AzSqlDatabase

## SYNOPSIS
Entfernt eine Azure SQL Datenbank.

## SYNTAX

```
Remove-AzSqlDatabase [-DatabaseName] <String> [-Force] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzSqlDatabase** entfernt eine Azure SQL Datenbank.
Dieses Cmdlet wird auch vom SQL Server Stretch Database Service in Azure unterstützt.

## BEISPIELE

### Beispiel 1: Entfernen einer Datenbank von einem Azure SQL Server
```
PS C:\>Remove-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Mit diesem Befehl wird die Datenbank mit dem Namen Datenbank01 von Server Server01 entfernt.
