# Get-AzSqlDatabaseDataMaskingPolicy

## SYNOPSIS
Ruft die Datenformatierungsrichtlinie für eine Datenbank ab.

## SYNTAX

```
Get-AzSqlDatabaseDataMaskingPolicy [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Get-AzSqlDatabaseDataMaskingPolicy** ruft die Datenformatierungsrichtlinie einer Azure-SQL ab.
Um dieses Cmdlet zu verwenden, verwenden Sie die *Parameter ResourceGroupName,* *ServerName* und *DatabaseName,* um die Datenbank zu identifizieren.
Dieses Cmdlet wird auch vom SQL Server Stretch Database Service in Azure unterstützt.

## BEISPIELE

### Beispiel 1: Datenformatierungsrichtlinie für eine Azure-SQL-Datenbank
```
PS C:\>Get-AzSqlDatabaseDataMaskingPolicy -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
DatabaseName      : Database01
ResourceGroupName : ResourceGroup01
ServerName        : Server01
DataMaskingState  : Enabled
PrivilegedUsers  :
```

Dieser Befehl ruft die Datenformatierungsrichtlinie aus Datenbankdatenbank01 in der Ressourcengruppe ResourceGroup01 auf Server Server01 ab.
