# Remove-AzStreamAnalyticsJob

## SYNOPSIS
Entfernt einen Stream Analytics-Auftrag.

## SYNTAX

```
Remove-AzStreamAnalyticsJob [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzStreamAnalyticsJob** löscht asynchron einen bestimmten Stream Analytics-Auftrag in Azure.

## BEISPIELE

### BEISPIEL 1: Entfernen eines Auftrags
```
PS C:\>Remove-AzStreamAnalyticsJob -ResourceGroupName "StreamAnalytics-Default-West-US" -Name "StreamingJob"
```

Mit diesem Befehl wird der Auftrag StreamingJob entfernt.
