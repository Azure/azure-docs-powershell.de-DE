# Remove-AzStreamAnalyticsInput

## SYNOPSIS
Löscht eine Eingabe aus einem Stream Analytics-Auftrag.

## SYNTAX

```
Remove-AzStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzStreamAnalyticsInput** löscht asynchron eine Eingabe aus einem Stream Analytics-Auftrag in Azure.

## BEISPIELE

### BEISPIEL 1: Entfernen eines Eingabedatenstroms aus einem Auftrag
```
PS C:\>Remove-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EventStream"
```

Mit diesem Befehl wird der Eingabedatenstrom aus StreamingJob entfernt.
