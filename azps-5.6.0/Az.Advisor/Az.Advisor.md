# Az.Advisor Module
## Beschreibung
Befehle für die Interaktion mit Azure Advisor.

## Az.Advisor-Cmdlets
### [Get-AzAdvisorRecommendation](Get-AzAdvisorRecommendation.md)
Ruft eine Liste der Azure Advisor-Empfehlungen ab.

### [Disable-AzAdvisorRecommendation](Disable-AzAdvisorRecommendation.md)
Deaktivieren Sie eine Azure Advisor-Empfehlung.

### [Enable-AzAdvisorRecommendation](Enable-AzAdvisorRecommendation.md)
Aktiviert Azure Advisor-Empfehlungen.

### [Get-AzAdvisorConfiguration](Get-AzAdvisorConfiguration.md)
Holen Sie sich die Azure Advisor-Konfigurationen für das angegebene Abonnement oder die angegebene Ressourcengruppe.

### [Set-AzAdvisorConfiguration](Set-AzAdvisorConfiguration.md)
Aktualisiert oder erstellt die Azure Advisor-Konfiguration.
