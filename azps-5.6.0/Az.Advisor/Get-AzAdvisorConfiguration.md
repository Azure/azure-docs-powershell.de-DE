# Get-AzAdvisorConfiguration

## SYNOPSIS
Holen Sie sich die Azure Advisor-Konfigurationen für das angegebene Abonnement oder die angegebene Ressourcengruppe.

## SYNTAX

```
Get-AzAdvisorConfiguration [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Die einem Abonnement zugeordneten Konfigurationen haben zwei Typen:

Konfiguration auf Abonnementebene: Es kann nur eine Konfiguration dieses Typs für ein Abonnement geben. LowCpuThreshold und Exclude sind die einzige Eigenschaft dieses Konfigurationstyps.

Konfiguration auf Ressourcengruppenebene: Es kann nur eine Konfiguration für jede Ressourcengruppe in einem Abonnement gibt. Exclude ist die einzige Eigenschaft dieses Konfigurationstyps.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\>$data = Get-AzAdvisorConfiguration
Id         : /subscriptions/{user_subscription}/providers/Microsoft.Advisor/configurations/{user_subscription}
Name       : {user_subscription}
Properties : Microsoft.Azure.Commands.Advisor.Cmdlets.Models.PsAzureAdvisorConfigurationProperties
Type       : Microsoft.Advisor/Configurations

Id         : /subscriptions/{user_subscription}/providers/Microsoft.Advisor/configurations/{user_subscription}-{resourceGroupName}
Name       : {user_subscription}-{resourceGroupName}
Properties : Microsoft.Azure.Commands.Advisor.Cmdlets.Models.PsAzureAdvisorConfigurationProperties
Type       : Microsoft.Advisor/Configurations

PS C:\>$data[0].Properties
AdditionalProperties :
Exclude              : False
LowCpuThreshold      : 20

PS C:\>$data[1].Properties
AdditionalProperties :
Exclude              : True
LowCpuThreshold      : null

```
Ruft eine Liste der Azure Advisor-Konfigurationen ab.
