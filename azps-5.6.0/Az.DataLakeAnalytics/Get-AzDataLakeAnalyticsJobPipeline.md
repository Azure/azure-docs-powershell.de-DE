# Get-AzDataLakeAnalyticsJobPipeline

## SYNOPSIS
Ruft eine Data Lake Analytics-Auftragspipeline oder -pipeline ab.

## SYNTAX

### GetAllInAccount (Standard)
```
Get-AzDataLakeAnalyticsJobPipeline [-Account] <String> [-SubmittedAfter <DateTimeOffset>]
 [-SubmittedBefore <DateTimeOffset>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetBySpecificJobPipeline
```
Get-AzDataLakeAnalyticsJobPipeline [-Account] <String> [-PipelineId] <Guid> [-SubmittedAfter <DateTimeOffset>]
 [-SubmittedBefore <DateTimeOffset>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Die **Get-AzDataLakeAnalyticsJobPipeline** ruft eine angegebene Azure Data Lake Analytics-Auftragspipeline oder eine Liste der Pipelines ab.

## BEISPIELE

### Beispiel 1: Erhalten einer angegebenen Pipeline
```
PS C:\>Get-AzDataLakeAnalyticsJobPipeline -Account "contosoadla" -PipelineId 83cb7ad2-3523-4b82-b909-d478b0d8aea3
```

Dieser Befehl ruft die angegebene Pipeline mit der ID "83cb7ad2-3523-4b82-b909-d478b0d8aea3" im Konto "contosoadla" ab.

### Beispiel 2: Erhalten einer Liste aller Pipelines im Konto
```
PS C:\>Get-AzDataLakeAnalyticsJobPipeline -AccountName "contosoadla"
```

Dieser Befehl ruft eine Liste aller Pipelines im Konto "contosoadla" ab.
