# Stop-AzDataLakeAnalyticsJob

## SYNOPSIS
Bricht einen Auftrag ab.

## SYNTAX

```
Stop-AzDataLakeAnalyticsJob [-Account] <String> [-JobId] <Guid> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Stop-AzDataLakeAnalyticsJob** bricht einen Azure Data Lake Analytics-Auftrag ab.

## BEISPIELE

### Beispiel 1: Abbrechen eines Auftrags
```
PS C:\>Stop-AzDataLakeAnalyticsJob -Account "ContosoAdlAccout" -JobId "a0a78d72-3fa8-4564-9b18-6becb3fda48a"
```

Mit diesem Befehl wird der Auftrag mit der angegebenen ID abgebrochen.
