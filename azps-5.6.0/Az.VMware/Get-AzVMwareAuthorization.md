# Get-AzVMWareAuthorization

## SYNOPSIS
Erhalten einer ExpressRoute-Schaltungsautorisierung nach Name in einer privaten Cloud

## SYNTAX

### Liste (Standard)
```
Get-AzVMWareAuthorization -PrivateCloudName <String> -ResourceGroupName <String> [-SubscriptionId <String[]>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### Erhalten
```
Get-AzVMWareAuthorization -Name <String> -PrivateCloudName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-AzVMWareAuthorization -InputObject <IVMWareIdentity> [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## BESCHREIBUNG
Erhalten einer ExpressRoute-Schaltungsautorisierung nach Name in einer privaten Cloud

## BEISPIELE

### Beispiel 1: Autorisierung erhalten
```powershell
PS C:\> Get-AzVMWareAuthorization -Name azps-test-auth -PrivateCloudName azps-test-cloud -ResourceGroupName azps-test-group



Name           Type
----           ----
azps-test-auth Microsoft.AVS/privateClouds/authorizations
```

Dieses Cmdlet erhält Autorisierung `azps-test-auth` in der privaten Cloud `azps-test-cloud`

### Beispiel 2: Listenautorisierung
```powershell
PS C:\> Get-AzVMWareAuthorization -PrivateCloudName azps-test-cloud -ResourceGroupName azps-test-group



Name           Type
----           ----
azps-test-auth Microsoft.AVS/privateClouds/authorizations
```

Dieses Cmdlet listet die Autorisierung `azps-test-auth` in der privaten Cloud auf. `azps-test-cloud`
