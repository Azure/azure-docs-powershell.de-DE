# Get-AzAvailabilitySet

## SYNOPSIS
Ruft Azure-Verfügbarkeitssätze in einer Ressourcengruppe ab.

## SYNTAX

```
Get-AzAvailabilitySet [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzAvailabilitySet-Cmdlet** ruft Azure-Verfügbarkeitssätze in einer Ressourcengruppe ab.
Sie können den Namen eines bestimmten Verfügbarkeitssets angeben, das Sie erhalten möchten.

## BEISPIELE
