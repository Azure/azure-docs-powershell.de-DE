# Remove-AzVMSecret

## SYNOPSIS
Entfernt (a) Geheime aus einem Objekt eines virtuellen Computers

## SYNTAX

```
Remove-AzVMSecret [-VM] <PSVirtualMachine> [[-SourceVaultId] <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das Remove-AzVMSecret cmdlet entfernt (a) geheime(n) Geheime aus einem Objekt eines virtuellen Computers.

## BEISPIELE

### Beispiel 1
```
PS C:\> Get-AzVM -ResourceGroupName "rg1" -Name "vm1" | Remove-AzVMSecret | Update-AzVM
```

Entfernt alle Geheiminformationen von einem virtuellen Computer "vm1" in der Ressourcengruppe "rg1" und aktualisiert die VM
