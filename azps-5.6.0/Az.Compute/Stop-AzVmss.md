# Stop-AzVmss

## SYNOPSIS
Beendet den VMSS oder eine Reihe virtueller Computer innerhalb des VMSS.

## SYNTAX

### Standardparameter (Standard)
```
Stop-AzVmss [-ResourceGroupName] <String> [-VMScaleSetName] <String> [[-InstanceId] <String[]>] [-Force]
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### FriendMethod
```
Stop-AzVmss [-ResourceGroupName] <String> [-VMScaleSetName] <String> [[-InstanceId] <String[]>] [-Force]
 [-StayProvisioned] [-SkipShutdown] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Stopp-AzVmss** stoppt alle virtuellen Computer im VmSS (Virtual Machine Scale Set) oder einer Reihe virtueller Computer.
Sie können den *Parameter InstanzId* verwenden, um eine Gruppe virtueller Computer auszuwählen.

## BEISPIELE

### Beispiel 1: Beenden aller virtuellen Computer im VMSS
```
PS C:\> Stop-AzVmss -ResourceGroupName "ContosoGroup" -VMScaleSetName "ContosoVMSS"
```

Mit diesem Befehl werden alle virtuellen Computer beendet, die zum VMSS mit dem Namen ContosoVMSS gehören.

### Beispiel 2: Beenden einer bestimmten Gruppe virtueller Computer innerhalb des VMSS
```
PS C:\> Stop-AzVmss -ResourceGroupName "ContosoGroup" -VMScaleSetName "ContosoVMSS" -InstanceId "3","5"
```

Mit diesem Befehl wird eine bestimmte Gruppe virtueller Computer beendet, die vom Array der Instanz-ID-Zeichenfolge angegeben werden, die zum VMSS mit dem Namen ContosoVMSS gehören.
