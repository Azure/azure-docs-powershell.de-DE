# Remove-AzVMAEMExtension

## SYNOPSIS
Entfernt die AEM-Erweiterung von einem virtuellen Computer.

## SYNTAX

```
Remove-AzVMAEMExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>]
 [[-OSType] <String>] [-NoWait] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzVMAEMExtension** entfernt die Azure Enhanced Monitoring (AEM)-Erweiterung von einem virtuellen Computer.

## BEISPIELE

### Beispiel 1: Entfernen der Erweiterung AEM
```
PS C:\> Remove-AzVMAEMExtension -ResourceGroupName "ResourceGroup11" -VMName "contoso-server"
```

Mit diesem Befehl wird die AEM-Erweiterung für den virtuellen Computer namens contoso-server entfernt.
