# Get-AzVMRunCommandDocument

## SYNOPSIS
Hier erhalten Sie ein Ausführungsbefehlsdokument.

## SYNTAX

```
Get-AzVMRunCommandDocument [-Location] <String> [[-CommandId] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Hier erhalten Sie ein Ausführungsbefehlsdokument.

## BEISPIELE

### Beispiel 1
```
PS C:\> Get-AzVMRunCommandDocument -Location 'westus' -CommandId 'RunPowerShellScript'
```

Ruft ein bestimmtes Ausführungsbefehlsdokument für "RunPowerShellScript" in "westus" ab.

### Beispiel 2
```
PS C:\> Get-AzVMRunCommandDocument -Location 'westus'
```

Listet alle verfügbaren Ausführungsbefehle in "westus" auf.
