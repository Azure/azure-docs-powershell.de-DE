# ConvertTo-AzVMManagedDisk

## SYNOPSIS
Wandelt einen virtuellen Computer mit blobbasierten Datenträgern in einen virtuellen Computer mit verwalteten Datenträgern um.

## SYNTAX

```
ConvertTo-AzVMManagedDisk [-ResourceGroupName] <String> [-VMName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet ConvertTo-AzVMManagedDisk** konvertiert einen virtuellen Computer mit blobbasierten Datenträgern in einen virtuellen Computer mit verwalteten Datenträgern.
Der virtuelle Computer muss vor dem Aufrufen dieses Vorgangs beendet werden.

## BEISPIELE

### Beispiel 1
```
PS C:\> ConvertTo-AzVMManagedDisk -ResourceGroupName 'ResourceGroup01' -VMName 'VM01'
```

Mit diesem Befehl werden die blobbasierten Datenträger des virtuellen Computers namens "VM01" in der Ressourcengruppe "ResourceGroup01" in verwaltete Datenträger konvertiert.
