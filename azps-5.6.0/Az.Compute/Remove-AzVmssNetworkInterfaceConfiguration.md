# Remove-AzVmssNetworkInterfaceConfiguration

## SYNOPSIS
Entfernt eine Netzwerkschnittstellenkonfiguration aus einem VMSS.

## SYNTAX

### NameParameterSet
```
Remove-AzVmssNetworkInterfaceConfiguration [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet>
 [-Name] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSet
```
Remove-AzVmssNetworkInterfaceConfiguration [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [-Id] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzVmssNetworkInterfaceConfiguration** entfernt eine Netzwerkschnittstellenkonfiguration aus einem VmSS (Virtual Machine Scale Set).

## BEISPIELE

### Beispiel 1: Entfernen einer Schnittstellenkonfiguration
```
PS C:\> $VMSS = Get-AzVmss -ResourceGroupName "ResourceGroup11" -VMScaleSetName "ContosoVMSS14"
PS C:\> Remove-AzVmssNetworkInterfaceConfiguration -VirtualMachineScaleSet $VMSS -Name "ContosoVmssInterface02"
```

Der erste Befehl ruft einen VMSS mithilfe des cmdlets Get-AzVmss ab und speichert ihn dann in der $VMSS Variablen.
Mit dem zweiten Befehl wird die Netzwerkschnittstellenkonfiguration contosoVmssInterface02 aus dem Satz in $VMSS.
