# Remove-AzVMSqlServerExtension

## SYNOPSIS
Entfernt eine SQL Server von einem virtuellen Computer.

## SYNTAX

```
Remove-AzVMSqlServerExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-NoWait]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzVMSqlServerExtension** entfernt eine AzureSQL Server-Erweiterung von einem virtuellen Computer.

## BEISPIELE

### Beispiel 1: Entfernen SQL Server Erweiterung
```
PS C:\> Remove-AzVMSqlServerExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM22" -Name "SqlIaaSAgent"
```

Mit diesem Befehl wird eine SQL Server vom virtuellen Computer namens ContosoVM22 entfernt.
