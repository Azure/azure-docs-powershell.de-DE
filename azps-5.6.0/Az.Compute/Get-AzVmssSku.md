# Get-AzVmssSku

## SYNOPSIS
Ruft die verfügbaren SKUs für den VMSS ab.

## SYNTAX

```
Get-AzVmssSku [-ResourceGroupName] <String> [-VMScaleSetName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzVmssSku-Cmdlet** ruft die verfügbaren SKUs für den Virtual Machine Scale Set (VMSS) ab.

## BEISPIELE

### Beispiel 1: Alle verfügbaren SKUs aus dem VMSS
```
PS C:\> Get-AzVmssSku -ResourceGroupName "ContosoGroup" -VMScaleSetName "ContosoVMSS"
```

Dieser Befehl ruft alle verfügbaren SKUs aus dem VMSS namens ContosoVMSS ab, das zur Ressourcengruppe "ContosoGroup" gehört.
