# Get-AzVmssRollingUpgrade

## SYNOPSIS
Zeigt den Status der neuesten Skalierungssatz für virtuelle Computer an.

## SYNTAX

```
Get-AzVmssRollingUpgrade [-ResourceGroupName] <String> [-VMScaleSetName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Zeigt den Status der neuesten Skalierungssatz für virtuelle Computer an.

## BEISPIELE

### Beispiel 1
```
PS C:\> Get-AzVmssRollingUpgrade -ResourceGroupName "Group001" -VMScaleSetName "VMSS001"
```

Dieser Befehl zeigt den Status des neuesten fortlaufenden Upgrades des VMSS namens VMSS001 an, das zur Ressourcengruppe "Group001" gehört.
