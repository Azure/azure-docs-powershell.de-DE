# Get-AzVmssVMDiskEncryption

## SYNOPSIS
Zeigt den Datenträgerverschlüsselungsstatus von VMs in einem VM-Skalierungssatz an.

## SYNTAX

```
Get-AzVmssVMDiskEncryption [-ResourceGroupName] <String> [-VMScaleSetName] <String> [[-InstanceId] <String>]
 [-ExtensionName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Zeigt den Datenträgerverschlüsselungsstatus des VM-Skalierungssets an.

## BEISPIELE

### Beispiel 1
```
PS C:\> Get-AzVmssVMDiskEncryption -ResourceGroupName "Group001" -VMScaleSetName "VMSS001" -InstanceId "1"
```

Zeigt den Datenträgerverschlüsselungsstatus der VM-Instanz 1 im VM-Skalierungssatz "VMSS001" an, der zur Ressourcengruppe "Group001" gehört.
