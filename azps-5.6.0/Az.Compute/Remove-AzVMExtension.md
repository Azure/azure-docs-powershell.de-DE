# Remove-AzVMExtension

## SYNOPSIS
Entfernt eine Erweiterung von einem virtuellen Computer.

## SYNTAX

```
Remove-AzVMExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzVMExtension** entfernt eine Erweiterung aus den Virtual Machine Extensions eines virtuellen Computers.

## BEISPIELE

### Beispiel 1: Entfernen einer Erweiterung von einem virtuellen Computer
```
PS C:\> Remove-AzVMExtension -ResourceGroupName "ResourceGroup11" -Name "ContosoTest" -VMName "VirtualMachine22"
```

Mit diesem Befehl wird die Erweiterung ContosoTest vom virtuellen Computer "VirtualMachine22" in ResourceGroup11 entfernt.
