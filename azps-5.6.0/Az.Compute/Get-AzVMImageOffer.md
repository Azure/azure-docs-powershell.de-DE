# Get-AzVMImageOffer

## SYNOPSIS
Ruft VMImage-Angebotstypen ab.

## SYNTAX

```
Get-AzVMImageOffer -Location <String> -PublisherName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Get-AzVMImageOffer** ruft die VMImage-Angebotstypen ab.

## BEISPIELE

### Beispiel 1: Erhalten von Angebotstypen für einen Herausgeber
```
PS C:\> Get-AzVMImageOffer -Location "Central US" -PublisherName "Fabrikam"
```

Dieser Befehl ruft die Angebotstypen für den angegebenen Herausgeber in der Region "Usa, Mitte" ab.
