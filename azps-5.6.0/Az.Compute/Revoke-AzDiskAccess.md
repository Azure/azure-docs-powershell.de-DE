# Revoke-AzDiskAccess

## SYNOPSIS
Widerruft den Zugriff auf einen Datenträger.

## SYNTAX

```
Revoke-AzDiskAccess [-ResourceGroupName] <String> [-DiskName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Revoke-AzDiskAccess** widerruft den Zugriff auf einen Datenträger.

## BEISPIELE

### Beispiel 1
```
PS C:\> Revoke-AzDiskAccess -ResourceGroupName 'ResourceGroup01' -DiskName 'Disk01'
```

Widerrufen des Zugriffs auf den Datenträger mit dem Namen "Disk01" in der Ressourcengruppe "ResourceGroup01"
