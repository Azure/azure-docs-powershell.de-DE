# Repair-AzVmssServiceFabricUpdateDomain

## SYNOPSIS
Manuelle Plattformaktualisierungsdomäne zum Aktualisieren virtueller Computer in einem Service Fabric-Skalierungssatz für virtuelle Computer.

## SYNTAX

### Standardparameter (Standard)
```
Repair-AzVmssServiceFabricUpdateDomain [-ResourceGroupName] <String> [-VMScaleSetName] <String>
 [-PlatformUpdateDomain] <Int32> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ResourceIdParameter
```
Repair-AzVmssServiceFabricUpdateDomain [-PlatformUpdateDomain] <Int32> [-ResourceId] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ObjectParameter
```
Repair-AzVmssServiceFabricUpdateDomain [-PlatformUpdateDomain] <Int32>
 [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Erzwingen Sie die manuelle Domänenaktualisierung der Plattform, um virtuelle Computer in einem Service Fabric-Skalierungssatz für virtuelle Computer zu aktualisieren.

## BEISPIELE

### Beispiel 1
```
PS C:\> Repair-AzVmssServiceFabricUpdateDomain -ResourceGroupName $rgname -VMScaleSetName $vmssName -PlatformUpdateDomain 0
```

Dieser Befehl erzwingt den Dienst fabric-Update-Walk auf UD 0 für den Skalierungssatz für virtuelle Computer, der durch den Namen der Ressourcengruppe und den Namen des Skalierungssatzs angegeben ist.

### Beispiel 2
```
PS C:\> $vmss = Get-AzVmss -ResourceGroupName $rgname -VMScaleSetName $vmssName
PS C:\> Repair-AzVmssServiceFabricUpdateDomain -VirtualMachineScaleSet $vmss -PlatformUpdateDomain 1
```

Dieser Befehl erzwingt den Dienst fabric-Update-Walk auf UD 1 für den Skalierungssatz für virtuelle Computer, der durch das VM-Skalierungssatzobjekt angegeben wurde.

### Beispiel 3
```
PS C:\> Repair-AzVmssServiceFabricUpdateDomain -ResourceId $resourceId  -PlatformUpdateDomain 2;
```

Dieser Befehl erzwingt den Dienst fabric-Update-Walk auf UD 2 für den Skalierungssatz für virtuelle Computer, der durch die Ressourcen-ID angegeben ist.
