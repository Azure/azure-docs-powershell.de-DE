# Stop-AzVmssRollingUpgrade

## SYNOPSIS
Bricht den aktuellen Skalierungssatz für virtuelle Computer ab.

## SYNTAX

```
Stop-AzVmssRollingUpgrade [-ResourceGroupName] <String> [-VMScaleSetName] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Bricht den aktuellen Skalierungssatz für virtuelle Computer ab.

## BEISPIELE

### Beispiel 1
```
PS C:\> Stop-AzVmssRollingUpgrade -ResourceGroupName "Group001" -VMScaleSetName "VMSS001"
```

Mit diesem Befehl wird das fortlaufende Rollup des VM-Skalierungssets "VMSS001" in der Ressourcengruppe "Group001" abgebrochen.
