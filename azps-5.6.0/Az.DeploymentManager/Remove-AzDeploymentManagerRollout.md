# Remove-AzDeploymentManagerRollout

## SYNOPSIS
Löscht das Rollout.

## SYNTAX

### Interaktiv (Standard)
```
Remove-AzDeploymentManagerRollout [-ResourceGroupName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Remove-AzDeploymentManagerRollout [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzDeploymentManagerRollout [-InputObject] <PSRollout> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzDeploymentManagerRollout** löscht ein Rollout in einem Terminalzustand.
Geben Sie das Rollout nach dem Namen und dem Namen der Ressourcengruppe an. Alternativ können Sie das Rolloutobjekt oder die ResourceId bereitstellen.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Remove-AzDeploymentManagerRollout -ResourceGroupName ContosoResourceGroup -Name ContosoRollout
```

Mit diesem Befehl wird ein Rollout mit dem Namen ContosoRollout in der ContosoResourceGroup gelöscht.

### Beispiel 2: Löschen eines Rollouts mithilfe des Ressourcenbezeichners
```powershell
PS C:\> Remove-AzDeploymentManagerRollout -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/rollouts/ContosoRollout"
```

Mit diesem Befehl wird ein Rollout mit dem Namen ContosoRollout in der ContosoResourceGroup gelöscht.

### Beispiel 3: Löschen eines Rollouts mithilfe des Rolloutobjekts.
```powershell
PS C:\> Remove-AzDeploymentManagerRollout -InputObject $rolloutObject
```

Mit diesem Befehl wird ein Rollout gelöscht, dessen Name und Ressourcengruppe den Eigenschaften Name und ResourceGroupName des $rolloutObject entsprechen.
