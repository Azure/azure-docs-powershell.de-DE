# Set-AzDeploymentManagerStep

## SYNOPSIS
Aktualisiert den Schritt.

## SYNTAX

```
Set-AzDeploymentManagerStep [-InputObject] <PSStepResource> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Set-AzDeploymentManagerStep** aktualisiert einen Schritt mit dem angegebenen Schrittobjekt.
Das Cmdlet gibt das aktualisierte Schrittobjekt zurück.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Set-AzDeploymentManagerStep -InputObject $stepObject
```

Mit diesem Befehl wird ein Schritt aktualisiert, dessen Name und Ressourcengruppe den Eigenschaften Name und ResourceGroupName des $stepObject entsprechen.
Der Schritt wird auf die eigenschaften aktualisiert, die in der Liste $stepObject.
