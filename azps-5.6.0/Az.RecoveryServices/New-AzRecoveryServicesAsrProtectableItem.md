# New-AzRecoveryServicesAsrProtectableItem

## SYNOPSIS
Fügen Sie (Entdecken) einen physischen Server zur Liste der schützenswürdigen Elemente hinzu.

## SYNTAX

```
New-AzRecoveryServicesAsrProtectableItem -ProtectionContainer <ASRProtectionContainer> -FriendlyName <String>
 -IPAddress <String> -OSType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Der **New-AzRecoveryServicesAsrProtectableItem** fügt der Liste der erkannten schützenden Elemente in einem Schutzcontainer in einer ASR-Fabric ein neues schützenbares Element hinzu (gilt nur für den VMware-Fabrictyp).

## BEISPIELE

### Beispiel 1

Fügen Sie (Entdecken) einen physischen Server zur Liste der schützenswürdigen Elemente hinzu. (automatisch generiert)

```powershell <!-- Aladdin Generated Example --> 
New-AzRecoveryServicesAsrProtectableItem -FriendlyName XXXXXXXXXX -IPAddress $ipaddresss -OSType Windows -ProtectionContainer $pc
```
