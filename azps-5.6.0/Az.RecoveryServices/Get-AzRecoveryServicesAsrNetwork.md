# Get-AzRecoveryServicesAsrNetwork

## SYNOPSIS
Ruft Informationen zu den Netzwerken ab, die von websitewiederherstellung für den aktuellen Tresor verwaltet werden.

## SYNTAX

### ByFabricObject (Standard)
```
Get-AzRecoveryServicesAsrNetwork -Fabric <ASRFabric> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByName
```
Get-AzRecoveryServicesAsrNetwork -Fabric <ASRFabric> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByFriendlyName
```
Get-AzRecoveryServicesAsrNetwork -Fabric <ASRFabric> -FriendlyName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzRecoveryServicesAsrNetwork-Cmdlet** ruft Informationen zu Azure Websitewiederherstellungsnetzwerken für den aktuellen Azure-Websitewiederherstellungstresor ab.

## BEISPIELE

### Beispiel 1
```
PS C:\> $Networks = Get-AzRecoveryServicesAsrNetwork -Fabric $Fabric
```

Ruft alle bekannten Netzwerke in der angegebenen Fabric ab.
