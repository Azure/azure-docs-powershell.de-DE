# New-AzRecoveryServicesAsrProtectionContainer

## SYNOPSIS
Erstellt einen Azure Site Recovery Protection-Container innerhalb der angegebenen Fabric.

## SYNTAX

```
New-AzRecoveryServicesAsrProtectionContainer -Name <String> -InputObject <ASRFabric>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das New-AzRecoveryServicesAsrProtectionContainer erstellt einen Schutzcontainer unter dem angegebenen Azure Site Recovery Fabric.

## BEISPIELE

### Beispiel 1
```
PS C:\> $job = New-AzRecoveryServicesAsrProtectionContainer -Name xyz -Fabric $fabric
PS C:\> Get-ASRJob -name $job.id
```

Startet die Erstellung des Schutzcontainers mit den angegebenen Parametern und gibt den ZUM Nachverfolgen des Vorgangs verwendeten ASR-Auftrag zurück.
