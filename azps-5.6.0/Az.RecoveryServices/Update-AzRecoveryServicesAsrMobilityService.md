# Update-AzRecoveryServicesAsrMobilityService

## SYNOPSIS
Push mobility service agent updates to protected machines.

## SYNTAX

```
Update-AzRecoveryServicesAsrMobilityService [-Account <ASRRunAsAccount>]
 -ReplicationProtectedItem <ASRReplicationProtectedItem> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Update-AzRecoveryServicesAsrMobilityService** versucht, Updates des Mobility Service Agent auf geschützte Computer zu schieben(wenn ein Update verfügbar ist).)

## BEISPIELE

### Beispiel 1
```
PS C:\> Update-AzRecoveryServicesAsrMobilityService -ReplicationProtectedItem $rpi -Account $fabric.fabricSpecificDetails.RunAsAccounts[0]
```

Aufgabe zum Nachverfolgen des Mobility Service Agents des geschützten Elements für die Aktualisierungsreplikation.
