# Remove-AzRecoveryServicesAsrStorageClassificationMapping

## SYNOPSIS
Löscht die angegebene ASR-Speicherklassifizierungszuordnung.

## SYNTAX

```
Remove-AzRecoveryServicesAsrStorageClassificationMapping -InputObject <ASRStorageClassificationMapping>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzRecoveryServicesAsrStorageClassificationMapping löscht** die angegebene Azure Site Recovery-Speicherklassifizierungszuordnung.

## BEISPIELE

### Beispiel 1
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrStorageClassificationMapping -StorageClassificationMapping $StorageClassificationMapping
```

Startet das Löschen der angegebenen Speicherklassifizierungszuordnung und gibt den ZUM Nachverfolgen des Vorgangs verwendeten ASR-Auftrag zurück.
