# Add-AzRecoveryServicesAsrReplicationProtectedItemDisk

## SYNOPSIS
Fügen Sie den Datenträger zum Schutz für den bereits geschützten virtuellen Azure-Computer hinzu.

## SYNTAX

```
Add-AzRecoveryServicesAsrReplicationProtectedItemDisk -InputObject <ASRReplicationProtectedItem>
 -AzureToAzureDiskReplicationConfiguration <ASRAzuretoAzureDiskReplicationConfig[]> [-WaitForCompletion]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Add-AzRecoveryServicesAsrReplicationProtectedItemDisk-Cmdlet** fügt den Datenträger zum Schutz für bereits geschützte virtuelle Azure-Computer hinzu.

## BEISPIELE

### Beispiel 1
```powershell
PS C:> Add-AzRecoveryServicesAsrReplicationProtectedItemDisk -ReplicationProtectedItem $rpi -AzureToAzureDiskReplicationConfiguration $disk1,$disk2
```

Starten Sie den Vorgang, um eine angegebene Datenträgerkonfiguration zum Schutz hinzuzufügen.

### Beispiel 2
```powershell
PS C:> $ReplicationProtectedItem |Add-AzRecoveryServicesAsrReplicationProtectedItemDisk -AzureToAzureDiskReplicationConfiguration $disk1,$disk2
```

Starten Sie den Vorgang, um eine angegebene Datenträgerkonfiguration zum Schutz hinzuzufügen. Geschütztes Element für die Pipingeingabereplikation.
