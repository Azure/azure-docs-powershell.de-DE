# Get-AzRedisCache

## SYNOPSIS
Ruft einen Redis-Cache ab.

## SYNTAX

```
Get-AzRedisCache [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzRedisCache-Cmdlet** ruft den angegebenen Azure Redis-Cache ab.
Wenn Sie keine Parameter angeben, ruft dieser Vorgang jeden Redis-Cache für das aktuelle Abonnement ab.

## BEISPIELE

### Beispiel 1: Erhalten eines Redis-Caches nach Name
```
PS C:\>Get-AzRedisCache -Name "myexists"

        Id                 : /subscriptions/a559b6fd-3a84-40bb-a450-b0db5ed37dfe/resourceGroups/myGroup/providers/Microsoft.Cache/Redis/myexists

        Name               : myexists
        Type               : Microsoft.Cache/Redis
        HostName           : myexists.redis.cache.windows.net
        Port               : 6379
        ProvisioningState  : succeeded
        SslPort            : 6380
        RedisConfiguration : {}
        EnableNonSslPort   : False
        RedisVersion       : 2.8
        Size               : 1GB
        Sku                : Basic
        Tag                : {}
        Zone               : []
```

Dieser Befehl ruft den Redis-Cache mit dem Namen myexists ab.
