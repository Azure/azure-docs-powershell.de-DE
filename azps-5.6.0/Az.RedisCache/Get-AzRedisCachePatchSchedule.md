# Get-AzRedisCachePatchSchedule

## SYNOPSIS
Ruft einen Patchplan ab.

## SYNTAX

```
Get-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzRedisCachePatchSchedule-Cmdlet** ruft einen Patchplan für einen Cache in Azure Redis Cache ab.

## BEISPIELE

### Beispiel 1: Erhalten des Patchzeitplans
```
PS C:\>Get-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06"
```

Dieser Befehl ruft den Patchzeitplan aus dem Cache mit dem Namen RedisCache06 ab.
