# Remove-AzRedisCache

## SYNOPSIS
Entfernt einen Redis-Cache.

## SYNTAX

```
Remove-AzRedisCache [-ResourceGroupName <String>] -Name <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzRedisCache** entfernt einen Azure Redis-Cache.

## BEISPIELE

### Beispiel 1: Entfernen eines Redis-Caches und Zurückgeben des Ergebnisses
```
PS C:\>Remove-AzRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force -PassThru
True
```

Mit diesem Befehl wird ein Redis-Cache entfernt und angezeigt, ob der Vorgang erfolgreich ist.

### Beispiel 2: Entfernen eines Redis-Caches und Nichtanzeige des Ergebnisses
```
PS C:\>Remove-AzRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force
```

Mit diesem Befehl wird ein Redis-Cache entfernt.
Da der *Parameter PassThru* nicht angegeben ist, wird das Ergebnis des Vorgangs nicht angezeigt.
