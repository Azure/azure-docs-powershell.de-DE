# Remove-AzRedisCacheLink

## SYNOPSIS
Entfernen Sie eine Verknüpfung zur Georeplikation zwischen zwei Redis Caches.

## SYNTAX

```
Remove-AzRedisCacheLink -PrimaryServerName <String> -SecondaryServerName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Entfernen Sie eine Verknüpfung zur Georeplikation zwischen zwei Redis Caches.

## BEISPIELE

### Beispiel 1: Entfernen eines Links zur Georeplikation
```
PS C:\>Remove-AzRedisCacheLink -PrimaryServerName "mycache1" -SecondaryServerName "mycache2"
```

Mit diesem Befehl wird eine Georeplikationsverknüpfung entfernt, bei der Der Redis-Cache mit dem Namen mycache1 primär und der Redis-Cache mit dem Namen mycache2 sekundär ist.
