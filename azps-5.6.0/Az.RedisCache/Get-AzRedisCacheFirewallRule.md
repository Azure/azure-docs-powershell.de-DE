# Get-AzRedisCacheFirewallRule

## SYNOPSIS
Firewallregeln für Den Redis Cache festlegen.

## SYNTAX

```
Get-AzRedisCacheFirewallRule [-ResourceGroupName <String>] -Name <String> [-RuleName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Wenn **der Parameter RuleName** angegeben ist, ruft **das Cmdlet Get-AzRedisCacheFirewallRule** Details zur angegebenen Firewallregel im Azure Redis Cache ab. Wenn nur **Name** angegeben ist, ruft dieser Vorgang alle Firewallregeln ab, die für diesen Redis-Cache verfügbar sind.

## BEISPIELE

### Beispiel 1: Erhalten einer einzelnen Firewallregel
```
PS C:\>Get-AzRedisCacheFirewallRule -Name "mycache" -RuleName "ruleone"

        ResourceGroupName : myGroup
        Name              : mycache
        FirewallRuleId    : /subscriptions/a559b6fd-3a84-40bb-a450-b0db5ed37dfe/resourceGroups/myGroup/providers/Microsoft.Cache/Redis/mycache/firewallRules/ruleone
        RuleName          : ruleone
        Type              : Microsoft.Cache/Redis/firewallRules
        StartIP           : 10.0.0.1
        EndIP             : 10.0.0.32
```

Dieser Befehl ruft die Firewallregel mit dem Namen ruleone aus dem Redis Cache mit dem Namen mycache ab.

### Beispiel 2: Alle Firewallregeln erhalten
```
PS C:\>Get-AzRedisCacheFirewallRule -Name "mycache"

        ResourceGroupName : myGroup
        Name              : mycache
        FirewallRuleId    : /subscriptions/a559b6fd-3a84-40bb-a450-b0db5ed37dfe/resourceGroups/myGroup/providers/Microsoft.Cache/Redis/mycache/firewallRules/ruleone
        RuleName          : ruleone
        Type              : Microsoft.Cache/Redis/firewallRules
        StartIP           : 10.0.0.1
        EndIP             : 10.0.0.32

        ResourceGroupName : myGroup
        Name              : mycache
        FirewallRuleId    : /subscriptions/a559b6fd-3a84-40bb-a450-b0db5ed37dfe/resourceGroups/myGroup/providers/Microsoft.Cache/Redis/mycache/firewallRules/ruletwo
        RuleName          : ruletwo
        Type              : Microsoft.Cache/Redis/firewallRules
        StartIP           : 10.0.0.33
        EndIP             : 10.0.0.64
```

Dieser Befehl ruft alle Firewallregeln aus dem Redis Cache mit dem Namen mycache ab.
