# Import-AzBlueprintWithArtifact

## SYNOPSIS
Importieren Sie eine Blaupausendatei im JSON-Format in ein Blaupausenobjekt, und speichern Sie sie in der angegebenen Abonnement- oder Verwaltungsgruppe.

## SYNTAX

```
Import-AzBlueprintWithArtifact -Name <String> [-SubscriptionId <String>] [-ManagementGroupId <String>]
 -InputPath <String> [-IncludeSubFolders] [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Importieren Sie eine Blaupausendefinition mit ihren Artefakten. 

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Import-AzBlueprintWithArtifact -Name MySimpleBlueprint -SubscriptionId 00000000-1111-0000-1111-000000000000 -InputPath  C:\Blueprints\SimpleBlueprint
```

Importieren Sie eine Blaupausendefinition mit ihren Artefakten, und speichern Sie sie in einem Abonnement.
