# Remove-AzBlueprintAssignment

## SYNOPSIS
Entfernen sie eine Blaupausenzuordnung aus einem Abonnement oder einer Verwaltungsgruppe.

## SYNTAX

### BySubscriptionAndName (Standard)
```
Remove-AzBlueprintAssignment [-Name] <String> [[-SubscriptionId] <String>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByManagementGroupAndName
```
Remove-AzBlueprintAssignment [-Name] <String> [-ManagementGroupId] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteBlueprintAssignmentByObject
```
Remove-AzBlueprintAssignment -InputObject <PSBlueprintAssignment> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Entfernen Sie eine Blaupause, die einem Abonnement zugewiesen wurde.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Remove-AzBlueprintAssignment -Name "myAssignment" -Subscription "00000000-1111-0000-1111-000000000000" -Confirm
```

Entfernen Sie die vom Namen angegebene Blaupausenzuordnung aus dem angegebenen Abonnement. Das Cmdlet wird zur Bestätigung aufgefordert, bevor der Befehl ausgeführt wird.
