# Get-AzBlueprintArtifact

## SYNOPSIS
Rufen Sie Artefakte aus einer Blaupausendefinition ab.

## SYNTAX

```
Get-AzBlueprintArtifact [-Name <String>] -Blueprint <PSBlueprintBase> [-BlueprintVersion <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Rufen Sie Artefakte aus einer Blaupausendefinition ab. Wenn keine Blaupausendefinitionsversion angegeben ist, wird die Entwurfsversion abgerufen. Ist keine Entwurfsversion verfügbar, wird die neueste veröffentlichte Blaupausendefinition zurückgegeben.
