# Get-AzIotSecurityAnalyticsAggregatedAlert

## SYNOPSIS
Zusammengefasste Benachrichtigung zur IoT-Sicherheit

## SYNTAX

### SolutionScope (Standard)
```
Get-AzIotSecurityAnalyticsAggregatedAlert -ResourceGroupName <String> -SolutionName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SolutionLevelResource
```
Get-AzIotSecurityAnalyticsAggregatedAlert -ResourceGroupName <String> -SolutionName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das Get-AzIotSecurityAnalyticsAggregatedAlert-Cmdlet gibt eine oder mehrere aggregierte Benachrichtigungen auf Geräten des iot-Hubs zurück.
Der Name der aggregierten Benachrichtigungen ist eine Kombination aus dem Warnungstyp und dem Benachrichtigungsaggragted-Datum, getrennt durch "/".

SystemSource: "Devices"

### Beispiel 2
```powershell
PS C:\> Get-AzIotSecurityAnalyticsAggregatedAlert -ResourceGroupName "MyResourceGroup" -SolutionName "MySolution"

Array of aggregated alert items as shown in example 1
```

Hier erhalten Sie eine Liste aller aggregierten Benachrichtigungen in der Projektmappe "MySolution" und der Ressourcengruppe "MyResourceGroup".
