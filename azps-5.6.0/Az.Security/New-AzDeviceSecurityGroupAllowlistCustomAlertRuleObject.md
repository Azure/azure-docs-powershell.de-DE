# New-AzDeviceSecurityGroupAllowlistCustomAlertRuleObject

## SYNOPSIS
Erstellen einer neuen benutzerdefinierten Warnungsregel für die Zulassenliste für Gerätesicherheitsgruppe (IoT Security)

## SYNTAX

```
New-AzDeviceSecurityGroupAllowlistCustomAlertRuleObject -Enabled <Boolean> -Type <String>
 -AllowlistValue <String[]> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das New-AzDeviceSecurityGroupAllowlistCustomAlertRuleObject-Cmdlet erstellt eine neue Liste der zulässigen benutzerdefinierten Warnungsregeln für Gerätesicherheitsgruppe in der IoT-Sicherheitslösung.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> New-AzDeviceSecurityGroupAllowlistCustomAlertRuleObject -Enabled $false -Type "LocalUserNotAllowed" -AllowlistValue @()

RuleType: "LocalUserNotAllowed"
DisplayName: "Login by a local user that isn't allowed"
Description: "Get an alert when a local user that isn't allowed logins to the device"
IsEnabled: false
ValueType: "String"
AllowlistValues: []
```

Erstellen einer neuen benutzerdefinierten Benachrichtigungsliste vom Typ "LocalUserNotAllowed" mit status set to disable
