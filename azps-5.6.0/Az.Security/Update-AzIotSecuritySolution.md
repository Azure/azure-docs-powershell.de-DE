# Update-AzIotSecuritySolution

## SYNOPSIS
Aktualisieren einer oder mehrere der folgenden Eigenschaften in der IoT-Sicherheitslösung: Tags, Empfehlungskonfiguration, benutzerdefinierte Ressourcen

## SYNTAX

### ResourceGroupLevelResource (Standard)
```
Update-AzIotSecuritySolution -Name <String> -ResourceGroupName <String> [-Tag <Hashtable>]
 [-UserDefinedResource <PSUserDefinedResources>]
 [-RecommendationsConfiguration <PSRecommendationConfiguration[]>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Update-AzIotSecuritySolution -ResourceId <String> [-Tag <Hashtable>]
 [-UserDefinedResource <PSUserDefinedResources>]
 [-RecommendationsConfiguration <PSRecommendationConfiguration[]>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Update-AzIotSecuritySolution -InputObject <PSIotSecuritySolution> [-Tag <Hashtable>]
 [-UserDefinedResource <PSUserDefinedResources>]
 [-RecommendationsConfiguration <PSRecommendationConfiguration[]>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das Update-AzIotSecuritySolution-Cmdlet eine oder mehrere der folgenden Eigenschaften in einer bestimmten IoT-Sicherheitslösung: Tags, Empfehlungskonfiguration, benutzerdefinierte Ressourcen.
Innerhalb der iot-Sicherheitslösung werden nur die angegebenen Eigenschaften aktualisiert.
Die IoT-Sicherheitslösung sammelt Sicherheitsdaten und -Ereignisse von iot-Geräten und iot-Hubs, um Bedrohungen zu verhindern und zu erkennen.
