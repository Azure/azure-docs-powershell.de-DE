# Remove-AzSentinelBookmark

## SYNOPSIS
Löschen sie eine Textmarke.

## SYNTAX

### BookmarkId. (Standard)
```
Remove-AzSentinelBookmark -ResourceGroupName <String> -WorkspaceName <String> -BookmarkId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzSentinelBookmark -InputObject <PSSentinelBookmark> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzSentinelBookmark** löscht eine Textmarke endgültig aus einem angegebenen Arbeitsbereich.
Sie können ein **Lesezeichenobjekt** mithilfe des Pipelineoperators übergeben oder die erforderlichen Parameter angeben.
Sie können den Parameter Bestätigen und die $ConfirmPreference Windows PowerShell verwenden, um zu steuern, ob das Cmdlet Sie zur Bestätigung aufgefordert.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Remove-AzSentinelBookmark -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -BookmarkId "MyBookmarkId"
```

Mit diesem Befehl wird die Textmarke aus dem Arbeitsbereich entfernt.
