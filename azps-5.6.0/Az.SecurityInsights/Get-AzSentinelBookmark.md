# Get-AzSentinelBookmark

## SYNOPSIS
Holen Sie sich eine Textmarke.

## SYNTAX

### WorkspaceScope (Standard)
```
Get-AzSentinelBookmark -ResourceGroupName <String> -WorkspaceName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### BookmarkId.
```
Get-AzSentinelBookmark -ResourceGroupName <String> -WorkspaceName <String> -BookmarkId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceId
```
Get-AzSentinelBookmark -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzSentinelBookmark-Cmdlet** ruft ein Lesezeichen aus dem angegebenen Arbeitsbereich ab.
Wenn Sie den *Parameter BookmarkId angeben,* wird ein einzelnes **Lesezeichenobjekt** zurückgegeben.
Wenn Sie den Parameter *BookmarkId* nicht angeben, wird ein Array zurückgegeben, das alle Lesezeichen im angegebenen Arbeitsbereich enthält.
Sie können das **Lesezeichenobjekt** verwenden, um die Textmarke zu aktualisieren, z. B. können Sie notizen die **Textmarke hinzufügen.**

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $Bookmarks = Get-AzSentinelBookmark -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName"
```

In diesem Beispiel werden alle Lesezeichen **im** angegebenen Arbeitsbereich ab- und dann in der $Bookmarks gespeichert.

### Beispiel 2
```powershell
PS C:\> $Bookmark = Get-AzSentinelBookmark -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -BookmarkId "MyBookmarkId"
```

In diesem Beispiel wird eine **Textmarke** im angegebenen Arbeitsbereich angezeigt und dann in der $Bookmark gespeichert.
