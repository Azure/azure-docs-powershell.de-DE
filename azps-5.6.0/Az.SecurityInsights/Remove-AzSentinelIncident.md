# Remove-AzSentinelIncident

## SYNOPSIS
Löschen eines Vorfalls

## SYNTAX

### IncidentId (Standard)
```
Remove-AzSentinelIncident -ResourceGroupName <String> -WorkspaceName <String> -IncidentId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzSentinelIncident -InputObject <PSSentinelIncident> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzSentinelIncident** löscht einen Vorfall endgültig aus einem angegebenen Arbeitsbereich.
Sie können ein **Incident-Objekt** mithilfe des Pipelineoperators übergeben oder die erforderlichen Parameter angeben.
Sie können den Parameter Bestätigen und die $ConfirmPreference Windows PowerShell verwenden, um zu steuern, ob das Cmdlet Sie zur Bestätigung aufgefordert.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Remove-AzSentinelIncident -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -IncidentId "MyIncidentId"
```

Mit diesem Befehl wird der Vorfall aus dem Arbeitsbereich entfernt.
