# Get-AzStorageSyncServer

## SYNOPSIS
Mit diesem Befehl werden alle Server aufgeführt, die bei einem bestimmten Speichersynchronisierungsdienst registriert sind.

## SYNTAX

### StringParameterSet (Standard)
```
Get-AzStorageSyncServer [-ResourceGroupName] <String> [-StorageSyncServiceName] <String> [-ServerId <Guid>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ObjectParameterSet
```
Get-AzStorageSyncServer [-ParentObject] <PSStorageSyncService> [-ServerId <Guid>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ParentStringParameterSet
```
Get-AzStorageSyncServer [-ParentResourceId] <String> [-ServerId <Guid>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Mit diesem Befehl werden alle Server aufgeführt, die bei einem bestimmten Speichersynchronisierungsdienst registriert sind. Es kann auch verwendet werden, um die Attribute der einzelnen registrierten Server auflisten.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Get-AzStorageSyncServer -ResourceGroupName "myResourceGroup" -StorageSyncServiceName "myStorageSyncServiceName"
```

Mit diesem Befehl werden alle Server bei einem bestimmten Speichersynchronisierungsdienst registriert.
