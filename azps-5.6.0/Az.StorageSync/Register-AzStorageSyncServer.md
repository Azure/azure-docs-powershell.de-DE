# Register-AzStorageSyncServer

## SYNOPSIS
Mit diesem Befehl wird ein Server bei einem Speichersynchronisierungsdienst registriert, der eine Vertrauensstellung erstellt. PowerShell oder das Azure-Portal können dann zum Konfigurieren der Synchronisierung auf diesem Server verwendet werden.

## SYNTAX

### StringParameterSet (Standard)
```
Register-AzStorageSyncServer [-ResourceGroupName] <String> [-StorageSyncServiceName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ObjectParameterSet
```
Register-AzStorageSyncServer [-ParentObject] <PSStorageSyncService> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ParentStringParameterSet
```
Register-AzStorageSyncServer [-ParentResourceId] <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Mit diesem Befehl wird ein Server bei einem Speichersynchronisierungsdienst registriert, der Ressource auf oberster Ebene für Azure File Sync. Es wird eine Vertrauensstellung zwischen Server und Speichersynchronisierungsdienst erstellt, die sichere Datenübertragungs- und Verwaltungskanäle sicherstellt. PowerShell oder das Azure-Portal können dann verwendet werden, um zu konfigurieren, welche Synchronisierungen auf diesem Server ausgeführt werden. Ein Server kann nur bei einem einzelnen Speichersynchronisierungsdienst registriert werden. Wenn Server jemals an der Synchronisierung derselben Gruppe von Dateien teilnehmen müssen, registrieren Sie sie beim gleichen Speichersynchronisierungsdienst.
Der Befehl muss lokal auf dem Server ausgeführt werden, der registriert werden soll – entweder direkt oder über eine Remote-PowerShell-Sitzung. Ein Remotecomputerobjekt kann nicht akzeptiert werden.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Register-AzStorageSyncServer -ResourceGroupName "myResourceGroup" -StorageSyncServiceName "myStorageSyncServiceName"
```

Mit diesem Befehl wird der lokale Server registriert, auf dem dieser Befehl ausgeführt wird.
