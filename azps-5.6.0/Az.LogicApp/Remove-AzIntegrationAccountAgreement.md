# Remove-AzIntegrationAccountAgreement

## SYNOPSIS
Entfernt einen Integrationskontovertrag.

## SYNTAX

```
Remove-AzIntegrationAccountAgreement -ResourceGroupName <String> -Name <String> -AgreementName <String>
 [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzIntegrationAccountAgreement** entfernt einen Integrationskontovertrag aus einer Azure-Ressourcengruppe.
Geben Sie den Namen des Integrationskontos, den Ressourcengruppennamen und den Vertragsnamen an.
Dieses Modul unterstützt dynamische Parameter.
Wenn Sie einen dynamischen Parameter verwenden möchten, geben Sie ihn in den Befehl ein.
Um die Namen dynamischer Parameter zu ermitteln, geben Sie nach dem Namen des Cmdlets einen Bindestrich (-) ein, und drücken Sie dann wiederholt die TAB-TASTE, um die verfügbaren Parameter zu durchkreisen.
Wenn Sie einen erforderlichen Vorlagenparameter weglassen, werden Sie vom Cmdlet zur Eingabe des Werts aufgefordert.

## BEISPIELE

### Beispiel 1: Entfernen eines Integrationskontovertrags nach Name
```
PS C:\>Remove-AzIntegrationAccountAgreement -ResourceGroupName "ResourceGroup11" -Name "IntegrationAccount31" -AgreementName "IntegrationAccountAgreement06" -Force
```

Mit diesem Befehl wird der Integrationskontovertrag mit dem Namen IntegrationAccountAgreement06 entfernt.
Der Befehl fordert Sie nicht zur Bestätigung auf.
