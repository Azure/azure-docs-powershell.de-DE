# Remove-AzIntegrationAccountCertificate

## SYNOPSIS
Entfernt ein Integrationskontozertifikat aus einer Ressourcengruppe.

## SYNTAX

```
Remove-AzIntegrationAccountCertificate -ResourceGroupName <String> -Name <String> -CertificateName <String>
 [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzIntegrationAccountCertificate** entfernt ein Integrationskontozertifikat aus einer Ressourcengruppe.
Geben Sie den Namen des Integrationskontos, den Ressourcengruppennamen und den Zertifikatnamen an.
Dieses Modul unterstützt dynamische Parameter.
Wenn Sie einen dynamischen Parameter verwenden möchten, geben Sie ihn in den Befehl ein.
Um die Namen dynamischer Parameter zu ermitteln, geben Sie nach dem Namen des Cmdlets einen Bindestrich (-) ein, und drücken Sie dann wiederholt die TAB-TASTE, um die verfügbaren Parameter zu durchkreisen.
Wenn Sie einen erforderlichen Vorlagenparameter weglassen, werden Sie vom Cmdlet zur Eingabe des Werts aufgefordert.

## BEISPIELE

### Beispiel 1: Entfernen eines Integrationskontozertifikats
```
PS C:\>Remove-AzIntegrationAccountCertificate -ResourceGroupName "ResourceGroup11" -Name "IntegrationAccount31" -CertificateName "IntegrationAccountCertificate01"
```

Mit diesem Befehl wird das Integrationskontozertifikat mit dem Namen IntegrationAccount31 entfernt.
