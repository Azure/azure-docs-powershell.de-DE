# Disable-AzStorageBlobDeleteRetentionPolicy

## SYNOPSIS
Deaktivieren Sie die Löschaufbewahrungsrichtlinie für den Azure Storage Blob-Dienst.

## SYNTAX

### AccountName (Standard)
```
Disable-AzStorageBlobDeleteRetentionPolicy [-ResourceGroupName] <String> [-StorageAccountName] <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AccountObject
```
Disable-AzStorageBlobDeleteRetentionPolicy -StorageAccount <PSStorageAccount> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### BlobServicePropertiesResourceId
```
Disable-AzStorageBlobDeleteRetentionPolicy [-ResourceId] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Disable-AzStorageBlobDeleteRetentionPolicy** deaktiviert die Löschaufbewahrungsrichtlinie für den Azure Storage Blob-Dienst.

## BEISPIELE

### Beispiel 1: Deaktivieren der Löschaufbewahrungsrichtlinie für den Blobdienst
```
C:\PS>Disable-AzStorageBlobDeleteRetentionPolicy -ResourceGroupName "myresourcegroup" -AccountName "mystorageaccount" -PassThru

Enabled Days
------- ----
  False
```

Mit diesem Befehl wird die Löschaufbewahrungsrichtlinie für den Blobdienst deaktiviert.
