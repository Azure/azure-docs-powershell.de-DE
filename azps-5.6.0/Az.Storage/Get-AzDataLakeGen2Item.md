# Get-AzDataLakeGen2Item

## SYNOPSIS
Ruft die Details einer Datei oder eines Verzeichnisses in einem Dateisystem ab.

## SYNTAX

```
Get-AzDataLakeGen2Item [-FileSystem] <String> [-Path <String>] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzDataLakeGen2Item-Cmdlet** ruft die Details einer Datei oder eines Verzeichnisses in einem Dateisystem in einem Azure-Speicherkonto ab.
Dieses Cmdlet funktioniert nur, wenn hierarchischer Namespace für das Speicherkonto aktiviert ist. Diese Art von Konto kann durch Ausführen des Cmdlets "New-AzStorageAccount" mit "-EnableHierarchicalNamespace $true" erstellt werden.
