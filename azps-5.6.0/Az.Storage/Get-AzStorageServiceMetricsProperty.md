# Get-AzStorageServiceMetricsProperty

## SYNOPSIS
Ruft Metrikeigenschaften für den Azure Storage-Dienst ab.

## SYNTAX

```
Get-AzStorageServiceMetricsProperty [-ServiceType] <StorageServiceType> [-MetricsType] <ServiceMetricsType>
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzStorageServiceMetricsProperty-Cmdlet** ruft Metrikeigenschaften für den Azure Storage-Dienst ab.

## BEISPIELE

### Beispiel 1: Erhalten von Metrikeigenschaften für den Blob-Dienst
```
C:\PS>Get-AzStorageServiceMetricsProperty -ServiceType Blob -MetricsType Hour
```

Dieser Befehl ruft Metrikeigenschaften für den Blobspeicher für den Metriktyp Stunde ab.
