# New-AzStorageAccountKey

## SYNOPSIS
Regeneriert einen Speicherschlüssel für ein Azure Storage-Konto.

## SYNTAX

```
New-AzStorageAccountKey [-ResourceGroupName] <String> [-Name] <String> [-KeyName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet New-AzStorageAccountKey** generiert einen Speicherschlüssel für ein Azure Storage-Konto neu.

## BEISPIELE

### Beispiel 1: Regenerieren eines Speicherschlüssels
```
PS C:\>New-AzStorageAccountKey -ResourceGroupName "MyResourceGroup" -Name "mystorageaccount" -KeyName "key1"
```

Mit diesem Befehl wird ein Speicherschlüssel für das angegebene Speicherkonto neu generiert.
