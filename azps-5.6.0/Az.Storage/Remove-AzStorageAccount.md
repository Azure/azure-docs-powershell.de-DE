# Remove-AzStorageAccount

## SYNOPSIS
Entfernt ein Speicherkonto aus Azure.

## SYNTAX

```
Remove-AzStorageAccount [-ResourceGroupName] <String> [-Name] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzStorageAccount** entfernt ein Speicherkonto aus Azure.

## BEISPIELE

### Beispiel 1: Entfernen eines Speicherkontos
```
PS C:\>Remove-AzStorageAccount -ResourceGroupName "RG01" -AccountName "mystorageaccount"
```

Mit diesem Befehl wird das angegebene Speicherkonto entfernt.
