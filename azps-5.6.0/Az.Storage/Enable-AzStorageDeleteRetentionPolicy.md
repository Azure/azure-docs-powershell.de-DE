# Enable-AzStorageDeleteRetentionPolicy

## SYNOPSIS
Aktivieren Sie die Aufbewahrungsrichtlinie zum Löschen für den Azure Storage Blob-Dienst.

## SYNTAX

```
Enable-AzStorageDeleteRetentionPolicy [-RetentionDays] <Int32> [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Enable-AzStorageDeleteRetentionPolicy** aktiviert die Löschaufbewahrungsrichtlinie für den Azure Storage Blob-Dienst.

## BEISPIELE

### Beispiel 1: Aktivieren der Löschaufbewahrungsrichtlinie für den Blobdienst
```
C:\PS>Enable-AzStorageDeleteRetentionPolicy -RetentionDays 3
```

Dieser Befehl aktiviert die Löschaufbewahrungsrichtlinie für den Blobdienst und setzt gelöschte Blob-Aufbewahrungstage auf 3.
