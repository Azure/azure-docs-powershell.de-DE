# New-AzStorageQueueSASToken

## SYNOPSIS
Generiert ein Signaturtoken für den freigegebenen Zugriff für eine Azure-Speicherwarteschlange.

## SYNTAX

### SasPolicy
```
New-AzStorageQueueSASToken [-Name] <String> -Policy <String> [-Protocol <SharedAccessProtocol>]
 [-IPAddressOrRange <String>] [-StartTime <DateTime>] [-ExpiryTime <DateTime>] [-FullUri]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SasPermission
```
New-AzStorageQueueSASToken [-Name] <String> [-Permission <String>] [-Protocol <SharedAccessProtocol>]
 [-IPAddressOrRange <String>] [-StartTime <DateTime>] [-ExpiryTime <DateTime>] [-FullUri]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **New-AzStorageQueueSASToken-Cmdlet** generiert ein Token für freigegebene Zugriffssignaturen für eine Azure-Speicherwarteschlange.

## BEISPIELE

### Beispiel 1: Generieren eines Warteschlangen-SAS-Tokens mit voller Berechtigung
```
PS C:\>New-AzStorageQueueSASToken -Name "Test" -Permission raup
```

In diesem Beispiel wird ein SAS-Warteschlangentoken mit voller Berechtigung generiert.
