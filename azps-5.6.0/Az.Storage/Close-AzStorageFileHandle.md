# Close-AzStorageFileHandle

## SYNOPSIS
Schließt die Dateihandles einer Dateifreigabe, eines Dateiverzeichnisses oder einer Datei.

## SYNTAX

### ShareNameCloseAll (Standard)
```
Close-AzStorageFileHandle [-ShareName] <String> [[-Path] <String>] [-Recursive] [-CloseAll]
 [-Context <IStorageContext>] [-PassThru] [-AsJob] [-ServerTimeoutPerRequest <Int32>]
 [-ClientTimeoutPerRequest <Int32>] [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Close-AzStorageFileHandle** schließt dateiziehpunkte einer Dateifreigabe, eines Dateiverzeichnisses oder einer Datei.

## BEISPIELE

### Beispiel 1: Listet alle Dateifreigaben des aktuellen Speicherkontos auf, und schließen Sie alle Dateiziehpunkte der Dateifreigaben rekursiv.
```
PS C:\>Get-AzStorageShare | Close-AzStorageFileHandle -CloseAll -Recursive
```

Mit diesem Befehl werden alle Dateifreigaben des aktuellen Speicherkontos aufgeführt, und alle Dateiziehpunkte der Dateifreigaben werden rekursiv geschlossen.

### Beispiel 2: Schließen Sie alle Dateiziehpunkte in einem Dateiverzeichnis rekursiv, und zeigen Sie die Anzahl der Ziehpunkte für geschlossene Dateien an.
```
PS C:\>Close-AzStorageFileHandle -ShareName "mysharename" -Path 'dir1/dir2' -Recursive -CloseAll -PassThru
10
```

Mit diesem Befehl werden alle Dateiziehpunkte in einem Dateiverzeichnis geschlossen und die Anzahl der geschlossenen Dateihandles angezeigt.

### Beispiel 3: Schließen aller Dateiziehpunkte, die vor 1 Tag in einem Dateiverzeichnis geöffnet wurden
```
PS C:\>Get-AzStorageFileHandle -ShareName "mysharename" -Path 'dir1/dir2' -Recursive | ? {$_.OpenTime.DateTime.AddDays(1) -lt (Get-Date)} | Close-AzStorageFileHandle -ShareName "mysharename"
```

Mit diesem Befehl werden alle Dateiziehpunkte in einem Dateiverzeichnis rekursiv aufgeführt, die Ziehpunkte, die vor einem Tag geöffnet wurden, herausfiltert und dann geschlossen.

### Beispiel 4: Schließen aller Dateiziehpunkte für eine Datei
```
PS C:\>Close-AzStorageFileHandle -ShareName "mysharename" -Path 'dir1/dir2/test.txt' -CloseAll
```

Mit diesem Befehl werden alle Dateihandles für eine Datei geschlossen.
