# Get-AzDtlVMsPerLabPolicy

## SYNOPSIS
Ruft die virtuellen Computer pro Lab-Richtlinie einer Übungseinheit in DevTest Labs ab.

## SYNTAX

```
Get-AzDtlVMsPerLabPolicy [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzDtlVMsPerLabPolicy-Cmdlet** ruft die virtuellen Computer pro Lab-Richtlinie einer Übung ab, wodurch Sie die Gesamtanzahl der in einer Übung zugelassenen virtuellen Computer festlegen können.
Das Cmdlet gibt den aktivierten oder deaktivierten Status der Richtlinie und die Gesamtanzahl der virtuellen Computer zurück, die in der in der Richtlinie festgelegten Übungseinheit zulässig sind.
