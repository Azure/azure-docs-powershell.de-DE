# Set-AzDtlAllowedVMSizesPolicy

## SYNOPSIS
Legt die Richtlinie für zulässige Größen virtueller Computer in einer Übungseinheit in DevTest Labs fest.

## SYNTAX

### Aktivieren (Standard)
```
Set-AzDtlAllowedVMSizesPolicy [[-VmSizes] <String[]>] [-Enable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Deaktivieren
```
Set-AzDtlAllowedVMSizesPolicy [[-VmSizes] <String[]>] [-Disable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Set-AzDtlAllowedVMSizesPolicy** legt die Richtlinie für zulässige Größen virtueller Computer fest, die eine Liste der in einer Übung zulässigen Größen virtueller Computer angibt.
Das Cmdlet verwendet die angegebene Ressourcengruppe und den Namen der Übungseinheit, um die Richtlinie zu festlegen.
