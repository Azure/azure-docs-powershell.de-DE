# Set-AzDtlAutoStartPolicy

## SYNOPSIS
Legt die Richtlinie für den automatischen Start einer Übungseinheit in DevTest Labs fest.

## SYNTAX

### Aktivieren (Standard)
```
Set-AzDtlAutoStartPolicy [[-Time] <DateTime>] [[-Days] <DayOfWeek[]>] [-Enable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Deaktivieren
```
Set-AzDtlAutoStartPolicy [[-Time] <DateTime>] [[-Days] <DayOfWeek[]>] [-Disable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Set-AzDtlAutoStartPolicy-Cmdlet** legt die Richtlinie für den automatischen Start einer Übung fest, die es virtuellen Laborautomaten ermöglicht, für den automatischen Start geplant zu werden.
Das Cmdlet verwendet die angegebene Ressourcengruppe und den Namen der Übungseinheit, um die Richtlinie zu festlegen.

## BEISPIELE

### Beispiel 1

Legt die Richtlinie für den automatischen Start einer Übungseinheit in DevTest Labs fest. (automatisch generiert)

```powershell <!-- Aladdin Generated Example --> 
Set-AzDtlAutoStartPolicy -Days Sunday -Enable -LabName <String> -ResourceGroupName MyResourceGroup -Time <DateTime>
```
