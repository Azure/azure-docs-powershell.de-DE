# Remove-AzRelayNamespace

## SYNOPSIS
Entfernt den Namespace aus der angegebenen Ressourcengruppe. 

## SYNTAX

```
Remove-AzRelayNamespace [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzRelayNamespace** entfernt den Namespace aus der angegebenen Ressourcengruppe.

## BEISPIELE

### Beispiel 1
```
PS C:\> Remove-AzRelayNamespace -ResourceGroupName Default-ServiceBus-WestUS -Name TestNameSpace-Relay1
```

Entfernt den Relay-Namespace `TestNameSpace-Relay1` aus der angegebenen Ressourcengruppe. `Default-ServiceBus-WestUS`
