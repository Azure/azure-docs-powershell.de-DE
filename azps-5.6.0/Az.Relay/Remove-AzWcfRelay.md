# Remove-AzWcfRelay

## SYNOPSIS
Entfernt wcfRelay aus dem angegebenen Relay-Namespace.

## SYNTAX

```
Remove-AzWcfRelay [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Remove-AzWcfRelay** entfernt wcfRelay aus dem angegebenen Relay-Namespace.

## BEISPIELE

### Beispiel 1
```
PS C:\> Remove-AzWcfRelay -ResourceGroupName Default-ServiceBus-WestUS -NamespaceName TestNameSpace-Relay1 -Name TestWCFRelay1
```

Entfernt wcfRelay `TestWCFRelay1` aus dem `TestNameSpace-Relay1` -Namespace.
