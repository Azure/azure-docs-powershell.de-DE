# Get-AzRelayOperation

## SYNOPSIS
Liste unterstützter Relayvorgänge

## SYNTAX

```
Get-AzRelayOperation [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Im **Get-AzRelayOperation-Cmdlet** werden die vom Relay unterstützten Vorgänge aufgeführt.
