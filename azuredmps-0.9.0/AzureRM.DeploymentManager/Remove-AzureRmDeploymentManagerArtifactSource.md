# Remove-AzureRmDeploymentManagerArtifactSource

## Synopsis
Löscht eine Artefakt-Quelle.

## Syntax

### Interaktiv (Standard)
```
Remove-AzureRmDeploymentManagerArtifactSource [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Remove-AzureRmDeploymentManagerArtifactSource [-ResourceId] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzureRmDeploymentManagerArtifactSource [-ArtifactSource] <PSArtifactSource> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Remove-AzureRmDeploymentManagerArtifactSource** " löscht eine Artefakt-Quelle mit dem Namen und dem Namen der Ressourcengruppe. Alternativ können Sie das ArtifactSource-Objekt oder die resourcecode-Objekt bereitstellen.

## Beispiele

### Beispiel 1: Löschen einer Artefakt-Quelle
```powershell
PS C:\> Remove-AzureRmDeploymentManagerArtifactSource -ResourceGroupName "ContosoResourceGroup" -Name "ContosoArtifactSource"
```

Dieser Befehl löscht eine Artefakt-Quelle namens ContosoArtifactSource in ContosoResourceGroup.

### Beispiel 2: Löschen einer Artefakt-Quelle mit dem Ressourcenbezeichner
```powershell
PS C:\> Remove-AzureRmDeploymentManagerArtifactSource -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/artifactSources/ContosoArtifactSource"
```

Dieser Befehl löscht eine Artefakt-Quelle namens ContosoArtifactSource in ContosoResourceGroup.

### Beispiel 3: Löschen einer Artefakt-Quelle mit einem Objekt
```powershell
PS C:\> Remove-AzureRmDeploymentManagerArtifactSource -ArtifactSource $artifactSourceObject
```

Dieser Befehl löscht eine Artefakt-Quelle, deren Name und ResourceGroup den Namen-und ResourceGroupName-Eigenschaften der $artifactSourceObject entsprechen.
