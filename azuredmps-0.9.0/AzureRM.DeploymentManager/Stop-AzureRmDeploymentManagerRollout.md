# Stop-AzureRmDeploymentManagerRollout

## Synopsis
Beendet einen Rollout in Bearbeitung.

## Syntax

### Interaktiv (Standard)
```
Stop-AzureRmDeploymentManagerRollout [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Stop-AzureRmDeploymentManagerRollout [-ResourceId] <String> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Stop-AzureRmDeploymentManagerRollout [-Rollout] <PSRollout> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Stop-AzureRmDeploymentManagerRollout** verhindert, dass ein Rollout in Bearbeitung ist, und gibt ein Objekt zurück, das den aktuellen Status des Rollouts darstellt.
Geben Sie den Rollout nach dem Namen und dem Namen der Ressourcengruppe an. Alternativ können Sie das Rollout-Objekt oder die "Resourcen-Nr" bereitstellen.

Beachten Sie, dass ein Rollout nicht mehr fortgesetzt oder neu gestartet werden kann, nachdem ein Rollout beendet wurde. Sie können nur einen neuen Rollout erstellen.

## Beispiele

### Beispiel 1
```powershell
PS C:\> Stop-AzureRmDeploymentManagerRollout -ResourceGroupName ContosoResourceGroup -Name ContosoRollout -SkipSucceeded
```

Dieser Befehl beendet ein Rollout mit dem Namen ContosoRollout in der ContosoResourceGroup. 

### Beispiel 2: Beenden eines Rollouts mit dem Ressourcenbezeichner
```powershell
PS C:\> Restart-AzureRmDeploymentManagerRollout -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/rollouts/ContosoRollout"
```

Dieser Befehl beendet ein Rollout mit dem Namen ContosoRollout in der ContosoResourceGroup.

### Beispiel 3: Beenden eines Rollouts mit dem Rollout-Objekt
```powershell
PS C:\> Get-AzureRmDeploymentManagerRollout -Rollout $rolloutObject
```

Mit diesem Befehl wird ein Rollout beendet, dessen Name und ResourceGroup den Namen-und ResourceGroupName-Eigenschaften der $rolloutObject entsprechen.
