# Remove-AzureRmDeploymentManagerRollout

## Synopsis
Löscht einen Rollout.

## Syntax

### Interaktiv (Standard)
```
Remove-AzureRmDeploymentManagerRollout [-ResourceGroupName] <String> [-Name] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Remove-AzureRmDeploymentManagerRollout [-ResourceId] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzureRmDeploymentManagerRollout [-Rollout] <PSRollout> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureRmDeploymentManagerRollout** löscht ein Rollout in einem Terminal Zustand.
Geben Sie den Rollout nach dem Namen und dem Namen der Ressourcengruppe an. Alternativ können Sie das Rollout-Objekt oder die "Resourcen-Nr" bereitstellen.

## Beispiele

### Beispiel 1
```powershell
PS C:\> Remove-AzureRmDeploymentManagerRollout -ResourceGroupName ContosoResourceGroup -Name ContosoRollout
```

Dieser Befehl löscht ein Rollout mit dem Namen ContosoRollout in der ContosoResourceGroup.

### Beispiel 2: Löschen eines Rollouts mit dem Ressourcenbezeichner
```powershell
PS C:\> Remove-AzureRmDeploymentManagerRollout -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/rollouts/ContosoRollout"
```

Dieser Befehl löscht ein Rollout mit dem Namen ContosoRollout in der ContosoResourceGroup.

### Beispiel 3: Löschen eines Rollouts mit dem Rollout-Objekt
```powershell
PS C:\> Remove-AzureRmDeploymentManagerRollout -Rollout $rolloutObject
```

Mit diesem Befehl wird ein Rollout gelöscht, dessen Name und ResourceGroup den Namen-und ResourceGroupName-Eigenschaften der $rolloutObject entsprechen.
