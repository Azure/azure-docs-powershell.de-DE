# Remove-AzureStorageQueue

## Synopsis
Entfernt eine Speicherwarteschlange.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Remove-AzureStorageQueue [-Name] <String> [-Force] [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureStorageQueue** entfernt eine Speicherwarteschlange.

## Beispiele

### Beispiel 1: Entfernen einer Speicherwarteschlange nach Namen
```
PS C:\>Remove-AzureStorageQueue "ContosoQueue01"
```

Dieser Befehl entfernt eine Warteschlange mit dem Namen ContosoQueue01.

### Beispiel 2: Entfernen mehrerer Speicher Warteschlangen
```
PS C:\>Get-AzureStorageQueue "Contoso*" | Remove-AzureStorageQueue
```

Mit diesem Befehl werden alle Warteschlangen mit Namen entfernt, die mit Contoso beginnen.
