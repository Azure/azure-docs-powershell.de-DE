# Remove-AzureRmADGroupMember

## Synopsis
Entfernt einen Benutzer aus einer Anzeigengruppe.

## Beschreibung
Entfernt einen Benutzer aus einer Anzeigengruppe.

## Beispiele

### Beispiel 1: Entfernen eines Benutzers aus einer Gruppe nach Objekt-ID

```
PS C:\> Remove-AzureRmADGroup -MemberObjectId D9076BBC-D62C-4105-9C78-A7F5BC4A3405 -GroupObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE
```

Entfernt den Benutzer mit der Objekt-ID "D9076BBC-D62C-4105-9C78-A7F5BC4A3405" aus der Gruppe mit der Objekt-ID "85F89C90-780E-4AA6-9F4F-6F268D322EEE".

### Beispiel 2 – Entfernen eines Benutzers aus einer Gruppe durch Verrohrung

```
PS C:\> Get-AzureRmADGroup -ObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE | Remove-AzureRmADGroupMember -MemberObjectId D9076BBC-D62C-4105-9C78-A7F5BC4A3405
```

Ruft die Gruppe mit der Objekt-ID "85F89C90-780E-4AA6-9F4F-6F268D322EEE" ab und leitet Sie an das Remove-AzureRmADGroupMember-Cmdlet weiter, um den Benutzer zu dieser Gruppe zu entfernen.
