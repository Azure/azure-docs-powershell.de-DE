# Register-AzureRmResourceProvider

## Synopsis
Registriert einen Ressourcenanbieter.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Register-AzureRmResourceProvider -ProviderNamespace <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Register-AzureRmResourceProvider** registriert einen Azure-Ressourcenanbieter.

## Beispiele

### Beispiel 1: Registrieren eines Anbieters
```
PS C:\>Register-AzureRmResourceProvider -ProviderNamespace Microsoft.Network
```

Dadurch wird der Microsoft. Network-Anbieter für Ihr Konto registriert.
