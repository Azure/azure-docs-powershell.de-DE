## Beschreibung
Das Cmdlet **New-AzureServiceADDomainExtensionConfig** generiert die Konfiguration für die Active Directory-Domänenerweiterung (AD) für Cloud-Dienste.

## Beispiele

### Beispiel 1: Angeben einer AD-Domänenkonfiguration
```
PS C:\> $ExtensionCfg = New-AzureServiceADDomainExtensionConfig -Role WorkerRole1 -DomainName $Domain -Credential $Cred -JoinOption 35;

PS C:\> New-AzureDeployment -ServiceName $CloudSvc -Slot "Production" -Package $Pkg -Configuration $Config -ExtensionConfiguration $ExtensionCfg;
```

Mit diesem Befehl wird eine Konfiguration für die AD-Domänenerweiterung generiert.
