# Set-AzureDns

## Synopsis
Ändert die IP-Adresse eines DNS-Servers.

## Syntax

```
Set-AzureDns [-Name] <String> [-IPAddress] <String> [-ServiceName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Satz-AzureDns** " ändert die IP-Adresse eines DNS-Servers für einen Azure-Dienst.

## Beispiele

### Beispiel 1: Ändern der IP-Adresse eines DNS-Servers
```
PS C:\> Set-AzureDns -ServiceName "ContosoService" -IPAddress 10.1.2.5 -Name "Dns07"
```

Dieser Befehl ändert die IP-Adresse des DNS-Servers mit dem Namen Dns07 für den Dienst mit dem Namen ContosoService.
