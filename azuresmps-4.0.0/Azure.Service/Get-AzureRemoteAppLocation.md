# Get-AzureRemoteAppLocation

## Synopsis
Listet die Speicherorte auf, die von Azure RemoteApp unterstützt werden.

## Syntax

```
Get-AzureRemoteAppLocation [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureRemoteAppLocation** " listet die Speicherorte auf, die von Azure RemoteApp unterstützt werden.

## Parameter

### -Profil

Wenn Sie kein Profil angeben, liest dieses Cmdlet aus dem lokalen Standardprofil.
