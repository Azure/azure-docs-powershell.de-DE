# Disable-AzureServiceProjectRemoteDesktop

## Synopsis
Deaktiviert den Remote Desktop Zugriff auf einen clouddienst.

## Syntax

```
Disable-AzureServiceProjectRemoteDesktop [-PassThru] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung
In diesem Thema wird das Cmdlet in der 0.8.10-Version des Microsoft Azure PowerShell-Moduls beschrieben.
Wenn Sie die Version des verwendeten Moduls abrufen möchten, geben Sie in der Azure PowerShell-Konsole `(Get-Module -Name Azure).Version` .

Mit der Funktion " **Disable-AzureServiceProjectRemoteDesktop** " wird der Remotedesktopzugriff auf einen gehosteten Dienst deaktiviert.
Sie müssen den Dienst mithilfe des Cmdlets **Publish-AzureServiceProject** veröffentlichen, nachdem Sie den Remote Desktop Zugriff deaktiviert haben, damit die Änderung wirksam wird.
