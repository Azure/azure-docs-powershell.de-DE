# Export-AzureVM

## Synopsis
Exportiert einen Zustand des Azure Virtual Machine in eine Datei.

## Syntax

```
Export-AzureVM [-ServiceName] <String> [-Name] <String> [-Path] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Export-AzureVM** exportiert den Zustand einer virtuellen Maschine in eine XML-Datei.

Beim Ausführen von " **Export-AzureVM** ", gefolgt von " **Remove-AzureVM** " und dann " **Import-AzureVM** ", um einen virtuellen Computer neu zu erstellen, kann die resultierende virtuelle Maschine eine andere IP-Adresse als das Original aufweisen.

## Beispiele

### Beispiel 1: Exportieren eines virtuellen Computers
```
PS C:\> Export-AzureVM -ServiceName "ContosoService" -Name "ContosoRole06" -Path "C:\vms\VMstate.xml"
```

Dieser Befehl exportiert den Zustand des angegebenen virtuellen Computers in die VMstate.xml-Datei.
