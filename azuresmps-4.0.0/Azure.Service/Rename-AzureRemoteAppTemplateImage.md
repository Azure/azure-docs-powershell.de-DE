# Rename-AzureRemoteAppTemplateImage

## Synopsis
Benennt ein Azure RemoteApp-Vorlagenbild um.

## Syntax

```
Rename-AzureRemoteAppTemplateImage [-ImageName] <String> [-NewName] <String> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **Rename-AzureRemoteAppTemplateImage** wird ein Azure RemoteApp-Vorlagenbild umbenannt.

## Beispiele

### Beispiel 1: Umbenennen eines Vorlagenbilds
```
PS C:\> Rename-AzureRemoteAppTemplateImage -ImageName "ContosoApps" -NewName "ContosoFinanceApps"
```

Mit diesem Befehl wird das Azure RemoteApp-Vorlagenbild mit dem Namen ContosoApps in ContosoFinanceApps umbenannt.
