# Remove-AzureAutomationAccount

## Synopsis

Entfernt ein Automatisierungs Konto.

## Syntax

```
Remove-AzureAutomationAccount -Name <String> [-Force] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung

[!INCLUDE [aa-deprecation](../include/aa-deprecation.md)]

Das Cmdlet **Remove-AzureAutomationAccount** entfernt ein Konto aus der Microsoft Azure-Automatisierung.

## Beispiele

### Beispiel 1: Entfernen eines Automatisierungs Kontos
```
PS C:\> Remove-AzureAutomationAccount -Name "MyAutomationAccount" -Force
```

Mit diesem Befehl wird ein Automatisierungs Kontonamens MyAutomationAccount entfernt, ohne dass die Benutzerüberprüfung angefordert wird.
