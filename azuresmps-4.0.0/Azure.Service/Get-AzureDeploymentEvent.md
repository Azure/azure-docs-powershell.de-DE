# Get-AzureDeploymentEvent

## Synopsis
Ruft Informationen zu den von Azure initiierten Ereignissen ab, die sich auf virtuelle Computer und Cloud-Dienste auswirken.

## Syntax

### GetDeploymentEventBySlot (Standard)
```
Get-AzureDeploymentEvent [-ServiceName] <String> [-StartTime] <DateTime> [-EndTime] <DateTime>
 [[-Slot] <String>] [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

### GetDeploymentEventByName
```
Get-AzureDeploymentEvent [-ServiceName] <String> [-StartTime] <DateTime> [-EndTime] <DateTime>
 [-DeploymentName] <String> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureDeploymentEvent** " Ruft Informationen zu Ereignissen ab, die Azure initiiert, die Auswirkungen auf virtuelle Computer und Cloud-Dienste haben.
Zu diesen Ereignissen gehören geplante Wartungsereignisse.
Dieses Cmdlet gibt eine Liste von Ereignissen zurück, die die Rolleninstanz oder den virtuellen Computer identifizieren, die sich auf den Grund für die Auswirkung und die Startzeit des Ereignisses auswirkt.

## Beispiele

### 1:
```
Get-AzureDeploymentEvent -DeploymentName "ConstosoDeployment" -ServiceName "ContosoService"
```
