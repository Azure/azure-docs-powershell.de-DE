# Get-AzureVirtualNetworkGateway

## Synopsis
Ruft ein virtuelles Netzwerkgateway ab.

## Syntax

```
Get-AzureVirtualNetworkGateway [-GatewayId <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureVirtualNetworkGateway** " Ruft ein Azure Virtual Network-Gateway ab.
