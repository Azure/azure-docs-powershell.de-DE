# Remove-AzureStaticVNetIP

## Synopsis
Entfernt die statischen virtuellen Netzwerk-IP-Adressinformationen aus einem virtuellen Computerobjekt (sofern vorhanden).

## Syntax

```
Remove-AzureStaticVNetIP -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureStaticVNetIP** entfernt die IP-Adressinformationen (static Virtual Network, VNet) aus einem virtuellen Computerobjekt (sofern vorhanden).
