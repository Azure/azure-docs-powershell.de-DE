# Save-AzureVMImage

## Synopsis
Erfasst und speichert das Bild eines gestoppten Azure Virtual Machine.

## Syntax

```
Save-AzureVMImage [-ServiceName] <String> [-Name] <String> [-ImageName] <String> [[-ImageLabel] <String>]
 [[-OSState] <String>] [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Save-AzureVMImage** erfasst und speichert das Bild eines gestoppten Azure Virtual Machine.
Führen Sie bei Windows-virtuellen Computern das Sysprep-Tool aus, um das Bild vorzubereiten, bevor es erfasst wird.
Nachdem das Bild aufgenommen wurde, wird der virtuelle Computer gelöscht.

## Beispiele

### Beispiel 1: Speichern einer vorhandenen virtuellen Maschine und anschließendes Löschen aus einer Bereitstellung
```
PS C:\> Save-AzureVMImage -ServiceName "MyService" -Name "MyVM" -NewImageName "MyBaseImage" -NewImageLabel "MyBaseVM"
```

Dieser Befehl zeichnet einen vorhandenen virtuellen Computer auf und löscht ihn aus der Bereitstellung.
