# Disable-AzureWebsiteApplicationDiagnostic

## Synopsis
Deaktiviert die Anwendungs Diagnose für eine Azure-Website.

## Syntax

```
Disable-AzureWebsiteApplicationDiagnostic [-PassThru] [-File] [-TableStorage] [-BlobStorage] [-Name <String>]
 [-Slot <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung
In diesem Thema wird das Cmdlet in der 0.8.10-Version des Microsoft Azure PowerShell-Moduls beschrieben.
Wenn Sie die Version des verwendeten Moduls abrufen möchten, geben Sie in der Azure PowerShell-Konsole `(Get-Module -Name Azure).Version` .

Deaktiviert die Anwendungs Diagnose für eine Azure-Website.
Deaktiviert die Protokollierung, die für die Speicherung in einem Dateisystem oder auf Azure konfiguriert ist.

## Beispiele

### Beispiel 1: Deaktivieren der Anwendungs Protokollierungsdatei
```
PS C:\> Disable-AzureWebsiteApplicationDiagnostic -Name MyWebsite -File
```

Im folgenden Beispiel wird die Anwendungsprotokollierung für das Dateisystem deaktiviert.

### Beispiel 2: Deaktivieren der Protokollierung mithilfe des Azure-Speichers
```
PS C:\> Disable-AzureWebsiteApplicationDiagnostic -Name MyWebsite -Storage
```

Im folgenden Beispiel wird die Anwendungsprotokollierung mithilfe von Speicher deaktiviert.
