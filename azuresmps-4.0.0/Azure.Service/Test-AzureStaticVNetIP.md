# Test-AzureStaticVNetIP

## Synopsis
Testet die Verfügbarkeit einer statischen virtuellen Netzwerk-IP-Adresse und ruft eine Liste mit Vorschlägen ab, wenn die abgefragte Adresse nicht verfügbar ist.

## Syntax

```
Test-AzureStaticVNetIP [-VNetName] <String> [-IPAddress] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Test-AzureStaticVNetIP** testet die Verfügbarkeit einer statischen virtuellen Netzwerk-IP-Adresse und ruft eine Liste mit Vorschlägen ab, wenn die abgefragte Adresse nicht verfügbar ist.
