# Remove-AzureVirtualNetworkGatewayConnection

## Synopsis
Entfernt eine virtuelle Netzwerkgateway-Verbindung.

## Syntax

```
Remove-AzureVirtualNetworkGatewayConnection -GatewayId <String> -ConnectedEntityId <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureVirtualNetworkGatewayConnection** entfernt eine virtuelle Netzwerkgateway-Verbindung.
