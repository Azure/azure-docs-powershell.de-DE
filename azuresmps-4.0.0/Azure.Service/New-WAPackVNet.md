# New-WAPackVNet

## Synopsis
Erstellt ein virtualisiertes Netzwerk.

## Syntax

```
New-WAPackVNet -LogicalNetwork <LogicalNetwork> -Name <String> [-Description <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung
Diese Themen sind veraltet und werden in Zukunft entfernt.
In diesem Thema wird das Cmdlet in der Version 0.8.1 des Microsoft Azure PowerShell-Moduls beschrieben.
Wenn Sie die Version des verwendeten Moduls ermitteln möchten, geben Sie aus der Azure PowerShell-Konsole ein `(Get-Module -Name Azure).Version` .

Mit dem Cmdlet **New-WAPackVNet** wird ein virtualisiertes Netzwerk erstellt.

## Beispiele

### Beispiel 1: Erstellen eines virtualisierten Netzwerks
```
PS C:\> $LogicalNetwork = Get-WAPackLogicalNetwork -Name "ContosoLogicalNetwork01"
PS C:\> New-WAPackVNet -LogicalNetwork $LogicalNetwork -Name "ContosoVNett01" -Description "A description"
```

Der erste Befehl ruft zuerst das logische Netzwerk ab, dem ein neues virtualisiertes Netzwerk hinzugefügt werden soll.
Dieses logische Netzwerk hat den Namen ContosoLogicalNetwork01.

Mit dem zweiten und letzten Befehl wird ein virtualisiertes Netzwerk mit dem zuvor abgerufenen logischen Netzwerk, einem Namen (ContosoVNett01) und einer Beschreibung (Beschreibung) erstellt.
