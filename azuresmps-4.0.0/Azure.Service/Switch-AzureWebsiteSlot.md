# Switch-AzureWebsiteSlot

## Synopsis
Tauscht den Produktions Steckplatz für eine Website mit einem anderen Steckplatz aus.

## Syntax

```
Switch-AzureWebsiteSlot [-Name <String>] [-Slot1 <String>] [-Slot2 <String>] [-Force]
 [-Profile <AzureSMProfile>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet " **Switch-AzureWebsiteSlot** " wird der Produktions Steckplatz für eine Website durch einen anderen Steckplatz ersetzt.
Dies funktioniert nur auf Websites mit zwei Slots.

## Beispiele

### Beispiel 1: Wechseln des Website Steckplatzes
```
PS C:\> Switch-AzureWebsiteSlot -Name MyWebsite
```

Wechseln Sie zum Sicherungs Steckplatz der Azure-Website mywebsite mit dem Produktions Steckplatz.
