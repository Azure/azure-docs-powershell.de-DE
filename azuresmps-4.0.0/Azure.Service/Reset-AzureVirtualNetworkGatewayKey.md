# Reset-AzureVirtualNetworkGatewayKey

## Synopsis
Setzt einen virtuellen Netzwerk-Gateway-Schlüssel zurück.

## Syntax

```
Reset-AzureVirtualNetworkGatewayKey -GatewayId <String> -ConnectedEntityId <String> -keyLength <Int32>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Reset-AzureVirtualNetworkGatewayKey** setzt einen virtuellen Netzwerk-Gateway-Schlüssel zurück.
