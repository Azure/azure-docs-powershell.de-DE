# Remove-AzureReservedIP

## Synopsis
Entfernt eine reservierte IP-Adresse mit dem Namen.

## Syntax

```
Remove-AzureReservedIP [-ReservedIPName] <String> [-Force] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureReservedIP** entfernt eine reservierte IP-Adresse mit dem Namen.

## Beispiele

### Beispiel 1: Entfernen einer reservierten IP-Adresse mit dem Namen
```
PS C:\> Remove-AzureReservedIP -ReservedIPName $name
```

Mit diesem Befehl wird eine reservierte IP-Adresse mit dem Namen entfernt.
