# Reset-AzureVirtualNetworkGatewayConnection

## Synopsis
Setzt eine virtuelle Netzwerkgateway-Verbindung zurück.

## Syntax

```
Reset-AzureVirtualNetworkGatewayConnection -GatewayId <String> -ConnectedEntityId <String>
 -RoutingWeight <Int32> [-SharedKey <String>] [-EnableBgp <Boolean>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Reset-AzureVirtualNetworkGatewayConnection** setzt eine virtuelle Netzwerkgateway-Verbindung zurück.
