# New-AzureStoreAddOn

## Synopsis
Kauft eine neue Add-on-Instanz.

## Syntax

```
New-AzureStoreAddOn -Name <String> -AddOn <String> -Plan <String> -Location <String> [-PromotionCode <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung
In diesem Thema wird das Cmdlet in der 0.8.10-Version des Microsoft Azure PowerShell-Moduls beschrieben.
Wenn Sie die Version des verwendeten Moduls abrufen möchten, geben Sie in der Azure PowerShell-Konsole `(Get-Module -Name Azure).Version` .

Kauft eine neue Add-on-Instanz aus dem Azure Store.

## Beispiele

### Beispiel 1
```
PS C:\> New-AzureStoreAddOn -Name MyAddOn -AddOn AddonId -Plan PlanId -Location "West US"
```

In diesem Beispiel wird ein Add-on mit dem Namen myaddon mit einer Planwert in West US-Position gekauft.

### Beispiel 2
```
PS C:\> New-AzureStoreAddOn -Name MyAddOn -AddOn AddonId -Plan PlanId -Location "West US" -PromotionCode MyPromoCode
```

In diesem Beispiel wird ein Werbe Code verwendet, um ein Add-on zu kaufen.
