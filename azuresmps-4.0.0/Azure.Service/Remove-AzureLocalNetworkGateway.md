# Remove-AzureLocalNetworkGateway

## Synopsis
Entfernt ein Azure-lokales Netzwerkgateway.

## Syntax

```
Remove-AzureLocalNetworkGateway -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureLocalNetworkGateway** entfernt ein Azure lokales Netzwerkgateway.
