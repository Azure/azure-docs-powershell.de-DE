# Restart-AzureWebsite

## Synopsis
Beendet die angegebene Website und startet Sie neu.

## Syntax

```
Restart-AzureWebsite [-PassThru] [-Name <String>] [-Slot <String>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Beschreibung
In diesem Thema wird das Cmdlet in der 0.8.10-Version des Microsoft Azure PowerShell-Moduls beschrieben.
Wenn Sie die Version des verwendeten Moduls abrufen möchten, geben Sie in der Azure PowerShell-Konsole `(Get-Module -Name Azure).Version` .

Mit dem Cmdlet " **Restart-AzureWebsite** " wird die angegebene Website beendet und neu gestartet.

## Beispiele

### Beispiel 1: Erneutes Starten einer Website
```
PS C:\> Restart-AzureWebsite -Name MyWebsite
```

In diesem Beispiel wird eine Website mit dem Namen "mywebsite" neu gestartet.
