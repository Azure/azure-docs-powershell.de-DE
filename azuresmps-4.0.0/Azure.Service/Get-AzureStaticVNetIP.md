# Get-AzureStaticVNetIP

## Synopsis
Ruft die statischen VNet-IP-Adressinformationen aus einem virtuellen Computerobjekt ab, sofern vorhanden.

## Syntax

```
Get-AzureStaticVNetIP -VM <IPersistentVM> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureStaticVNetIP** " Ruft die IP-Adressinformationen (static Virtual Network, VNet) von einem virtuellen Computerobjekt ab, sofern vorhanden.
