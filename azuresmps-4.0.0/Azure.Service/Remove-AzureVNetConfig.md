# Remove-AzureVNetConfig

## Synopsis
Löscht die Netzwerkkonfiguration aus dem aktuellen Azure-Abonnement.

## Syntax

```
Remove-AzureVNetConfig [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureVNetConfig** entfernt alle virtuellen Netzwerkeinstellungen aus dem aktuellen Azure-Abonnement.

## Beispiele

### Beispiel 1: Entfernen einer virtuellen Netzwerkkonfiguration aus dem aktuellen Abonnement
```
PS C:\> Remove-AzureVNetConfig
```

Mit diesem Befehl wird die Konfiguration des virtuellen Netzwerks aus dem aktuellen Abonnement entfernt.
