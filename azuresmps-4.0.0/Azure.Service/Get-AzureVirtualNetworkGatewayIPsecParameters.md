# Get-AzureVirtualNetworkGatewayIPsecParameters

## Synopsis
Ruft die IPSec-Parameter für ein Azure Virtual Network-Gateway ab.

## Syntax

```
Get-AzureVirtualNetworkGatewayIPsecParameters -GatewayId <String> -ConnectedEntityId <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureVirtualNetworkGatewayIPsecParameters** " Ruft die IPSec-Parameter für ein Azure Virtual Network-Gateway ab.
