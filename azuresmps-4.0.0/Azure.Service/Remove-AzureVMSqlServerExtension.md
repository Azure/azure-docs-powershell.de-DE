# Remove-AzureVMSqlServerExtension

## Synopsis
Entfernt eine Azure Virtual Machine-SQL Server-Erweiterung von einem virtuellen Computerobjekt.

## Syntax

```
Remove-AzureVMSqlServerExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureVMSqlServerExtension** entfernt eine Azure Virtual Machine-SQL Server-Erweiterung von einem virtuellen Computerobjekt.
