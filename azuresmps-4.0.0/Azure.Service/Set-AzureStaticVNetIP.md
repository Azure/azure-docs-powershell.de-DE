# Set-AzureStaticVNetIP

## Synopsis
Legt die statischen VNet-IP-Adressinformationen für ein Objekt eines virtuellen Computers fest.

## Syntax

```
Set-AzureStaticVNetIP [-IPAddress] <String> -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Set-AzureStaticVNetIP** " legt die IP-Adressinformationen (static Virtual Network, VNet) für ein Objekt eines virtuellen Computers fest.

## Beispiele

### Beispiel 1: Einrichten der IP-Adresse des virtuellen Netzwerks, die einem virtuellen Computer zugeordnet ist
```
PS C:\> # Prerequisite: VNet has been set up with SubNet
          # Set-AzureVNetConfig -ConfigurationPath $vnetConfigPath;

          $vm = New-AzureVMConfig -Name $vmname -ImageName $img -InstanceSize $sz | Add-AzureProvisioningConfig -Windows -Password $pwd -AdminUsername $usr;
          $vm = Set-AzureSubNet -VM $vm -SubNetNames $sn;
          Set-AzureStaticVNetIP -IPAddress $ip -VM $vm;
          New-AzureVM -ServiceName $svc -VMs $vm -VNetName $vnetName -Location $loc;
```

Der erste Befehl legt den Konfigurationspfad für ein virtuelles Netzwerk fest.

Mit dem zweiten Befehl wird eine Konfiguration für einen virtuellen Computer erstellt.

Der dritte Befehl legt das Subnetz für den virtuellen Computer fest.

Der vierte Befehl legt die IP-Adresse für den virtuellen Computer fest.

Der fünfte Befehl erstellt einen virtuellen Computer, der den virtuellen Computer verwendet.
