# Set-AzureRole

## Synopsis
Legt die Anzahl der Instanzen einer Azure-Rolle fest, die ausgeführt werden soll.

## Syntax

```
Set-AzureRole [-ServiceName] <String> [-Slot] <String> [-RoleName] <String> [-Count] <Int32>
 [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Set-AzureRole** " legt fest, wie viele Instanzen einer angegebenen Rolle in einer Azure-Bereitstellung ausgeführt werden.

## Beispiele

### 1: die Anzahl der Instanzen für eine Rolle einstellen
```
PS C:\> Set-AzureRole -ServiceName "MySvc01" -Slot "Production" -RoleName "MyTestRole03" -Count 3
```

Mit diesem Befehl wird die MyTestRole03-Rolle festgelegt, die in Production für den MySvc01-Dienst ausgeführt wird, sodass drei Instanzen vorhanden sind.
