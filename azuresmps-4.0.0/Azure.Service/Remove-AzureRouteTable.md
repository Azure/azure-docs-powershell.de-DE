# Remove-AzureRouteTable

## Synopsis
Entfernt eine Route-Tabelle.

## Syntax

```
Remove-AzureRouteTable -Name <String> [-PassThru] [-Force] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureRouteTable** entfernt eine Route-Tabelle aus Ihrem Abonnement.
Wenn eine Routentabelle einem Subnetz zugeordnet ist, können Sie die Tabelle nicht entfernen.

## Beispiele

### Beispiel 1: Entfernen einer Routentabelle
```
PS C:\> Remove-AzureRouteTable -Name "PublicRouteTable"
```

Dieser Befehl entfernt die Route-Tabelle mit dem Namen PublicRouteTable.
