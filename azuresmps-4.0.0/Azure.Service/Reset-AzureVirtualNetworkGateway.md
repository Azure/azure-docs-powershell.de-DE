# Reset-AzureVirtualNetworkGateway

## Synopsis
Setzt ein virtuelles Netzwerkgateway zurück.

## Syntax

```
Reset-AzureVirtualNetworkGateway -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Reset-AzureVirtualNetworkGateway** setzt ein virtuelles Netzwerkgateway zurück.
