# Get-AzureVirtualNetworkGatewayConnection

## Synopsis
Ruft eine Verbindung mit einem virtuellen Netzwerkgateway ab.

## Syntax

```
Get-AzureVirtualNetworkGatewayConnection [-GatewayId <String>] [-ConnectedEntityId <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureVirtualNetworkGatewayConnection** " erhält eine Azure Virtual Network Gateway-Verbindung.
