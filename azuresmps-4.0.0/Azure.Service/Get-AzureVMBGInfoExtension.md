# Get-AzureVMBGInfoExtension

## Synopsis
Ruft die BGInfo-Erweiterung ab, die auf einem virtuellen Computer angewendet wurde.

## Syntax

```
Get-AzureVMBGInfoExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureVMBGInfoExtension** " Ruft die BGInfo-Erweiterung ab, die auf einem virtuellen Computer angewendet wurde.

## Beispiele

### Beispiel 1: Abrufen der auf einem angegebenen virtuellen Computer angewendeten BGInfo-Erweiterung
```
PS C:\> Get-AzureVMBGInfoExtension -VM $VM;
```

Dieser Befehl ruft die BGInfo-Erweiterung ab, die auf einem angegebenen virtuellen Computer angewendet wurde.
