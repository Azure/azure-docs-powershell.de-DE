# Remove-AzureVMPuppetExtension

## Synopsis
Entfernt die auf einem virtuellen Computer angewendete Marionetten Erweiterung.

## Syntax

```
Remove-AzureVMPuppetExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureVMPuppetExtension** entfernt die auf einem virtuellen Computer angewendete Marionetten Erweiterung.

## Beispiele

### Beispiel 1: Entfernen der auf einem virtuellen Computer angewendeten Marionetten Erweiterung
```
PS C:\> Remove-AzureVMPuppetExtension -VM $VM;
```

Mit diesem Befehl wird die auf dem angegebenen virtuellen Computer angewendete Marionetten Erweiterung entfernt.
