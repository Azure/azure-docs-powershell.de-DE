# Remove-AzureVirtualNetworkGateway

## Synopsis
Entfernt ein virtuelles Netzwerkgateway.

## Syntax

```
Remove-AzureVirtualNetworkGateway -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureVirtualNetworkGateway** entfernt ein virtuelles Netzwerkgateway.
