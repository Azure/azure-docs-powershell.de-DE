# Get-AzureVMImageDiskConfigSet

## Synopsis
Ruft ein Festplatten-Konfigurationssatz Objekt aus dem Kontext des Eingabebilds ab.

## Syntax

```
Get-AzureVMImageDiskConfigSet [[-ImageContext] <OSImageContext>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureVMImageDiskConfigSet** " Ruft ein Festplatten-Konfigurationssatz Objekt aus dem Kontext des Eingabebilds ab.

## Beispiele

### Beispiel 1: Abrufen eines Festplatten-Konfigurationssatz Objekts von einem virtuellen Computer
```
PS C:\> Get-AzureVMImage -ImageName $Img | Get-AzureVMImageDiskConfigSet
```

Dieser Befehl ruft ein Festplatten-Konfigurationssatz Objekt von einem virtuellen Computer ab.
