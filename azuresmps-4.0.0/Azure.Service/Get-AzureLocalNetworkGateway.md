# Get-AzureLocalNetworkGateway

## Synopsis
Ruft ein lokales Netzwerkgateway ab.

## Syntax

```
Get-AzureLocalNetworkGateway [-GatewayId <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureLocalNetworkGateway** " Ruft ein lokales Netzwerkgateway ab.
