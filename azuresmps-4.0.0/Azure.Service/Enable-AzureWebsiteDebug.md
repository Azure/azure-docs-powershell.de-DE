# Enable-AzureWebsiteDebug

## Synopsis
Ermöglicht das Debuggen der Website.

## Syntax

```
Enable-AzureWebsiteDebug [-PassThru] -Version <String> [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung
Aktiviert das Debug-Feature der Website in Visual Studio.

## Beispiele

### Aktivieren des Debuggens von Visual Studio 2013
```
PS C:\> Enable-AzureWebsiteDebug -Name MyWebsite -Version VS2013
```

Ermöglicht das Debuggen auf vs 2013.
