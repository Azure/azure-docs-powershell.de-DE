# Add-AzureDns

## Synopsis
Fügt einen DNS-Server zu einem Azure-Dienst hinzu.

## Syntax

```
Add-AzureDns [-Name] <String> [-IPAddress] <String> [-ServiceName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **Add-AzureDns** wird ein DNS-Server zu einem Azure-Dienst hinzugefügt.

## Beispiele

### Beispiel 1: Hinzufügen eines DNS-Servers
```
PS C:\> Add-AzureDns -ServiceName "ContosoService" -IPAddress 10.1.2.4 -Name "Dns07"
```

Mit diesem Befehl wird der DNS-Server mit der Adresse 10.1.2.4 zu ContosoService hinzugefügt.
