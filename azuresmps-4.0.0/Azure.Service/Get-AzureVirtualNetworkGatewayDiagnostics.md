# Get-AzureVirtualNetworkGatewayDiagnostics

## Synopsis
Ruft die Ergebnisse der Azure Virtual Network Gateway-Diagnose ab.

## Syntax

```
Get-AzureVirtualNetworkGatewayDiagnostics -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureVirtualNetworkGatewayDiagnostics** " Ruft die Ergebnisse der Azure Virtual Network Gateway-Diagnose ab.
