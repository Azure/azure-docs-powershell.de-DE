# Get-AzureWebsiteLocation

## Synopsis
Ruft alle verfügbaren Websitespeicher Orte ab.

## Syntax

```
Get-AzureWebsiteLocation [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Beschreibung
In diesem Thema wird das Cmdlet in der 0.8.10-Version des Microsoft Azure PowerShell-Moduls beschrieben.
Wenn Sie die Version des verwendeten Moduls abrufen möchten, geben Sie in der Azure PowerShell-Konsole `(Get-Module -Name Azure).Version` .

Ruft alle verfügbaren Websitespeicher Orte für das aktuelle Abonnement ab.

## Beispiele

### Beispiel 1: Abrufen aller verfügbaren Speicherorte
```
PS C:\> Get-AzureWebsiteLocations
```

In diesem Beispiel werden alle verfügbaren Websitespeicher Orte für das aktuelle Abonnement abgerufen.
