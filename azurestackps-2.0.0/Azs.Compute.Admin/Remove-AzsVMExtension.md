# Remove-AzsVMExtension

## Synopsis
Löscht das angegebene Image der virtuellen Computererweiterung.

## Syntax

### Löschen (Standard)
```
Remove-AzsVMExtension -Publisher <String> -Type <String> -Version <String> [-Location <String>]
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzsVMExtension -InputObject <IComputeAdminIdentity> [-DefaultProfile <PSObject>] [-PassThru] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

## Beschreibung
Löscht das angegebene Image der virtuellen Computererweiterung.

## Beispiele

### Beispiel 1: Entfernen einer vorhandenen VM-Erweiterung 
```powershell
PS C:\> Remove-AzsVMExtension -Location local -Publisher Microsoft -Type MicroExtension -Version 0.1.0
```

Ein erfolgreicher Aufruf zum Entfernen eines Compute-Kontingents gibt keine Ausgabe zurück.

### Beispiel 2: Entfernen einer VM-Erweiterung, die nicht vorhanden ist
```powershell
PS C:\> Remove-AzsVMExtension -Location local -Publisher Microsoft -Type DoesntExist -Version 9.8.7
```

Ein erfolgreicher Anruf zum Entfernen eines nicht vorhandenen Platt Form Bilds gibt keine Ausgabe zurück.
