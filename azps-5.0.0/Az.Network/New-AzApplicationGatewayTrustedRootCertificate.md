# New-AzApplicationGatewayTrustedRootCertificate

## Synopsis
Erstellt ein vertrauenswürdiges Stammzertifikat für ein Anwendungsgateway.

## Syntax

```
New-AzApplicationGatewayTrustedRootCertificate -Name <String> -CertificateFile <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **New-AzApplicationGatewayTrustedRootCertificate** erstellt ein vertrauenswürdiges Stammzertifikat für ein Azure Application Gateway.

## Beispiele

### Beispiel 1
```powershell
PS C:\> $certFilePath = ".\rootCA.cer"
PS C:\> $trc = New-AzApplicationGatewayTrustedRootCertificate -Name "trc1" -CertificateFile $certFilePath
```

Dieser Befehl erstellt ein vertrauenswürdiges Stammzertifikat mit dem Namen "trc1" und speichert das Ergebnis in der Variablen mit dem Namen $TRC.
