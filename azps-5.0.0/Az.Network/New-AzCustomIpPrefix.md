# New-AzCustomIpPrefix

## Synopsis
Erstellt eine CustomIpPrefix-Ressource

## Syntax

```
New-AzCustomIpPrefix -Name <String> -ResourceGroupName <String> -Location <String> -Cidr <String>
 [-Zone <String[]>] [-Tag <Hashtable>] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **New-AzCustomIpPrefix** erstellt eine CustomIpPrefix-Ressource.

## Beispiele

### Beispiel 1
```powershell
PS C:\> $myCustomIpPrefix = New-AzCustomIpPrefix -Name $prefixName -ResourceGroupName $rgName -Cidr 40.40.40.0/24 -Location westus
```

Dieser Befehl erstellt eine neue CustomIpPrefix-Ressource mit dem Namen $prefixName in der Ressourcengruppe $rgName mit einem CIDR von 40.40.40.0/24 in westus.
