# New-AzPrivateDnsZoneConfig

## Synopsis
Erstellt eine DNS-Zonenkonfiguration der privaten DNS-Zonengruppe.

## Syntax

```
New-AzPrivateDnsZoneConfig -Name <String> [-PrivateDnsZoneId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **New-AzPrivateDnsZoneConfig** können Sie ein neues DNS Zone Configuration-Objekt erstellen, das für die DNS-Zonengruppe eingerichtet wird.

## Beispiele

### Erstellt eine DNS-Zonenkonfiguration
```powershell
PS C:\> $dnsZone = New-AzPrivateDnsZone -ResourceGroupName "rg" -Name "test.vault.azure.com"
PS C:\> $config = New-AzPrivateDnsZoneConfig -Name "test-vault-azure-com" -PrivateDnsZoneId $dnsZone.ResourceId
```

Im obigen Beispiel wird die DNS-Zone erstellt und dann die DNS-Zonenkonfiguration erstellt. `New-AzPrivateDnsZone` das Cmdlet wird vom Modul AZ. PrivateDns geprüft.
