# Get-AzLoadBalancerInboundNatPoolConfig

## Synopsis
Ruft mindestens eine eingehende NAT-Poolkonfiguration von einem Load Balancer ab.

## Syntax

```
Get-AzLoadBalancerInboundNatPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzLoadBalancerInboundNatPoolConfig** " ruft mindestens eine eingehende NAT-Poolkonfiguration von einem Load Balancer ab.

## Beispiele

### 1: Abrufen
```
PS C:\> $slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> $slb | Get-AzLoadBalancerInboundNatPoolConfig -Name myInboundNatPool
```
