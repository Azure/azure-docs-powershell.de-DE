# New-AzApplicationGatewayFirewallPolicyManagedRuleSet

## Synopsis
Erstellt eine ManagedRuleSet für die firewallPolicy

## Syntax

```
New-AzApplicationGatewayFirewallPolicyManagedRuleSet -RuleSetType <String> -RuleSetVersion <String>
 [-RuleGroupOverride <PSApplicationGatewayFirewallPolicyManagedRuleGroupOverride[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das **neue AzApplicationGatewayFirewallPolicyManagedRuleSet** erstellt eine Managed-Rules für eine Firewall-Richtlinie.

## Beispiele

### Beispiel 1
```powershell
PS C:\> $managedRuleSet = New-AzApplicationGatewayFirewallPolicyManagedRuleSet -RuleSetType $ruleSetType 
-RuleSetVersion $ruleSetVersion -RuleGroupOverrides $ruleGroupOverride1, $ruleGroupOverride2
```

Erstellt eine ManagedRuleSet mit rulesettype als $ruleSetType, ruleSetVersion als $ruleSetVersion und RuleGroupOverrides als eine Liste mit ganzen als $ruleGroupOverride 1, $ruleGroupOverride 2 der neuen ManagedRuleSet zugeordnet ist $managedRuleSet
