# New-AzFirewallPublicIpAddress

## Synopsis
Dies ist der Platzhalter für die IP-Adresse, die für Multi PIP auf Azure Firewall verwendet werden kann.

## Syntax

```
New-AzFirewallPublicIpAddress [-Address <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Beschreibung
Dies ist der Platzhalter für die IP-Adresse, die für Multi PIP auf Azure Firewall verwendet werden kann.

## Beispiele

### Beispiel 1
```powershell
PS C:\> $publicIp = New-AzFirewallPublicIpAddress -Address 20.2.3.4
```

$publicIp ist der Platzhalter für die IP-Adresse 20.2.3.4
