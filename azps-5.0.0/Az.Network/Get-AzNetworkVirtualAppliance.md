# Get-AzNetworkVirtualAppliance

## Synopsis
Abrufen oder auflisten virtueller Netzwerk-Appliances.

## Syntax

### ResourceNameParameterSet (Standard)
```
Get-AzNetworkVirtualAppliance [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzNetworkVirtualAppliance -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Die Get-AzNetworkVirtualAppliance-Befehle ruft Netzwerk-Virtual Appliance-Ressourcen ab oder listet Sie auf.
