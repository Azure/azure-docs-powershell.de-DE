# Get-AzApplicationGatewayAuthenticationCertificate

## Synopsis
Ruft ein Authentifizierungszertifikat für ein Anwendungsgateway ab.

## Syntax

```
Get-AzApplicationGatewayAuthenticationCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzApplicationGatewayAuthenticationCertificate** " Ruft ein Authentifizierungszertifikat für ein Azure Application Gateway ab.

## Beispiele

### Beispiel 1: Abrufen eines angegebenen Authentifizierungszertifikats
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $pool = Get-AzApplicationGatewayBackendAddressPool -Name "pool01" -ApplicationGateway $appgw
```

Der erste Befehl ruft das Anwendungsgateway mit dem Namen appGwName ab und speichert es in der $appgw-Variablen.
Der zweite Befehl ruft das Authentifizierungszertifikat mit dem Namen pool01 ab und speichert es in der $Pool-Variablen.
