# Get-AzP2sVpnGatewayDetailedConnectionHealth

## Synopsis
Ruft die detaillierten Informationen zu aktuellen Point-to-Site-Verbindungen von P2SVpnGateway ab.

## Syntax

### ByP2SVpnGatewayName (Standard)
```
Get-AzP2sVpnGatewayDetailedConnectionHealth [-Name <String>] -ResourceGroupName <String>
 -OutputBlobSasUrl <String> [-VpnUserNamesFilter <String[]>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByP2SVpnGatewayObject
```
Get-AzP2sVpnGatewayDetailedConnectionHealth -InputObject <PSP2SVpnGateway> -OutputBlobSasUrl <String>
 [-VpnUserNamesFilter <String[]>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByP2SVpnGatewayResourceId
```
Get-AzP2sVpnGatewayDetailedConnectionHealth -ResourceId <String> -OutputBlobSasUrl <String>
 [-VpnUserNamesFilter <String[]>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **Get-AzP2sVpnGatewayDetailedConnectionHealth** können Sie die detaillierten Informationen zu aktuellen Point-to-Site-Verbindungen von P2SVpnGateway abrufen. Der Kunde muss die SAS-URL weiterleiten, wo wir diese detaillierten Gesundheitsinformationen einfügen können.

## Beispiele

### Beispiel 1
```powershell
PS C:\> $blobSasUrl = New-AzStorageBlobSASToken -Container contp2stesting -Blob emptyfile.txt -Context $context -Permission "rwd" -StartTime $now.AddHours(-1) -ExpiryTime $now.AddDays(1) -FullUri
PS C:\> $blobSasUrl
SignedSasUrl
PS C:\> Get-AzP2sVpnGatewayDetailedConnectionHealth -Name 683482ade8564515aed4b8448c9757ea-westus-gw -ResourceGroupName P2SCortexGATesting -OutputBlobSasUrl $blobSasUrl
SasUrl : SignedSasUrl
```

Mit dem Cmdlet **Get-AzP2sVpnGatewayDetailedConnectionHealth** können Sie die detaillierten Informationen zu aktuellen Point-to-Site-Verbindungen von P2SVpnGateway abrufen. Der Kunde kann Integritätsinformationen vom übergebenen SAS-URL-Download herunterladen. Dadurch werden Informationen zu den einzelnen Punkt-zu-Standort-Verbindungen mit Benutzernamen, Bytes in, Bytes aus, zugewiesene IP-Adresse usw. angezeigt.
