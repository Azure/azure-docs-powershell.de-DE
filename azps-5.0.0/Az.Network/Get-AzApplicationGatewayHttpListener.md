# Get-AzApplicationGatewayHttpListener

## Synopsis
Ruft den HTTP-Listener eines Anwendungs Gateways ab.

## Syntax

```
Get-AzApplicationGatewayHttpListener [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzApplicationGatewayHttpListener** " Ruft den HTTP-Listener eines Anwendungs Gateways ab.

## Beispiele

### Beispiel 1: Abrufen eines bestimmten HTTP-Listeners
```
PS C:\>$Appgw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Listener = Get-AzApplicationGatewayHttpListener -Name "Listener01" -ApplicationGateway $Appgw
```

Dieser Befehl ruft einen HTTP-Listener mit dem Namen Listener01 ab.

### Beispiel 2: Abrufen einer Liste mit http-Listenern
```
PS C:\>$Appgw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Listeners = Get-AzApplicationGatewayHttpListener -ApplicationGateway $Appgw
```

Dieser Befehl ruft eine Liste der HTTP-Listener ab.
