# New-AzApplicationGatewayFirewallCondition

## Synopsis
Erstellt eine übereinstimmungsbedingung für die benutzerdefinierte Regel

## Syntax

```
New-AzApplicationGatewayFirewallCondition -MatchVariable <PSApplicationGatewayFirewallMatchVariable[]>
 -Operator <String> [-NegationCondition <Boolean>] -MatchValue <String[]> [-Transform <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das **AzApplicationGatewayFirewallCondition** erstellt eine übereinstimmungsbedingung für die benutzerdefinierte Firewall-Regel.

## Beispiele

### Beispiel 1
```powershell
PS C:\> $condition = New-AzApplicationGatewayFirewallCondition -MatchVariable $variable -Operator Contains -NegationCondition false -Transforms Lowercase, Trim -MatchValue abc, cde
```

Der Befehl erstellt eine neue übereinstimmungsbedingung mithilfe der in der $Variable definierten Übereinstimmungsvariablen, der Operator ist Contains und die Negations Bedingung ist falsch, umgeht einschließlich klein-und Zuschneiden, der Übereinstimmungs Wert ist ABC und CDE. Die neue übereinstimmungsbedingung wird in $Condition gespeichert.
