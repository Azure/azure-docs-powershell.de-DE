# Get-AzServiceEndpointPolicyDefinition

## Synopsis
Ruft eine Dienstendpunkt-Richtliniendefinition ab.

## Syntax

```
Get-AzServiceEndpointPolicyDefinition [-Name <String>] -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzServiceEndpointPolicyDefinition** " Ruft eine Dienstendpunkt-Richtliniendefinition ab.

## Beispiele

### Beispiel 1
```
$policydef= Get-AzServiceEndpointPolicyDefinition -Name "ServiceEndpointPolicyDefinition1" -ServiceEndpointPolicy $Policy
```

Dieser Befehl ruft die Dienstendpunkt-Richtliniendefinition mit dem Namen ServiceEndpointPolicyDefinition1 in ServiceEndpointPolicy ab $Policy speichert Sie in der $policydef-Variablen.
