# Get-AzExpressRoutePortIdentity

## Synopsis
Abrufen einer einem ExpressRoutePort zugewiesenen Identität

## Syntax

```
Get-AzExpressRoutePortIdentity -ExpressRoutePort <PSExpressRoutePort>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzExpressRoutePortIdentity** " Ruft eine Identität ab, die einem lokalen Azure ExpressRoutePort-Objekt zugewiesen ist.

## Beispiele

### Beispiel 1
```powershell
PS C:\> $exrPort = Get-AzExpressRoutePort -Name $exrPortName -ResourceGroupName $resgpName
PS C:\> $identity = Get-AzExpressRoutePortIdentity -ExpressRoutePort $exrPort
```
