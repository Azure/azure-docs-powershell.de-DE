# Remove-AzRouteTable

## Synopsis
Entfernt eine Route-Tabelle.

## Syntax

```
Remove-AzRouteTable -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzRouteTable** entfernt eine Azure Route-Tabelle.

## Beispiele

### Beispiel 1: Entfernen einer Routentabelle
```
PS C:\>Remove-AzRouteTable -ResourceGroupName "ResourceGroup11 -Name "RouteTable01"
Confirm
Are you sure you want to remove resource 'RouteTable01'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

Dieser Befehl entfernt die Routingtabelle mit dem Namen RouteTable01 in der Ressourcengruppe mit dem Namen ResourceGroup11.
Das Cmdlet fordert Sie zur Bestätigung auf, bevor die Tabelle entfernt wird.
