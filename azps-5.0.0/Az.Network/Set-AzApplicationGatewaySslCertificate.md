# Set-AzApplicationGatewaySslCertificate

## Synopsis
Aktualisiert ein SSL-Zertifikat für ein Anwendungsgateway.

## Syntax

```
Set-AzApplicationGatewaySslCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 [-CertificateFile <String>] [-Password <SecureString>] [-KeyVaultSecretId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Satz-AzApplicationGatewaySslCertificate** " aktualisiert ein SSL-Zertifikat für ein Anwendungsgateway.

## Beispiele

### Beispiel 1: Aktualisieren eines vorhandenen SSL-Zertifikats auf dem Anwendungs Gateway
```
PS C:\> $appGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $password = ConvertTo-SecureString "P@ssw0rd" -AsPlainText -Force
PS C:\> $cert = Set-AzApplicationGatewaySslCertificate -ApplicationGateway $AppGW -Name "Cert01" -CertificateFile "D:\cert01.pfx" -Password $password
```

Aktualisieren Sie ein vorhandenes SSL-Zertifikat für das Anwendungsgateway mit dem Namen ApplicationGateway01.

### Beispiel 2: Aktualisieren eines vorhandenen SSL-Zertifikats unter Verwendung von keyvault Secret (Version-less-Secret-Code) auf dem Anwendungs Gateway
```
PS C:\> $secret = Get-AzKeyVaultSecret -VaultName "keyvault01" -Name "sslCert01"
PS C:\> $secretId = $secret.Id.Replace($secret.Version, "") # https://<keyvaultname>.vault.azure.net/secrets/
PS C:\> $cert = Set-AzApplicationGatewaySslCertificate -ApplicationGateway $AppGW -Name "Cert01" -KeyVaultSecretId $secretId
```

Besorgen Sie sich das Geheimnis und aktualisieren Sie ein vorhandenes SSL-Zertifikat mit `Set-AzApplicationGatewaySslCertificate` .

### Beispiel 3: Aktualisieren eines vorhandenen SSL-Zertifikats mithilfe von keyvault Secret auf dem Anwendungs Gateway
```
PS C:\> $secret = Get-AzKeyVaultSecret -VaultName "keyvault01" -Name "sslCert01"
PS C:\> $secretId = $secret.Id # https://<keyvaultname>.vault.azure.net/secrets/<hash>
PS C:\> $cert = Set-AzApplicationGatewaySslCertificate -ApplicationGateway $AppGW -Name "Cert01" -KeyVaultSecretId $secretId
```

Besorgen Sie sich das Geheimnis und aktualisieren Sie ein vorhandenes SSL-Zertifikat mit `Set-AzApplicationGatewaySslCertificate` .
Hinweis: Wenn es erforderlich ist, dass Application Gateway das Zertifikat mit dem keyvault synchronisiert, geben Sie bitte den versionslosen Geheimdienst an.
