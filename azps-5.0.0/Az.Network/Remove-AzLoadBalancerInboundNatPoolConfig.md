# Remove-AzLoadBalancerInboundNatPoolConfig

## Synopsis
Entfernt eine eingehende NAT-Poolkonfiguration von einem Lastenausgleichsmodul.

## Syntax

```
Remove-AzLoadBalancerInboundNatPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzLoadBalancerInboundNatPoolConfig** entfernt eine eingehende NAT-Poolkonfiguration von einem Lastenausgleichsmodul.

## Beispiele

### 1: Entfernen
```
PS C:\> $slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Remove-AzLoadBalancerInboundNatPoolConfig -Name myinboundnatpool -LoadBalancer $slb
```
