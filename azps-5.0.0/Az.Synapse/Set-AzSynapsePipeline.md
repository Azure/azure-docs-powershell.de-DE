# Set-AzSynapsePipeline

## Synopsis
Erstellt eine Pipeline im Arbeitsbereich.

## Syntax

### SetByName (Standard)
```
Set-AzSynapsePipeline -WorkspaceName <String> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByObject
```
Set-AzSynapsePipeline -WorkspaceObject <PSSynapseWorkspace> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Satz-AzSynapsePipeline** " erstellt eine Pipeline im Arbeitsbereich.

## Beispiele

### Beispiel 1
```powershell
PS C:\> Set-AzSynapsePipeline -WorkspaceName ContosoWorkspace -Name ContosoPipeline -DefinitionFile "C:\pipeline.json"
```

Dieser Befehl erstellt eine Pipeline mit dem Namen ContosoPipeline im Arbeitsbereich mit dem Namen ContosoWorkspace.
Der Befehl basiert auf Informationen in der pipeline.jsfür die Datei.
Diese Datei enthält Informationen zu Aktivitäten.

### Beispiel 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Set-AzSynapsePipeline -Name ContosoPipeline -DefinitionFile "C:\pipeline.json"
```

Dieser Befehl erstellt eine Pipeline mit dem Namen ContosoPipeline im Arbeitsbereich mit dem Namen ContosoWorkspace through Pipeline.
Der Befehl basiert auf Informationen in der pipeline.jsfür die Datei.
Diese Datei enthält Informationen zu Aktivitäten.
