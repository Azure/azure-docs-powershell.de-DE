# Set-AzApiManagementTenantAccess

## Synopsis
Aktiviert oder deaktiviert den Mandanten Zugriff.

## Syntax

```
Set-AzApiManagementTenantAccess -Context <PsApiManagementContext> -Enabled <Boolean> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Satz-AzApiManagementTenantAccess** " aktiviert oder deaktiviert den Mandanten Zugriff.

## Beispiele

### Beispiel 1: Aktivieren des Mandanten Zugriffs
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Set-AzApiManagementTenantAccess -Context $apimContext -Enabled $True
```

Dieser Befehl ermöglicht Mandanten Zugriff im angegebenen Kontext.
