# New-AzApiManagementResourceLocationObject

## Synopsis
Erstellen eines neuen Ressourcen Standort Vertrags (wird in Gateways verwendet)

## Syntax

```
New-AzApiManagementResourceLocationObject -Name <String> [-City <String>] [-District <String>]
 [-CountryOrRegion <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **New-AzApiManagementResourceLocationObject** " erstellt einen neuen Ressourcen Standort Vertrag (wird in Gateways verwendet).

## Beispiele

### Beispiel 1: Erstellen eines Ressourcen Standort Vertrags
```powershell
PS C:\>$location = New-AzApiManagementResourceLocationObject -Name "n1" -City "c1" -District "d1" -CountryOrRegion "r1"
```

Dieser Befehl erstellt einen Ressourcen Standort.
