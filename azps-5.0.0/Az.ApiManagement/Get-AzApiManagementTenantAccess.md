# Get-AzApiManagementTenantAccess

## Synopsis
Ruft die Zugriffskonfiguration für einen Mandanten ab.

## Syntax

```
Get-AzApiManagementTenantAccess -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzApiManagementTenantAccess** " Ruft die Mandanten Zugriffskonfiguration für einen Mandanten ab.
Schlüssel werden nicht in Ergebnisdetails berücksichtigt. Verwenden **Sie Get-AzApiManagementTenantAccessSecret** , um Client Secret zu erhalten.

## Beispiele

### Beispiel 1: Abrufen der Mandanten Zugriffskonfiguration
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantAccess -Context $apimContext
```

Dieser Befehl ruft die Mandanten Zugriffskonfiguration für den angegebenen Kontext ab.
