# New-AzCosmosDBCassandraClusterKey

## Synopsis
Erstellt einen neuen CosmosDB Cassandra-Cluster Schlüssel.

## Syntax

```
New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Der **New-AzCosmosDBCassandraClusterKey** erstellt einen neuen CosmosDB Cassandra-Cluster Schlüssel.

## Beispiele

### Beispiel 1
```powershell
PS C:\> New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String>

Name   OrderBy
----   -------
{name}  Asc
```
