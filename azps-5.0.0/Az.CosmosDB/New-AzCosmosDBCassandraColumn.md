# New-AzCosmosDBCassandraColumn

## Synopsis
Erstellt eine neue CosmosDB Cassandra-Spalte.

## Syntax

```
New-AzCosmosDBCassandraColumn -Name <String> -Type <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Das **New-AzCosmosDBCassandraColumn** erstellt eine neue CosmosDB Cassandra-Spalte.

## Beispiele

### Beispiel 1
```powershell
PS C:\> New-AzCosmosDBCassandraColumn -Name "name" -Type int

Name Type
---- ----
name int
```
