# Get-AzFrontDoorRulesEngine

## Synopsis
Abrufen von Regelmodul Konfigurationen

## Syntax

```
Get-AzFrontDoorRulesEngine -ResourceGroupName <String> -FrontDoorName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzFrontDoorRulesEngine** " Ruft eine bestimmte Regelmodul Konfiguration ab oder ruft alle Regelmodul Konfigurationen ab, die mit einer Haustür verknüpft sind. 

## Beispiele

### Beispiel 1
```powershell
PS C:\> Get-AzFrontDoorRulesEngine -ResourceGroupName $resourceGroupName -FrontDoorName $frontDoorName -Name rulesengine3

Name         RulesEngineRules
----         ----------------
rulesEngine3 {rules1}
```

Abrufen einer bestimmten Regelmodul Konfiguration

### Beispiel 2

```powershell
PS C:\> Get-AzFrontDoorRulesEngine -ResourceGroupName $resourceGroupName -FrontDoorName $frontDoorName

Name         RulesEngineRules
----         ----------------
rulesEngine1 {Rule1}
rulesEngine2 {Rule1}
rulesEngine3 {rules1}
```

Alle Regelmodul Konfigurationen in einer Haustür abrufen.

### Beispiel 3

```powershell
PS C:\> Get-AzFrontDoorRulesEngine -ResourceGroupName $resourceGroupName -FrontDoorName $frontDoorName -Name nonexistent
```
