# Add-AzDataLakeAnalyticsFirewallRule

## Synopsis
Fügt eine Firewallregel zu einem Data Lake Analytics-Konto hinzu.

## Syntax

```
Add-AzDataLakeAnalyticsFirewallRule [-Account] <String> [-Name] <String> [-StartIpAddress] <String>
 [-EndIpAddress] <String> [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Add-AzDataLakeAnalyticsFirewallRule** fügt eine Firewallregel zu einem Azure Data Lake Analytics-Konto hinzu.

## Beispiele

### Beispiel 1: Hinzufügen einer Firewall-Regel
```
PS C:\>Add-AzDataLakeAnalyticsFirewallRule -Account "ContosoAdlAcct" -Name "My firewall rule" -StartIpAddress 127.0.0.1 -EndIpAddress 127.0.0.10
```

Mit diesem Befehl wird die Firewallregel mit dem Namen "meine Firewall-Regel" aus dem Konto "ContosoAdlAcct" mit dem IP-Bereich hinzugefügt: 127.0.0.1-127.0.0.10
