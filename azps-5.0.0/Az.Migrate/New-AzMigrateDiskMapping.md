# New-AzMigrateDiskMapping

## Synopsis
Erstellt eine neue Datenträgerzuordnung

## Syntax

```
New-AzMigrateDiskMapping -DiskID <String> -DiskType <DiskAccountType> -IsOSDisk <String>
 [-DiskEncryptionSetID <String>] [<CommonParameters>]
```

## Beschreibung
Das New-AzMigrateDiskMapping-Cmdlet erstellt eine Zuordnung des Quelldatenträgers, der an den zu migrierenden Server angefügt ist.

## Beispiele

### Beispiel 1: Erstellen von Datenträgern
```powershell
PS C:\> New-AzMigrateDiskMapping -DiskID a -DiskType Standard -IsOSDisk 'true'

DiskEncryptionSetId DiskId   DiskType  IsOSDisk LogStorageAccountId LogStorageAccountSasSecretName  
------------------- ------   --------  -------- ------------------- ------------------------------   
                      a      Standard  true  
```

Abrufen von Datenträger Objekten zum Bereitstellen von Eingaben für New-AzMigrateServerReplication
