## Beschreibung
Das Cmdlet " **Get-AzManagedHsmKey** " ruft Azure Managed HSM-Schlüssel ab.
Dieses Cmdlet ruft ein bestimmtes **Microsoft. Azure. Commands. keyvault. Models. keybundle** oder eine Liste aller **keybundle** -Objekte in einem verwalteten HSM oder einer Version ab.
