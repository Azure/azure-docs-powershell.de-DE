# Remove-AzDataLakeStoreItemAcl

## Synopsis
Löscht die ACL einer Datei oder eines Ordners im Data Lake Store.

## Syntax

```
Remove-AzDataLakeStoreItemAcl [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Default] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzDataLakeStoreItemAcl** löscht die Zugriffssteuerungsliste (ACL) einer Datei oder eines Ordners im Data Lake Store.

## Beispiele

### Beispiel 1: Entfernen der ACL aus einem Ordner
```
PS C:\>Remove-AzDataLakeStoreItemAcl -AccountName "ContosoADL" -Path "/"
```

Dieser Befehl entfernt die ACL für das Stammverzeichnis für das ContosoADL-Konto.
