# AZ. KubernetesConfiguration-Modul
## Beschreibung
Microsoft Azure PowerShell: KubernetesConfiguration-Cmdlets

## AZ. KubernetesConfiguration-Cmdlets
### [Get-AzKubernetesConfiguration](Get-AzKubernetesConfiguration.md)
Ruft Details der Quell Code Verwaltungskonfiguration ab.

### [Neu – AzKubernetesConfiguration](New-AzKubernetesConfiguration.md)
Erstellen Sie eine neue Kubernetes-Quell Code Verwaltungskonfiguration.

### [Remove-AzKubernetesConfiguration](Remove-AzKubernetesConfiguration.md)
Dadurch wird die YAML-Datei gelöscht, die zum Einrichten der Quellcodeverwaltungs-Konfiguration verwendet wird, wodurch die zukünftige Synchronisierung aus dem Quell-Repo gestoppt wird.
