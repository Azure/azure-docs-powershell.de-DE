# New-AzPeerAsnContactDetail

## Synopsis
Erstellt eine in-Memory-Kontakt Detail für PeerAsn. 

## Syntax

```
New-AzPeerAsnContactDetail -Role <String> -Email <String> [-Phone <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Erstellen eines in-Memory-PeerAsn-Kontaktdetails

## Beispiele

### Kontaktdetails erstellen und zu PeerAsn hinzufügen
```powershell
PS C:\> $nocContact = New-AzPeerAsnContactDetail -Role Noc -Email "noc@contoso.com" -Phone "+1 (887) 888-8088"
PS C:\> $customerContact = New-AzPeerAsnContactDetail -Role Noc -Email "noc@contoso.com" -Phone "+1 (887) 888-8088"
PS C:\> New-AzPeerAsn -Name $name -PeerName "Contoso Networks Limited" -PeerAsn 65000 -ContactDetail $nocContact,$customerContact
```

Rolle und e-Mail sind erforderlich. Telefon ist optional. Telefon unterstützt +-() oder Leerzeichen. Ein PeerAsn muss mindestens ein Kontakt Detail des Typs "NOC" aufweisen.
