# Stop-AzDeployment

## Synopsis
Abbrechen einer ausgeführten Bereitstellung

## Syntax

### StopByDeploymentName (Standard)
```
Stop-AzDeployment [-Name] <String> [-PassThru] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### StopByDeploymentId
```
Stop-AzDeployment -Id <String> [-PassThru] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### StopByInputObject
```
Stop-AzDeployment -InputObject <PSDeployment> [-PassThru] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Stop-AzDeployment** " bricht eine Bereitstellung im Abonnement Bereich ab, die gestartet, aber nicht abgeschlossen wurde.
Damit eine Bereitstellung beendet werden kann, muss die Bereitstellung einen unvollständigen Bereitstellungsstatus aufweisen, beispielsweise die Bereitstellung und nicht den Status abgeschlossen, wie bereitgestellt oder fehlgeschlagen.

Verwenden Sie das New-AzDeployment-Cmdlet, um eine Bereitstellung im Abonnement Bereich zu erstellen.

Dieses Cmdlet beendet nur eine ausgeführte Bereitstellung. Verwenden Sie den Parameter *Name* , um eine bestimmte Bereitstellung zu beenden.

## Beispiele

### Beispiel 1
```
PS C:\>Stop-AzDeployment -Name "deployment01"
```

Mit diesem Befehl wird eine ausgeführte Bereitstellung "deployment01" im aktuellen Abonnement Bereich abgebrochen.

### Beispiel 2
```
PS C:\>Get-AzDeployment -Name "deployment01" | Stop-AzDeployment
```

Mit diesem Befehl wird die Bereitstellung "deployment01" im aktuellen Abonnement Bereich abgerufen und abgebrochen.
