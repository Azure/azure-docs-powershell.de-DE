# Register-AzResourceProvider

## Synopsis
Registriert einen Ressourcenanbieter.

## Syntax

```
Register-AzResourceProvider -ProviderNamespace <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Register-AzResourceProvider** registriert einen Azure-Ressourcenanbieter.

## Beispiele

### Beispiel 1: Registrieren eines Anbieters
```
PS C:\>Register-AzResourceProvider -ProviderNamespace Microsoft.Network
```

Dadurch wird der Microsoft. Network-Anbieter für Ihr Konto registriert.
