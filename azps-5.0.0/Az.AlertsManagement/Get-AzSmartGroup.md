# Get-AzSmartGroup

## Synopsis
Ruft Informationen zu intelligenten Gruppen ab

## Syntax

### SmartGroupsListByFilter (Standard)
```
Get-AzSmartGroup [-SortBy <String>] [-SortOrder <String>] [-TimeRange <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SmartGroupById
```
Get-AzSmartGroup -SmartGroupId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzSmartGroup** " Ruft Informationen zu Smart Groups ab.

## Beispiele

### Beispiel 1
```powershell
PS C:\> Get-AzSmartGroup -TimeRange "1h"
```

Auflisten aller intelligenten Gruppen, die in der letzten 1 Stunde gebildet wurden Verwenden Sie Format-List, um die vollständigen Details jeder intelligenten Gruppe in der Liste abzurufen.

### Beispiel 2
```powershell
PS C:\> Get-AzSmartGroup -SmartGroupId "afbf1b3a-0a6c-4f19-9c9b-644ccd7b1529" | Format-List
```

Abrufen intelligenter Gruppendetails nach ID (GUID) oder Ressourcen-ID (vollständige Arm-ID)
