# Invoke-AzIotHubQuery

## Synopsis
Abfragen eines vielseitigen Hubs mithilfe einer leistungsfähigen SQL-ähnlichen Sprache.

## Syntax

### ResourceSet (Standard)
```
Invoke-AzIotHubQuery [-ResourceGroupName] <String> [-IotHubName] <String> [-Query] <String> [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectSet
```
Invoke-AzIotHubQuery [-InputObject] <PSIotHub> [-Query] <String> [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdSet
```
Invoke-AzIotHubQuery [-ResourceId] <String> [-Query] <String> [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Abfragen eines großen Hubs mithilfe einer leistungsfähigen SQL-ähnlichen Sprache, um Informationen zu Geräte-und Modul Zwillingen, Aufträgen und dem Nachrichtenrouting abzurufen.
Weitere Informationen finden Sie unter https://docs.microsoft.com/azure/iot-hub/iot-hub-devguide-query-language .

## Beispiele

### Beispiel 1
```powershell
PS C:\> Invoke-AzIotHubQuery -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Query "select * from devices"
```

Abfragen aller Geräte Zwillings Daten in einem Azure-viel-Hub

### Beispiel 2
```powershell
PS C:\> Invoke-AzIotHubQuery -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Query "select * from devices.modules where devices.deviceId = 'myDevice1'" -Top 2
```

Abfrage der obersten zwei Modul Daten auf einem Zielgerät
