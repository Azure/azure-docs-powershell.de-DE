# Remove-AzStorageQueue

## Synopsis
Entfernt eine Speicherwarteschlange.

## Syntax

```
Remove-AzStorageQueue [-Name] <String> [-Force] [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzStorageQueue** entfernt eine Speicherwarteschlange.

## Beispiele

### Beispiel 1: Entfernen einer Speicherwarteschlange nach Namen
```
PS C:\>Remove-AzStorageQueue "ContosoQueue01"
```

Dieser Befehl entfernt eine Warteschlange mit dem Namen ContosoQueue01.

### Beispiel 2: Entfernen mehrerer Speicher Warteschlangen
```
PS C:\>Get-AzStorageQueue "Contoso*" | Remove-AzStorageQueue
```

Mit diesem Befehl werden alle Warteschlangen mit Namen entfernt, die mit Contoso beginnen.
