# Disable-AzStorageStaticWebsite

## Synopsis
Deaktivieren Sie die statische Website für das Azure Storage-Konto.

## Syntax

```
Disable-AzStorageStaticWebsite [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Disable-AzStorageStaticWebsite** deaktiviert die statische Website für das Azure-Speicherkonto.

## Beispiele

### Beispiel 1: Deaktivieren der statischen Website für ein Azure Storage-Konto
```
C:\PS>Disable-AzStorageStaticWebsite
```

Mit diesem Befehl wird die statische Website für ein Azure Storage-Konto deaktiviert.
