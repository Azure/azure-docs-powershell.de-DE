# Disable-AzStorageBlobRestorePolicy

## Synopsis
Deaktiviert die BLOB-Wiederherstellungsrichtlinie für ein Speicherkonto.

## Syntax

### Kontoname (Standard)
```
Disable-AzStorageBlobRestorePolicy [-ResourceGroupName] <String> [-StorageAccountName] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Kontoobject
```
Disable-AzStorageBlobRestorePolicy -StorageAccount <PSStorageAccount> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### BlobServicePropertiesResourceId
```
Disable-AzStorageBlobRestorePolicy [-ResourceId] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Disable-AzStorageBlobRestorePolicy** deaktiviert die BLOB-Wiederherstellungsrichtlinie für den Azure Storage-BLOB-Dienst.

## Beispiele

### Beispiel 1: deaktiviert die BLOB-Wiederherstellungsrichtlinie für den Azure Storage-BLOB-Dienst auf einem Speicherkonto
```powershell
PS C:\> Disable-AzStorageBlobRestorePolicy -ResourceGroupName "myresourcegoup" -StorageAccountName "mystorageaccount"
```

Mit diesem Befehl wird die BLOB-Wiederherstellungsrichtlinie für den Azure Storage-BLOB-Dienst auf einem Speicherkonto deaktiviert.
