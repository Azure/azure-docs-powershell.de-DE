# AZ. Sicherheitsmodul
## Beschreibung
Das Azure Security Center bietet Ihnen die Kontrolle über die Sicherheit Ihrer Azure-Abonnements und anderer Computer, mit denen Sie außerhalb von Azure verbunden sind.
