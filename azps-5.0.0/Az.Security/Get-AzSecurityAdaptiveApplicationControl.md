# Get-AzSecurityAdaptiveApplicationControl

## Synopsis
Ruft eine Liste der Application Control VM/Server-Gruppen für das Abonnement ab.

## Syntax

### SubscriptionScope (Standard)
```
Get-AzSecurityAdaptiveApplicationControl [-SubscriptionId <String>] [-IncludePathRecommendation <Boolean>] [-Summary <Boolean>] 
[-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Steuerelemente für die Adaptive Anwendung werden automatisch vom Azure Security Center berechnet, verwenden Sie dieses Cmdlet, um eine Liste der Steuerelemente für Adaptive Anwendungen im Umfang eines Abonnements abzurufen.

### Beispiel 1
```powershell
PS C:\> Get-AzSecurityAdaptiveApplicationControl

Location   : centralus

Id         : /subscriptions/3eeab341-f466-499c-a8be-85427e154baf7612f869/providers/Microsoft.Security/locations/centralus/applicationWhitelistings/GROUP3
Name       : GROUP3

Location   : centralus
```

Ruft eine Liste der Application Control VM/Server-Gruppen für das Abonnement ab.
