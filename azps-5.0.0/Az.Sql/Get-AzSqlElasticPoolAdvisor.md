# Get-AzSqlElasticPoolAdvisor

## Synopsis
Ruft einen oder mehrere Ratgeber für einen Azure SQL-elastischen Pool ab.

## Syntax

```
Get-AzSqlElasticPoolAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -ElasticPoolName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzSqlElasticPoolAdvisor** " Ruft einen oder mehrere Azure SQL Elastic Pool Advisor für einen Azure SQL-elastischen Pool ab.
