# Get-AzSqlElasticPoolRecommendedAction

## Synopsis
Ruft eine oder mehrere Empfohlene Aktionen für einen Azure SQL Elastic Pool Advisor ab.

## Syntax

```
Get-AzSqlElasticPoolRecommendedAction [-RecommendedActionName <String>] -ServerName <String>
 -ElasticPoolName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzSqlElasticPoolRecommendedAction** " Ruft eine oder mehrere Empfohlene Aktionen für einen Azure SQL Elastic Pool Advisor ab.
