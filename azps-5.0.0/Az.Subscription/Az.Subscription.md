# AZ. Subscription-Modul
## Beschreibung
In den Themen in diesem Abschnitt werden die Azure PowerShell-Cmdlets für Azure-Abonnements im Azure Resource Manager (Arm)-Framework dokumentiert. Die Cmdlets sind im Microsoft. Azure. Commands. Subscription-Namespace vorhanden.

## AZ. Subscription-Cmdlets
### [Get-AzSubscriptionAlias](Get-AzSubscriptionAlias.md)
Ruft Alias Details für Abonnements ab.

### [Neu – AzSubscriptionAlias](New-AzSubscriptionAlias.md)
Erstellt einen neuen Alias und ein neues Abonnement

### [Remove-AzSubscriptionAlias](Remove-AzSubscriptionAlias.md)
Löscht den Abonnement-Alias

### [Update-AzSubscription](Update-AzSubscription.md)
Aktualisiert ein Azure-Abonnement
