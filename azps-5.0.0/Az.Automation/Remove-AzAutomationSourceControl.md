# Remove-AzAutomationSourceControl

## Synopsis
Entfernt ein Azure Automation-Quellcodeverwaltungs-Steuerelement.

## Syntax

```
Remove-AzAutomationSourceControl [-Name] <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Das Remove-AzAutomationSourceControl-Cmdlet entfernt eine Quellcodeverwaltung aus der Azure-Automatisierung.

## Beispiele

### Beispiel 1
Dieser Befehl entfernt das Automatisierungs Quellen-Steuerelement mit dem Namen VSTSNative im Konto mit dem Namen devAccount.
Mit diesem Befehl wird der *Force* -Parameter angegeben. Daher werden Sie nicht zur Bestätigung aufgefordert.

```powershell
PS C:\> Remove-AzAutomationSourceControl -ResourceGroupName "rg1" `
                                              -AutomationAccountName "devAccount" `
                                              -Name "VSTSNative" `
                                              -Force
```
