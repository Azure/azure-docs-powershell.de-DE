# Set-AzEventHubNamespace

## Synopsis
Aktualisiert den angegebenen Event Hubs-Namespace.

## Syntax

## Beschreibung
Das Set-AzEventHubNamespace-Cmdlet aktualisiert die Eigenschaften des angegebenen Event Hubs-Namespace.

## Beispiele

### Beispiel 1
