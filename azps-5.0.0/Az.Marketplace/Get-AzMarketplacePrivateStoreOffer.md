# Get-AzMarketplacePrivateStoreOffer

## Synopsis
Besorgen Sie sich ein oder mehrere private Store-Angebote.

## Syntax

```
Get-AzMarketplacePrivateStoreOffer -PrivateStoreId <String> [-OfferId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Besorgen Sie sich ein oder mehrere private Store-Angebote mit öffentlich-privaten Plänen, die unter MANDANTENBEREICH hinzugefügt wurden. Wenn die Abonnement-ID präsentiert wird, besorgen Sie sich das Angebot eines oder mehrerer privater Stores mit privaten Plänen nur unter dem Abonnementumfang.
