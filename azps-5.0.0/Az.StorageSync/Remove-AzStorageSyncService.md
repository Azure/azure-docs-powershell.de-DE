# Remove-AzStorageSyncService

## Synopsis
Mit diesem Befehl wird der angegebene Speicher Synchronisierungsdienst gelöscht.

## Syntax

### StringParameterSet (Standard)
```
Remove-AzStorageSyncService [-ResourceGroupName] <String> [-Name] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzStorageSyncService [-InputObject] <PSStorageSyncService> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzStorageSyncService [-ResourceId] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit diesem Befehl wird der angegebene Speicher Synchronisierungsdienst gelöscht. Ein Speicher Synchronisierungsdienst kann nur entfernt werden, wenn alle enthaltenen Synchronisierungsgruppen und registrierten Server zuerst gelöscht werden.

## Beispiele

### Beispiel 1
```powershell
PS C:\> Remove-AzStorageSyncService -Force -ResourceGroupName "myResourceGroup" -Name "myStorageSyncServiceName"
```

Mit diesem Befehl wird der Speicher Synchronisierungsdienst entfernt.
