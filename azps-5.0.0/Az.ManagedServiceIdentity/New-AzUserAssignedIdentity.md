# New-AzUserAssignedIdentity

## Synopsis
Erstellt einen neuen Benutzer, dem eine Identität zugewiesen wurde, oder aktualisiert eine vorhandene Benutzer zugewiesene Identität.

## Syntax

```
New-AzUserAssignedIdentity [-ResourceGroupName] <String> [-Name] <String> [-Location <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-Tag <Hashtable>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **New-AzUserAssignedIdentity** erstellt eine neue Benutzer zugewiesene Identität. Bei Verwendung mit einer bereits vorhandenen Identität wurde die Identität aktualisiert.
Wenn Sie der Identität Azure Resource Manager-Tags hinzufügen möchten, verwenden Sie das Set-AzResource-Cmdlet.
