# Add-AzServiceFabricNode

## Synopsis
Fügen Sie dem jeweiligen Knotentyp im Clusterknoten hinzu.

## Syntax

```
Add-AzServiceFabricNode -NumberOfNodesToAdd <Int32> [-ResourceGroupName] <String> [-Name] <String>
 -NodeType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Verwenden **Sie Add-AzServiceFabricNode** , um dem jeweiligen Knotentyp Knoten hinzuzufügen. Sie müssen lediglich die Anzahl der Knoten angeben, die Sie einem Knotentyp hinzufügen möchten.

## Beispiele

### Beispiel 1
```powershell
PS c:> Add-AzServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NumberOfNodesToAdd 2 -NodeTypeName 'nt1'
```

Mit diesem Befehl werden dem Knotentyp "N1" 2 Knoten hinzugefügt.
