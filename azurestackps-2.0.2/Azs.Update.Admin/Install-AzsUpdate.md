# Install-AzsUpdate

## Synopsis
Wenden Sie ein bestimmtes Update an einem Aktualisierungsspeicherort an.

## Syntax

### Übernehmen (Standard)
```
Install-AzsUpdate -Name <String> [-Location <String>] [-ResourceGroupName <String>] [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### ApplyViaIdentity
```
Install-AzsUpdate -InputObject <IUpdateAdminIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Beschreibung
Wenden Sie ein bestimmtes Update an einem Aktualisierungsspeicherort an.

## Beispiele

### Beispiel 1: Install-AzsUpdate nach Name
```powershell
PS C:\> Install-AzsUpdate -Name Microsoft1.1907.0.10
```

Unifiedgroup ermöglicht es Ihnen, bestimmte Updates nach Namen zu installieren.
Beachten Sie, dass die Update Version unbedingt größer als die aktuelle Version ist.
