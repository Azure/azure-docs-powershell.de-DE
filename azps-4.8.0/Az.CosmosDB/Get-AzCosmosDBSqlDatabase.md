# Get-AzCosmosDBSqlDatabase

## Synopsis
Ruft die CosmosDB-SQL-Datenbank ab.

## Syntax

### ByNameParameterSet (Standard)
```
Get-AzCosmosDBSqlDatabase -ResourceGroupName <String> -AccountName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Get-AzCosmosDBSqlDatabase [-Name <String>] -ParentObject <PSDatabaseAccountGetResults>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzCosmosDBSqlDatabase** " Ruft die Liste aller vorhandenen CosmosDB SQL-Datenbanken f端r einen bestimmten ResourceGroupName, Accountname, ab und ruft eine einzelne CosmosDB-SQL-Datenbank f端r eine bestimmte ResourceGroupName, Accountname, DatabaseName und Containername ab.
