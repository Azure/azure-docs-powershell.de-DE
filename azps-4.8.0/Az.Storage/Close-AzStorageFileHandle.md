# Close-AzStorageFileHandle

## Synopsis

## Beschreibung
Mit dem Cmdlet " **Close-AzStorageFileHandle** " werden Dateihandles einer Dateifreigabe oder eines Dateiverzeichnisses oder einer Datei geschlossen.

## Beispiele

### Beispiel 1: Listet alle Dateifreigaben des aktuellen speicherkontos auf und schließt alle Dateihandles der Dateifreigaben rekursiv.
```
PS C:\>Get-AzStorageShare | Close-AzStorageFileHandle -CloseAll -Recursive
```

Dieser Befehl listet alle Dateifreigaben des aktuellen speicherkontos auf und schließt alle Dateihandles der Dateifreigaben rekursiv.

### Beispiel 2: Schließen Sie alle Dateihandles in einem Dateiverzeichnis rekursiv, und zeigen Sie die Anzahl der geschlossenen Dateihandles an.
```
PS C:\>Close-AzStorageFileHandle -ShareName "mysharename" -Path 'dir1/dir2' -Recursive -CloseAll -PassThru
10
```

Mit diesem Befehl werden alle Dateihandles in einem Dateiverzeichnis geschlossen und die Anzahl der geschlossenen Dateihandles angezeigt.

### Beispiel 3: Schließen Sie alle Dateihandles, die vor 1 Tag in einem Dateiverzeichnis geöffnet sind.
```
PS C:\>Get-AzStorageFileHandle -ShareName "mysharename" -Path 'dir1/dir2' -Recursive | ? {$_.OpenTime.DateTime.AddDays(1) -lt (Get-Date)} | Close-AzStorageFileHandle -ShareName "mysharename"
```

Dieser Befehl listet alle Dateihandles in einem Dateiverzeichnis rekursiv auf, filtert die Handles, die vor 1 Tag geöffnet sind, und schließt Sie dann.

### Beispiel 4: Schließen aller Dateihandles für eine Datei
```
PS C:\>Close-AzStorageFileHandle -ShareName "mysharename" -Path 'dir1/dir2/test.txt' -CloseAll
```

Mit diesem Befehl werden alle Dateihandles für eine Datei geschlossen.
