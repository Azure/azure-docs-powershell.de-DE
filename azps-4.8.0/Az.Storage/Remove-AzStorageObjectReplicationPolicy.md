# Remove-AzStorageObjectReplicationPolicy

## Synopsis
Entfernt die angegebene Objekt Replikationsrichtlinie von einem Speicherkonto.

## Syntax

### Kontoname (Standard)
```
Remove-AzStorageObjectReplicationPolicy [-ResourceGroupName] <String> [-StorageAccountName] <String>
 -PolicyId <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Kontoobject
```
Remove-AzStorageObjectReplicationPolicy -StorageAccount <PSStorageAccount> -PolicyId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Richtlinienobject
```
Remove-AzStorageObjectReplicationPolicy -InputObject <PSObjectReplicationPolicy> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzStorageObjectReplicationPolicy** entfernt die angegebene Objekt Replikationsrichtlinie von einem Speicherkonto.

## Beispiele

### Beispiel 1: Entfernen einer Objekt Replikationsrichtlinie mit einer bestimmten Richtlinien-Nr von einem Speicherkonto
```
Remove-AzStorageObjectReplicationPolicy -ResourceGroupName "myresourcegroup" -AccountName "mystorageaccount" -PolicyId $policyId
```

Mit diesem Befehl wird eine Objekt Replikationsrichtlinie mit einer bestimmten Richtlinien-Nr von einem Speicherkonto entfernt.
