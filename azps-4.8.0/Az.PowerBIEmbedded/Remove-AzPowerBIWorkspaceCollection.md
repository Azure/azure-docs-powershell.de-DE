# Remove-AzPowerBIWorkspaceCollection

## Synopsis
Entfernt eine Power BI-Arbeitsbereichs Sammlung.

## Syntax

```
Remove-AzPowerBIWorkspaceCollection [-ResourceGroupName] <String> [-WorkspaceCollectionName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzPowerBIWorkspaceCollection** entfernt eine Power BI-Arbeitsbereichs Sammlung aus ihrer Azure-Abonnement-und Ressourcengruppe.

## Beispiele

### Beispiel 1: Entfernen einer Arbeitsbereichs Sammlung
```
PS C:\>Remove-AzPowerBIWorkspaceCollection -ResourceGroupName "ResourceGroup17" -WorkspaceCollectionName "WCN11"
```

Dieser Befehl entfernt die Arbeitsbereichs Sammlung mit dem Namen WCN11 in der angegebenen Ressourcengruppe.
