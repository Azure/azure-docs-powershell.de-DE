# AZ. ConnectedKubernetes-Modul
## Beschreibung
Microsoft Azure PowerShell: ConnectedKubernetes-Cmdlets

## AZ. ConnectedKubernetes-Cmdlets
### [Get-AzConnectedKubernetes](Get-AzConnectedKubernetes.md)
Gibt die Eigenschaften des angegebenen verbundenen Clusters zurück, einschließlich Name, Identität, Eigenschaften und zusätzlicher Cluster Details.

### [Neu – AzConnectedKubernetes](New-AzConnectedKubernetes.md)
API zum Registrieren eines neuen K8s-Clusters und damit zum Erstellen einer nachverfolgten Ressource in Arm

### [Remove-AzConnectedKubernetes](Remove-AzConnectedKubernetes.md)
Löschen eines verbundenen Clusters und Entfernen der nachverfolgten Ressource im Azure Resource Manager (Arm).

### [Update-AzConnectedKubernetes](Update-AzConnectedKubernetes.md)
API zum Aktualisieren bestimmter Eigenschaften der verbundenen Clusterressource
