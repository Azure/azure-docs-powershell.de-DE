# Disable-AzHDInsightMonitoring

## Synopsis
Deaktiviert die Überwachung in einem HDInsight-Cluster, und relevante Protokolle werden nicht mehr in den während der Aktivierung angegebenen Überwachungs Arbeitsbereich überfließen.

## Syntax

```
Disable-AzHDInsightMonitoring [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Disable-AzHDInsightMonitoring** deaktiviert die Überwachung in einem Azure HDInsight-Cluster.

## Beispiele

### Beispiel 1
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster

True
```

Die Überwachung wird im HDInsight-Cluster deaktiviert, und relevante Protokolle werden nicht mehr in den Arbeitsbereich "Überwachung" fließen.

### Beispiel 2
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster -ResourceGroupName testrg

True
```

Die Überwachung wird im HDInsight-Cluster deaktiviert, und relevante Protokolle werden nicht mehr in den Arbeitsbereich "Überwachung" fließen.
