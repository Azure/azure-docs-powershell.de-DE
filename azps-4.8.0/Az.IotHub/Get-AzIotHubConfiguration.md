# Get-AzIotHubConfiguration

## Synopsis
Listet alle oder eine bestimmte viel automatische Geräte Verwaltungskonfiguration auf.

## Syntax

### ResourceSet (Standard)
```
Get-AzIotHubConfiguration [-ResourceGroupName] <String> [-IotHubName] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### InputObjectSet
```
Get-AzIotHubConfiguration [-InputObject] <PSIotHub> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIdSet
```
Get-AzIotHubConfiguration [-ResourceId] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Besorgen Sie sich die Details einer viel automatischen Device Management-Konfiguration, oder Listen Sie viel automatische Device Management-Konfigurationen in einem viel Hub auf.
Weitere Informationen finden Sie unter https://docs.microsoft.com/azure/iot-hub/iot-hub-automatic-device-management .

## Beispiele

### Beispiel 1
```powershell
PS C:\> Get-AzIotHubConfiguration -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "config1"
```

Rufen Sie die Details einer viel automatischen Geräte Verwaltungskonfiguration ab.

### Beispiel 2
```powershell
PS C:\> Get-AzIotHubConfiguration -ResourceGroupName "myresourcegroup" -IotHubName "myiothub"
```

Listen Sie viele automatische Geräte Verwaltungs Konfigurationen in einem viel Hub auf.
