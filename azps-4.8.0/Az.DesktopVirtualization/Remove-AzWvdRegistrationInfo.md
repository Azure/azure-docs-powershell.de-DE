# Remove-AzWvdRegistrationInfo

## Synopsis
Entfernen Sie die Registrierungsinformationen für Windows Virtual Desktop.

## Syntax

```
Remove-AzWvdRegistrationInfo -HostPoolName <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Beschreibung
Entfernen Sie die Registrierungsinformationen für Windows Virtual Desktop.

## Beispiele

### Beispiel 1: Löschen eines Windows Virtual Desktop-Registrierungs Tokens
```powershell
PS C:\> Remove-AzWvdRegistrationInfo -ResourceGroupName ResourceGroupName -HostPoolName HostPoolName
```

Mit diesem Befehl wird ein Windows Virtual Desktop-Registrierungs Token in einem Host Pool gelöscht.
