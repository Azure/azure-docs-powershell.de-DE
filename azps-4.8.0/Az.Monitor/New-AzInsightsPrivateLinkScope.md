# New-AzInsightsPrivateLinkScope

## Synopsis
Erstellen eines privaten Verknüpfungsbereichs

## Syntax

```
New-AzInsightsPrivateLinkScope -Location <String> -ResourceGroupName <String> -Name <String> [-Tags <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Erstellen eines privaten Verknüpfungsbereichs

## Beispiele

### Beispiel 1
```powershell
New-AzInsightsPrivateLinkScope -Location "eastus" -ResourceGroupName "rg_name" -Name "scope_name"
```

Erstellen eines privaten Verknüpfungsbereichs mit dem Namen "scope_name" unter Ressourcengruppe "rg_name" an Position "eastus"
