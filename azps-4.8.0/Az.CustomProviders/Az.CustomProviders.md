# AZ. CustomProviders-Modul
## Beschreibung
Microsoft Azure PowerShell: Cmdlets für benutzerdefinierte Anbieter

## AZ. CustomProviders-Cmdlets
### [Get-AzCustomProvider](Get-AzCustomProvider.md)
Ruft das benutzerdefinierte Ressourcenanbieter Manifest ab.

### [Get-AzCustomProviderAssociation](Get-AzCustomProviderAssociation.md)
Besorgen Sie sich eine Zuordnung.

### [Neu – AzCustomProvider](New-AzCustomProvider.md)
Erstellt oder aktualisiert den benutzerdefinierten Ressourcenanbieter.

### [Neu – AzCustomProviderAssociation](New-AzCustomProviderAssociation.md)
Erstellen oder Aktualisieren einer Zuordnung

### [Remove-AzCustomProvider](Remove-AzCustomProvider.md)
Löscht den benutzerdefinierten Ressourcenanbieter.

### [Remove-AzCustomProviderAssociation](Remove-AzCustomProviderAssociation.md)
Löschen einer Zuordnung

### [Update-AzCustomProvider](Update-AzCustomProvider.md)
Aktualisiert einen vorhandenen benutzerdefinierten Ressourcenanbieter.
Der einzige Wert, der über Patch aktuell aktualisiert werden kann, sind die Tags.
