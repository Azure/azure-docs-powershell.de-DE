# Set-AzIoTDeviceProvisioningServiceEnrollmentGroup

## Synopsis
Aktualisieren einer Geräte Registrierungsgruppe


## Beschreibung
Aktualisieren einer Registrierungsgruppe in einem Azure viel-Bereitstellungsdienst für Hub-Geräte

## Beispiele

### Beispiel 1
```powershell
PS C:\> Set-AzIoTDeviceProvisioningServiceEnrollmentGroup -ResourceGroupName "myresourcegroup" -DpsName "mydps" -Name "enroll1" -AllocationPolicy Hashed -IotHub "hub1","hub2"
```

Aktualisieren von Zuordnungsrichtlinien und Hubs für eine Registrierungsgruppe.
