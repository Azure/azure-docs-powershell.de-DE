# Remove-AzSynapseLinkedService

## Synopsis
Entfernt einen verknüpften Dienst aus dem Arbeitsbereich.

## Syntax

### RemoveByName (Standard)
```
Remove-AzSynapseLinkedService -WorkspaceName <String> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByObject
```
Remove-AzSynapseLinkedService -WorkspaceObject <PSSynapseWorkspace> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByInputObject
```
Remove-AzSynapseLinkedService -InputObject <PSLinkedServiceResource> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **Remove-AzSynapseLinkedService** wird ein verknüpfter Dienst aus dem Arbeitsbereich entfernt.

## Beispiele

### Beispiel 1
```powershell
PS C:\> Remove-AzSynapseLinkedService -WorkspaceName ContosoWorkspace -Name ContosoLinkedService
```

Mit diesem Befehl wird der verknüpfte Dienst mit dem Namen ContosoLinkedService aus dem Arbeitsbereich mit dem Namen ContosoWorkspace entfernt.

### Beispiel 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Remove-AzSynapseLinkedService -Name ContosoLinkedService
```

Mit diesem Befehl wird der verknüpfte Dienst mit dem Namen ContosoLinkedService aus dem Arbeitsbereich mit dem Namen ContosoWorkspace through Pipeline entfernt.

### Beispiel 3
```powershell
PS C:\> $linkedService = Get-AzSynapseLinkedService -WorkspaceName ContosoWorkspace -Name ContosoLinkedService
PS C:\> $linkedService | Remove-AzSynapseLinkedService
```

Mit diesem Befehl wird der verknüpfte Dienst mit dem Namen ContosoLinkedService aus dem Arbeitsbereich mit dem Namen ContosoWorkspace through Pipeline entfernt.
