# Get-AzSynapseDataFlow

## Synopsis
Ruft Informationen zu Datenflüssen im Arbeitsbereich ab.

## Syntax

### GetByName (Standard)
```
Get-AzSynapseDataFlow -WorkspaceName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByObject
```
Get-AzSynapseDataFlow -WorkspaceObject <PSSynapseWorkspace> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzSynapseDataFlow** " Ruft Informationen zu Datenflüssen im Arbeitsbereich ab.
Wenn Sie den Namen eines Datenflusses angeben, ruft dieses Cmdlet Informationen zu diesem Datenfluss ab.
Wenn Sie keinen Namen angeben, ruft dieses Cmdlet Informationen zu allen Datenflüssen im Arbeitsbereich ab.

## Beispiele

### Beispiel 1
```powershell
PS C:\> Get-AzSynapseDataFlow -WorkspaceName ContosoWorkspace
```

Dieser Befehl ruft Informationen zu allen Datenflüssen im Arbeitsbereich mit dem Namen ContosoWorkspace ab.

### Beispiel 2
```powershell
PS C:\> Get-AzSynapseDataFlow -WorkspaceName ContosoWorkspace -Name ContosoDataFlow
```

Dieser Befehl ruft Informationen über den Datenfluss mit dem Namen ContosoDataFlow im Arbeitsbereich mit dem Namen ContosoWorkspace ab.
