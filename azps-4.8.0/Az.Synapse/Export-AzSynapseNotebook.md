# Export-AzSynapseNotebook

## Synopsis
Exportiert Notbooks.

## Syntax

### ExportByName (Standard)
```
Export-AzSynapseNotebook -WorkspaceName <String> [-Name <String>] -OutputFolder <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ExportByObject
```
Export-AzSynapseNotebook -WorkspaceObject <PSSynapseWorkspace> [-Name <String>] -OutputFolder <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ExportByInputObject
```
Export-AzSynapseNotebook -InputObject <PSNotebookResource> -OutputFolder <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Export-AzSynapseNotebook** exportiert ein Azure Synapse-Notizbuch in eine Notizbuch-Datei (. ipynb). Der Name des Notizbuchs wird zum Namen der exportierten Datei. Wenn Sie den Namen eines Notizbuchs angeben, wird dieses Notizbuch vom Cmdlet exportiert. Wenn Sie keinen Namen angeben, werden alle Notizbücher im Arbeitsbereich vom Cmdlet exportiert.

## Beispiele

### Beispiel 1
```powershell
PS C:\> Export-AzSynapseNotebook -WorkspaceName ContosoWorkspace -OutputFolder "C:\Notebook"
```

Exportiert alle Notizbücher im Arbeitsbereich ContosoWorkspace in den Ordner "C:\Notebook".

### Beispiel 2
```powershell
PS C:\> Export-AzSynapseNotebook -WorkspaceName ContosoWorkspace -Name ContosoNotebook -OutputFolder "C:\Notebook"
```

Exportiert ein einzelnes Notizbuch namens ContosoNotebook im Arbeitsbereich ContosoWorkspace in den Ordner "C:\Notebook".

### Beispiel 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Export-AzSynapseNotebook -Name ContosoNotebook -OutputFolder "C:\Notebook"
```

Exportiert ein einzelnes Notizbuch mit dem Namen ContosoNotebook im Arbeitsbereich ContosoWorkspace in den Ordner "C:\Notebook" bis zur Pipeline.

### Beispiel 4
```powershell
PS C:\> $notebook = Get-AzSynapseNotebook -WorkspaceName ContosoWorkspace -Name ContosoNotebook
PS C:\> $notebook | Export-AzSynapseNotebook -OutputFolder "C:\Notebook"
```

Exportiert ein einzelnes Notizbuch mit dem Namen ContosoNotebook im Arbeitsbereich ContosoWorkspace in den Ordner "C:\Notebook" bis zur Pipeline.
