# Get-AzSynapseWorkspace

## Synopsis
Ruft einen Synapse-Analyse Arbeitsbereich ab.

## Syntax

### GetByNameParameterSet (Standard)
```
Get-AzSynapseWorkspace [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzSynapseWorkspace -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzSynapseWorkspace** " Ruft Informationen zu einem Azure Synapse-Analyse Arbeitsbereich ab.

## Beispiele

### Beispiel 1
```powershell
PS C:\> Get-AzSynapseWorkspace
```

Dieser Befehl ruft alle Azure Synapse-Analyse Arbeitsbereiche unter dem aktuellen Abonnement ab.

### Beispiel 2
```powershell
PS C:\> Get-AzSynapseWorkspace -ResourceGroupName ContosoResourceGroup
```

Dieser Befehl ruft alle Azure Synapse-Analyse Arbeitsbereiche unter dem aktuellen Abonnement ab.

### Beispiel 3
```powershell
PS C:\> Get-AzSynapseWorkspace -Name ContosoWorkspace
```

Mit diesem Befehl wird der Azure Synapse Analytics-Arbeitsbereich mit dem angegebenen Namen abgerufen.

### Beispiel 4
```powershell
PS C:\> Get-AzSynapseWorkspace -ResourceId /subscriptions/21686af7-58ec-4f4d-9c68-f431f4db4edd/resourceGroups/ContosoResourceGroup/providers/Microsoft.Synapse/workspaces/ContosoWorkspace
```

Mit diesem Befehl wird der Azure Synapse Analytics-Arbeitsbereich mit der angegebenen Ressourcen-ID abgerufen.
