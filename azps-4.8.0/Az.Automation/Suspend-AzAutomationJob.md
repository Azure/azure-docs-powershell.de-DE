# Suspend-AzAutomationJob

## Synopsis
Unterbricht einen Automatisierungs Auftrag.

## Syntax

```
Suspend-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **Suspend-AzAutomationJob** wird ein Azure-Automatisierungs Auftrag angehalten.
Geben Sie einen ausgeführten Automatisierungs Auftrag an.
Verwenden Sie das Resume-AzAutomationJob-Cmdlet, um einen angehalten Job fortzusetzen.

## Beispiele

### Beispiel 1: Anhalten eines Auftrags
```
PS C:\>Suspend-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Mit diesem Befehl wird der Auftrag mit der angegebenen ID angehalten.
