# Remove-AzAutomationWebhook

## Synopsis
Entfernt einen webhook aus einem Automatisierungs-runbooks.

## Syntax

```
Remove-AzAutomationWebhook [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzAutomationWebhook** entfernt einen webhook aus einem Azure Automation-runbooks.
Der webhook wird gelöscht.

## Beispiele

### Beispiel 1: Entfernen eines webhooks
```
PS C:\>Remove-AzAutomationWebhook -Name "Webhook11" -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01" -Force
```

Dieser Befehl entfernt einen webhook mit dem Namen Webhook11 im Automatisierungs Konto mit dem Namen AutomationAccount01.
Der Befehl gibt den Parameter *Force* an.
Daher werden Sie nicht zur Bestätigung aufgefordert.
