# Disable-AzureRmAlias

## Synopsis
Deaktiviert AzureRm-Präfix Aliase für AZ-Module.

## Syntax

```
Disable-AzureRmAlias [-Scope <String>] [-Module <String[]>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Deaktiviert AzureRm-Präfix Aliase für AZ-Module. Wenn-Module angegeben ist, werden nur in den aufgeführten Modulen Aliase deaktiviert. Andernfalls sind alle AzureRm-Aliase deaktiviert.

## Beispiele

### Beispiel 1
```
PS C:\> Disable-AzureRmAlias
```

Deaktiviert alle AzureRm-Präfixe für die aktuelle PowerShell-Sitzung.

### Beispiel 1
```
PS C:\> Disable-AzureRmAlias -Module Az.Accounts -Scope CurrentUser
```

Deaktiviert AzureRm-Aliase für das AZ. Accounts-Modul sowohl für den aktuellen Prozess als auch für den aktuellen Benutzer.
