# Test-AzAnalysisServicesServer

## Synopsis
Testet das vorhanden sein einer Instanz des Analysis Services-Servers

## Syntax

```
Test-AzAnalysisServicesServer [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Test-AzAnalysisServicesServer-Cmdlet testet das vorhanden sein einer Instanz des Analysis Services-Servers.

## Beispiele

### Beispiel 1
```
PS C:\> Test-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Dieser Befehl testet, ob ein Server mit dem Namen "Testserver" in der testgroup der ResourceGroup vorhanden ist.
