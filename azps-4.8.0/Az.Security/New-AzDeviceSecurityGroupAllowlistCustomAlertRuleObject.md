# New-AzDeviceSecurityGroupAllowlistCustomAlertRuleObject

## Synopsis
Erstellen einer benutzerdefinierten Warnungsregel für eine neue Zulassungsliste für die Geräte Sicherheitsgruppe (viel Sicherheit)

## Syntax

```
New-AzDeviceSecurityGroupAllowlistCustomAlertRuleObject -Enabled <Boolean> -Type <String>
 -AllowlistValue <String[]> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das New-AzDeviceSecurityGroupAllowlistCustomAlertRuleObject-Cmdlet erstellt eine neue Liste zulässiger benutzerdefinierter Warnungsregeln für die Geräte Sicherheitsgruppe in der Sicherheitslösung.

## Beispiele

### Beispiel 1
```powershell
PS C:\> New-AzDeviceSecurityGroupAllowlistCustomAlertRuleObject -Enabled $false -Type "LocalUserNotAllowed" -AllowlistValue @()

RuleType: "LocalUserNotAllowed"
DisplayName: "Login by a local user that isn't allowed"
Description: "Get an alert when a local user that isn't allowed logins to the device"
IsEnabled: false
ValueType: "String"
AllowlistValues: []
```

Erstellen einer benutzerdefinierten Warnungs Rull vom Typ "LocalUserNotAllowed" mit dem Status "deaktiviert"
