# Set-AzCdnEndpoint

## Synopsis
Aktualisiert einen CDN-Endpunkt.

## Syntax

```
Set-AzCdnEndpoint -CdnEndpoint <PSEndpoint> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Satz-AzCdnEndpoint** " aktualisiert einen Azure Content Delivery Network (CDN)-Endpunkt.
