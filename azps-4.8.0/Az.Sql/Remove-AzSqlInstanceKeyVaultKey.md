# Remove-AzSqlInstanceKeyVaultKey

## Synopsis
Entfernt einen Key Vault-Schlüssel aus einer verwalteten SQL-Instanz

## Syntax

### AddAzureRmSqlManagedInstanceKeyVaultKeyDefaultParameterSet (Standard)
```
Remove-AzSqlInstanceKeyVaultKey [-ResourceGroupName] <String> [-InstanceName] <String> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddAzureRmSqlManagedInstanceKeyVaultKeyInputObjectParameterSet
```
Remove-AzSqlInstanceKeyVaultKey [-Instance] <AzureSqlManagedInstanceModel> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddAzureRmSqlManagedInstanceKeyVaultKeyResourceIdParameterSet
```
Remove-AzSqlInstanceKeyVaultKey [-InstanceResourceId] <String> [-KeyId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Remove-AzSqlInstanceKeyVaultKey-Cmdlet entfernt den schlüsseltresor Schlüssel aus der angegebenen verwalteten Instanz. Beachten Sie, dass die Berechtigungen der verwalteten SQL-Instanz für den Tresor des Schlüssels nicht geändert werden. Wenn Sie Berechtigungen ändern möchten, verwenden Sie die Einstellung-AzKeyVaultAccessPolicy. Beachten Sie, dass dieses Cmdlet keine Änderungen an Key Vault vornimmt. Wenn Sie einen Schlüssel aus dem Schlüsselspeicher entfernen möchten, verwenden Sie Remove-AzureKeyVaultKey.
