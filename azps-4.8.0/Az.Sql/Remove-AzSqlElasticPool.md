# Remove-AzSqlElasticPool

## Synopsis
Löscht einen elastischen datenbankpool.

## Syntax

```
Remove-AzSqlElasticPool [-ElasticPoolName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzSqlElasticPool** löscht einen Azure SQL-Datenbank-Elastic-Pool.

## Beispiele

### Beispiel 1: Löschen eines elastischen Pools
```
PS C:\>Remove-AzSqlElasticPool -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Dieser Befehl löscht einen elastischen Pool mit dem Namen ElasticPool01.
