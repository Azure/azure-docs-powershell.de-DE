# New-AzFirewallHubIpAddress

## Synopsis
IP-Adressen assoicated der Firewall auf dem virtuellen Hub

## Syntax

```
New-AzFirewallHubIpAddress [-PrivateIPAddress <String>] [-PublicIPs <PSAzureFirewallHubPublicIpAddresses>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
IP-Adressen assoicated der Firewall auf dem virtuellen Hub. Hierbei kann es sich um öffentliche und private Adressen handeln.

## Beispiele

### Beispiel 1
```powershell
PS C:\> $fwpips = New-AzFirewallHubPublicIpAddress -Count 2
PS C:\> New-AzFirewallHubIpAddress -PublicIPs $fwpips
```

In diesem Beispiel wird ein Hub-IP-Adressen Objekt mit einer Anzahl von zwei öffentlichen IPS erstellt. Das HubIPAddress-Objekt wird mit der Firewall auf dem virtuellen Hub ssociated.
