# Remove-AzNetworkWatcherPacketCapture

## SYNOPSIS
Entfernt eine Paketerfassungsressource.

## SYNTAX

### SetByResource (Standard)
```
Remove-AzNetworkWatcherPacketCapture -NetworkWatcher <PSNetworkWatcher> -PacketCaptureName <String> [-PassThru]
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByName
```
Remove-AzNetworkWatcherPacketCapture -NetworkWatcherName <String> -ResourceGroupName <String>
 -PacketCaptureName <String> [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### SetByLocation
```
Remove-AzNetworkWatcherPacketCapture -Location <String> -PacketCaptureName <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Der Remove-AzNetworkWatcherPacketCapture entfernt eine Paketaufnahmeressource. Es wird empfohlen, die Stop-AzNetworkWatcherPacketCapture, bevor Sie Remove-AzNetworkWatcherPacketCapture aufrufen. Wenn die Paketaufzeichnungssitzung ausgeführt wird, Remove-AzNetworkWatcherPacketCapture als Paketaufzeichnung bezeichnet wird, wird die Paketaufzeichnung möglicherweise nicht gespeichert. Wenn die Sitzung vor dem Entfernen beendet wird, wird die CAP-Datei, die Aufnahmedaten enthält, nicht entfernt. 

## BEISPIELE

### Beispiel 1: Entfernen einer Paketaufzeichnungssitzung
```
Remove-AzNetworkWatcherPacketCapture -NetworkWatcher $networkWatcher -PacketCaptureName "PacketCaptureTest"
```

In diesem Beispiel entfernen wir eine vorhandene Paketaufzeichnungssitzung namens "PacketCaptureTest".
