# Update-AzVirtualApplianceSite

## Synopsis
Ändern oder Ändern einer virtuellen Appliance-Website, die mit einer virtuellen Netzwerk-Appliance-Ressource verbunden ist

## Syntax

```
Update-AzVirtualApplianceSite -Name <String> -ResourceGroupName <String> -NetworkVirtualApplianceId <String>
 [-AddresssPrefix <String>] [-O365Policy <PSOffice365PolicyProperties>] [-Tag <Hashtable>] [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit dem Befehl Update-AzVirtualApplianceSite wird eine Website Ressource einer virtuellen Appliance geändert.

## Beispiele

### Beispiel 1
```powershell
PS C:\> $nva=Get-AzNetworkVirtualAppliance -ResourceGroupName testrg -Name nva
PS C:\> Update-AzVirtualApplianceSite -Name testsite -ResourceGroupName testrg -AddresssPrefix 10.0.4.0/24 -NetworkVirtualApplianceId $nva.Id
```

Ändern Sie das Adresspräfix für eine Website Ressource einer virtuellen Appliance.
