# Get-AzPeeringServiceCountry

## Synopsis
Listet die Länder auf, die für peeringdienst verfügbar sind.

## Syntax

```
Get-AzPeeringServiceCountry [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Listet die Länder auf, die für peeringdienst verfügbar sind.

## Beispiele

### Beispiel 1
```powershell
PS C:\> Get-AzPeeringServiceCountry
```

Liste der Länder, die für peeringdienst verfügbar sind.
