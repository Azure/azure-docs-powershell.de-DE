# Add-AzEventHubIPRule

## Synopsis
Hinzufügen einer einzelnen IP-Regel zum NetworkRuleSet des angegebenen Namespaces

## Syntax

### IPRulePropertiesParameterSet (Standard)
```
Add-AzEventHubIPRule [-ResourceGroupName] <String> [-Name] <String> [-IpMask] <String> [-Action <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IPRuleInputObjectParameterSet
```
Add-AzEventHubIPRule [-ResourceGroupName] <String> [-Name] <String>
 [-IpRuleObject] <PSNWRuleSetIpRulesAttributes> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Hinzufügen einer einzelnen IP-Regel zum NetworkRuleSet des angegebenen Namespaces

## Beispiele

### Beispiel 1
```powershell
PS C:\> Add-AzEventHubIPRule -ResourceGroupName v-ajnavtest -Namespace Eventhub-Namespace1-2389 -IpMask "11.22.33.44" -Action Allow
```
Name: standardmäßige Standard: Allow ID:/Subscriptions/SubscriptionId/resourceGroups/RSG-TestAzEventhub/Providers/Microsoft.Eventhub/Namespaces/Eventhub-Namespace-2389/networkRuleSets/default Typ: Microsoft. Eventhub/Namespaces/NetworkRuleSet IpRules: {11.22.33.44, allow} VirtualNetworkRules: 

Fügen Sie die IPRule mit IpMask "11.22.33.44" hinzu, und die Aktion ermöglicht den angegebenen Namespace.

### Beispiel 2
```powershell
PS C:\> Add-AzEventHubIPRule -ResourceGroupName v-ajnavtest -Namespace Eventhub-Namespace1-2389 -IpRuleObject $ipruleobject
```
Name: standardmäßige Standard: Allow ID:/Subscriptions/SubscriptionId/resourceGroups/RSG-TestAzEventhub/Providers/Microsoft.Eventhub/Namespaces/Eventhub-Namespace-2389/networkRuleSets/default Typ: Microsoft. Eventhub/Namespaces/NetworkRuleSet IpRules: {11.22.33.44, allow} VirtualNetworkRules: 

Fügen Sie die IPRule mit IpMask "11.22.33.44" hinzu, und die Aktion ermöglicht den angegebenen Namespace.
