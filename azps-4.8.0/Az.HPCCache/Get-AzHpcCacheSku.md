# Get-AzHpcCacheSku

## Synopsis
Ruft alle im Abonnement verfügbaren SKUs ab.

## Syntax

```
Get-AzHpcCacheSku [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzHpcCacheSku** " gibt eine Liste der in Subscription verfügbaren SKUs zurück.

## Beispiele

### Beispiel 1
```powershell
PS C:\> Get-AzHpcCacheSku
```
