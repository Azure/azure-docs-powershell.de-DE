# New-AzDiskAccess

## Synopsis
Erstellt eine Datenträgerzugriffs Ressource

## Syntax

```
New-AzDiskAccess [-ResourceGroupName] <String> [-Name] <String> [-Location] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **New-AzDiskAccess** " erstellt eine Datenträgerzugriffs Ressource

## Beispiele

### Beispiel 1
```
PS C:\> New-AzDiskAccess -ResourceGroupName "ResourceGroup01" -Name "DiskAccess01" -Location "NorthCentralUS"
```

Mit diesem Befehl wird ein Datenträgerzugriff mit bestimmten Eigenschaften erstellt.
