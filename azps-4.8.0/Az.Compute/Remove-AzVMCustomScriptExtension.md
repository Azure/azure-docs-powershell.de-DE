# Remove-AzVMCustomScriptExtension

## Synopsis
Entfernt eine benutzerdefinierte Skripterweiterung von einem virtuellen Computer.

## Syntax

```
Remove-AzVMCustomScriptExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Force]
 [-NoWait] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Remove-AzVMCustomScriptExtension** " entfernt eine benutzerdefinierte Skript Virtual Machine-Erweiterung von einem virtuellen Computer.

## Beispiele

### Beispiel 1

Entfernt eine benutzerdefinierte Skripterweiterung von einem virtuellen Computer. automatisch

```powershell <!-- Aladdin Generated Example --> 
Remove-AzVMCustomScriptExtension -Name 'AgentPool01' -ResourceGroupName myresourcegroup -VMName 'VM01'
```
