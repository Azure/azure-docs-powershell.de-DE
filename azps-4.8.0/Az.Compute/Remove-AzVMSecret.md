# Remove-AzVMSecret

## Synopsis
Entfernt (a) Secret (s) aus einem virtuellen Computerobjekt

## Syntax

```
Remove-AzVMSecret [-VM] <PSVirtualMachine> [[-SourceVaultId] <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Remove-AzVMSecret-Cmdlet entfernt (a) Secret (s) aus einem virtuellen Computerobjekt.

## Beispiele

### Beispiel 1
```
PS C:\> Get-AzVM -ResourceGroupName "rg1" -Name "vm1" | Remove-AzVMSecret | Update-AzVM
```

Entfernt alle Geheimnisse von einem virtuellen Computer "VM1" in der Ressourcengruppe "RG1" und aktualisiert die VM.
