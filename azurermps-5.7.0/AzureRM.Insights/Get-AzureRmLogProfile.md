# Get-AzureRmLogProfile

## Synopsis
Ruft ein Protokoll Profil ab.

```
Get-AzureRmLogProfile [-Name <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureRmLogProfile** " Ruft ein Protokoll Profil ab.
