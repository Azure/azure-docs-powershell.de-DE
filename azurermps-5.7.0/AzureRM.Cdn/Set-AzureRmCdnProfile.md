# Set-AzureRmCdnProfile

## Synopsis
Aktualisiert ein CDN-Profil.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Set-AzureRmCdnProfile -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Satz-AzureRmCdnProfile** " aktualisiert ein Azure Content Delivery Network (CDN)-Profil.
