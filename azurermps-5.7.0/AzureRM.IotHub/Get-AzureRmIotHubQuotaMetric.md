# Get-AzureRmIotHubQuotaMetric

## Synopsis
Ruft die Kontingent Metrik für eine IotHub ab.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Get-AzureRmIotHubQuotaMetric [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Ruft die Kontingent Metrik für eine IotHub ab.

## Beispiele

### Beispiel 1 Abrufen der Kontingent Metriken
```
PS C:\> Get-AzureRmIotHubQuotaMetric -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Ruft die Kontingent metrischen Informationen für das IotHub mit dem Namen "myiothub" ab.
