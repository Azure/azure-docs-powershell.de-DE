# AzureRM. LocationBasedServices-Modul
## Beschreibung
In diesem Thema wird die Hilfe zu den Azure Location based Services-Cmdlets angezeigt.

## AzureRM. LocationBasedServices-Cmdlets
### [Get-AzureRmLocationBasedServicesAccount](Get-AzureRmLocationBasedServicesAccount.md)
Ruft das Konto ab.

### [Get-AzureRmLocationBasedServicesAccountKey](Get-AzureRmLocationBasedServicesAccountKey.md)
Ruft die API-Schlüssel für ein Konto ab. Diese Schlüssel sind der Authentifizierungsmechanismus, der bei nachfolgenden Aufrufen von Azure Location based Services verwendet wird.

### [Neu – AzureRmLocationBasedServicesAccount](New-AzureRmLocationBasedServicesAccount.md)
Erstellt ein standortbasiertes Dienstkonto.

### [Neu – AzureRmLocationBasedServicesAccountKey](New-AzureRmLocationBasedServicesAccountKey.md)
Generiert einen Kontoschlüssel erneut.

### [Remove-AzureRmLocationBasedServicesAccount](Remove-AzureRmLocationBasedServicesAccount.md)
Löscht ein standortbasiertes Dienstkonto.
