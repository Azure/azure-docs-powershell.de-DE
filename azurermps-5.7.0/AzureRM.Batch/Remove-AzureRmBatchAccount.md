# Remove-AzureRmBatchAccount

## Synopsis
Entfernt ein Stapelverarbeitungs Konto.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Remove-AzureRmBatchAccount [-AccountName] <String> [[-ResourceGroupName] <String>] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureRmBatchAccount** entfernt ein Azure-Batch Konto.
Dieses Cmdlet fordert Sie auf, bevor ein Konto entfernt wird, es sei denn, Sie geben den Parameter *Force* an.

## Beispiele

### Beispiel 1: Entfernen eines Stapelverarbeitungs Kontos
```
PS C:\>Remove-AzureRmBatchAccount -AccountName "pfuller"
```

Dieser Befehl entfernt das Batch Konto mit dem Namen pfuller.
Mit diesem Befehl werden Sie zur Bestätigung aufgefordert, bevor Sie das Konto löschen.
