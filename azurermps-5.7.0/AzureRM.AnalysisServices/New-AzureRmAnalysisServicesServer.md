# New-AzureRmAnalysisServicesServer

## Synopsis
Erstellt einen neuen Analysis Services-Server

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
New-AzureRmAnalysisServicesServer [-ResourceGroupName] <String> [-Name] <String> [-Location] <String>
 [-Sku] <String> [[-Tag] <Hashtable>] [[-Administrator] <String>] [[-BackupBlobContainerUri] <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das New-AzureRmAnalysisServicesServer-Cmdlet erstellt einen neuen Analysis Services-Server.

## Beispiele

### Beispiel 1
```
PS C:\> New-AzureRmAnalysisServicesServer -ResourceGroupName "testresourcegroup" -Name "testserver" -Location "West-US" -Sku "S1"
```

Erstellt einen Server mit dem Namen Testserver im Azure Region West-US und in der Ressourcengruppe testresrourcegroup. Die SKU-Ebene für den Server ist S1.
