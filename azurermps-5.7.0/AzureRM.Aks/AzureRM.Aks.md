# AzureRM. AKS-Modul
## Beschreibung
Befehle für die Interaktion mit Azure Managed Kubernetes-Clustern

## AzureRM. AKS-Cmdlets
### [Get-AzureRmAks](Get-AzureRmAks.md)
Liste Kubernetes verwalteter Cluster

### [Importieren-AzureRmAksCredential](Import-AzureRmAksCredential.md)
Importieren und Zusammenführen der Kubectl-Konfiguration für einen verwalteten Kubernetes-Cluster

### [Neu – AzureRmAks](New-AzureRmAks.md)
Erstellen eines neuen verwalteten Kubernetes-Clusters

### [Remove-AzureRmAks](Remove-AzureRmAks.md)
Löschen eines verwalteten Kubernetes-Clusters

### [Satz-AzureRmAks](Set-AzureRmAks.md)
Aktualisieren oder Erstellen eines verwalteten Kubernetes-Clusters

### [Anfang-AzureRmAksDashboard](Start-AzureRmAksDashboard.md)
Erstellen Sie einen Kubectl-SSH-Tunnel zum Dashboard des verwalteten Clusters.

### [Stopp-AzureRmAksDashboard](Stop-AzureRmAksDashboard.md)
Beenden Sie den Kubectl-SSH-Tunnel, der in Start-AzureRmKubernetesDashboard erstellt wurde.
