# New-AzureRmRecoveryServicesAsrvCenter

## Synopsis
Fügt einen vCenter Server hinzu, um geschützte Elemente zu erkennen.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
New-AzureRmRecoveryServicesAsrvCenter -Fabric <ASRFabric> -Name <String> -Account <ASRRunAsAccount>
 -Port <Int32> -IpOrHostName <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **New-AzureRmRecoveryServicesAsrvCenter** wird ein vCenter Server hinzugefügt, in dem die geschützten Elemente erkannt werden. Mit diesem Cmdlet wird der vCenter Server für die Ermittlung mit dem Konfigurationsserver registriert.

## Beispiele

### Beispiel 1
```
PS C:\> New-AzureRmRecoveryServicesAsrvCenterServer -Account $ConfigServer.FabricSpecificDetails.RunAsAccounts[1] -Fabric $ConfigServer -Name InmTest59 -Port 443 -Server 10.150.209.6

Asr Job for vCenter creation.
```

Fügt einen vCenter Server hinzu, um geschützte Elemente zu erkennen.
