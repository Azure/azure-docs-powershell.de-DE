# Stop-AzureRmSqlSyncGroupSync

## Synopsis
Beendet die Synchronisierung einer Synchronisierungsgruppe.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Stop-AzureRmSqlSyncGroupSync [-SyncGroupName] <String> [-PassThru] [-ServerName] <String>
 [-DatabaseName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Stop-AzureRmSqlSyncGroupSync** " beendet die Synchronisierung der Synchronisierungsgruppe.

## Beispiele

### Beispiel 1
```
PS C:\> Stop-AzureRmSqlSyncGroupSync -SyncGroupName mysg [-ServerName] mysrv [-DatabaseName] mydb [-ResourceGroupName] myrg
```

Dieser Befehl beendet die Synchronisierung, die für die Synchronisierungsgruppe mysg fortgesetzt wird.
