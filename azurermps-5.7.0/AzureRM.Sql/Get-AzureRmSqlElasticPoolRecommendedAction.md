# Get-AzureRmSqlElasticPoolRecommendedAction

## Synopsis
Ruft eine oder mehrere Empfohlene Aktionen für einen Azure SQL Elastic Pool Advisor ab.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Get-AzureRmSqlElasticPoolRecommendedAction [-RecommendedActionName <String>] -ServerName <String>
 -ElasticPoolName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureRmSqlElasticPoolRecommendedAction** " Ruft eine oder mehrere Empfohlene Aktionen für einen Azure SQL Elastic Pool Advisor ab.
