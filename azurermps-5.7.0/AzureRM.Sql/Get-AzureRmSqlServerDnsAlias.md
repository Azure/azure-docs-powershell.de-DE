# Get-AzureRmSqlServerDnsAlias

## Synopsis
Ruft den Azure SQL Server-DNS-Alias ab oder listet ihn auf.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Get-AzureRmSqlServerDnsAlias [-Name <String>] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Abrufen des bestimmten Azure SQL Server-DNS-Alias oder Auflisten aller Server-DNS-Aliase für den Server

## Beispiele

### Beispiel 1
```
PS C:\> $serverDNSAliases = Get-AzureRmSqlServerDNSAlias -ServerName servername -ResourceGroupName rgname

ResourceGroupName  ServerName   DnsAliasName
-----------------  ----------   ------------------
rgname             servername   dnsaliasname
rgname             servername   dnsaliasname2
```

Listet alle Server-DNS-Aliase für den jeweiligen Server auf

### Beispiel 2
```
PS C:\> $serverDNSAliases = Get-AzureRmSqlServerDNSAlias -DnsAliasName dnsaliasname -ServerName servername -ResourceGroupName rgname

ResourceGroupName  ServerName   DnsAliasName
-----------------  ----------   ------------------
rgname             servername   dnsaliasname
```

Ruft den Server-DNS-Alias ab, der vom Server und Aliasnamen angegeben wird.
