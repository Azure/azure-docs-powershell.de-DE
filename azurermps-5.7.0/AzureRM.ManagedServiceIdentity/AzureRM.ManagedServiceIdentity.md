# AzureRM. ManagedServiceIdentity-Modul
## Beschreibung
In diesem Thema werden die Hilfethemen für die Azure Managed Service Identity-Cmdlets angezeigt.

## AzureRM. ManagedServiceIdentity-Cmdlets
### [Get-AzureRmUserAssignedIdentity](Get-AzureRmUserAssignedIdentity.md)
Ruft Benutzer zugewiesene Identitäten/Identitäten ab.

### [Neu – AzureRmUserAssignedIdentity](New-AzureRmUserAssignedIdentity.md)
Erstellt einen neuen Benutzer, dem eine Identität zugewiesen wurde, oder aktualisiert eine vorhandene Benutzer zugewiesene Identität.

### [Remove-AzureRmUserAssignedIdentity](Remove-AzureRmUserAssignedIdentity.md)
Entfernt eine Benutzer zugewiesene Identität.
