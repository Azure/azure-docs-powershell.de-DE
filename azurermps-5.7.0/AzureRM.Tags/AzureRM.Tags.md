# AzureRM. Tags-Modul
## Beschreibung
In diesem Thema werden die Hilfethemen für die Azure Tag-Cmdlets angezeigt.

## AzureRM. Tags-Cmdlets
### [Get-AzureRmTag](Get-AzureRmTag.md)
Ruft vordefinierte Azure-Tags ab.

### [Neu – AzureRmTag](New-AzureRmTag.md)
Erstellt eine vordefinierte Azure-Kategorie oder fügt Werte zu einem vorhandenen Tag hinzu.

### [Remove-AzureRmTag](Remove-AzureRmTag.md)
Löscht vordefinierte Azure-Tags oder-Werte.
