# Get-AzureRmSiteRecoveryVaultSettings

## Synopsis
Ruft Einstellungsinformationen für den aktuellen Standort Wiederherstellungs Tresor ab.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Get-AzureRmSiteRecoveryVaultSettings [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureRmSiteRecoveryVaultSettings** " ruft Einstellungsinformationen im Zusammenhang mit dem aktuellen Azure Site Recovery Vault ab.
