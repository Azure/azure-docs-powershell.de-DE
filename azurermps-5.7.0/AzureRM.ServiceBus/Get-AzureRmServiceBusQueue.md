# Get-AzureRmServiceBusQueue

## Synopsis
Gibt eine Beschreibung für die angegebene ServiceBus-Warteschlange zurück.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Get-AzureRmServiceBusQueue [-ResourceGroupName] <String> [-Namespace] <String> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureRmServiceBusQueue** " gibt eine Beschreibung der angegebenen ServiceBus-Warteschlange zurück.
