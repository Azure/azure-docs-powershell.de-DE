# Remove-AzureRmRedisCacheLink

## Synopsis
Entfernen einer Geo-Replikationsverknüpfung zwischen zwei 2-a-Caches

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Remove-AzureRmRedisCacheLink -PrimaryServerName <String> -SecondaryServerName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Entfernen einer Geo-Replikationsverknüpfung zwischen zwei 2-a-Caches

## Beispiele

### Beispiel 1: Entfernen einer Geo-Replikationsverknüpfung
```
PS C:\>Remove-AzureRmRedisCacheLink -PrimaryServerName "mycache1" -SecondaryServerName "mycache2"
```

Dieser Befehl entfernt eine Geo-Replication-Links, wobei der mycache1-Cache mit dem Namen "Primär" und der "mycache2.
