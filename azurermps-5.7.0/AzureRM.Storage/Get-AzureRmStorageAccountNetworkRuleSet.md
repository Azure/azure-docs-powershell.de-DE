# Get-AzureRmStorageAccountNetworkRuleSet

## Synopsis
Abrufen der NetWorkRule-Eigenschaft eines speicherkontos

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Get-AzureRmStorageAccountNetworkRuleSet [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureRmStorageAccountNetworkRuleSet** " Ruft die NetworkRule-Eigenschaft eines speicherkontos ab.

## Beispiele

### Beispiel 1: Abrufen der NetworkRule-Eigenschaft eines angegebenen speicherkontos
```
PS C:\> Get-AzureRmStorageAccountNetworkRuleSet  -ResourceGroupName "rg1" -AccountName "mystorageaccount"
```

Dieser Befehl ruft die NetworkRule-Eigenschaft eines angegebenen speicherkontos ab.
