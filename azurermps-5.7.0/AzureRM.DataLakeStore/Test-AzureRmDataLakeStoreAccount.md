# Test-AzureRmDataLakeStoreAccount

## Synopsis
Testet das vorhanden sein eines Data Lake Store-Kontos.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Test-AzureRmDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Test-AzureRmDataLakeStoreAccount** testet das vorhanden sein eines Data Lake Store-Kontos.

## Beispiele

### Beispiel 1: Testen eines Kontos
```
PS C:\>Test-AzureRmDataLakeStoreAccount -Name "ContosoADL"
```

Dieser Befehl testet, ob das Konto mit dem Namen ContosoADL vorhanden ist.
