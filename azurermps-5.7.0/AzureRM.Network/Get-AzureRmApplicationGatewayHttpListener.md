# Get-AzureRmApplicationGatewayHttpListener

## Synopsis
Ruft den HTTP-Listener eines Anwendungs Gateways ab.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Get-AzureRmApplicationGatewayHttpListener [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureRmApplicationGatewayHttpListener** " Ruft den HTTP-Listener eines Anwendungs Gateways ab.

## Beispiele

### Beispiel 1: Abrufen eines bestimmten HTTP-Listeners
```
PS C:\>$Appgw = Get-AzureRmApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Listener = Get-AzureRmApplicationGatewayHttpListener -Name "Listener01" -ApplicationGateway $Appgw
```

Dieser Befehl ruft einen HTTP-Listener mit dem Namen Listener01 ab.

### Beispiel 2: Abrufen einer Liste mit http-Listenern
```
PS C:\>$Appgw = Get-AzureRmApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Listeners = Get-AzureRmApplicationGatewayHttpListener -ApplicationGateway $Appgw
```

Dieser Befehl ruft eine Liste der HTTP-Listener ab.
