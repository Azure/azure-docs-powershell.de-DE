# Remove-AzureRmExpressRouteCircuit

## Synopsis
Entfernt einen Express Route-Schaltkreis.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Remove-AzureRmExpressRouteCircuit -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **Remove-AzureRmExpressRouteCircuit** wird ein Express Route-Schaltkreis entfernt.

## Beispiele

### Beispiel 1: Löschen eines Express Route-Schaltkreises
```
Remove-AzureRmExpressRouteCircuit -Name $CircuitName -ResourceGroupName $rg
```

### Beispiel 2: Löschen eines Express Route-Schaltkreises mithilfe der Pipeline
```
Get-AzureRmExpressRouteCircuit -Name $CircuitName -ResourceGroupName $rg | Remove-AzureRmExpressRouteCircuit
```
