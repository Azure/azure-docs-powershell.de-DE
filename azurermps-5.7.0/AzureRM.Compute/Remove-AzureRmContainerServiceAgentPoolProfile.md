# Remove-AzureRmContainerServiceAgentPoolProfile

## Synopsis
Entfernt ein Profil eines Agenten Pools aus einem Containerdienst.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Remove-AzureRmContainerServiceAgentPoolProfile [-ContainerService] <ContainerService> [-Name] <String>
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureRmContainerServiceAgentPoolProfile** entfernt ein Agenten Pool Profil aus einem Containerdienst.

## Beispiele

### Beispiel 1: Entfernen eines Profils aus einem Containerdienst
```
PS C:\> $Container = Get-AzureRmContainerService -ResourceGroupName "ResourceGroup17" -Name "CSResourceGroup17" 
PS C:\> Remove-AzureRmContainerServiceAgentPoolProfile -ContainerService $Container -Name "AgentPool01"
```

Der erste Befehl ruft einen Containerdienst mit dem Namen CSResourceGroup17 mithilfe des Get-AzureRmContainerService-Cmdlets ab.
Der Befehl speichert den Dienst in der $Container-Variablen.

Der zweite Befehl entfernt das Profil mit dem Namen AgentPool01 aus dem Containerdienst in $Container.
