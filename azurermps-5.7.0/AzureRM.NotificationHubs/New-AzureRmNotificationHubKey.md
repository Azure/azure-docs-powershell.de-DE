# New-AzureRmNotificationHubKey

## Synopsis
Generieren Sie den Autorisierungsregel Schlüssel für eine NotificationHub erneut.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
New-AzureRmNotificationHubKey [-ResourceGroup] <String> [-Namespace] <String> [-NotificationHub] <String>
 [[-AuthorizationRule] <String>] [-PolicyKey] <String> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
New-AzureRmNotificationHubKey-Cmdlet generiert den Primärschlüssel/sekundären Schlüssel für die NotificationHub-Autorisierungsregel erneut.
