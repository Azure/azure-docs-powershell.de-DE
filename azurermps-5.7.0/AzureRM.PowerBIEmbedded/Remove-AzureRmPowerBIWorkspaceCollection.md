# Remove-AzureRmPowerBIWorkspaceCollection

## Synopsis
Entfernt eine Power BI-Arbeitsbereichs Sammlung.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Remove-AzureRmPowerBIWorkspaceCollection [-ResourceGroupName] <String> [-WorkspaceCollectionName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureRmPowerBIWorkspaceCollection** entfernt eine Power BI-Arbeitsbereichs Sammlung aus ihrer Azure-Abonnement-und Ressourcengruppe.

## Beispiele

### Beispiel 1: Entfernen einer Arbeitsbereichs Sammlung
```
PS C:\>Remove-AzureRmPowerBIWorkspaceCollection -ResourceGroupName "ResourceGroup17" -WorkspaceCollectionName "WCN11"
```

Dieser Befehl entfernt die Arbeitsbereichs Sammlung mit dem Namen WCN11 in der angegebenen Ressourcengruppe.
