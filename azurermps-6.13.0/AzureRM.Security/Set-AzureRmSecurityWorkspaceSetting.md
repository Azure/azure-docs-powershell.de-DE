# Set-AzureRmSecurityWorkspaceSetting

## Synopsis
Aktualisiert die Arbeitsbereichseinstellungen für das Abonnement.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Set-AzureRmSecurityWorkspaceSetting -Name <String> -Scope <String> -WorkspaceId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Aktualisiert die Arbeitsbereichseinstellungen für das Abonnement.
Der konfigurierte Arbeitsbereich enthält die Sicherheitsdaten, die vom Sicherheitsagenten erfasst wurden, der in VMs innerhalb dieses Abonnements installiert ist.

## Beispiele

### Beispiel 1
```powershell
PS C:\> Set-AzureRmSecurityWorkspaceSetting -Name "default" -Scope "/subscriptions/487bb485-b5b0-471e-9c0d-10717612f869" -WorkspaceId  "/subscriptions/487bb485-b5b0-471e-9c0d-10717612f869/resourcegroups/mainws/providers/microsoft.operationalinsights/workspaces/securityuserws"

Id                                                                                                         Name    WorkspaceId 
--                                                                                                         ----    ----
/subscriptions/487bb485-b5b0-471e-9c0d-10717612f869/providers/Microsoft.Security/workspaceSettings/default default /...
```

Legt den Arbeitsbereich "Mein Arbeitsbereich" so fest, dass alle Sicherheitsdaten enthalten sind, die von den Sicherheits-Agents erfasst wurden.
