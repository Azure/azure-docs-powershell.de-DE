# Set-AzureRmAutomationDscNode

## Synopsis
Ändert die Knoten Konfiguration, der ein DSC-Knoten zugeordnet ist.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Set-AzureRmAutomationDscNode -Id <Guid> -NodeConfigurationName <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Satz-AzureRmAutomationDscNode** " ändert eine APS-Knoten Konfiguration (Desired State Configuration, DSC).
Azure Automation speichert die Konfiguration des DSC-Knotens als MOF-Konfigurationsdokument (Managed Object Format).

## Beispiele

### Beispiel 1: Ändern der Knoten Konfigurations Zuordnung
```
PS C:\>Set-AzureRmAutomationDscNode -NodeConfigurationName "Contoso.NodeConfiguration01" -ResourceGroupName "ResourceGroup01" -Id 064a8929-c98b-25e4-80hh-111c8a6067j8
```

Dieser Befehl weist die Knoten Konfiguration mit dem Namen "contoso. NodeConfiguration01" dem Knoten mit der angegebenen GUID zu.
