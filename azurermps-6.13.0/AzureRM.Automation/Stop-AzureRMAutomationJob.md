# Stop-AzureRmAutomationJob

## Synopsis
Beendet einen Automatisierungs Auftrag.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Stop-AzureRmAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Stop-AzureRmAutomationJob** beendet einen Azure-Automatisierungs Auftrag.
Geben Sie einen ausgeführten Automatisierungs Auftrag an.

## Beispiele

### Beispiel 1: Beenden eines Auftrags
```
PS C:\>Stop-AzureRmAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Mit diesem Befehl wird der Auftrag beendet, der die angegebene ID aufweist.
