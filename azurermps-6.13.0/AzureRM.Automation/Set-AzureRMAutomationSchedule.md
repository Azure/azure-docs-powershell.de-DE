# Set-AzureRmAutomationSchedule

## Synopsis
Ändert einen Automatisierungs Zeitplan.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Set-AzureRmAutomationSchedule [-Name] <String> [-IsEnabled <Boolean>] [-Description <String>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **festlegen-AzureRmAutomationSchedule** " ändert einen Zeitplan in Azure-Automatisierung.

## Beispiele

### Beispiel 1: Ändern eines Zeitplans
```
PS C:\>Set-AzureRmAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule01" -Description "Automation Schedule" -ResourceGroupName "ResourceGroup01"
```

Mit diesem Befehl wird die Beschreibung des Zeitplans mit dem Namen Schedule01 geändert.
