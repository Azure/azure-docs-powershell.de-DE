# Remove-AzureRmDataLakeStoreVirtualNetworkRule

## Synopsis
Entfernt die angegebene virtuelle Netzwerkregel im angegebenen Data Lake Store.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Remove-AzureRmDataLakeStoreVirtualNetworkRule [-Account] <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureRmDataLakeStoreVirtualNetworkRule** entfernt die angegebene virtuelle Netzwerkregel im angegebenen Data Lake Store.

## Beispiele

### Beispiel 1
```powershell
PS C:\> Remove-AzureRmDataLakeStoreVirtualNetworkRule -Account "dls" -Name "myVNET"
```

Entfernt die virtuelle Netzwerkregel "myVNET" aus dem Konto "DLS"
