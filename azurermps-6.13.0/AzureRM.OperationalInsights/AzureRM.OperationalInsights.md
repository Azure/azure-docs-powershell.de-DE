# AzureRM. OperationalInsights-Modul
## Beschreibung
In diesem Thema werden Hilfethemen zu den Azure-Cmdlets für operationelle Einblicke angezeigt.

## AzureRM. OperationalInsights-Cmdlets
### [Deaktivieren-AzureRmOperationalInsightsIISLogCollection](Disable-AzureRmOperationalInsightsIISLogCollection.md)
Beendet die Sammlung von IIS-Protokollen von Computern.

### [Deaktivieren-AzureRmOperationalInsightsLinuxCustomLogCollection](Disable-AzureRmOperationalInsightsLinuxCustomLogCollection.md)
Beendet die Sammlung benutzerdefinierter Protokolle von Linux-Computern.

### [Deaktivieren-AzureRmOperationalInsightsLinuxPerformanceCollection](Disable-AzureRmOperationalInsightsLinuxPerformanceCollection.md)
Beendet die Sammlung von Leistungsindikatoren auf Linux-Computern.

### [Deaktivieren-AzureRmOperationalInsightsLinuxSyslogCollection](Disable-AzureRmOperationalInsightsLinuxSyslogCollection.md)
Beendet die Sammlung von syslog-Daten von Linux-Computern.

### [Enable-AzureRmOperationalInsightsIISLogCollection](Enable-AzureRmOperationalInsightsIISLogCollection.md)
Startet die Sammlung von IIS-Protokollen von Computern in einem Arbeitsbereich.

### [Enable-AzureRmOperationalInsightsLinuxCustomLogCollection](Enable-AzureRmOperationalInsightsLinuxCustomLogCollection.md)
Startet eine Sammlung von benutzerdefinierten Protokollen von Linux-Computern.

### [Enable-AzureRmOperationalInsightsLinuxPerformanceCollection](Enable-AzureRmOperationalInsightsLinuxPerformanceCollection.md)
Startet die Sammlung von Leistungsindikatoren auf Linux-Computern.

### [Enable-AzureRmOperationalInsightsLinuxSyslogCollection](Enable-AzureRmOperationalInsightsLinuxSyslogCollection.md)
Startet die Sammlung von syslog-Daten von Linux-Computern.

### [Get-AzureRmOperationalInsightsDataSource](Get-AzureRmOperationalInsightsDataSource.md)
Abrufen von DataSources unter Azure Log Analytics-Arbeitsbereich

### [Get-AzureRmOperationalInsightsIntelligencePacks](Get-AzureRmOperationalInsightsIntelligencePacks.md)
Ruft die verfügbaren Intelligence Packs ab.

### [Get-AzureRmOperationalInsightsLinkTargets](Get-AzureRmOperationalInsightsLinkTargets.md)
Ruft Konten ab, die keinem Abonnement zugeordnet sind.

### [Get-AzureRmOperationalInsightsSavedSearch](Get-AzureRmOperationalInsightsSavedSearch.md)
Gibt alle gespeicherten Suchvorgänge für einen angegebenen Arbeitsbereich zurück.

### [Get-AzureRmOperationalInsightsSavedSearchResults](Get-AzureRmOperationalInsightsSavedSearchResults.md)
Gibt die Ergebnisse einer Abfrage zurück.

### [Get-AzureRmOperationalInsightsSchema](Get-AzureRmOperationalInsightsSchema.md)
Gibt das einem Arbeitsbereich zugeordnete Schema zurück.

### [Get-AzureRmOperationalInsightsSearchResults](Get-AzureRmOperationalInsightsSearchResults.md)
Gibt Suchergebnisse basierend auf den angegebenen Parametern zurück.

### [Get-AzureRmOperationalInsightsStorageInsight](Get-AzureRmOperationalInsightsStorageInsight.md)
Ruft Informationen zu einem Speicher Einblick ab.

### [Get-AzureRmOperationalInsightsWorkspace](Get-AzureRmOperationalInsightsWorkspace.md)
Ruft Informationen zu einem Arbeitsbereich ab.

### [Get-AzureRmOperationalInsightsWorkspaceManagementGroups](Get-AzureRmOperationalInsightsWorkspaceManagementGroups.md)
Ruft Details zu Verwaltungsgruppen ab, die mit einem Arbeitsbereich verbunden sind.

### [Get-AzureRmOperationalInsightsWorkspaceSharedKeys](Get-AzureRmOperationalInsightsWorkspaceSharedKeys.md)
Ruft die freigegebenen Schlüssel für einen Arbeitsbereich ab.

### [Get-AzureRmOperationalInsightsWorkspaceUsage](Get-AzureRmOperationalInsightsWorkspaceUsage.md)
Ruft die Verwendungsdaten für einen Arbeitsbereich ab.

### [Invoke-AzureRmOperationalInsightsQuery](Invoke-AzureRmOperationalInsightsQuery.md)
Gibt Suchergebnisse basierend auf den angegebenen Parametern zurück.

### [Neu – AzureRmOperationalInsightsAzureActivityLogDataSource](New-AzureRmOperationalInsightsAzureActivityLogDataSource.md)
Das Azure-Aktivitätsprotokoll wird von einem bestimmten Abonnement erfasst.

### [Neu – AzureRmOperationalInsightsComputerGroup](New-AzureRmOperationalInsightsComputerGroup.md)
Erstellt eine Computergruppe.

### [Neu – AzureRmOperationalInsightsCustomLogDataSource](New-AzureRmOperationalInsightsCustomLogDataSource.md)
Definiert eine benutzerdefinierte Protokoll Sammlungs Richtlinie.

### [Neu – AzureRmOperationalInsightsLinuxPerformanceObjectDataSource](New-AzureRmOperationalInsightsLinuxPerformanceObjectDataSource.md)
Fügt allen Linux-Computern in einem Arbeitsbereich Leistungsindikatoren hinzu.

### [Neu – AzureRmOperationalInsightsLinuxSyslogDataSource](New-AzureRmOperationalInsightsLinuxSyslogDataSource.md)
Fügt eine Datenquelle zu Linux-Computern hinzu.

### [Neu – AzureRmOperationalInsightsSavedSearch](New-AzureRmOperationalInsightsSavedSearch.md)
Erstellt eine neue gespeicherte Suche mit den angegebenen Parametern.

### [Neu – AzureRmOperationalInsightsStorageInsight](New-AzureRmOperationalInsightsStorageInsight.md)
Erstellt einen Speicher Einblick in einen Arbeitsbereich.

### [Neu – AzureRmOperationalInsightsWindowsEventDataSource](New-AzureRmOperationalInsightsWindowsEventDataSource.md)
Sammelt Ereignisprotokolle von Computern, auf denen das Windows-Betriebssystem ausgeführt wird.

### [Neu – AzureRmOperationalInsightsWindowsPerformanceCounterDataSource](New-AzureRmOperationalInsightsWindowsPerformanceCounterDataSource.md)
Fügt eine Windows-Leistungsindikator-Datenquelle für verbundene Computer hinzu, auf denen das Windows-Betriebssystem ausgeführt wird.

### [Neu – AzureRmOperationalInsightsWorkspace](New-AzureRmOperationalInsightsWorkspace.md)
Erstellt einen Arbeitsbereich.

### [Remove-AzureRmOperationalInsightsDataSource](Remove-AzureRmOperationalInsightsDataSource.md)
Löscht eine Datenquelle.

### [Remove-AzureRmOperationalInsightsSavedSearch](Remove-AzureRmOperationalInsightsSavedSearch.md)
Entfernt eine gespeicherte Suche aus dem Arbeitsbereich.

### [Remove-AzureRmOperationalInsightsStorageInsight](Remove-AzureRmOperationalInsightsStorageInsight.md)
Entfernt eine Speicher Einsicht.

### [Remove-AzureRmOperationalInsightsWorkspace](Remove-AzureRmOperationalInsightsWorkspace.md)
Entfernt einen Arbeitsbereich.

### [Satz-AzureRmOperationalInsightsDataSource](Set-AzureRmOperationalInsightsDataSource.md)
Aktualisiert eine Datenquelle.

### [Satz-AzureRmOperationalInsightsIntelligencePack](Set-AzureRmOperationalInsightsIntelligencePack.md)
Aktiviert oder deaktiviert das angegebene Intelligence Pack.

### [Satz-AzureRmOperationalInsightsSavedSearch](Set-AzureRmOperationalInsightsSavedSearch.md)
Aktualisiert eine gespeicherte Suche, die bereits vorhanden ist.

### [Satz-AzureRmOperationalInsightsStorageInsight](Set-AzureRmOperationalInsightsStorageInsight.md)
Aktualisiert eine Speicher-Insight.

### [Satz-AzureRmOperationalInsightsWorkspace](Set-AzureRmOperationalInsightsWorkspace.md)
Aktualisiert einen Arbeitsbereich.
