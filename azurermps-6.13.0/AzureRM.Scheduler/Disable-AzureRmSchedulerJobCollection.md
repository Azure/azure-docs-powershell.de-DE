# Disable-AzureRmSchedulerJobCollection

## Synopsis
Deaktiviert eine Auftrags Sammlung.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Disable-AzureRmSchedulerJobCollection -ResourceGroupName <String> -JobCollectionName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Disable-AzureRmSchedulerJobCollection** deaktiviert eine Auftrags Sammlung in Azure Scheduler.
