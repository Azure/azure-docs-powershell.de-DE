# Get-AzureRmIotHubRegistryStatistic

## Synopsis
Ruft die RegistryStatistics für eine IotHub.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Get-AzureRmIotHubRegistryStatistic [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Ruft die RegistryStatistics für eine IotHub.
Damit erhalten Sie Informationen über die Anzahl der insgesamt aktivierten und deaktivierten Geräte in einem IotHub.

## Beispiele

### Beispiel 1 Abrufen des RegistryStatistics
```
PS C:\> Get-AzureRmIotHubRegistryStatistic -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Ruft die RegistryStatictics für die IotHub mit dem Namen "myiothub" ab.
