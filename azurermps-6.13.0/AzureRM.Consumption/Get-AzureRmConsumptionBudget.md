# Get-AzureRmConsumptionBudget

## Synopsis
Rufen Sie eine Liste der Budgets in einem Abonnement oder einer Ressourcengruppe ab.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Get-AzureRmConsumptionBudget [-DefaultProfile <IAzureContextContainer>] [-ResourceGroupName <String>]
 [-Name <String>] [<CommonParameters>]
```

## Beschreibung
Das Get-AzureRmConsumptionBudget-Cmdlet ruft eine Liste der Budgets entweder in einem Abonnement oder in einer Ressourcengruppe ab.

## Beispiele

### Beispiel 1: Abrufen einer Liste der Budgets auf Abonnementebene

             StartDate:  6/1/2018 12:00:00 AM
