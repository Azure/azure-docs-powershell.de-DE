# Reset-AzureRmRedisCache

## Synopsis
Startet Knoten eines Caches neu.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Reset-AzureRmRedisCache [-ResourceGroupName <String>] -Name <String> -RebootType <String> [-ShardId <Int32>]
 [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **Reset-AzureRmRedisCache** werden die Knoten einer Azure-a-Cache-Instanz neu gestartet.

## Beispiele

### Beispiel 1: beide Knoten neu starten
```
PS C:\>Reset-AzureRmRedisCache -ResourceGroupName "ResourceGroup13" -Name "RedisCache06" -RebootType "AllNodes" -Force
```

Dieser Befehl startet beide Knoten für den Cache mit dem Namen "RedisCache06" neu.
