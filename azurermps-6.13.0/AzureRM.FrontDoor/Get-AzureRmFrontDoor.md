# Get-AzureRmFrontDoor

## Synopsis
Herunterladen des Front-Door-Lastenausgleichs

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Get-AzureRmFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Die **Get-AzureRmFrontDoor-** cmdletGet Ruft alle vorhandenen Front Türen im aktuellen Abonnement ab, die den angegebenen Informationen entsprechen.
