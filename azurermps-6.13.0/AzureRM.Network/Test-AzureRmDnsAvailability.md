# Test-AzureRmDnsAvailability

## Synopsis
Überprüft, ob ein Domänenname in der cloudapp.Azure.com-Zone für die Verwendung verfügbar ist.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Test-AzureRmDnsAvailability -DomainNameLabel <String> -Location <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Überprüft, ob ein Domänenname in der cloudapp.Azure.com-Zone für die Verwendung verfügbar ist.

## Beispiele

### Beispiel 1: Überprüfen Sie, ob contoso.cloudapp.Azure.com zur Verwendung verfügbar ist.
```
Test-AzureRmDnsAvailability -DomainNameLabel contoso -Location westus
```
