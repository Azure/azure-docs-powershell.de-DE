# New-AzureRmApplicationGatewayAutoscaleConfiguration

## Synopsis
Erstellt eine AutoScale-Konfiguration für das Anwendungs Gateway.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
New-AzureRmApplicationGatewayAutoscaleConfiguration -MinCapacity <Int32>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **New-AzureRmApplicationGatewayAutoscaleConfiguration** erstellt die Konfiguration von AutoScale für ein Azure Application Gateway.

## Beispiele

### Beispiel 1
```powershell
PS C:\> $autoscaleConfig = New-AzureRmApplicationGatewayAutoscaleConfiguration -MinCapacity 3
PS C:\> $gw = New-AzureRmApplicationGateway -Name $appgwName -ResourceGroupName $rgname ..  -AutoscaleConfiguration $autoscaleConfig
```

Der erste Befehl erstellt eine AutoScale-Konfiguration mit minimaler Kapazität 3.
Mit dem zweiten Befehl wird ein Anwendungsgateway mit der Konfiguration für das AutoSkalieren erstellt.
