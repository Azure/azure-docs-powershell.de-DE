# Set-AzureRmApplicationGatewaySslCertificate

## Synopsis
Legt den Zielstatus eines SSL-Zertifikats fest.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Set-AzureRmApplicationGatewaySslCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> -Password <SecureString> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Set-AzureRmApplicationGatewaySslCertificate** " legt den Zielstatus eines SSL-Zertifikats fest.

## Beispiele

### Beispiel 1: Festlegung des Zielstatus eines SSL-Zertifikats
```
PS C:\> $appGW = Get-AzureRmApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $password = ConvertTo-SecureString "P@ssw0rd" -AsPlainText -Force
PS C:\> $cert = Set-AzureRmApplicationGatewaySslCertificate -ApplicationGateway $AppGW -Name "Cert01" -CertificateFile "D:\cert01.pfx" -Password $password
```

Mit diesem Befehl wird der Zielstatus für ein SSL-Zertifikat vom Anwendungsgateway mit dem Namen ApplicationGateway01 festgelegt.
