# Set-AzureRmVirtualNetworkTap

## Synopsis
Legt den Zielstatus für einen virtuellen Netzwerk Hahn fest.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Set-AzureRmVirtualNetworkTap -VirtualNetworkTap <PSVirtualNetworkTap> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das **Set-AzureRmVirtualNetworkTap-** Element legt den Zielstatus für ein Azure Virtual Network Tap fest.

## Beispiele

### Beispiel 1: Konfigurieren eines virtuellen Netzwerk Hahns
```
PS C:\>$vTap = Get-AzureRmVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
PS C:\>$vTap.DestinationNetworkInterfaceIPConfiguration = $newDestinationNic.IpConfigurations[0]
PS C:\>Set-AzureRmVirtualNetworkTap -VirtualNetworkTap $vTap
```

Der Befehl aktualisiert das Ziel IPKonfigurationsdateiAddress und aktualisiert das virtuelle Netzwerk Tap.
Wenn es auf eine Tap-Konfiguration ankommt, die darauf verweist, wird der gesamte Quell Verkehr nicht mit dem neuen Ziel-IP-Konfigurations Post-Update gespiegelt.
