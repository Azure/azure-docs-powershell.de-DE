# AzureRM. keyvault-Modul
## Beschreibung
In diesem Thema werden Hilfethemen zu den Azure Key Vault-Cmdlets angezeigt.

## AzureRM. keyvault-Cmdlets
### [Add-AzureKeyVaultCertificate](Add-AzureKeyVaultCertificate.md)
Fügt ein Zertifikat zu einem schlüsseltresor hinzu.

### [Add-AzureKeyVaultCertificateContact](Add-AzureKeyVaultCertificateContact.md)
Fügt einen Kontakt für Zertifikat Benachrichtigungen hinzu.

### [Add-AzureKeyVaultKey](Add-AzureKeyVaultKey.md)
Erstellt einen Schlüssel in einem schlüsseltresor oder importiert einen Schlüssel in einen schlüsseltresor.

### [Add-AzureKeyVaultManagedStorageAccount](Add-AzureKeyVaultManagedStorageAccount.md)
Fügt dem angegebenen schlüsseltresor ein vorhandenes Azure Storage-Konto hinzu, dessen Schlüssel vom Schlüssel-Vault-Dienst verwaltet werden.

### [Backup-AzureKeyVaultCertificate](Backup-AzureKeyVaultCertificate.md)
Sichern eines Zertifikats in einem schlüsseltresor

### [Backup-AzureKeyVaultKey](Backup-AzureKeyVaultKey.md)
Sie können einen Schlüssel in einem schlüsseltresor sichern.

### [Backup-AzureKeyVaultManagedStorageAccount](Backup-AzureKeyVaultManagedStorageAccount.md)
Sichern eines von keyvault verwalteten speicherkontos

### [Backup-AzureKeyVaultSecret](Backup-AzureKeyVaultSecret.md)
Sichern eines Geheimnisses in einem schlüsseltresor

### [Get-AzureKeyVaultCertificate](Get-AzureKeyVaultCertificate.md)
Ruft ein Zertifikat aus einem schlüsseltresor ab.

### [Get-AzureKeyVaultCertificateContact](Get-AzureKeyVaultCertificateContact.md)
Ruft Kontakte ab, die für Zertifikat Benachrichtigungen für einen schlüsseltresor registriert sind.

### [Get-AzureKeyVaultCertificateIssuer](Get-AzureKeyVaultCertificateIssuer.md)
Ruft einen Zertifikataussteller für einen schlüsseltresor ab.

### [Get-AzureKeyVaultCertificateOperation](Get-AzureKeyVaultCertificateOperation.md)
Ruft den Status eines Zertifikat Vorgangs ab.

### [Get-AzureKeyVaultCertificatePolicy](Get-AzureKeyVaultCertificatePolicy.md)
Ruft die Richtlinie für ein Zertifikat in einem schlüsseltresor ab.

### [Get-AzureKeyVaultKey](Get-AzureKeyVaultKey.md)
Ruft schlüsseltresor Schlüssel ab.

### [Get-AzureKeyVaultManagedStorageAccount](Get-AzureKeyVaultManagedStorageAccount.md)
Ruft wichtige Vault Managed Azure-Speicherkonten ab.

### [Get-AzureKeyVaultManagedStorageSasDefinition](Get-AzureKeyVaultManagedStorageSasDefinition.md)
Ruft Schlüsselspeicher-SAS-Definitionen für verwalteten Speicher ab.

### [Get-AzureKeyVaultSecret](Get-AzureKeyVaultSecret.md)
Ruft die Geheimnisse in einem schlüsseltresor ab.

### [Get-AzureRmKeyVault](Get-AzureRmKeyVault.md)

### [Importieren-AzureKeyVaultCertificate](Import-AzureKeyVaultCertificate.md)
Importiert ein Zertifikat in einen schlüsseltresor.

### [Neu – AzureKeyVaultCertificateAdministratorDetails](New-AzureKeyVaultCertificateAdministratorDetails.md)
Erstellt ein Zertifikat Administrator Details-Objekt im Arbeitsspeicher.

### [Neu – AzureKeyVaultCertificateOrganizationDetails](New-AzureKeyVaultCertificateOrganizationDetails.md)
Erstellt ein Objekt im Arbeitsspeicher Zertifikat Organisation Details.

### [Neu – AzureKeyVaultCertificatePolicy](New-AzureKeyVaultCertificatePolicy.md)
Erstellt ein Zertifikatrichtlinien Objekt im Arbeitsspeicher.

### [Neu – AzureRmKeyVault](New-AzureRmKeyVault.md)
Erstellt einen schlüsseltresor.

### [Remove-AzureKeyVaultCertificate](Remove-AzureKeyVaultCertificate.md)

### [Remove-AzureKeyVaultSecret](Remove-AzureKeyVaultSecret.md)

### [Satz-AzureRmKeyVaultAccessPolicy](Set-AzureRmKeyVaultAccessPolicy.md)

### [Stopp-AzureKeyVaultCertificateOperation](Stop-AzureKeyVaultCertificateOperation.md)
