# AzureRM. IotCentral-Modul
## Beschreibung
Bietet CRUD-Verwaltung für viele zentrale Anwendungen.

## AzureRM. IotCentral-Cmdlets
### [Get-AzureRmIotCentralApp](Get-AzureRmIotCentralApp.md)
Ruft Eigenschaften für eine oder mehrere zentrale Anwendungen ab.

### [Neu – AzureRmIotCentralApp](New-AzureRmIotCentralApp.md)
Erstellt eine neue viel zentrale Anwendung.

### [Remove-AzureRmIotCentralApp](Remove-AzureRmIotCentralApp.md)
Löscht eine viel zentrale Anwendung.

### [Satz-AzureRmIotCentralApp](Set-AzureRmIotCentralApp.md)
Aktualisiert die Metadaten für eine viel zentrale Anwendung.
