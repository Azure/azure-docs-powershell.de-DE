# AzureRM. RecoveryServices-Modul
## Beschreibung
In diesem Thema werden Hilfethemen zu den Azure Recovery Services-Cmdlets angezeigt.

## AzureRM. RecoveryServices-Cmdlets
### [Get-AzureRmRecoveryServicesBackupProperty](Get-AzureRmRecoveryServicesBackupProperty.md)
Ruft Sicherungseigenschaften ab.

### [Get-AzureRmRecoveryServicesVault](Get-AzureRmRecoveryServicesVault.md)
Ruft eine Liste der Vaults für Wiederherstellungsdienste ab.

### [Get-AzureRmRecoveryServicesVaultSettingsFile](Get-AzureRmRecoveryServicesVaultSettingsFile.md)
Ruft die Azure Site Recovery Vault Settings-Datei ab.

### [Neu – AzureRmRecoveryServicesVault](New-AzureRmRecoveryServicesVault.md)
Erstellt einen neuen Vault für Wiederherstellungsdienste.

### [Remove-AzureRmRecoveryServicesVault](Remove-AzureRmRecoveryServicesVault.md)
Löscht einen Vault für Wiederherstellungsdienste.

### [Satz-AzureRmRecoveryServicesBackupProperties](Set-AzureRmRecoveryServicesBackupProperties.md)
Legt die Eigenschaften für die Sicherungsverwaltung fest.

### [Satz-AzureRmRecoveryServicesVaultContext](Set-AzureRmRecoveryServicesVaultContext.md)
Legt den Vault-Kontext fest.
