# AzureRM. ContainerInstance-Modul
## Beschreibung
In diesem Thema werden Hilfethemen für die Azure Container instance-Cmdlets angezeigt.

## AzureRM. ContainerInstance-Cmdlets
### [Get-AzureRmContainerGroup](Get-AzureRmContainerGroup.md)
Ruft Container Gruppen ab.

### [Get-AzureRmContainerInstanceLog](Get-AzureRmContainerInstanceLog.md)
Rufen Sie die Protokolle einer Containerinstanz in einer Containergruppe ab.

### [Neu – AzureRmContainerGroup](New-AzureRmContainerGroup.md)
Erstellt eine Containergruppe.

### [Remove-AzureRmContainerGroup](Remove-AzureRmContainerGroup.md)
Entfernt eine Containergruppe.
