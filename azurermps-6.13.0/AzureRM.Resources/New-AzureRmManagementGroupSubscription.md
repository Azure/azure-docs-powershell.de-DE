# New-AzureRmManagementGroupSubscription

## Synopsis
Fügt ein Abonnement zu einer Verwaltungsgruppe hinzu.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
New-AzureRmManagementGroupSubscription [-GroupName] <String> [-SubscriptionId] <Guid> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **New-AzureRMManagementGroupSubscription** wird einer Verwaltungsgruppe ein Abonnement hinzugefügt.

## Beispiele

### Beispiel 1: Hinzufügen eines Abonnements zu einer Verwaltungsgruppe
```
PS C:\> New-AzureRMManagementGroupSubscription -GroupName "TestGroup" -SubscriptionId 2120692d-35c3-44c8-81f5-631fa7351726
```
