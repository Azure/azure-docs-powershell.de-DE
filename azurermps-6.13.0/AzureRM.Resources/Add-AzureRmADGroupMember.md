# Add-AzureRmADGroupMember

## Synopsis
Fügt einen Benutzer zu einer vorhandenen Anzeigengruppe hinzu.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

### MemberObjectIdWithGroupObjectId (Standard)
```
Add-AzureRmADGroupMember -MemberObjectId <Guid[]> -TargetGroupObjectId <Guid> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Fügt einen Benutzer zu einer vorhandenen Anzeigengruppe hinzu.

## Beispiele

### Beispiel 1: Hinzufügen eines Benutzers zu einer Gruppe nach Objekt-ID

```
PS C:\> Add-AzureRmADGroupMember -MemberObjectId D9076BBC-D62C-4105-9C78-A7F5BC4A3405 -TargetGroupObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE
```

Fügt den Benutzer mit der Objekt-ID "D9076BBC-D62C-4105-9C78-A7F5BC4A3405" der Gruppe mit der Objekt-ID "85F89C90-780E-4AA6-9F4F-6F268D322EEE" hinzu.

### Beispiel 2 – Hinzufügen eines Benutzers zu einer Gruppe durch Verrohrung

```
PS C:\> Get-AzureRmADGroup -ObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE | Add-AzureRmADGroupMember -MemberObjectId D9076BBC-D62C-4105-9C78-A7F5BC4A3405
```

Ruft die Gruppe mit der Objekt-ID "85F89C90-780E-4AA6-9F4F-6F268D322EEE" ab und leitet Sie an das Add-AzureRmADGroupMember-Cmdlet weiter, um den Benutzer dieser Gruppe hinzuzufügen.
