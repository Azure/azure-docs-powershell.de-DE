# Get-AzureRmSqlServerBackupLongTermRetentionVault

## Synopsis
Ruft einen langfristigen Aufbewahrungs Tresor für Server ab.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Get-AzureRmSqlServerBackupLongTermRetentionVault [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureRmSqlServerBackupLongTermRetentionVault** " Ruft das langfristige Aufbewahrungs Depot ab, das auf diesem Server registriert ist.
Bei Vault handelt es sich um eine Azure-Sicherungs Ressource, die zum Speichern von Sicherungsdaten verwendet wird.
