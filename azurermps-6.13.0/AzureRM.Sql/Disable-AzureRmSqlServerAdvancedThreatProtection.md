# Disable-AzureRmSqlServerAdvancedThreatProtection

## Synopsis
Deaktiviert den erweiterten Bedrohungsschutz auf einem Server.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Disable-AzureRmSqlServerAdvancedThreatProtection [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Disable-AzureRmSqlServerAdvancedThreatProtection** deaktiviert den erweiterten Bedrohungsschutz auf einem Server.

## Beispiele

### Beispiel 1 – Schutz vor erweiterten Bedrohungen durch Server deaktivieren
```powershell
PS C:\>  Disable-AzureRmSqlServerAdvancedThreatProtection `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : False
```

### Beispiel 2 – deaktivieren Sie den erweiterten Schutz vor Bedrohungen durch Serverressourcen
```powershell
PS C:\>  Get-AzureRmSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Disable-AzureRmSqlServerAdvancedThreatProtection

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : False
```
