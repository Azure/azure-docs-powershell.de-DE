# Resume-AzureRmSqlDatabase

## Synopsis
Setzt eine SQL Data Warehouse-Datenbank fort.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Resume-AzureRmSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **Resume-AzureRmSqlDatabase** wird eine Azure SQL Data Warehouse-Datenbank fortgesetzt.

## Beispiele

### Beispiel 1: Fortsetzen einer Azure SQL Data Warehouse-Datenbank
```
PS C:\>Resume-AzureRmSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Mit diesem Befehl wird eine angehaltene Azure SQL Data Warehouse-Datenbank fortgesetzt.
