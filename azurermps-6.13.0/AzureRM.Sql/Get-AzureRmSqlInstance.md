# Get-AzureRmSqlInstance

## Synopsis
Gibt Informationen zu Azure SQL-Datenbankinstanz zurück.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

### GetInstanceByResourceGroup (Standard)
```
Get-AzureRmSqlInstance [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetInstanceByNameAndResourceGroup
```
Get-AzureRmSqlInstance [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureRmSqlInstance** " gibt Informationen zu mindestens einem Azure SQL-verwalteten Instanzen zurück.
Geben Sie den Namen einer Instanz an, um Informationen nur für diese Instanz anzuzeigen.
