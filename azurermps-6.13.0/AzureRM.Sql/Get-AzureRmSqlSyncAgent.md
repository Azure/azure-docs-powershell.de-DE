# Get-AzureRmSqlSyncAgent

## Synopsis
Gibt Informationen zu Azure SQL-Synchronisierungs-Agents zurück.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Get-AzureRmSqlSyncAgent [[-Name] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureRmSqlSyncAgent** " gibt Informationen zu einem oder mehreren Azure SQL-Synchronisierungs-Agents zurück.
Geben Sie den Namen eines Synchronisierungs-Agents an, um Informationen nur für diesen Synchronisierungs-Agent anzuzeigen.
