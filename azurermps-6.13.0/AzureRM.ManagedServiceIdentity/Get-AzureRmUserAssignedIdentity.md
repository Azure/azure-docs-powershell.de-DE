# Get-AzureRmUserAssignedIdentity

## Synopsis
Ruft Benutzer zugewiesene Identitäten/Identitäten ab.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

### SuscriptionParameterSet (Standard)
```
Get-AzureRmUserAssignedIdentity [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceGroupParameterSet
```
Get-AzureRmUserAssignedIdentity -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Der **Get-AzureRmUserAssignedIdentity** ruft vorhandene Benutzer zugewiesene Identitäten ab.
