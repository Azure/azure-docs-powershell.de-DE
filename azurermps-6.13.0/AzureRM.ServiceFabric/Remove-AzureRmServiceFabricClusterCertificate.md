# Remove-AzureRmServiceFabricClusterCertificate

## Synopsis
Entfernen Sie ein Cluster Zertifikat, das für die Cluster Sicherheit verwendet wird.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Remove-AzureRmServiceFabricClusterCertificate -Thumbprint <String> [-Name] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Verwenden **Sie Remove-AzureRmServiceFabricClusterCertificate** , um ein Cluster Zertifikat aus dem Cluster zu entfernen, sofern noch ein weiteres gültiges Zertifikat vorhanden ist, das bereits im Cluster verwendet wird.

## Beispiele

### Beispiel 1
```
PS C:\> Remove-AzureRmServiceFabricClusterCertificate -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -Thumbprint '5F3660C715EBBDA31DB1FFDCF508302348DE8E7A
```

Mit diesem Befehl wird das Zertifikat mit dem Fingerabdruck-5F3660C715EBBDA31DB1FFDCF508302348DE8E7A für die Cluster Sicherheit entfernt.
