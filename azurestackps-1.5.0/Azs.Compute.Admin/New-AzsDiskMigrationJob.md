# New-AzsDiskMigrationJob

## Synopsis
Startet einen Auftrag für die verwaltete Datenträger Migration, um verwaltete Datenträger zur angegebenen Zielfreigabe zu migrieren.

## Syntax

```
New-AzsDiskMigrationJob [-Disks] <Disk[]> [-TargetShare] <String> [[-Location] <String>] [-Name] <String>
 [<CommonParameters>]
```

## Beschreibung
Erstellen eines Datenträger Migrationsauftrags

## Beispiele

### --------------------------Beispiel 1--------------------------
```
New-AzsDiskMigrationJob -Name "MyMigrationJob" -Disks $disks -location local -TargetShare "\\SU1FileServer.azurestack.local\SU1_ObjStore"
```

Starten eines verwalteten Datenträger Migrationsauftrags für die ersten 20 Datenträger
