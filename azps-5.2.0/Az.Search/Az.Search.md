# Az.Search Module
## Beschreibung
In diesem Thema werden Hilfethemen für die Azure Search-Cmdlets angezeigt.

## Az.Search-Cmdlets
### [Get-AzSearchAdminKeyPair](Get-AzSearchAdminKeyPair.md)
Ruft das Administratorschlüsselpaar des Azure Search-Diensts ab.

### [Get-AzSearchQueryKey](Get-AzSearchQueryKey.md)
Ruft die Abfrageschlüssel des Azure Search-Diensts ab.

### [Get-AzSearchService](Get-AzSearchService.md)
Ruft einen Azure Search-Dienst ab.

### [New-AzSearchAdminKey](New-AzSearchAdminKey.md)
Generiert einen Administratorschlüssel des Azure Search-Diensts erneut.

### [New-AzSearchQueryKey](New-AzSearchQueryKey.md)
Erstellen Sie einen neuen Abfrageschlüssel für den Azure Search-Dienst.

### [New-AzSearchService](New-AzSearchService.md)
Erstellt einen Azure Search-Dienst.

### [Remove-AzSearchQueryKey](Remove-AzSearchQueryKey.md)
Entfernen Sie den Abfrageschlüssel aus dem Azure Search-Dienst.

### [Remove-AzSearchService](Remove-AzSearchService.md)
Entfernen Sie einen Azure Search-Dienst.

### [Set-AzSearchService](Set-AzSearchService.md)
Aktualisieren Sie einen Azure Search-Dienst.
