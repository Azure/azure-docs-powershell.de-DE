# Get-AzEventHubNamespace

## SYNOPSIS
Ruft die Details eines Event Hubs-Namespaces ab, oder ruft eine Liste aller Ereignishubs-Namespaces im aktuellen Abonnement von Azure ab.

## SYNTAX

```
Get-AzEventHubNamespace [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das Get-AzEventHubNamespace-Cmdlet ruft entweder die Details eines angegebenen Ereignishubs-Namespaces oder eine Liste aller Ereignishubs-Namespaces im aktuellen Abonnement von Azure ab.
Wenn der Namespacename angegeben wird, werden die Details eines einzelnen Ereignishubs-Namespaces zur端ckgegeben.
Wenn der Namespacename nicht angegeben wird, wird eine Liste mit Namespaces zur端ckgegeben.

Ruft die Details des \` Namespaces "MyNamespaceName" \` in der Ressourcengruppe \` "MyResourceGroupName" \` ab.
