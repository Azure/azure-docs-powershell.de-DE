# Remove-AzAutomationSourceControl

## SYNOPSIS
Entfernt ein Azure Automation-Quellsteuerelement.

## SYNTAX

```
Remove-AzAutomationSourceControl [-Name] <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das Remove-AzAutomationSourceControl entfernt ein Quellsteuerelement aus der Azure-Automatisierung.

## BEISPIELE

### Beispiel 1
Mit diesem Befehl wird das Automatisierungsquellensteuerelement mit dem Namen VSTSNative im Konto "devAccount" entfernt.
Dieser Befehl gibt den Parameter *"Force"* an. Daher werden Sie nicht zur Bestätigung aufgefordert.

```powershell
PS C:\> Remove-AzAutomationSourceControl -ResourceGroupName "rg1" `
                                              -AutomationAccountName "devAccount" `
                                              -Name "VSTSNative" `
                                              -Force
```
