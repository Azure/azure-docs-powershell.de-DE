# Get-AzAutomationJobOutput

## SYNOPSIS
Ruft die Ausgabe eines Automatisierungsauftrags ab.

## SYNTAX

```
Get-AzAutomationJobOutput [-Id] <Guid> [-Stream <StreamType>] [-StartTime <DateTimeOffset>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzAutomationJobOutput"** ruft die Ausgabe eines Azure-Automatisierungsauftrags ab.

## BEISPIELE

### Beispiel 1: Erhalten der Ausgabe eines Automatisierungsauftrags
```
PS C:\>Get-AzAutomationJobOutput -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01" -Stream "Any"
```

Dieser Befehl ruft die Ausgabe des Auftrags ab, der die angegebene ID hat.
