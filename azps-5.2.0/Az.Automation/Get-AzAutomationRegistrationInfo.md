# Get-AzAutomationRegistrationInfo

## SYNOPSIS
Ruft Registrierungsinformationen für das Onboarding eines DSC-Knotens oder hybriden Workers in die Automatisierung ab.

## SYNTAX

```
Get-AzAutomationRegistrationInfo [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzAutomationRegistrationInfo"** ruft den Endpunkt und die Schlüssel ab, die zum Integrieren eines Knotens (Desired State Configuration, DSC) oder eines Hybrid-Workers in ein Azure-Automatisierungskonto erforderlich sind.

## BEISPIELE

### Beispiel 1: Registrierungsinformationen erhalten
```
PS C:\>Get-AzAutomationRegistrationInfo -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01"
```

Dieser Befehl ruft die Registrierungsinformationen für das Automatisierungskonto namens AutomationAccount01 in der Ressourcengruppe mit dem Namen "ResourceGroup01" ab.
