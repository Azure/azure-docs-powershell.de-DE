# Reset-AzPowerBIWorkspaceCollectionAccessKey

## SYNOPSIS
Setzt die angegebene Zugriffstaste zurück.

## SYNTAX

```
Reset-AzPowerBIWorkspaceCollectionAccessKey [-ResourceGroupName] <String> [-WorkspaceCollectionName] <String>
 [-Key1] [-Key2] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Reset-AzPowerBIWorkspaceCollectionAccessKey"** setzt die angegebene Zugriffstaste in Ihrer Power BI-Arbeitsbereichssammlung zurück.

## BEISPIELE

### Beispiel 1: Zurücksetzen des primären Zugriffsschlüssels
```
PS C:\>Reset-AzPowerBIWorkspaceCollectionAccessKey -ResourceGroupName "ResourceGroup17" -WorkspaceCollectionName "WCN11" -Key1
```

Mit diesem Befehl wird der primäre Zugriffsschlüssel für die Arbeitsbereichssammlung mit dem Namen WCN11 in der angegebenen Ressourcengruppe zurückgesetzt.
