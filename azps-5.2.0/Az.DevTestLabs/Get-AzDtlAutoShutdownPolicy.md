# Get-AzDtlAutoShutdownPolicy

## SYNOPSIS
Ruft die Richtlinie für das automatische Herunterfahren einer Übungseinheit in DevTest Labs ab.

## SYNTAX

```
Get-AzDtlAutoShutdownPolicy [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzDtlAutoShutdownPolicy"** ruft die Richtlinie für das automatische Herunterfahren eines Labors ab, mit der Sie alle virtuellen Computer in einem Labor zu einer bestimmten Zeit des Tages automatisch herunterfahren können.
Das Cmdlet gibt zurück, ob der Status der Richtlinie aktiviert ist, und die Tageszeit, die Sie für das automatische Herunterfahren der virtuellen Computer des Labors festgelegt haben.

## BEISPIELE

### Beispiel 1

Ruft die Richtlinie für das automatische Herunterfahren einer Übungseinheit in DevTest Labs ab. (automatisch generiert)

```powershell <!-- Aladdin Generated Example --> 
Get-AzDtlAutoShutdownPolicy -LabName <String> -ResourceGroupName MyResourceGroup
```
