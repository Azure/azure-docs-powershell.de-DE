# Get-AzDtlAllowedVMSizesPolicy

## SYNOPSIS
Ruft die zulässige Größenrichtlinie für virtuelle Computer in einer Übungseinheit in DevTest Labs ab.

## SYNTAX

```
Get-AzDtlAllowedVMSizesPolicy [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzDtlAllowedVMSizesPolicy"** ruft die Richtlinie für zulässige Größen für virtuelle Computer ab, mit der Sie eine Liste der im Labor zulässigen Größen virtueller Computer angeben können.
Das Cmdlet gibt den Status "Aktiviert" oder "Deaktiviert" der Richtlinie und eine Liste aller zulässigen Größen für virtuelle Computer zurück, die Sie in der angegebenen Richtlinie festgelegt haben.
