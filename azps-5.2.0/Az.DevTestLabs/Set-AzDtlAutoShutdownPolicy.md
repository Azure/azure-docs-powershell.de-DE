# Set-AzDtlAutoShutdownPolicy

## SYNOPSIS
Legt die Richtlinie für das automatische Herunterfahren einer DevTest-Übungseinheit fest.

## SYNTAX

### Aktivieren (Standard)
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Enable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Deaktivieren
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Disable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Set-AzDtlAutoShutdownPolicy** legt die Richtlinie für das automatische Herunterfahren eines Labors fest, das alle virtuellen Computer im Labor zu einer bestimmten Zeit des Tages automatisch heruntergefahren.
Das Cmdlet verwendet die angegebene Ressourcengruppe und den Namen der Übungseinheit zum Festlegen der Richtlinie.

## BEISPIELE

### Beispiel 1

Legt die Richtlinie für das automatische Herunterfahren einer DevTest-Übungseinheit fest. (automatisch generiert)

```powershell <!-- Aladdin Generated Example --> 
Set-AzDtlAutoShutdownPolicy -Enable -LabName <String> -ResourceGroupName MyResourceGroup -Time <DateTime>
```
