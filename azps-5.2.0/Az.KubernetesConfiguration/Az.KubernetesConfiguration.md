# Az.KubernetesConfiguration Module
## Beschreibung
Microsoft Azure PowerShell: KubernetesConfiguration-Cmdlets

## Az.KubernetesConfiguration-Cmdlets
### [Get-AzTätigernetesConfiguration](Get-AzKubernetesConfiguration.md)
Ruft Details zur Quellcodeverwaltungskonfiguration ab.

### [New-AzAkiernetesConfiguration](New-AzKubernetesConfiguration.md)
Erstellen Sie eine neue Konfiguration der Kubernetes-Quellsteuerung.

### [Remove-AzAkiernetesConfiguration](Remove-AzKubernetesConfiguration.md)
Dadurch wird die ZUM Einrichten der Quellsteuerungskonfiguration verwendete YAML-Datei gelöscht, was die zukünftige Synchronisierung aus dem Quell-Repository beendet.
