# Get-AzApiManagementLogger

## SYNOPSIS
Ruft APIs-Verwaltungsprotokollierungsobjekte ab.

## SYNTAX

### GetAllLoggers (Standard)
```
Get-AzApiManagementLogger -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByLoggerId
```
Get-AzApiManagementLogger -Context <PsApiManagementContext> -LoggerId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzApiManagementLogger"** ruft einen Azure API Management **Logger** oder alle Logger ab.

## BEISPIELE

### Beispiel 1: Alle Protokoller erhalten
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementLogger -Context $apimContext
```

Dieser Befehl ruft alle Logger f체r den angegebenen Kontext ab.

### Beispiel 2: Einen bestimmten Protokollger erhalten
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementLogger -Context $apimContext -LoggerId "Logger123"
```

Mit diesem Befehl wird ein Protokoller entfernt, der den ID Logger123 enth채lt.
