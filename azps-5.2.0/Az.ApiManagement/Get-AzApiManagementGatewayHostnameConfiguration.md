# Get-AzApiManagementGatewayHostnameConfiguration

## SYNOPSIS
Ruft die Konfiguration des ganzen oder spezifischen Hostnamens für das vorhandene Gateway ab.

## SYNTAX

### GetByGatewayId (Standard)
```
Get-AzApiManagementGatewayHostnameConfiguration -Context <PsApiManagementContext> -GatewayId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByGatewayHostnameId
```
Get-AzApiManagementGatewayHostnameConfiguration -Context <PsApiManagementContext> -GatewayId <String>
 -GatewayHostnameConfigurationId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzApiManagementGatewayHostnameConfiguration"** ruft eine ganz oder eine bestimmte Hostnamenkonfiguration für das vorhandene Gateway ab.

## BEISPIELE

### Beispiel 1: Alle Hostnamenkonfigurationen für das Gateway erhalten
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementGatewayHostnameConfiguration -Context $apimContext  -GatewayId "0123456789"
```

Dieser Befehl ruft alle Hostnamenkonfigurationen für ein "0123456789"-Gateway ab.

### Beispiel 2: Erhalten einer Hostnamenkonfiguration für das Gateway
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementGatewayHostnameConfiguration -Context $apimContext -GatewayId "0123456789" -GatewayHostnameConfigurationId "123"
```

Dieser Befehl ruft die Konfiguration des Hostnamens "123" für ein "0123456789"-Gateway ab.
