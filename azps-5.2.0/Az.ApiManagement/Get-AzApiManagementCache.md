# Get-AzApiManagementCache

## SYNOPSIS
Hier finden Sie die Details des Caches.

## SYNTAX

### ContextParameterSet (Standard)
```
Get-AzApiManagementCache -Context <PsApiManagementContext> [-CacheId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzApiManagementCache -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Erfahren Sie mehr 端ber den im Api-Verwaltungsdienst konfigurierten Cache.

## BEISPIELE

### Beispiel 1: Alle Caches erhalten
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementCache -Context $apimContext
```

```

ServiceName       : contoso
```

Ruft eine Liste aller im Api-Verwaltungsdienst konfigurierten Caches ab.
