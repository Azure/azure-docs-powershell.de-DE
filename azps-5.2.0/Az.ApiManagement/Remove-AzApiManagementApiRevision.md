# Remove-AzApiManagementApiRevision

## SYNOPSIS
Eine bestimmte VERSION der API wurde entfernt

## SYNTAX

### ByApiId (Standard)
```
Remove-AzApiManagementApiRevision -Context <PsApiManagementContext> -ApiId <String> -ApiRevision <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzApiManagementApiRevision -InputObject <PsApiManagementApi> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das Cmdlet **"Remove-AzApiManagementApiRevision"** entfernt eine bestimmte API-Überarbeitung.

## BEISPIELE

### Beispiel 1: Entfernen einer API-Überarbeitung
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzAzureRmApiManagementApiRevision -Context $apimContext -ApiId "echo-api" -ApiRevision "2"
```

Mit diesem Befehl wird die `2` Überarbeitung der API aus dem `echo-api` API-Verwaltungsdienst entfernt.

### Beispiel 2

Mit diesem Befehl wird die zwei Überarbeitung der API-Echo-API aus dem API-Verwaltungsdienst entfernt. (automatisch generiert)

```powershell
<!-- Aladdin Generated Example --> 
Remove-AzApiManagementApiRevision -ApiId '0001' -ApiRevision 6 -Context <PsApiManagementContext>
```
