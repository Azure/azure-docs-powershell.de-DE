# New-AzRelayHybridConnection

## SYNOPSIS
Erstellt eine HybridVerbindung im angegebenen Relay-Namespace.

## SYNTAX

### HybridConnectionInputObjectSet
```
New-AzRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-InputObject <PSHybridConnectionAttributes>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### HybridConnectionPropertiesSet
```
New-AzRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-RequiresClientAuthorization <Boolean>] [-UserMetadata <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das New-AzRelayHybridConnection erstellt eine HybridVerbindung im angegebenen Relay-Namespace.

PS C:\> $getHybirdConnection.RequiresClientAuthorization = $False
