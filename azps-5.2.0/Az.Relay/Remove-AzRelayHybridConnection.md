# Remove-AzRelayHybridConnection

## SYNOPSIS
Entfernt die HybridVerbindung aus dem angegebenen HybridConnection-Namespace.

## SYNTAX

```
Remove-AzRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzRelayHybridConnection"** entfernt die HybridVerbindung aus dem angegebenen Relay-Namespace.

## BEISPIELE

### Beispiel 1
```
PS C:\> Remove-AzRelayHybridConnection -ResourceGroupName Default-ServiceBus-WestUS -Namespace TestNameSpace-Relay1 -Name TestHybridConnection
```

Entfernt hybridconnection `TestHybridConnection` aus dem `TestNameSpace-Relay1` Namespace.
