# Test-AzAnalysisServicesServer

## SYNOPSIS
Überprüft das Vorhandensein einer Instanz des Analysis Services-Servers.

## SYNTAX

```
Test-AzAnalysisServicesServer [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das Test-AzAnalysisServicesServer cmdlet überprüft das Vorhandensein einer Instanz des Analysis Services-Servers.

## BEISPIELE

### Beispiel 1
```
PS C:\> Test-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Dieser Befehl testt, ob die Testgruppe der Ressourcengruppe einen Server mit dem Namen "Testserver" hat.
