# Az.Subscription Module
## Beschreibung
In den Themen in diesem Abschnitt werden die Azure -PowerShell-Cmdlets für das Abonnement von Azure im Azure Resource Manager (ARM) dokumentiert. Die Cmdlets sind im Microsoft.Azure.Commands.Subscription-Namespace vorhanden.

## Az.Subscription-Cmdlets
### [Get-AzSubscriptionAlias](Get-AzSubscriptionAlias.md)
Ruft Abonnementaliasdetails ab

### [New-AzSubscriptionAlias](New-AzSubscriptionAlias.md)
Erstellt einen neuen Alias und ein neues Abonnement.

### [Remove-AzSubscriptionAlias](Remove-AzSubscriptionAlias.md)
Löscht den Abonnementalias.

### [Update-AzSubscription](Update-AzSubscription.md)
Aktualisiert ein Azure-Abonnement.
