# Update-AzActionRule

## SYNOPSIS
Aktualisiert Eigenschaften von Aktionsregel.

## SYNTAX

### ByNameSimplifiedPatch (Standard)
```
Update-AzActionRule -Name <String> -ResourceGroupName <String> [-Status <String>] [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Update-AzActionRule -ResourceId <String> [-Status <String>] [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Update-AzActionRule -InputObject <PSActionRule> [-Status <String>] [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
**Das Cmdlet "Update-AzActionRule"** aktualisiert Eigenschaften von Aktionsregel – Status und Tags.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Update-AzActionRule -ResourceGroupName "test-rg" -Name "Test-ActionRule" -Status "Disabled"
```

Dieses Cmdlet deaktiviert die Aktionsregel.
