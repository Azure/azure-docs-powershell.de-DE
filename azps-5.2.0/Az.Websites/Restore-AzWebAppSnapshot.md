# Restore-AzWebAppSnapshot

## SYNOPSIS
Stellt eine Web-App-Momentaufnahme wieder zur Verfügung.

## SYNTAX

### FromResourceName
```
Restore-AzWebAppSnapshot [-RecoverConfiguration] [-UseDisasterRecovery] [-Force] [-AsJob]
 [-ResourceGroupName] <String> [-Name] <String> [[-Slot] <String>] [-DefaultProfile <IAzureContextContainer>]
 [-InputObject] <AzureWebAppSnapshot> [-WhatIf] [-Confirm] [<CommonParameters>]
```

### FromWebApp
```
Restore-AzWebAppSnapshot [-RecoverConfiguration] [-UseDisasterRecovery] [-Force] [-AsJob] [-WebApp] <PSSite>
 [-DefaultProfile <IAzureContextContainer>] [-InputObject] <AzureWebAppSnapshot> [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Stellt eine Web-App-Momentaufnahme in der Web App wieder zur Verfügung. Durch das Wiederherstellen einer Momentaufnahme werden alle Dateien in einer Web App mit den in der Momentaufnahme enthaltenen Dateien überschrieben. Verwenden Sie zum Wiederherstellen von Einstellungen auch den Parameter des Switchparameters "RecoverConfiguration". Eine Momentaufnahme aus einer Web App kann in jede andere Web App im selben Abonnement wiederhergestellt werden.

## BEISPIELE

### Beispiel 1
```
PS C:\> $snapshot = (Get-AzWebAppSnapshot -ResourceGroupName "Default-Web-WestUS" -Name "ContosoApp" -Slot "Staging")[0]
PS C:\> Restore-AzWebAppSnapshot -ResourceGroupName "Default-Web-WestUS" -Name "ContosoApp" -Slot "Restore" -InputObject $snapshot -RecoverConfiguration
```

Ruft die neueste Momentaufnahme einer Web App namens "ContosoApp" mit dem Slot "Staging" in der Ressourcengruppe "Default-Web-WestUS" ab. Stellt die Momentaufnahme auf dem "Wiederherstellen"-Platz der Web App wieder zur Verfügung.
