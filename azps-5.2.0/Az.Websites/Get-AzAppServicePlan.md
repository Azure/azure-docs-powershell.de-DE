# Get-AzAppServicePlan

## SYNOPSIS
Ruft einen Azure App Service Plan in der angegebenen Ressourcengruppe ab.

## SYNTAX

### S1
```
Get-AzAppServicePlan [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### S2
```
Get-AzAppServicePlan [-Location] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzAppServicePlan"** ruft einen Azure App Service Plan in der angegebenen Ressourcengruppe ab.

## BEISPIELE

### Beispiel 1: Erhalten eines App -Dienstplans aus einer Ressourcengruppe
```
PS C:\>Get-AzAppServicePlan -ResourceGroupName "Default-Web-WestUS" -Name "ContosoASP"
```

Dieser Befehl ruft den App-Service-Plan "ContosoASP" ab, der zur Ressourcengruppe "Default-Web-WestUS" gehört.

### Beispiel 2: Alle App -Dienstpläne an einem Speicherort erhalten
```
PS C:\>Get-AzAppServicePlan -Location "West US"
```

Dieser Befehl ruft alle App Service-Pläne ab, die sich in der Region "USA, Westen" befinden.
