# Get-AzWebAppSlotConfigName

## SYNOPSIS
Liste der Web App-Slot-Konfigurationsnamen

## SYNTAX

### S1
```
Get-AzWebAppSlotConfigName [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### S2
```
Get-AzWebAppSlotConfigName [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzWebAppSlotConfigName"** ruft die Liste der Namen von "App-Einstellung" und "Verbindungszeichenfolge" ab, die derzeit als Sloteinstellungen markiert sind.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\>Get-AzWebAppSlotConfigName -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Dieser Befehl ruft Die App-Einstellungen und Verbindungszeichenfolgen ab, die sich auf die Web App namens "ContosoSite" bezieht, die der Ressourcengruppe "Default-Web-WestUS" zugeordnet ist.
