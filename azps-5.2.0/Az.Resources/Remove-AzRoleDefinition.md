# Remove-AzRoleDefinition

## SYNOPSIS
Löscht eine benutzerdefinierte Rolle in Azure RBAC.
Die zu löschende Rolle wird mithilfe der Eigenschaft "ID" der Rolle angegeben.
Beim Löschen ist ein Fehler vorhanden, wenn der benutzerdefinierten Rolle Rollenzuweisungen vorgenommen wurden.

## SYNTAX

### RoleDefinitionIdParameterSet (Standard)
```
Remove-AzRoleDefinition -Id <Guid> [-Scope <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RoleDefinitionNameParameterSet
```
Remove-AzRoleDefinition [-Name] <String> [-Scope <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzRoleDefinition -InputObject <PSRoleDefinition> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das Remove-AzRoleDefinition cmdlet löscht eine benutzerdefinierte Rolle in Azure Role-Based Access Control.
Geben Sie den Parameter "ID" einer vorhandenen benutzerdefinierten Rolle an, um diese benutzerdefinierte Rolle zu löschen.
Standardmäßig werden Sie Remove-AzRoleDefinition Bestätigungsaufforderung angezeigt.
Verwenden Sie den Parameter "Erzwingen", um die Eingabeaufforderung zu unterdrücken.
Wenn für die benutzerdefinierte Rolle vorhandene Rollenzuweisungen vorhanden sind, die gelöscht werden sollen, wird der Lösch fehlschlagen.

## BEISPIELE

### Beispiel 1
```
Get-AzRoleDefinition -Name "Virtual Machine Operator" | Remove-AzRoleDefinition
```

### Beispiel 2
```
Remove-AzRoleDefinition -Id "52a6cc13-ff92-47a8-a39b-2a8205c3087e"
```
