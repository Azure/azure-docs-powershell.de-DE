# Remove-AzManagedApplication

## SYNOPSIS
Entfernt eine verwaltete Anwendung

## SYNTAX

### RemoveByNameAndResourceGroup (Standard)
```
Remove-AzManagedApplication -Name <String> -ResourceGroupName <String> [-Force] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveById
```
Remove-AzManagedApplication -Id <String> [-Force] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzManagedApplication"** entfernt eine verwaltete Anwendung.

## BEISPIELE

### Beispiel 1: Entfernen einer verwalteten Anwendung nach Ressourcen-ID
```
PS C:\>$Application = Get-AzManagedApplication -Name "myApp" -ResourceGroupName "myRG"
PS C:\>Remove-AzManagedApplication -Id $Application.ResourceId -Force
```

Der erste Befehl ruft mithilfe des Cmdlets "Get-AzManagedApplication" eine verwaltete Anwendung namens "myApp" ab.
Der Befehl speichert sie in der $Application Variable.
Der zweite Befehl entfernt die verwaltete Anwendung, die durch die **Eigenschaft "ResourceId"** von "$Application.
