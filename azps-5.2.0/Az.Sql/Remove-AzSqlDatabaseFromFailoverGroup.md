# Remove-AzSqlDatabaseFromFailoverGroup

## SYNOPSIS
Entfernt eine oder mehrere Datenbanken aus einer Azure SQL-Datenbank-Failovergruppe.

## SYNTAX

```
Remove-AzSqlDatabaseFromFailoverGroup [-ServerName] <String> [-FailoverGroupName] <String>
 -Database <System.Collections.Generic.List`1[Microsoft.Azure.Commands.Sql.Database.Model.AzureSqlDatabaseModel]>
 [-Force] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Entfernt eine oder mehrere Datenbanken aus der angegebenen Azure SQL-Failovergruppe. Die Datenbanken und Replikationsbeziehungen bleiben intakt, sind aber nicht mehr über die Failovergruppenendpunkte zugänglich.
Zum Abrufen von Datenbankobjekten, mit denen der Parameter "-Database" auffüllt werden soll, verwenden Sie (beispielsweise) das Get-AzSqlDatabase Cmdlet.
Der primäre Server der Failovergruppe muss zum Ausführen des Befehls verwendet werden.

## BEISPIELE

### Beispiel 1
```
PS C:\> $failoverGroup = Get-AzSqlDatabase -ResourceGroupName rg -ServerName primaryserver -DatabaseName db1 | Remove-AzSqlDatabaseFromFailoverGroup -ResourceGroupName rg -ServerName primaryserver -FailoverGroupName fg
```

Mit diesem Befehl wird eine Datenbank aus einer Failovergruppe entfernt, indem sie in die Datenbank einpipeniert wird.

### Beispiel 2
```
PS C:\> $primaryServer = Get-AzSqlServer -ResourceGroupName rg -ServerName primaryserver
PS C:\> $failoverGroup = $primaryServer | Remove-AzSqlDatabaseFromFailoverGroup -FailoverGroupName fg -Database ($primaryServer | Get-AzSqlDatabase)
```

Mit diesem Befehl werden alle Datenbanken aus einer Failovergruppe entfernt.

### Beispiel 3
```
PS C:\> $failoverGroup = Get-AzSqlDatabaseFailoverGroup -ResourceGroupName rg -ServerName primaryserver -FailoverGroupName fg
PS C:\> $databases = Get-AzSqlElasticPoolDatabase -ResourceGroupName rg -ServerName primaryserver -ElasticPoolName pool1
PS C:\> $failoverGroup = $failoverGroup | Remove-AzSqlDatabaseFromFailoverGroup -Database $databases
```

Mit diesem Befehl werden alle Datenbanken in einem Pool mit Bändern aus einer Failovergruppe entfernt.
