# Get-AzSqlDatabaseExpanded

## SYNOPSIS
Ruft eine Datenbank und die erweiterten Eigenschaftswerte ab.

## SYNTAX

```
Get-AzSqlDatabaseExpanded [-ServerName] <String> [[-DatabaseName] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzSqlDatabaseExpanded"** ruft eine Datenbank und die erweiterten Eigenschaftswerte ab.
Dieses Cmdlet wird auch vom Dienst SQL Server Stretch Database in Azure unterstützt.

## BEISPIELE

### Beispiel 1: Datenbankobjekt mit Dienststufenratgeberinformationen erhalten
```
PS C:\> Get-AzSqlDatabaseExpanded -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Dieser Befehl gibt die Datenbank mit erweiterten Eigenschaften zurück, die die Informationen zum Dienststufenratgeber enthalten.

### Beispiel 2: Auflisten von Datenbankobjekten mithilfe von Filtern
```
PS C:\> Get-AzSqlDatabaseExpanded -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database*"
```

Dieser Befehl gibt ein erweitertes Datenbankobjekt für alle Datenbanken in Server01 zurück, die mit "Datenbank" beginnen.
