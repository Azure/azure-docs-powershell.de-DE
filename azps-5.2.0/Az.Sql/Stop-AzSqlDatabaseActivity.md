# Stop-AzSqlDatabaseActivity

## SYNOPSIS
Bricht den asynchronen Aktualisierungsvorgang für die Datenbank ab.

## SYNTAX

```
Stop-AzSqlDatabaseActivity [-ServerName] <String> [-ElasticPoolName <String>] -DatabaseName <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Stop-AzSqlDatabaseActivity"** bricht den asynchronen Aktualisierungsvorgang für die Datenbank ab.

## BEISPIELE

### Beispiel 1: Abbrechen des Vorgangs für asynchrone Updates in der Datenbank

Dieser Befehl bricht den Vorgang für asynchrone Aktualisierungen in der Datenbank ab.
