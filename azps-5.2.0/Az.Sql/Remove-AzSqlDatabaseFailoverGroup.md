# Remove-AzSqlDatabaseFailoverGroup

## SYNOPSIS
Entfernt eine Azure SQL-Datenbank-Failovergruppe.

## SYNTAX

```
Remove-AzSqlDatabaseFailoverGroup [-ServerName] <String> [-FailoverGroupName] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Mit diesem Befehl wird die Failovergruppe mit dem angegebenen Namen entfernt, und alle Datenbanken und Replikationsbeziehungen bleiben erhalten. Die Registrierung des Listenerendpunkts wird im DNS aufgehoben.
Der primäre Server der Failovergruppe sollte zum Ausführen des Befehls verwendet werden.

## BEISPIELE

### Beispiel 1
```
PS C:\> Remove-AzSqlDatabaseFailoverGroup -ResourceGroupName rg -ServerName primaryserver -FailoverGroupName fg
```

Entfernen einer Failovergruppe
