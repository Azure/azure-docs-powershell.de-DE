# Remove-AzSqlSyncAgent

## SYNOPSIS
Entfernt einen Azure SQL-Synchronisierungs-Agent.

## SYNTAX

```
Remove-AzSqlSyncAgent [-Name] <String> [-Force] [-PassThru] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzSqlSyncAgent"** entfernt einen Azure SQL-Agent.

## BEISPIELE

### Beispiel 1: Entfernen eines Synchronisierungs-Agents
```
PS C:\>Remove-AzSqlSyncAgent -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -Name "syncAgent01"
```

Mit diesem Befehl wird der Azure SQL-Synchronisierungs-Agent namens "syncAgent01" entfernt.
