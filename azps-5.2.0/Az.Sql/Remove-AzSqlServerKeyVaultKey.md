# Remove-AzSqlServerKeyVaultKey

## SYNOPSIS
Entfernt einen Schlüssel im Tresor von einem SQL Server.

## SYNTAX

```
Remove-AzSqlServerKeyVaultKey [-KeyId] <String> [-AsJob] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das Remove-AzSqlServerKeyVaultKey cmdlet entfernt den Schlüssel des Tresorschlüssels vom angegebenen SQL Server.
Beachten Sie, SQL die Berechtigungen des Servers für den Tresor des Schlüssels nicht geändert werden.
Verwenden Sie Set-AzKeyVaultAccessPolicy, um Berechtigungen zu ändern.
Beachten Sie, dass dieses Cmdlet keine Änderungen am Schlüsseltresor vor nimmt.
Verwenden Sie "Remove-AzKeyVaultKey", um einen Schlüssel aus dem Schlüsseltresor zu entfernen.

## BEISPIELE

### Beispiel 1: Entfernen eines Schlüsseltresorschlüssels
```
PS C:\> Remove-AzSqlServerKeyVaultKey -KeyId 'https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901' -ServerName 'ContosoServer' -ResourceGroupName 'ContosoResourceGroup'
```

Mit diesem Befehl wird der Schlüssel "Schlüssel im Tresor" mit der ID ' https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901 vom angegebenen Server entfernt.
