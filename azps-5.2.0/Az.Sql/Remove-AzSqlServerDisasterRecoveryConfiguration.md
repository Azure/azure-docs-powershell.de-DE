# Remove-AzSqlServerDisasterRecoveryConfiguration

## SYNOPSIS
Entfernt eine wiederherstellungskonfiguration SQL Datenbankserversystems.

## SYNTAX

```
Remove-AzSqlServerDisasterRecoveryConfiguration [-VirtualEndpointName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzSqlServerDisasterRecoveryConfiguration"** entfernt eine SQL Datenbankserversystemwiederherstellungskonfiguration.
