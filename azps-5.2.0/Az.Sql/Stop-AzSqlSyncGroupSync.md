# Stop-AzSqlSyncGroupSync

## SYNOPSIS
Beendet die Synchronisierung einer Synchronisierungsgruppe.

## SYNTAX

```
Stop-AzSqlSyncGroupSync [-SyncGroupName] <String> [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Stop-AzSqlSyncGroupSync"** beendet die Synchronisierung einer Synchronisierungsgruppe.

## BEISPIELE

### Beispiel 1
```
PS C:\> Stop-AzSqlSyncGroupSync -SyncGroupName mysg [-ServerName] mysrv [-DatabaseName] mydb [-ResourceGroupName] myrg
```

Mit diesem Befehl wird die Synchronisierung beendet, die für die Synchronisierungsgruppe "mysg" ausgeführt wird.
