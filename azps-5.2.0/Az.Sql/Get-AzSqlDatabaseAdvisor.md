# Get-AzSqlDatabaseAdvisor

## SYNOPSIS
Ruft einen oder mehrere Berater für eine Azure SQL-Datenbank ab.

## SYNTAX

```
Get-AzSqlDatabaseAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -DatabaseName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Get-AzSqlDatabaseAdvisor** ruft mindestens einen Azure SQL Datenbankratgeber für eine Azure SQL ab.
