# Az.IotCentral Module
## Beschreibung
Bietet die CRUD-Verwaltung für IoT-Zentralanwendungen.

## Az.IotCentral-Cmdlets
### [Get-AzIotCentralApp](Get-AzIotCentralApp.md)
Ruft Eigenschaften für eine oder mehrere zentrale IoT-Anwendungen ab.

### [New-AzIotCentralApp](New-AzIotCentralApp.md)
Erstellt eine neue IoT-Zentralanwendung.

### [Remove-AzIotCentralApp](Remove-AzIotCentralApp.md)
Löscht eine IoT-Zentralanwendung.

### [Set-AzIotCentralApp](Set-AzIotCentralApp.md)
Aktualisiert die Metadaten für eine IoT-Zentralanwendung.
