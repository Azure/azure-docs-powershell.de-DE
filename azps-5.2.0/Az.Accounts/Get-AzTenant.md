# Get-AzTenant

## SYNOPSIS
Ruft Mandanten ab, die für den aktuellen Benutzer autorisiert sind.

## SYNTAX

```
Get-AzTenant [[-TenantId] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das Get-AzTenant cmdlet bekommt Mandanten, die für den aktuellen Benutzer autorisiert sind.

## BEISPIELE

### Beispiel 1: Abrufen aller Mandanten
```
PS C:\> Connect-AzAccount
PS C:\> Get-AzTenant

Id                                   Name        Category Domains
--                                   ----------- -------- -------
xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx Microsoft   Home     {test0.com, test1.com, test2.microsoft.com, test3.microsoft.com...}
yyyyyyyy-yyyy-yyyy-yyyy-yyyyyyyyyyyy Testhost    Home     testhost.onmicrosoft.com
```

Dieses Beispiel zeigt, wie Sie alle autorisierten Mandanten eines Azure-Kontos erhalten.

### Beispiel 2: Abrufen eines bestimmten Mandanten
```
PS C:\> Connect-AzAccount
PS C:\> Get-AzTenant -TenantId xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx

Id                                   Name        Category Domains
--                                   ----------- -------- -------
xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx Microsoft   Home     {test0.com, test1.com, test2.microsoft.com, test3.microsoft.com...}
```

Dieses Beispiel zeigt, wie Sie einen bestimmten autorisierten Mandanten eines Azure-Kontos erhalten.
