# Clear-AzContext

## SYNOPSIS
Entfernen Sie alle Azure-Anmeldeinformationen, das Konto und die Abonnementinformationen.

## SYNTAX

```
Clear-AzContext [-PassThru] [-Force] [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Entfernen Sie alle Azure-Anmeldeinformationen, das Konto und die Abonnementinformationen.

## BEISPIELE

### Beispiel 1: Löschen des globalen Kontexts
```powershell
PS C:\> Clear-AzContext -Scope CurrentUser
```

Entfernen Sie alle Konto-, Abonnement- und Anmeldeinformationen für jede Powershell-Sitzung.
