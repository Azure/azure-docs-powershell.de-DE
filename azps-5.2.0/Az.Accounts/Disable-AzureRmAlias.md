# Disable-AzureRmAlias

## SYNOPSIS
Deaktiviert AzureRm-Präfixaliase für Az-Module.

## SYNTAX

```
Disable-AzureRmAlias [-Scope <String>] [-Module <String[]>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Deaktiviert AzureRm-Präfixaliase für Az-Module. Wenn "-Module" angegeben ist, sind die Aliase nur für die aufgelisteten Module deaktiviert. Andernfalls werden alle AzureRm-Aliase deaktiviert.

## BEISPIELE

### Beispiel 1
```
PS C:\> Disable-AzureRmAlias
```

Deaktiviert alle AzureRm-Präfixe für die aktuelle PowerShell-Sitzung.

### Beispiel 1
```
PS C:\> Disable-AzureRmAlias -Module Az.Accounts -Scope CurrentUser
```

Deaktiviert die AzureRm-Aliase für das Modul "Az.Accounts" sowohl für den aktuellen Prozess als auch für den aktuellen Benutzer.
