# Get-AzReservationOrderId

## SYNOPSIS
Hier finden Sie eine Liste der `ReservationOrder` anwendbaren IDs.

## SYNTAX

```
Get-AzReservationOrderId [-SubscriptionId <Guid>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Erhalten Sie ids of applicable `ReservationOrder` s, die auf dieses Abonnement angewendet werden können.

## BEISPIELE

### Beispiel 1
```
PS C:\> Get-AzReservationOrderId
```

Angewendet für `ReservationOrder` Standardabonnement

### Beispiel 2
```
PS C:\> Get-AzReservationOrderId -SubscriptionId "1111aaaa-b1b2-c0c2-d0d2-00000fffff"
```

Für `ReservationOrder` angegebenes Abonnement angewendet werden
