# Update-AzRecoveryServicesAsrRecoveryPlan

## SYNOPSIS
Aktualisiert den Inhalt eines Azure-Websitewiederherstellungsplans.

## SYNTAX

### ByRPObject (Standard)
```
Update-AzRecoveryServicesAsrRecoveryPlan -InputObject <ASRRecoveryPlan>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByRPFile
```
Update-AzRecoveryServicesAsrRecoveryPlan -Path <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Update-AzRecoveryServicesAsrRecoveryPlan"** aktualisiert den Inhalt eines Wiederherstellungsplans mithilfe der Inhalte des angegebenen ASR-Wiederherstellungsplanobjekts oder der JSON-Datei mit der ASR-Wiederherstellungsplandefinition.

## BEISPIELE

### Beispiel 1: Aktualisieren eines Wiederherstellungsplans
```
PS C:\> $currentJob = Update-AzRecoveryServicesAsrRecoveryPlan -RecoveryPlan $RP
```

Starten Sie den Vorgang zum Aktualisieren eines Wiederherstellungsplans mithilfe des Inhalts des angegebenen ASR-Wiederherstellungsplanobjekts, und gibt den ASR-Auftrag zurück, mit dem der Vorgang nachverfolgt wird.
