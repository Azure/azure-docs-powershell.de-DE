# Get-AzRecoveryServicesAsrStorageClassificationMapping

## SYNOPSIS
Ruft die Zuordnungen der ASR-Speicherklassifizierung ab.

## SYNTAX

### ByObject (Standard)
```
Get-AzRecoveryServicesAsrStorageClassificationMapping -StorageClassification <ASRStorageClassification>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByObjectWithName
```
Get-AzRecoveryServicesAsrStorageClassificationMapping -Name <String>
 -StorageClassification <ASRStorageClassification> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzRecoveryServicesAsrStorageClassificationMapping"** ruft die Details einer ASR-Speicherklassifizierungszuordnung ab.

## BEISPIELE

### Beispiel 1
```
PS C:\> $StorageClassificationMappings = Get-AzRecoveryServicesAsrStorageClassificationMapping -StorageClassification $StorageClassification
```

Listet alle Speicherklassifizierungszuordnungen auf, die der angegebenen Speicherklassifizierung zugeordnet sind.
