# Remove-AzRecoveryServicesAsrFabric

## SYNOPSIS
Löscht das angegebene Azure Site Recovery Fabric aus dem Recovery Services-Tresor.

## SYNTAX

```
Remove-AzRecoveryServicesAsrFabric -InputObject <ASRFabric> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzRecoveryServicesAsrFabric"** entfernt das angegebene Azure Site Recovery-Fabric aus dem Recovery Services-Tresor.

## BEISPIELE

### Beispiel 1
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrFabric -Fabric $Fabric
```

Startet das Löschen des angegebenen Fabric und gibt den ASR-Auftrag zurück, der zum Nachverfolgen des Vorgangs verwendet wird.
