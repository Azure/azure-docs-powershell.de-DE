# New-AzFrontDoorBackendPoolsSettingObject

## SYNOPSIS
Erstellen Sie ein "PSBackendPoolsSetting"-Objekt für die Erstellung von Front Door.

## SYNTAX

```
New-AzFrontDoorBackendPoolsSettingObject [-EnforceCertificateNameCheck <PSEnabledState>]
 [-SendRecvTimeoutInSeconds <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "New-AzFrontD selbstBackendpoolsSettingObject"** erstellt ein neues PSBackendPoolsSettings-Objekt für die Erstellung von Front Door.

## BEISPIELE

### Beispiel 1: Erstellen eines Back-EndPoolsSettings-Objekts mit Standardeinstellungen
```powershell
PS C:\> New-AzFrontDoorBackendpoolsSettingObject


EnforceCertificateNameCheck : Enabled
SendRecvTimeoutInSeconds      : 30
Id                          :
Name                        :
Type                        :
```

### Beispiel 2: Erstellen eines Back-EndPoolsSettings-Objekts mit vom Benutzer angegebenen Werten
```powershell
PS C:\> New-AzFrontDoorBackendpoolsSettingObject -SendRecvTimeoutInSeconds 60 -EnforceCertificateNameCheck Enabled


EnforceCertificateNameCheck : Enabled
SendRecvTimeoutInSeconds      : 60
Id                          :
Name                        :
Type                        :
```

### -EnforceCertificateNameCheck
