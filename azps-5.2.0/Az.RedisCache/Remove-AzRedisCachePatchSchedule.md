# Remove-AzRedisCachePatchSchedule

## SYNOPSIS
Entfernt den Patchzeitplan.

## SYNTAX

```
Remove-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzRedisCachePatchSchedule"** entfernt den Patchzeitplan aus einem Cache in Azure Redis Cache.

## BEISPIELE

### Beispiel 1: Entfernen des Patchzeitplans
```
PS C:\>Remove-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06"
```

Mit diesem Befehl wird der Patchzeitplan aus dem Cache namens "RedisCache06" entfernt.
