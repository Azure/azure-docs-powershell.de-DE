# Remove-AzRedisCacheDiagnostic

## SYNOPSIS
Deaktiviert die Diagnose für einen Azure Redis-Cache.

## SYNTAX

```
Remove-AzRedisCacheDiagnostic [-ResourceGroupName <String>] -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzRedisCacheDiagnostic"** deaktiviert die Diagnose in einem Azure Redis-Cache.

## BEISPIELE

### Beispiel 1: Deaktivieren der Diagnose
```
PS C:\>Remove-AzRedisCacheDiagnostic -ResourceGroupName "ContosoResourceGroup" -Name "PeakCache" -Force
```

Mit diesem Befehl wird die Diagnose für einen angegebenen Azure Redis Cache deaktiviert.
Dadurch wird die Diagnose für alle Azure Redis Caches in derselben Region für das Abonnement deaktiviert.
