# Remove-AzRedisCacheLink

## SYNOPSIS
Entfernen Sie eine Georeplikationsverknüpfung zwischen zwei Redis-Caches.

## SYNTAX

```
Remove-AzRedisCacheLink -PrimaryServerName <String> -SecondaryServerName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Entfernen Sie eine Georeplikationsverknüpfung zwischen zwei Redis-Caches.

## BEISPIELE

### Beispiel 1: Entfernen einer Georeplikationsverknüpfung
```
PS C:\>Remove-AzRedisCacheLink -PrimaryServerName "mycache1" -SecondaryServerName "mycache2"
```

Mit diesem Befehl werden Georeplikationsverknüpfungen entfernt, bei denen "Redis-Cache" mit dem Namen "mycache1" primär und "Redis-Cache" mit dem Namen "mycache2" sekundär ist.
