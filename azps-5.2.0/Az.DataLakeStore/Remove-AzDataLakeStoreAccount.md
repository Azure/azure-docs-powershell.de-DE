# Remove-AzDataLakeStoreAccount

## SYNOPSIS
Löscht ein Data Lake Store-Konto endgültig.

## SYNTAX

```
Remove-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzDataLakeStoreAccount"** löscht ein Data Lake Store-Konto dauerhaft.

## BEISPIELE

### Beispiel 1: Entfernen eines Data Lake Store-Kontos
```
PS C:\>Remove-AzDataLakeStoreAccount -Name "ContosoADL"
```

Mit diesem Befehl wird das Konto "ContosoADL" aus dem Data Lake Store entfernt.
