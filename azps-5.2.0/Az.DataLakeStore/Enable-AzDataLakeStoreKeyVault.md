# Enable-AzDataLakeStoreKeyVault

## SYNOPSIS
Versucht, einen vom Benutzer verwalteten Schlüsseltresor für die Verschlüsselung des angegebenen Data Lake Store-Kontos zu aktivieren.

## SYNTAX

```
Enable-AzDataLakeStoreKeyVault [-Account] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Enable-AzDataLakeStoreKeyVault"** versucht, einen vom Benutzer verwalteten Schlüsseltresor für die Verschlüsselung des angegebenen Data Lake Store-Kontos zu aktivieren.

## BEISPIELE

### Beispiel 1: Aktivieren des Schlüsseltresor für das ContosoADLS-Konto
```
PS C:\>Enable-AzDataLakeStoreKeyVault -Name "ContosoADLS"
```

Mit diesem Befehl wird versucht, den vom Benutzer verwalteten Schlüsseltresor für das Data Lake Store-Konto namens ContosoADLS zu aktivieren.
