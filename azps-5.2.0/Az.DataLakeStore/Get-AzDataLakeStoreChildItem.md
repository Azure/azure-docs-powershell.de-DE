# Get-AzDataLakeStoreChildItem

## SYNOPSIS
Ruft die Liste der Elemente in einem Ordner im Data Lake Store ab.

## SYNTAX

```
Get-AzDataLakeStoreChildItem [-Account] <String> [-Path] <DataLakeStorePathInstance>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzDataLakeStoreChildItem"** ruft die Liste der Elemente in einem Ordner im Data Lake Store ab.

## BEISPIELE

### Beispiel 1: Erhalten der untergeordneten Elemente f端r einen Ordner
```
PS C:\>Get-AzDataLakeStoreChildItem -AccountName "ContosoADL" -Path "/MyFiles/"
```

Dieser Befehl ruft die untergeordneten Elemente f端r den Ordner "MyFiles" ab.
