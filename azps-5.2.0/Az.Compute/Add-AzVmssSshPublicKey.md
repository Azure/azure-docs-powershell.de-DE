# Add-AzVmssSshPublicKey

## SYNOPSIS
Fügt der VMSS öffentliche SSH-Schlüssel hinzu.

## SYNTAX

```
Add-AzVmssSshPublicKey [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [[-Path] <String>]
 [[-KeyData] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Add-AzVmssSshPublicKey"** fügt die öffentlichen Schlüssel hinzu, die Sie zum Herstellen einer Verbindung mit virtuellen Computern des Virtual Machine Scale Set (VMSS) über Secure Shell (SSH) verwenden können.

## BEISPIELE

### Beispiel 1: Hinzufügen eines öffentlichen SSH-Schlüssels zu VMSS
```
PS C:\> $VMSS = New-AzVmssConfig
PS C:\> Add-AzVmssSshPublicKey -VirtualMachineScaleSet $VMSS -KeyData "MIIDszCCApugAwIBAgIJALBV9YJCF/tAMA0GCSq12Ib3DQEB21QUAMEUxCzAJBgNV" -Path "/home/admin/.ssh/authorized_keys"
```

In diesem Beispiel wird der VMSS ein öffentlicher SSH-Schlüssel hinzufügt.
Der erste Befehl verwendet das **Cmdlet "New-AzVmssConfig"** zum Erstellen eines VMSS-Konfigurationsobjekts und speichert das Ergebnis in der Variablen namens $VMSS.
Der zweite Befehl fügt einen SSH-Schlüssel mit den angegebenen Schlüsseldaten hinzu und speichert den Schlüssel am angegebenen Pfad auf dem virtuellen Computer.
