# Remove-AzVMSecret

## SYNOPSIS
Entfernt (a) geheime(n) Schlüssel aus einem Objekt des virtuellen Computers.

## SYNTAX

```
Remove-AzVMSecret [-VM] <PSVirtualMachine> [[-SourceVaultId] <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das Remove-AzVMSecret cmdlet entfernt (a) geheime(n) Schlüssel aus einem Objekt des virtuellen Computers.

## BEISPIELE

### Beispiel 1
```
PS C:\> Get-AzVM -ResourceGroupName "rg1" -Name "vm1" | Remove-AzVMSecret | Update-AzVM
```

Entfernt alle geheimen Daten von einem virtuellen Computer "vm1" in der Ressourcengruppe "rg1" und aktualisiert die VM
