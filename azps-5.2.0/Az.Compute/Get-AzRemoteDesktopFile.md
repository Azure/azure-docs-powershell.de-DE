# Get-AzRemoteDesktopFile

## SYNOPSIS
Ruft eine RDP-Datei ab.

## SYNTAX

### Herunterladen
```
Get-AzRemoteDesktopFile [-ResourceGroupName] <String> [-Name] <String> [-LocalPath] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Starten
```
Get-AzRemoteDesktopFile [-ResourceGroupName] <String> [-Name] <String> [[-LocalPath] <String>] [-Launch]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzRemoteDesktopFile"** ruft eine Remotedesktopprotokolldatei (RDP) ab.

## BEISPIELE

### Beispiel 1: Herunterladen einer Remotedesktopdatei
```
PS C:\> Get-AzRemoteDesktopFile -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07" -LocalPath "D:\RemoteDesktopFile07.rdp"
```

Dieser Befehl ruft die Remotedesktopdatei für den virtuellen Computer "VirtualMachine07" ab.
Der Befehl speichert das Ergebnis in der Datei "D:\RemoteDesktopFile07.rdp".
