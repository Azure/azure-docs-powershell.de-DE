# New-AzMetricFilter

## SYNOPSIS
Erstellt einen metrischen Dimensionsfilter, der zum Abfragen von Metriken verwendet werden kann.

## SYNTAX

```
New-AzMetricFilter [-Dimension] <String> [-Operator] <String> [-Value] <String[]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "New-AzMetricFilter"** erstellt einen metrischen Dimensionsfilter, der zum Abfragen von Metriken verwendet werden kann.

## BEISPIELE

### Beispiel 1: Erstellen eines metrischen Dimensionsfilters
```
PS C:\>New-AzMetricFilter -Dimension City -Operator eq -Value "Seattle","New York"
```

Mit diesem Befehl wird ein metrischer Dimensionsfilter im Format "Stadt eq 'Seattle' oder City eq 'New York'" erstellt.
