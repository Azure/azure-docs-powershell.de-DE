# Get-AzAutoscaleSetting

## SYNOPSIS
Ruft die Einstellungen für die Automatische Skalierung ab.

## SYNTAX

```
Get-AzAutoscaleSetting -ResourceGroupName <String> [-Name <String>] [-DetailedOutput]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzAutoscaleSetting"** ruft alle Autoskaleneinstellungen ab, die einer Ressourcengruppe oder einer angegebenen Einstellung für die Autoskala zugeordnet sind.
