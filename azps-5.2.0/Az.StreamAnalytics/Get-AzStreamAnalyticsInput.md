# Get-AzStreamAnalyticsInput

## SYNOPSIS
Ruft Eingaben für Stream Analytics-Auftragseingaben ab.

## SYNTAX

```
Get-AzStreamAnalyticsInput [-JobName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Im **Cmdlet "Get-AzStreamAnalyticsInput"** werden alle Eingaben aufgeführt, die in einem Stream Analytics-Auftrag definiert sind oder Informationen zu einer bestimmten Eingabe erhalten.

## BEISPIELE

### Beispiel 1: Erhalten von Informationen zu den für einen Auftrag definierten Eingaben
```powershell
PS C:\>Get-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob"
```

Dieser Befehl gibt Informationen zu allen Eingaben zurück, die im Auftrag von StreamingJob definiert sind.

### Beispiel 2: Erhalten von Informationen zu einer bestimmten, für einen Auftrag definierten Eingabe
```powershell
PS C:\>Get-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EntryStream"
```

Dieser Befehl gibt Informationen über die Eingabe mit dem Namen EntryStream zurück, die im Auftrag von StreamingJob definiert wurde.
