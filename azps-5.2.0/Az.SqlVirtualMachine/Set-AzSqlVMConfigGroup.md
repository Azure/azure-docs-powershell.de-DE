# Set-AzSqlVMConfigGroup

## SYNOPSIS
Legen Sie die Informationen relativ zu einer Sql Virtual Machine-Gruppe in einer Konfiguration für einen virtuellen Sql-Computer.

## SYNTAX

```
Set-AzSqlVMConfigGroup [-SqlVM] <AzureSqlVMModel> [-SqlVMGroup] <AzureSqlVMGroupModel>
 -ClusterOperatorAccountPassword <SecureString> -SqlServiceAccountPassword <SecureString>
 [-ClusterBootstrapAccountPassword <SecureString>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das Set-AzSqlVMConfigGroup cmdlet die informationen, die erforderlich sind, um einer Gruppe virtueller Sql-Computer für eine Konfiguration des virtuellen Sql Computers bei beitreten zu können.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $config = Set-AzSqlVMConfigGroup -SqlVM $config -SqlVMGroup $group -ClusterOperatorAccountPassword 'password' -SqlServiceAccountPassword 'password'
```

Aktualisieren Sie die Gruppeninformationen einer Konfiguration des virtuellen Sql-Computers.
