# Set-AzCdnProfile

## SYNOPSIS
Aktualisiert ein CDN-Profil.

## SYNTAX

```
Set-AzCdnProfile -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Set-AzCdnProfile-Cmdlet** aktualisiert ein Azure Content Delivery Network (CDN)-Profil.
