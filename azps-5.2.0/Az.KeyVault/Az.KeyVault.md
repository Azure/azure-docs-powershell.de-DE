# Az.KeyVault Module
## Beschreibung
In diesem Thema werden Hilfethemen für die Azure Key Vault-Cmdlets angezeigt.

## Az.KeyVault-Cmdlets
### [Add-AzKeyVaultCertificate](Add-AzKeyVaultCertificate.md)
Fügt einem Schlüsseltresor ein Zertifikat hinzu.

### [Add-AzKeyVaultCertificateContact](Add-AzKeyVaultCertificateContact.md)
Fügt einen Kontakt für Zertifikatbenachrichtigungen hinzu.

### [Add-AzKeyVaultKey](Add-AzKeyVaultKey.md)
Erstellt einen Schlüssel in einem Schlüsseltresor oder importiert einen Schlüssel in einen Schlüsseltresor.

### [Add-AzKeyVaultManagedStorageAccount](Add-AzKeyVaultManagedStorageAccount.md)
Fügt dem angegebenen Schlüsseltresor ein vorhandenes Azure -Speicherkonto hinzu, dessen Schlüssel vom Schlüsseltresordienst verwaltet werden sollen.

### [Add-AzKeyVaultNetworkRule](Add-AzKeyVaultNetworkRule.md)
Fügt eine Regel hinzu, die den Zugriff auf einen Schlüsseltresor basierend auf der Internetadresse des Clients einschränken soll.
