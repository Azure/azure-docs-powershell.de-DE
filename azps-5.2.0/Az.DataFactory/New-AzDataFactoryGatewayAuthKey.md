# New-AzDataFactoryGatewayAuthKey

## SYNOPSIS
Erstellt einen Authentifizierungsschlüssel für ein Azure Data Factory-Gateway.

## SYNTAX

### ByFactoryName (Standard)
```
New-AzDataFactoryGatewayAuthKey [-DataFactoryName] <String> [-GatewayName] <String> [-KeyName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByFactoryObject
```
New-AzDataFactoryGatewayAuthKey [-InputObject] <PSDataFactory> [-GatewayName] <String> [-KeyName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "New-AzDataFactoryGatewayAuthKey"** erstellt den Gatewayauthentifizierungsschlüssel für ein angegebenes Azure Data Factory-Gateway.
Mit diesem Schlüssel registrieren Sie das Gateway bei einem Clouddienst.

## BEISPIELE

### Beispiel 1: Erstellt einen Authentifizierungsschlüssel für das Gateway für Schlüssel1
```
PS C:\> New-AzDataFactoryGatewayAuthKey -ResourceGroup ADFResource -GatewayName 'MyGateway' -DataFactoryName MyADF -KeyName key1
Key1 : DMG@632e739e-1053-4070-9102-8591f067526e@41fcbc45-c594-4152-a8f1-fcbcd6452aea@wu@BH0EV9hu/o2IYGQzfYYD203XhdS6Tty
       fkYwYFbG6wBU=
Key2 :
```

Mit diesem Befehl wird ein Authentifizierungsschlüssel von Schlüssel1 für das Daten factory-Gateway namens "MyGateway" erstellt.
