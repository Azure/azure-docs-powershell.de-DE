# New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride

## SYNOPSIS
Erstellt den Eintrag "RuleGroupOverride" in "ManagedRuleSets" für die Firewallrichtlinie.

## SYNTAX

```
New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride -RuleGroupName <String>
 -Rule <PSApplicationGatewayFirewallPolicyManagedRuleOverride[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Die **Neue-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride** erstellt einen Regeleintrag "GroupOverride" in einem managedRuleSet für eine Firewallrichtlinie.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $overrideEntry = New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride -RuleGroupName $ruleName -Rules $rule1,$rule2
```

Erstellt einen Eintrag "RuleGroupOverride" mit dem Gruppennamen $ruleName und Regeln als $rule 1, $rule 2. Weist die gleiche Zuordnung zu $overrideEntry
