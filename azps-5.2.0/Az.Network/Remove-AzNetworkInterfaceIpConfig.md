# Remove-AzNetworkInterfaceIpConfig

## SYNOPSIS
Entfernt eine Netzwerkschnittstellen-IP-Konfiguration von einer Netzwerkschnittstelle.

## SYNTAX

```
Remove-AzNetworkInterfaceIpConfig -Name <String> -NetworkInterface <PSNetworkInterface>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzNetworkInterfaceIpConfig"** entfernt eine Netzwerkschnittstellen-IP-Konfiguration von einer Azure-Netzwerkschnittstelle.

## BEISPIELE

### Beispiel 1: Löschen einer IP-Konfiguration von einer Netzwerkschnittstelle
```powershell
$nic = Get-AzNetworkInterface -Name mynic -ResourceGroupName myrg

Remove-AzNetworkInterfaceIpConfig -Name IPConfig-1 -NetworkInterface $nic

Set-AzNetworkInterface -NetworkInterface $nic
```

Der erste Befehl ruft eine Netzwerkschnittstelle namens "mynic" ab und speichert sie in der variablen $nic. Mit dem zweiten Befehl wird die dieser Netzwerkschnittstelle zugeordnete IP-Konfiguration namens IPConfig-1 entfernt. Der dritte Befehl legt Änderungen fest, die an der Netzwerkschnittstelle vorgenommen werden.
