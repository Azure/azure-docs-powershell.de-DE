# Get-AzLoadBalancerOutboundRuleConfig

## SYNOPSIS
Ruft eine ausgehende Regelkonfiguration in einem Lastenausgleich ab.

## SYNTAX

```
Get-AzLoadBalancerOutboundRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzLoadBalancerOutboundRuleConfig"** ruft eine ausgehende Regelkonfiguration oder eine Liste der ausgehenden Regelkonfigurationen in einem Lastenausgleich ab.

## BEISPIELE

### Beispiel 1: Erhalten einer ausgehenden Regelkonfiguration in einem Lastenausgleich
```powershell
PS C:\>$slb = Get-AzLoadBalancer -ResourceGroupName "MyResourceGroup" -Name "MyLoadBalancer"
PS C:\>Get-AzLoadBalancerOutboundRuleConfig -LoadBalancer $slb -Name "MyRule"
```

Der erste Befehl ruft den Lastenausgleich namens "MyLoadBalancer" ab und speichert ihn dann im Variablenspeicher $slb.
Der zweite Befehl ruft die ausgehende Regelkonfiguration namens "MyRule" ab, die diesem Lastenausgleich zugeordnet ist.
