# Remove-AzApplicationGatewayBackendAddressPool

## SYNOPSIS
Entfernt einen Back-End-Adresspool von einem Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewayBackendAddressPool -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzApplicationGatewayBackendAddressPool"** entfernt einen Back-End-Adresspool aus einem Azure-Anwendungsgateway.

## BEISPIELE

### Beispiel 1: Entfernen eines Back-End-Adresspools von einem Anwendungsgateway
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayBackendAddressPool -ApplicationGateway $AppGw -Name "BackEndPool02"
```

Der erste Befehl ruft das Anwendungsgateway namens ApplicationGateway01 ab, das zur Ressourcengruppe "ResourceGroup01" gehört, und speichert es in der $AppGw Variable.
Mit dem zweiten Befehl wird der Back-End-Adresspool mit dem Namen BackEndPool02 vom Anwendungsgateway entfernt.
