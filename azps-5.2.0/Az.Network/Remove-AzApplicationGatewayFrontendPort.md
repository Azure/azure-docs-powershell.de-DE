# Remove-AzApplicationGatewayFrontendPort

## SYNOPSIS
Entfernt einen Front-End-Port von einem Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewayFrontendPort -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzApplicationGatewayFrontendPort"** entfernt einen Front-End-Port von einem Azure-Anwendungsgateway.

## BEISPIELE

### Beispiel: Entfernen eines Front-End-Ports von einem Anwendungsgateway
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayFrontendPort -ApplicationGateway $AppGw -Name "FrontEndPort02"
```

Der erste Befehl ruft ein Anwendungsgateway namens ApplicationGateway01 ab, das zur Ressourcengruppe "ResourceGroup01" gehört und das Gateway in einer $AppGw speichert.
Der zweite Befehl entfernt den Port namens "FrontEndPort02" vom Anwendungsgateway.
