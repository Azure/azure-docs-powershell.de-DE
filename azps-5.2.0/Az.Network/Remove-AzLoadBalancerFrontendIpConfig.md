# Remove-AzLoadBalancerFrontendIpConfig

## SYNOPSIS
Entfernt eine Front-End-IP-Konfiguration aus einem Lastenausgleich.

## SYNTAX

```
Remove-AzLoadBalancerFrontendIpConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzLoadBalancerFrontendIpConfig"** entfernt eine Front-End-IP-Konfiguration aus einem Azure Load Balancer.

## BEISPIELE

### Beispiel 1: Entfernen einer Front-End-IP-Konfiguration aus einem Lastenausgleich
```
PS C:\>$loadbalancer = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:> Remove-AzLoadBalancerFrontendIpConfig -Name "frontendName" -LoadBalancer $loadbalancer
```

Der erste Befehl ruft den Lastenausgleich ab, der der Front-End-IP-Konfiguration zugeordnet ist, die Sie entfernen möchten, und speichert ihn dann in der $loadbalancer Variable.
Der zweite Befehl entfernt die zugehörige Frontend-IP-Konfiguration aus dem Lastenausgleich in $loadbalancer.
