# Get-AzEffectiveRouteTable

## SYNOPSIS
Ruft die effektive Routentabelle einer Netzwerkschnittstelle ab.

## SYNTAX

```
Get-AzEffectiveRouteTable -NetworkInterfaceName <String> [-ResourceGroupName <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzEffectiveRouteTable"** gibt die effektive Routentabelle zurück, die auf eine Netzwerkschnittstelle angewendet wird.

## BEISPIELE

### Beispiel 1: Erstellen der Tabelle für effektive Routen auf einer Netzwerkschnittstelle
```
PS C:\>Get-AzEffectiveRouteTable -NetworkInterfaceName "MyNetworkInterface" -ResourceGroupName "MyResourceGroup"
```

Dieser Befehl ruft die effektive Routentabelle ab, die der Netzwerkschnittstelle namens "MyNetworkInterface" in der Ressourcengruppe "MyResourceGroup" zugeordnet ist.
