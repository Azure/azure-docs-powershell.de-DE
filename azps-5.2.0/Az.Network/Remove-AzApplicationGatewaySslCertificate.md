# Remove-AzApplicationGatewaySslCertificate

## SYNOPSIS
Entfernt ein Zertifikat aus einem Azure-Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewaySslCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzApplicationGatewaySslCertificate"** entfernt ein Secure Sockets Layer (SSL)-Zertifikat aus einem Azure-Anwendungsgateway.

## BEISPIELE

### Beispiel 1: Entfernen eines SSL-Zertifikats von einem Anwendungsgateway
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewaySslCertificate -ApplicationGateway $AppGW -Name "Cert02"
```

Der erste Befehl ruft das Anwendungsgateway namens ApplicationGateway01 ab und speichert das Ergebnis in der Variablen namens $AppGW.
Mit dem zweiten Befehl wird das #A0 namens Cert02 aus dem Anwendungsgateway entfernt, das in der #A1 $AppGW ist.
