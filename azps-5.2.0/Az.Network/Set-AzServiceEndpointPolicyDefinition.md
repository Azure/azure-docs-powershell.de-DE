# Set-AzServiceEndpointPolicyDefinition

## SYNOPSIS
Aktualisiert die Richtliniendefinition für einen Dienstendpunkt.

## SYNTAX

```
Set-AzServiceEndpointPolicyDefinition -Name <String> -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-Description <String>] [-ServiceResource <String[]>] [-Service <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Set-AzServiceEndpointPolicyDefinition"** erstellt eine Richtlinie für den Dienstendpunkt.

## BEISPIELE

### Beispiel 1: Aktualisiert die Definition einer Dienstendpunktrichtlinie in einer Dienstendpunktrichtlinie
```
$serviceEndpointPolicy = Set-AzServiceEndpointPolicyDefinition -Name "Policydef1" -ServiceEndpointPolicy $serviceEndpointPolicy
```

Mit diesem Befehl wird eine Richtlinie für den Dienstendpunkt namens "Policydef1" in der vom Objektobjektobjekt definierten Dienstendpunktrichtlinie $ServiceEndpointPolicy.
