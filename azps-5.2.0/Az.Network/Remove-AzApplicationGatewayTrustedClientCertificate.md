# Remove-AzApplicationGatewayTrustedClientCertificate

## SYNOPSIS
Entfernt das Zertifikatkettenobjekt der vertrauenswürdigen Client-Zertifizierungsstelle aus einem Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewayTrustedClientCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzApplicationGatewayTrustedClientCertificate"** entfernt das Zertifikatkettenobjekt der vertrauenswürdigen Client-Zertifizierungsstelle aus einem Anwendungsgateway.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name "TrustedClientCertificate01"
```

Der erste Befehl ruft ein Anwendungsgateway ab und speichert es in der $gw Variable. Mit dem zweiten Befehl wird die Zertifikatkette der vertrauenswürdigen Client-Zertifizierungsstelle namens "TrustedClientCertificate01" aus dem Anwendungsgateway entfernt, das auf der $gw.
