# Get-AzLoadBalancerRuleConfig

## SYNOPSIS
Ruft die Regelkonfiguration f端r einen Lastenausgleich ab.

## SYNTAX

```
Get-AzLoadBalancerRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzLoadBalancerRuleConfig"** ruft eine oder mehrere Regelkonfigurationen f端r einen Lastenausgleich ab.

## BEISPIELE

### Beispiel 1: Regelkonfiguration eines Lastenausgleichs
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerRuleConfig -Name "MyLBrulename" -LoadBalancer $slb
```

Der erste Befehl ruft den Lastenausgleich namens "MyLoadBalancer" ab und speichert ihn dann im Variablenspeicher $slb.
Der zweite Befehl ruft die zugeordnete Regelkonfiguration namens "MyLBrulename" aus dem Lastenausgleich in $slb.
