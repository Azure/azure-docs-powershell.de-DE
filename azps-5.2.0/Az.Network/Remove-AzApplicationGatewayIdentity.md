# Remove-AzApplicationGatewayIdentity

## SYNOPSIS
Entfernt eine Identität von einem Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewayIdentity -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
**Mit dem Cmdlet "Remove-AzApplicationGatewayIdentity"** wird die Identität von einem Anwendungsgateway entfernt.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $appgw = Remove-AzApplicationGatewayIdentity -ApplicationGateway $appgw
PS C:\> $updatedgateway = Set-AzApplicationGateway -ApplicationGateway $appgw
```

In diesem Beispiel wird die Identität aus einem vorhandenen Anwendungsgateway entfernt.
Hinweis: Wenn das Gateway auf einen geheimen Schlüsselverweis verweist, ist es auch wichtig, diese Ssl-Zertifikatverweise zusammen mit diesem Vorgang zu entfernen.
