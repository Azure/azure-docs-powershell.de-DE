# New-AzLoadBalancerBackendAddressPoolConfig

## SYNOPSIS
Erstellt eine Back-End-Adresspoolkonfiguration für einen Lastenausgleich.

## SYNTAX

```
New-AzLoadBalancerBackendAddressPoolConfig -Name <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "New-AzLoadBalancerBackendAddressPoolConfig"** erstellt eine Back-End-Adresspoolkonfiguration für einen Azure Load Balancer.

## BEISPIELE

### Beispiel 1: Erstellen einer Back-End-Adresspoolkonfiguration für einen Lastenausgleich
```
PS C:\>New-AzLoadBalancerBackendAddressPoolConfig -Name "BackendAddressPool02"
```

Mit diesem Befehl wird eine Back-End-Adresspool-Konfiguration namens Back-EndAddressPool02 für einen Lastenausgleich erstellt.
