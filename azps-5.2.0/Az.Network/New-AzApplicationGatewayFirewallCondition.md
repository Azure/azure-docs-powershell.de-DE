# New-AzApplicationGatewayFirewallCondition

## SYNOPSIS
Erstellt eine Übereinstimmungsbedingung für eine benutzerdefinierte Regel.

## SYNTAX

```
New-AzApplicationGatewayFirewallCondition -MatchVariable <PSApplicationGatewayFirewallMatchVariable[]>
 -Operator <String> [-NegationCondition <Boolean>] -MatchValue <String[]> [-Transform <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
**"New-AzApplicationGatewayFirewallCondition"** erstellt eine Übereinstimmungsbedingung für die benutzerdefinierte Firewallregel.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $condition = New-AzApplicationGatewayFirewallCondition -MatchVariable $variable -Operator Contains -NegationCondition false -Transforms Lowercase, Trim -MatchValue abc, cde
```

Der Befehl erstellt eine neue Übereinstimmungsbedingung mithilfe der im $variable definierten Übereinstimmungsvariablen, der Operator ist "Contains" und "negation condition is false", "Transfroms", einschließlich Kleinbuchstaben und "trim", der Übereinstimmungswert "abc" und "cde". Die neue Übereinstimmungsbedingung wird in $condition.
