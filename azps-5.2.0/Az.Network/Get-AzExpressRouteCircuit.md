# Get-AzExpressRouteCircuit

## SYNOPSIS
Ruft einen Azure -ExpressRoute-Schaltkreis aus Azure ab.

## SYNTAX

```
Get-AzExpressRouteCircuit [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzExpressRouteCircuit"** dient zum Abrufen eines ExpressRoute-Schaltkreisobjekts aus Ihrem Abonnement. Das zurückgegebene Schaltkreisobjekt kann als Eingabe für andere Cmdlets verwendet werden, die mit ExpressRoute-Schaltkreisen arbeiten.
