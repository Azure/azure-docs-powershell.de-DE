# Remove-AzFirewall

## SYNOPSIS
Entfernen Sie eine Firewall.

## SYNTAX

```
Remove-AzFirewall -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzFirewall"** entfernt eine Azure-Firewall.

## BEISPIELE

### Beispiel 1: Erstellen und Löschen einer Firewall
```powershell
New-AzFirewall -Name "azFw" -ResourceGroupName "rgName" -Location centralus 

Remove-AzFirewall -Name "azFw" -ResourceGroupName "rgName"
Confirm
Are you sure you want to remove resource 'azFw'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

In diesem Beispiel wird eine Firewall erstellt und dann gelöscht. Verwenden Sie das Kennzeichen "-Force", um die Eingabeaufforderung beim Löschen der Firewall zu unterdrücken.
