# Remove-AzApplicationGatewayHttpListenerCustomError

## SYNOPSIS
Entfernt einen benutzerdefinierten Fehler aus einem Http-Listener eines Anwendungsgateways.

## SYNTAX

```
Remove-AzApplicationGatewayHttpListenerCustomError -StatusCode <String>
 -HttpListener <PSApplicationGatewayHttpListener> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzApplicationGatewayCustomError"** entfernt einen benutzerdefinierten Fehler aus einem http-Listener eines Anwendungsgateways.

## BEISPIELE

### Beispiel 1: Entfernt einen benutzerdefinierten Fehler aus einem HTTP-Listener.
```powershell
PS C:\> $updatedlistener = Remove-AzApplicationGatewayCustomError -HttpListener $listener01 -StatusCode HttpStatus502
```

Dieser Befehl entfernt den benutzerdefinierten Fehler von http-Statuscode 502 aus dem http-Listener $listener 01 und gibt den aktualisierten Listener zur端ck.
