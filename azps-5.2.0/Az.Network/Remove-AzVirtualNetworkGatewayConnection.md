# Remove-AzVirtualNetworkGatewayConnection

## SYNOPSIS
Löscht eine Virtuelle Netzwerkgatewayverbindung

## SYNTAX

```
Remove-AzVirtualNetworkGatewayConnection -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Die Virtuelle Netzwerkgatewayverbindung ist das Objekt, das den mit dem virtuellen Netzwerkgateway in Azure verbundenen IPsec-Tunnel (Site-to-Site oder Vnet-to-Vnet) darstellt.
Das **Cmdlet "Remove-AzVirtualNetworkGatewayConnection"** löscht das Objekt Ihrer Verbindung basierend auf "Name" und "Ressourcengruppenname".

## BEISPIELE

### Beispiel 1: Löschen einer virtuellen Netzwerkgatewayverbindung
```powershell
Remove-AzVirtualNetworkGatewayConnection -Name myTunnel -ResourceGroupName myRG
```

Löscht das Objekt der Virtuellen Netzwerkgatewayverbindung mit dem Namen "myKlasse" in der Ressourcengruppe "myRG".
