# Get-AzCognitiveServicesAccountType

## SYNOPSIS
Ruft die verfügbaren Cognitive Services-Kontotypen ab.

## SYNTAX

### GetAccountTypes (Standard)
```
Get-AzCognitiveServicesAccountType [-Location <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetAccountTypeWithName
```
Get-AzCognitiveServicesAccountType -TypeName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Get-AzCognitiveServicesAccountType** ruft die verfügbaren Cognitive Services-Kontotypen unter diesem Abonnement ab.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Get-AzCognitiveServicesAccountType
```

Hier finden Sie eine Liste der verfügbaren Typen.

### Beispiel 2
```powershell
PS C:\> Get-AzCognitiveServicesAccountType -Location westus
```

Hier finden Sie eine Liste der verfügbaren Typen in WestUs.

### Beispiel 3
```powershell
PS C:\> Get-AzCognitiveServicesAccountType -TypeName Face

Face
```

Überprüfen Sie, ob es sich um einen gültigen Typnamen handelt, wird der Name zurückgegeben, `Face` wenn es sich um einen gültigen Namen handelt.
