# New-AzDigitalTwinsInstance

## SYNOPSIS
Erstellen oder aktualisieren Sie die Metadaten einer DigitalTwinsInstance.
Das übliche Muster zum Ändern einer Eigenschaft besteht im Abrufen der Metadaten "DigitalTwinsInstance" und der Sicherheitsmetadaten, die dann mit den geänderten Werten in einem neuen Textkörper kombiniert werden, um "DigitalTwinsInstance" zu aktualisieren.

## SYNTAX

### CreateExpanded (Standard)
```
New-AzDigitalTwinsInstance -ResourceGroupName <String> -ResourceName <String> -Location <String>
 [-SubscriptionId <String>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

### Erstellen
```
New-AzDigitalTwinsInstance -ResourceGroupName <String> -ResourceName <String>
 -DigitalTwinsCreate <IDigitalTwinsDescription> [-SubscriptionId <String>] [-DefaultProfile <PSObject>]
 [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## BESCHREIBUNG
Erstellen oder aktualisieren Sie die Metadaten einer DigitalTwinsInstance.

```powershell

```

Standardmäßiges Erstellen einer AzDigitalTwinsInstance

### Beispiel 2: Erstellen eines AzDigitalTwinsInstance von AzDigitalTwins-Objekt.
```powershell
PS C:\> $GetAzDigTwin = Get-AzDigitalTwinsInstance -ResourceGroupName youritemp -ResourceName youriDigitalTwinsTest
```
