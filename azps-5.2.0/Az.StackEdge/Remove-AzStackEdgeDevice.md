# Remove-AzStackEdgeDevice

## SYNOPSIS
Entfernt ein Stack Edge-Gerät.

## SYNTAX

### DeleteByNameParameterSet (Standard)
```
Remove-AzStackEdgeDevice [-ResourceGroupName] <String> [-Name] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByResourceIdParameterSet
```
Remove-AzStackEdgeDevice -ResourceId <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-PassThru]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByInputObjectParameterSet
```
Remove-AzStackEdgeDevice -DeviceObject <PSStackEdgeDevice> [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzStackEdgeDevice"** entfernt die Konfiguration für ein Stack Edge-Gerät.
Beachten Sie, dass das Gerät erst gelöscht werden kann, nachdem Sie die Bestellung bestellt haben und bevor das Gerät von Microsoft für die Sendung vorbereitet wird.

Informationen zum Löschen der Ressource vor der Verwendung dieses [Cmdlets](https://docs.microsoft.com/en-us/azure/databox-online/data-box-edge-return-device#delete-the-resource) finden Sie in der Dokumentation

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Remove-AzStackEdgeDevice ResourceGroupName resourceGroupName -Name deviceName
```
