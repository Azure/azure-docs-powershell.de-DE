# New-AzBatchApplication

## SYNOPSIS
Fügt dem angegebenen Batchkonto eine Anwendung hinzu.

## SYNTAX

```
New-AzBatchApplication [-AccountName] <String> [-ResourceGroupName] <String> [-ApplicationName] <String>
 [[-AllowUpdates] <Boolean>] [[-DisplayName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "New-AzBatchApplication"** fügt dem angegebenen Azure -Batchkonto eine Anwendung hinzu.

## BEISPIELE

### Beispiel 1: Hinzufügen einer leeren Anwendung zu einem Batchkonto
```
PS C:\>New-AzBatchApplication -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationName "Litware" -AllowUpdates $True -DisplayName "Litware Advanced Reticulator"
```

Mit diesem Befehl wird die Anwendung Litware im ContosoBatch-Konto erstellt.
Die Anwendung enthält zunächst keine Pakete.
