# Az.Batch Module
## Beschreibung
Mit den Azure -Batch-Cmdlets im Azure-Modul können Sie Microsoft Azure Batch services in Azure PowerShell verwalten.

## Az.Batch-Cmdlets
### [Disable-AzBatchAutoScale](Disable-AzBatchAutoScale.md)
Deaktiviert die automatische Skalierung eines Pools.

### [Disable-AzBatchComputeNodeScheduling](Disable-AzBatchComputeNodeScheduling.md)
Deaktiviert die Aufgabenplanung für den angegebenen Rechenknoten.

### [Disable-AzBatchJob](Disable-AzBatchJob.md)
Deaktiviert einen Stapelauftrag.

### [Disable-AzBatchJobSchedule](Disable-AzBatchJobSchedule.md)
Deaktiviert einen Stapelauftragszeitplan.

### [Enable-AzBatchAutoScale](Enable-AzBatchAutoScale.md)
Ermöglicht die automatische Skalierung eines Pools.

Aktiviert einen Stapelauftragszeitplan.

### [Enable-AzBatchTask](Enable-AzBatchTask.md)
