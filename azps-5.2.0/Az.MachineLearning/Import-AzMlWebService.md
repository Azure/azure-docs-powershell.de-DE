# Import-AzMlWebService

## SYNOPSIS
Importiert ein JSON-Objekt in eine Webdienstdefinition.

## SYNTAX

### ImportFromJSONFile
```
Import-AzMlWebService -InputFile <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ImportFromJSONString.
```
Import-AzMlWebService -JsonString <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das Import-AzMlWebService importiert , wird entweder direkt oder in einer Referenzdatei angegeben und erstellt ein Webdienstdefinitionsobjekt, das an das cmdlet New-AzMlWebService übergeben werden kann.

## BEISPIELE

### Beispiel 1: Importieren aus einer Zeichenfolge
```
Import-AzMlWebService -JsonString $jsonDefinition
```

### Beispiel 2: Importieren aus dateipfad
```
Import-AzMlWebService -InputFile "C:\mlservice.json"
```
