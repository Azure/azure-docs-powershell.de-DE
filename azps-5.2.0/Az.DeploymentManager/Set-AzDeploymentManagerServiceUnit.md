# Set-AzDeploymentManagerServiceUnit

## SYNOPSIS
Aktualisiert die Diensteinheit.

## SYNTAX

```
Set-AzDeploymentManagerServiceUnit [-InputObject] <PSServiceUnitResource>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Set-AzDeploymentManagerServiceUnit"** aktualisiert eine Diensteinheit mit dem angegebenen Diensteinheitsobjekt.
Das Cmdlet gibt das aktualisierte Diensteinheitsobjekt zurück.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Set-AzDeploymentManagerServiceUnit -InputObject $serviceUnitObject
```

Mit diesem Befehl wird eine Diensteinheit aktualisiert, deren Name, Dienstname, Name der Diensttopologie und Ressourcengruppe mit den Eigenschaften "Name", "ServiceName", "ServiceTopologyName" bzw. "ResourceGroupName" der $serviceUnitObject übereinstimmen.
Der Befehl gibt das aktualisierte Diensteinheitsobjekt zurück.
