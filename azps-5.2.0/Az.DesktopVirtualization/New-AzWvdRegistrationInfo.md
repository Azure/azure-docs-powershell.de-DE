# New-AzWvdRegistrationInfo

## SYNOPSIS
Erstellen Sie Informationen zur Registrierung des virtuellen Windows-Desktops.

## SYNTAX

```
New-AzWvdRegistrationInfo -ExpirationTime <String> -HostPoolName <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## BESCHREIBUNG
Erstellen Sie Informationen zur Registrierung des virtuellen Windows-Desktops.

## BEISPIELE

### Beispiel 1: Erstellen eines Tokens für die Registrierung des virtuellen Windows-Desktops
```powershell
PS C:\> New-AzWvdRegistrationInfo -ResourceGroupName ResourceGroupName -HostPoolName HostPoolName -ExpirationTime $((get-date).ToUniversalTime().AddDays(1).ToString('yyyy-MM-ddTHH:mm:ss.fffffffZ'))

ExpirationTime       RegistrationTokenOperation Token
```

Mit diesem Befehl wird ein Windows Virtual Desktop Registration Token in einem Hostpool erstellt.
