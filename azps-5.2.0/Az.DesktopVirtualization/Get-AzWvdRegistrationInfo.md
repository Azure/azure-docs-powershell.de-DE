# Get-AzWvdRegistrationInfo

## SYNOPSIS
Hier finden Sie Informationen zur Registrierung des virtuellen Windows-Desktops.

## SYNTAX

```
Get-AzWvdRegistrationInfo -HostPoolName <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## BESCHREIBUNG
Hier finden Sie Informationen zur Registrierung des virtuellen Windows-Desktops.

## BEISPIELE

### Beispiel 1: Erhalten eines Tokens für die Registrierung des virtuellen Windows-Desktops
```powershell
PS C:\> Get-AzWvdRegistrationInfo -ResourceGroupName ResourceGroupName -HostPoolName HostPoolName

ExpirationTime       RegistrationTokenOperation Token
--------------       -------------------------- -----
4/1/2020 10:19:33 PM None                       eyJhbGciOiJSUzI1NiIsImtpZCI6IkMyRjU1RUYxNzg0MEFCNzkzMDk2RUYzRjdEMkNBRDk0NThGNDhEOTQiLCJ0eXAiOiJKV1QifQ.eyJSZWdpc3RyYXRpb25JZCI6IjU5NGJjZWUwLTk5MjQtNDg3ZC1iOW...
```

Dieser Befehl ruft ein Windows Virtual Desktop Registration Token in einem Hostpool ab.
