# Add-AzServiceFabricNode

## SYNOPSIS
Fügen Sie dem jeweiligen Knotentyp im Cluster Knoten hinzu.

## SYNTAX

```
Add-AzServiceFabricNode -NumberOfNodesToAdd <Int32> [-ResourceGroupName] <String> [-Name] <String>
 -NodeType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Verwenden **Sie Add-AzServiceFabricNode,** um dem jeweiligen Knotentyp Knoten hinzuzufügen. Sie müssen nur die Anzahl der Knoten angeben, die Sie einem Knotentyp hinzufügen möchten.

## BEISPIELE

### Beispiel 1
```powershell
PS c:> Add-AzServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NumberOfNodesToAdd 2 -NodeTypeName 'nt1'
```

Mit diesem Befehl werden dem Knotentyp "n1" zwei Knoten hinzugefügt.
