# Set-AzStorageServiceMetricsProperty

## SYNOPSIS
Ändert die Metrikeigenschaften für den Azure Storage-Dienst.

## SYNTAX

```
Set-AzStorageServiceMetricsProperty [-ServiceType] <StorageServiceType> [-MetricsType] <ServiceMetricsType>
 [-Version <Double>] [-RetentionDays <Int32>] [-MetricsLevel <MetricsLevel>] [-PassThru]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Set-AzStorageServiceMetricsProperty"** ändert die Metrikeigenschaften für den Azure Storage-Dienst.

## BEISPIELE

### Beispiel 1: Ändern der Metrikeigenschaften für den Blob-Dienst
```
C:\PS>Set-AzStorageServiceMetricsProperty -ServiceType Blob -MetricsType Hour -MetricsLevel Service -PassThru -RetentionDays 10 -Version 1.0
```

Mit diesem Befehl werden Die Version 1.0-Metriken für den BLOB-Speicher auf eine Dienstebene geändert.
Die Azure Storage -Dienstmetriken bewahren Einträge 10 Tage lang auf.
Da dieser Befehl den Parameter *"PassThru"* angibt, zeigt der Befehl die geänderten Metrikeigenschaften an.
