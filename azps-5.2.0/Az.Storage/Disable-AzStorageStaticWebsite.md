# Disable-AzStorageStaticWebsite

## SYNOPSIS
Deaktivieren Sie die statische Website für das Azure Storage-Konto.

## SYNTAX

```
Disable-AzStorageStaticWebsite [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Disable-AzStorageStaticWebsite"** deaktiviert die statische Website für das Azure Storage-Konto.

## BEISPIELE

### Beispiel 1: Deaktivieren der statischen Website für ein Azure -Speicherkonto
```
C:\PS>Disable-AzStorageStaticWebsite
```

Mit diesem Befehl wird die statische Website für ein Azure -Speicherkonto deaktiviert.
