# New-AzCosmosDBGremlinCompositePath

## SYNOPSIS
Erstellt ein neues Objekt vom Typ "PSCompositePath". Sie kann als Parameterwert für Set-AzCosmosDBGremlinGraph übergeben werden.

## SYNTAX

```
New-AzCosmosDBGremlinCompositePath [-Path <String>] [-Order <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Objekt, das dem CompositePath-Objekt der Gremlin-API entspricht.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> New-AzCosmosDBGremlinCompositePath -Path "/abc" -Order Ascending

Path Order
---- -----
/abc Ascending
```
