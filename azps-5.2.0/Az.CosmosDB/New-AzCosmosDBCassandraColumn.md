# New-AzCosmosDBCassandraColumn

## SYNOPSIS
Erstellt eine neue Spalte des UnternamensDB.

## SYNTAX

```
New-AzCosmosDBCassandraColumn -Name <String> -Type <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Die **Neue-AzCosmosDBCassenColumn** erstellt eine neue Spalte aus Demmerz.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> New-AzCosmosDBCassandraColumn -Name "name" -Type int

Name Type
---- ----
name int
```
