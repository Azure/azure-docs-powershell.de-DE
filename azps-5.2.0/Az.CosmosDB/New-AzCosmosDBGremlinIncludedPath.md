# New-AzCosmosDBGremlinIncludedPath

## SYNOPSIS
Erstellt ein neues Objekt vom Typ "PSIncludedPath". Sie kann als Parameterwert für "Set-AzCosmosDBGremlinGraph" übergeben werden.

## SYNTAX

```
New-AzCosmosDBGremlinIncludedPath [-Path <String>] [-Index <PSIndexes[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Objekt, das dem IncludedPath-Objekt der Gremlin-API entspricht.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $index1 = New-AzCosmosDBGremlinIncludedPathIndex -DataType String -Precision -1 -Kind Hash
New-AzCosmosDBGremlinIncludedPath -Path "/*" -Index $index1
Path Indexes
---- -------
/*   {Microsoft.Azure.Commands.CosmosDB.Models.PSIndexes}
```
