# New-AzDataMigrationAzureActiveDirectoryApp

## Synopsis
Erstellen Sie eine neue Instanz datamigration Azure ActiveDirectory-Anwendungsdetails.

## Syntax

```
New-AzDataMigrationAzureActiveDirectoryApp -ApplicationId <String> -AppKey <SecureString>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Erstellen Sie eine neue Instanz datamigration Azure ActiveDirectory-Anwendungsdetails.

## Beispiele

### Beispiel 1
```powershell
PS C:\> $secpasswd = ConvertTo-SecureString "Your Secret Key Here" -AsPlainText -Force
C:\> New-AzDmsAadApp -ApplicationId "Your AppId/Service Principal ID here" -AppKey $secpasswd
```
ApplicationId: "Ihre Benutzerkennung/Dienstprinzipal-ID hier" AppKey: System. Security. SecureString-Mandantenkennung: "Mandanten-ID"
