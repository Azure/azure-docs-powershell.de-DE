# Grant-AzHDInsightRdpServicesAccess

## Synopsis
Gewährt RDP-Zugriff auf den Windows-Cluster.

## Syntax

```
Grant-AzHDInsightRdpServicesAccess [-ClusterName] <String> [-RdpCredential] <PSCredential>
 [-RdpAccessExpiry] <DateTime> [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Grant-AzHDInsightRdpServicesAccess** ermöglicht das Remote Desktop Protokoll (RDP) für den Zugriff auf einen Windows-basierten Azure HDInsight-Cluster.

## Beispiele

### Beispiel 1: Gewähren des RDP-Zugriffs auf einen Azure HDInsight-Cluster
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential

PS C:\> Grant-AzHDInsightRdpServicesAccess `
            -ClusterName $clusterName `
            -RdpCredential $newRdpCreds `
            -RdpAccessExpiry $newRdpExpiry
```

Dieser Befehl gewährt RDP-Zugriff auf den Cluster mit dem Namen "Your-Hadoop-001".
