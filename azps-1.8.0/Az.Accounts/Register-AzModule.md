# Register-AzModule

## Synopsis
Nur für interne Verwendung – Bereitstellen von Laufzeitunterstützung für generierte autoruhe-Cmdlets

## Syntax

```
Register-AzModule [<CommonParameters>]
```

## Beschreibung
Nur für interne Verwendung – Bereitstellen von Laufzeitunterstützung für generierte autoruhe-Cmdlets

## Beispiele

### Beispiel 1
```powershell
PS C:\> Register-AzModule
```

Intern von von autoruhe generierten Cmdlets verwendet
