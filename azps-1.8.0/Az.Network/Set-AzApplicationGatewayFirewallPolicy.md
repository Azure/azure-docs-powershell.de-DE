## Beschreibung
Das Cmdlet " **Satz-AzApplicationGatewayFirewallPolicy** " aktualisiert eine Azure Application Gateway-Firewall-Richtlinie.

## Beispiele

### Beispiel 1
```powershell
PS C:\> $UpdatedAppGwFirewallPolicy = Set-AzApplicationGatewayFirewallPolicy -ApplicationGateway $AppGwFirewallPolicy
```

Dieser Befehl aktualisiert die Firewall-Richtlinie des Anwendungs Gateways mit den Einstellungen in der $AppGwFirewallPolicy-Variable und speichert das aktualisierte Gateway in der $UpdatedAppGwFirewallPolicy-Variablen.
