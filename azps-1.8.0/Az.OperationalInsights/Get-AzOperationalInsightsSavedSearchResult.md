# Get-AzOperationalInsightsSavedSearchResult

## Synopsis
Gibt die Ergebnisse einer Abfrage zurück.

## Syntax

```
Get-AzOperationalInsightsSavedSearchResult [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-SavedSearchId] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzOperationalInsightsSavedSearchResult** " gibt die Ergebnisse aus der Abfrage zurück, die durch die Such-ID angegeben wird.

## Beispiele

### Beispiel 1: Abrufen aller Suchergebnisse für eine gespeicherte Suche
```
PS C:\>Get-AzOperationalInsightSavedSearchResult -ResourceGroupName "ContosoResourceGroup" -WorkspaceName "ContosoWorkspace" -SavedSearchId "ContosoSavedSearchId"
```

Dieser Befehl ruft alle Suchergebnisse für eine gespeicherte Suche ab.
