# Remove-AzIntegrationAccountSchema

## Synopsis
Entfernt ein Schema des Integrations Kontos.

## Syntax

```
Remove-AzIntegrationAccountSchema -ResourceGroupName <String> -Name <String> -SchemaName <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzIntegrationAccountSchema** entfernt ein Schema eines Integrations Kontos aus einer Ressourcengruppe.
Geben Sie den Namen des Integrations Kontos, den Ressourcengruppennamen und den Schemanamen an.
Dieses Modul unterstützt dynamische Parameter.
Wenn Sie einen dynamischen Parameter verwenden möchten, geben Sie ihn in den Befehl ein.
Wenn Sie die Namen der dynamischen Parameter ermitteln möchten, geben Sie nach dem Cmdlet-Namen einen Bindestrich (-) ein, und drücken Sie dann wiederholt die Tab-Taste, um die verfügbaren Parameter zu durchlaufen.
Wenn Sie einen erforderlichen Vorlagenparameter nicht angeben, werden Sie vom Cmdlet zur Eingabe des Werts aufgefordert.

## Beispiele

### Beispiel 1: Entfernen eines Schemas für ein Integrations Konto
```
PS C:\>Remove-AzIntegrationAccountSchema -ResourceGroupName "ResourceGroup11" -Name "IntegrationAccount31" -SchemaName "IntegrationAccountSchema43"
```

Dieser Befehl entfernt ein Schema des Integrations Kontos mit dem Namen IntegrationAccountSchema43.
