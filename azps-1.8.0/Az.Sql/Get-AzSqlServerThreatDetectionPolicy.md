# Get-AzSqlServerThreatDetectionPolicy

## Synopsis
Ruft die Richtlinie für die Bedrohungserkennung für einen Server ab.

## Syntax

```
Get-AzSqlServerThreatDetectionPolicy -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzSqlServerThreatDetectionPolicy** " Ruft die Richtlinie für die Bedrohungserkennung eines Azure SQL Server ab.
Um dieses Cmdlet zu verwenden, geben Sie die Parameter *ResourceGroupName* und Server *Name* an, um den Server zu identifizieren, für den dieses Cmdlet die Richtlinie abruft.

## Beispiele

### Beispiel 1: Abrufen der Bedrohungs Erkennungsrichtlinie für einen Server
```
PS C:\>Get-AzSqlServerThreatDetectionPolicy -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
ResourceGroupName            : ResourceGroup11
ServerName                   : Server01
ThreatDetectionState         : Enabled
NotificationRecipientsEmails : admin@myCompany.com
StorageAccountName           : mystorage
EmailAdmins                  : True
ExcludedDetectionTypes       : {}
RetentionInDays              : 0
```

Dieser Befehl ruft die Richtlinie für die Bedrohungserkennung für einen Server mit dem Namen SERVER01 ab.
Der Server ist der Ressourcengruppe ResourceGroup11 zugeordnet.
