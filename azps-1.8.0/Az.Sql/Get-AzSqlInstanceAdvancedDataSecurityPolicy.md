# Get-AzSqlInstanceAdvancedDataSecurityPolicy

## Synopsis
Ruft erweiterte Datensicherheitsrichtlinien einer verwalteten Instanz ab.

## Syntax

```
Get-AzSqlInstanceAdvancedDataSecurityPolicy [-InputObject <AzureSqlManagedInstanceModel>]
 -InstanceName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzSqlInstanceAdvancedDataSecurityPolicy** " retrives die erweiterte Datensicherheitsrichtlinie einer verwalteten Instanz.

## Beispiele

### Beispiel 1 – ruft die erweiterte Datensicherheit für verwaltete Instanzen ab
```powershell
PS C:\>  Get-AzSqlInstanceAdvancedDataSecurityPolicy `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### Beispiel 2 – Ruft die erweiterte Datensicherheit verwalteter Instanzen von der Ressource für verwaltete Instanzen ab
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Get-AzSqlInstanceAdvancedDataSecurityPolicy

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```
