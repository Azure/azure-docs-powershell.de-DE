# Enable-AzSqlServerAdvancedThreatProtection

## Synopsis
Ermöglicht erweiterten Bedrohungsschutz auf einem Server.

## Syntax

```
Enable-AzSqlServerAdvancedThreatProtection [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **enable-AzSqlServerAdvancedThreatProtection** ermöglicht erweiterten Bedrohungsschutz auf einem Server.

## Beispiele

### Beispiel 1: Aktivieren von erweitertem Bedrohungsschutz für Server
```powershell
PS C:\>  Enable-AzSqlServerAdvancedThreatProtection `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```

### Beispiel 2 – Aktivieren des erweiterten Bedrohungsschutzes für Server von Serverressourcen
```powershell
PS C:\>  Get-AzSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Enable-AzSqlServerAdvancedThreatProtection

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```
