# Get-AzSqlInstance

## Synopsis
Gibt Informationen zu Azure SQL-Datenbankinstanz zurück.

## Syntax

```
Get-AzSqlInstance [[-Name] <String>] [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzSqlInstance** " gibt Informationen zu mindestens einem Azure SQL-verwalteten Instanzen zurück.
Geben Sie den Namen einer Instanz an, um Informationen nur für diese Instanz anzuzeigen.
