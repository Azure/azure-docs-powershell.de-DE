# Remove-AzAutomationHybridWorkerGroup

## Synopsis
Entfernt eine Hybrid Arbeitskraft Gruppe aus Automatisierung.

## Syntax

```
Remove-AzAutomationHybridWorkerGroup [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Das Remove-AzAutomationHybridWorkerGroup-Cmdlet entfernt eine Hybrid Arbeitskraft Gruppe aus Automatisierung.

## Beispiele

### Beispiel 1
Mit diesem Befehl wird ein Hybrid Worker nach Name entfernt.

```powershell
PS C:\> Remove-AzAutomationHybridWorkerGroup -ResourceGroupName "rg1" `
                                                  -AutomationAccountName "devAccount" `
                                                  -Name "GroupName" `
                                                  -Force
```
