# Get-AzAutomationSourceControlSyncJob

## Synopsis
Ruft Synchronisierungsaufträge für die Azure Automation-Quellcodeverwaltung ab.

## Syntax

```
Get-AzAutomationSourceControlSyncJob -SourceControlName <String> [-JobId <Guid>] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet "Get-AzAutomationSourceControlSyncJob" ruft Synchronisierungsaufträge für die Azure Automation-Quellcodeverwaltung ab. Um einen bestimmten Synchronisierungsauftrag für die Quellcodeverwaltung abzurufen, geben Sie dessen ID an.

## Beispiele

### Beispiel 1
Dieser Befehl ruft alle Automatisierungs-Quellcodeverwaltungs-Synchronisierungsaufträge für das Quell Code Verwaltungs VSTSNative ab.


```powershell
PS C:\> Get-AzAutomationSourceControlSyncJob -ResourceGroupName "rg1" `
                                                  -AutomationAccountName "devAccount" `
                                                  -Name "VSTSNative"

SourceControlSyncJobId               SyncType Status StartTime           EndTime
----------------------               -------- ------ ---------           -------
08d6d266-27b6-463c-beea-bc48a67ace15 FullSync Failed 08/15/2018 09:17 AM 08/15/2018 09:18 AM
b566d564-878a-4641-8c44-25bf7850531e FullSync Failed 08/15/2018 09:09 AM 08/15/2018 09:10 AM
```

### Beispiel 2
Dieser Befehl ruft den Synchronisierungsauftrag für die Quellcodeverwaltung mit ID 08d6d266-27b6-463c-beea-bc48a67ace15 für das Quell Code Verwaltungs VSTSNative ab.
