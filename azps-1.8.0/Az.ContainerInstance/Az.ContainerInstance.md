# AZ. ContainerInstance-Modul
## Beschreibung
In diesem Thema werden Hilfethemen für die Azure Container instance-Cmdlets angezeigt.

## AZ. ContainerInstance-Cmdlets
### [Get-AzContainerGroup](Get-AzContainerGroup.md)
Ruft Container Gruppen ab.

### [Get-AzContainerInstanceLog](Get-AzContainerInstanceLog.md)
Rufen Sie die Protokolle einer Containerinstanz in einer Containergruppe ab.

### [Neu – AzContainerGroup](New-AzContainerGroup.md)
Erstellt eine Containergruppe.

### [Remove-AzContainerGroup](Remove-AzContainerGroup.md)
Entfernt eine Containergruppe.
