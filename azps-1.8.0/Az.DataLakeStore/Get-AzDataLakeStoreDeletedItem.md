# Get-AzDataLakeStoreDeletedItem

## Synopsis
Sucht nach gelöschten Einträgen im Papierkorb, die dem Filter entsprechen.

## Syntax

```
Get-AzDataLakeStoreDeletedItem [-Account] <String> [-Filter] <String> [-Count <Int32>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzDataLakeStoreDeletedItem** " durchsucht die gelöschten Dateien oder Ordner im Data Lake Store, die dem angegebenen Filter entsprechen.
Es werden die folgenden Attribute der gelöschten Elemente angezeigt: OriginalPath, TrashDirPath, Typ, Erstellungsdatum.
Dies kann ein langwieriger Vorgang sein, da er möglicherweise Millionen von Dateien im Papierkorb durchsuchen und als Job ausführen kann.
