# Get-AzRecoveryServicesAsrAlertSetting

## Synopsis
Ruft die konfigurierten Einstellungen für die Azure Site Recovery-Benachrichtigung für den Tresor ab.

## Syntax

```
Get-AzRecoveryServicesAsrAlertSetting [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzRecoveryServicesAsrAlertSetting** " Ruft die konfigurierten Azure Site Recovery-Benachrichtigungseinstellungen für den Tresor ab.

## Beispiele

### Beispiel 1
```
PS C:\> Get-AzRecoveryServicesAsrAlertSetting

CustomEmailAddress     EmailSubscriptionOwner Locale
------------------     ---------------------- ------
{abcxxxx@xxxx.com} On                     en-US
```

Benachrichtigungs-und Benachrichtigungseinstellung für Azure Site Recovery
