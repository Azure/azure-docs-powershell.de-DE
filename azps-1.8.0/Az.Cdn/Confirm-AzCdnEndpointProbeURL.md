# Confirm-AzCdnEndpointProbeURL

## Synopsis
Überprüft eine Prüf Punkt-URL.

## Syntax

```
Confirm-AzCdnEndpointProbeURL -ProbeUrl <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Confirm-AzCdnEndpointProbeURL** bestätigt, ob die bereitgestellte Prüf Punkt-URL für die dynamische Website Beschleunigung verwendet werden kann.

## Beispiele

### Beispiel 1
```powershell
PS C:\> Confirm-AzCdnEndpointProbeURL -ProbeUrl "http://www.bing.com/images"
IsValid: true
ErrorCode: None
Message:
```

Validiert die Prüf Punkt-URL " http://www.bing.com/images "
