# Remove-AzRedisCacheLink

## Synopsis
Entfernen einer Geo-Replikationsverknüpfung zwischen zwei 2-a-Caches

## Syntax

```
Remove-AzRedisCacheLink -PrimaryServerName <String> -SecondaryServerName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Entfernen einer Geo-Replikationsverknüpfung zwischen zwei 2-a-Caches

## Beispiele

### Beispiel 1: Entfernen einer Geo-Replikationsverknüpfung
```
PS C:\>Remove-AzRedisCacheLink -PrimaryServerName "mycache1" -SecondaryServerName "mycache2"
```

Dieser Befehl entfernt eine Geo-Replication-Links, wobei der mycache1-Cache mit dem Namen "Primär" und der "mycache2.
