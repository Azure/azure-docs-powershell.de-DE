# AZ. GuestConfiguration-Modul
## Beschreibung
Modul mit Cmdlets, mit denen Benutzer den Kompatibilitätszustand Ihrer virtuellen Computer in Azure basierend auf den Ergebnissen der Azure-Richtlinie verstehen können.

## AZ. GuestConfiguration-Cmdlets
### [Get-AzVMGuestPolicyStatus](Get-AzVMGuestPolicyStatus.md)
Ruft den Status der gastkonfigurations Richtlinien (detailliert) für eine Initiative des Typs "Gast Konfiguration" ab, die einem virtuellen Computer zugewiesen ist.
Eine Initiative ist eine Richtlinie des Definitions Typs "Initiative".

### [Get-AzVMGuestPolicyStatusHistory](Get-AzVMGuestPolicyStatusHistory.md)
Ruft den Kompatibilitätsstatus des Gast Konfigurationsrichtlinien für eine Initiative des Typs "Gast Konfiguration" ab, die einem virtuellen Computer zugewiesen ist.
Eine Initiative ist eine Richtlinie des Definitions Typs "Initiative".
