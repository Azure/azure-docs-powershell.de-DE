# Invoke-AzIotHubQuery

## SYNOPSIS
Fragen Sie einen IoT Hub mithilfe einer leistungsfähigen SQL sprache ab.

## SYNTAX

### ResourceSet (Standard)
```
Invoke-AzIotHubQuery [-ResourceGroupName] <String> [-IotHubName] <String> [-Query] <String> [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectSet
```
Invoke-AzIotHubQuery [-InputObject] <PSIotHub> [-Query] <String> [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdSet
```
Invoke-AzIotHubQuery [-ResourceId] <String> [-Query] <String> [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Fragen Sie einen IoT Hub mithilfe einer leistungsfähigen SQL-like-Sprache ab, um Informationen zu Geräte- und Modulmodulen, Aufträgen und Nachrichtenrouting abzurufen.
Weitere https://docs.microsoft.com/azure/iot-hub/iot-hub-devguide-query-language Informationen finden Sie hier.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Invoke-AzIotHubQuery -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Query "select * from devices"
```

Fragen Sie alle Gerätedaten in einem Azure IoT Hub ab.

### Beispiel 2
```powershell
PS C:\> Invoke-AzIotHubQuery -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Query "select * from devices.modules where devices.deviceId = 'myDevice1'" -Top 2
```

Fragen Sie Top-2-Moduldaten auf einem Zielgerät ab.
