# Get-AzIotHubKey

## SYNOPSIS
Ruft einen IotHub-Schlüssel ab.

## SYNTAX

### ResourceSet (Standard)
```
Get-AzIotHubKey [-ResourceGroupName] <String> [-Name] <String> [[-KeyName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdSet
```
Get-AzIotHubKey [-HubId] <String> [[-KeyName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Ruft einen IotHub-Schlüssel ab.
Sie können entweder alle Schlüssel auflisten oder die Liste nach einem bestimmten Schlüsselnamen filtern.

## BEISPIELE

### Beispiel 1: Alle Schlüssel erhalten
```
PS C:\> Get-AzIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Ruft alle Schlüssel für den IotHub namens "myiothub" ab.

### Beispiel 2: Informationen zu einem bestimmten Schlüssel erhalten
```
PS C:\> Get-AzIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub" -KeyName "iothubowner"
```

Ruft die Informationen für einen Schlüssel namens "iothubowner" für IotHub mit dem Namen "myiothub" ab.
