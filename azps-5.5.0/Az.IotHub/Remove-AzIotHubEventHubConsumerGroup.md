# Remove-AzIotHubEventHubConsumerGroup

## SYNOPSIS
Löscht eine Eventhub-Consumergruppe.

## SYNTAX

```
Remove-AzIotHubEventHubConsumerGroup [-ResourceGroupName] <String> [-Name] <String>
 [-EventHubConsumerGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Löscht eine Eventhub-Consumergruppe.

## BEISPIELE

### Beispiel 1 Entfernen der Eventhub-Consumergruppe aus dem Telemetrieereignishub
```
PS C:\> Remove-AzIotHubEventHubConsumerGroup -ResourceGroupName "myresourcegroup" -Name "myiothub" -EventHubConsumerGroupName myconsumergroup
```

Entfernt die Consumergruppe mit dem Namen "myconsumergroup" aus dem IotHub mit dem Namen "myiothub".
