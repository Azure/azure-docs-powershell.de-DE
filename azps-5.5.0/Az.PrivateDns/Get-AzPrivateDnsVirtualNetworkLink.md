# Get-AzPrivateDnsVirtualNetworkLink

## SYNOPSIS
Ruft eine virtuelle Netzwerkverknüpfung ab, die der angegebenen privaten DNS-Zone zugeordnet ist.

## SYNTAX

```
Get-AzPrivateDnsVirtualNetworkLink -ResourceGroupName <String> -ZoneName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzPrivateDnsVirtualNetworkLink"** ruft aus der angegebenen Ressourcengruppe virtuelle Netzwerklinks ab, die einer bestimmten privaten DNS-Zone zugeordnet sind.
Wenn Sie den *Parameter "Name"* angeben, wird ein **einzelnes "PSPrivateDnsVirtualNetworkLink"-Objekt** zurückgegeben.
Wenn Sie den Parameter *"Name"* nicht angeben, wird ein Array zurückgegeben, das alle Verknüpfungen enthält, die der Zone in der angegebenen Ressourcengruppe zugeordnet sind.
Sie können das **Objekt "PSPrivateDnsVirtualNetworkLink"** verwenden, um die Verknüpfung zu aktualisieren.
