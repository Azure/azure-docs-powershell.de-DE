# New-AzCosmosDBSqlCompositePath

## SYNOPSIS
Erstellt ein neues Objekt vom Typ "PSCompositePath". Sie kann als Parameterwert für "Set-AzCosmosDBSqlContainer" übergeben werden.

## SYNTAX

```
New-AzCosmosDBSqlCompositePath [-Path <String>] [-Order <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Objekt, das dem ZusammengesetztPath-Objekt der Sql-API entspricht.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> New-AzCosmosDBSqlCompositePath -Path "/abc" -Order Ascending

Path Order
---- -----
/abc Ascending
```
