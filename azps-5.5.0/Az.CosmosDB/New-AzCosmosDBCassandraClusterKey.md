# New-AzCosmosDBCassandraClusterKey

## SYNOPSIS
Erstellt einen neuen Clusterschlüssel aus Unterningen.

## SYNTAX

```
New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Der **Neue-AzCosmosDBCasskonfigurationsclusterKey** erstellt einen neuen Clusterschlüssel aus Demkka.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String>

Name   OrderBy
----   -------
{name}  Asc
```
