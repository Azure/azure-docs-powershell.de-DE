# New-AzDataFactoryHub

## SYNOPSIS
Erstellt einen Hub für eine Azure Data Factory.

## SYNTAX

### ByFactoryName (Standard)
```
New-AzDataFactoryHub [-Name] <String> [-File] <String> [-Force] [-DataFactoryName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByFactoryObject
```
New-AzDataFactoryHub [-Name] <String> [-File] <String> [-Force] [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "New-AzDataFactoryHub"** erstellt einen Hub für Azure Data Factory in der angegebenen Azure-Ressourcengruppe und in der angegebenen Datenfactory mit der angegebenen Dateidefinition.
Nachdem Sie den Hub erstellt haben, können Sie ihn verwenden, um verknüpfte Dienste in einer Gruppe zu speichern und zu verwalten, und Sie können dem Hub Pipelines hinzufügen.

## BEISPIELE

### Beispiel 1: Erstellen eines Hubs
```
PS C:\>New-AzDataFactoryHub -ResourceGroupName "ADFResourceGroup" -DataFactoryName "ADFDataFactory" -Name "ContosoDataHub" -File "C:\Hub.json"
```

Mit diesem Befehl wird ein Hub mit dem Namen "ContosoDataHub" in der Ressourcengruppe "ADFResourceGroup" und der Datenfactory mit dem Namen ADFDataFactory erstellt.
