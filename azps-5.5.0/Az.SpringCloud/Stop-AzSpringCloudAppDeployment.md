# Stop-AzSpringCloudAppDeployment

## SYNOPSIS
Beenden sie die Bereitstellung.

## SYNTAX

### Stopp (Standard)
```
Stop-AzSpringCloudAppDeployment -AppName <String> -Name <String> -ResourceGroupName <String>
 -ServiceName <String> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

### StopViaIdentity
```
Stop-AzSpringCloudAppDeployment -InputObject <ISpringCloudIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## BESCHREIBUNG
Beenden sie die Bereitstellung.

## BEISPIELE

### Beispiel 1: Beenden des Quell-Cloud-Diensts nach Name.
```powershell
PS C:\> Stop-AzSpringCloudAppDeployment -ResourceGroupName spring-cloud-rg -ServiceName spring-cloud-service -AppName gateway -DeploymentName default
```

Beenden Sie den Spring Cloud Service nach Name.

### Beispiel 2: Beenden des Spring Cloud Service von Pipe.
```powershell
PS C:\> Get-AzSpringCloudAppDeployment -ResourceGroupName spring-cloud-rg -ServiceName spring-cloud-service -AppName gateway -DeploymentName default | Stop-AzSpringCloud
```

Beenden Sie den Spring Cloud Service von Pipe.
