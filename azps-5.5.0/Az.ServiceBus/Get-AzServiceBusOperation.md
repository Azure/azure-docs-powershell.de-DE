# Get-AzServiceBusOperation

## SYNOPSIS
Liste unterstützter ServiceBus-Vorgänge

## SYNTAX

```
Get-AzServiceBusOperation [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzServiceBusOperation"** listet die von ServiceBus unterstützten Vorgänge auf.

## BEISPIELE

### Beispiel 1
```
PS C:\> Get-AzServiceBusOperation
```

Listet die von ServiceBus unterstützten Vorgänge auf
