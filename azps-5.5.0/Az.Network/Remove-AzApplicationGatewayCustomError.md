# Remove-AzApplicationGatewayCustomError

## SYNOPSIS
Entfernt einen benutzerdefinierten Fehler von einem Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewayCustomError -StatusCode <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzApplicationGatewayCustomError"** entfernt einen benutzerdefinierten Fehler von einem Anwendungsgateway.

## BEISPIELE

### Beispiel 1: Entfernt benutzerdefinierten Fehler von einem Anwendungsgateway
```
PS C:\> $updatedgateway = Remove-AzApplicationGatewayCustomError -ApplicationGateway $appgw -StatusCode HttpStatus502
```

Dieser Befehl entfernt den benutzerdefinierten Fehler des Http-Statuscodes 502 aus dem $appgw und gibt das aktualisierte Gateway zurück.
