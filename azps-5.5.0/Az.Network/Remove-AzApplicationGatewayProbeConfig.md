# Remove-AzApplicationGatewayProbeConfig

## SYNOPSIS
Entfernt einen Integritätstest aus einem vorhandenen Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewayProbeConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das Remove-AzApplicationGatewayProbeConfig cmdlet entfernt eine Wärmeprobe von einem vorhandenen Anwendungsgateway.

## BEISPIELE

### Beispiel 1: Entfernen einer Integritätsprobe von einem vorhandenen Anwendungsgateway
```
PS C:\>$Gateway = Remove-AzApplicationGatewayProbeConfig -ApplicationGateway Gateway -Name "Probe04"
```

Mit diesem Befehl wird der Integritätstest namens "Probe04" aus dem Anwendungsgateway namens "Gateway" entfernt.
