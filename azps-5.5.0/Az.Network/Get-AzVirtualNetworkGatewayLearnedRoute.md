# Get-AzVirtualNetworkGatewayLearnedRoute

## SYNOPSIS
Listet die von einem virtuellen Azure-Netzwerkgateway erlernten Routen auf.

## SYNTAX

```
Get-AzVirtualNetworkGatewayLearnedRoute -VirtualNetworkGatewayName <String> -ResourceGroupName <String>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Aufzählen der von einem virtuellen Azure-Netzwerkgateway erlernten Routen aus verschiedenen Quellen. Dies schließt über BGP erlernte Routen sowie statische Routen ein. 

## BEISPIELE

### Beispiel 1
```
PS C:\> Get-AzVirtualNetworkGatewayLearnedRoute -ResourceGroupName resourceGroup -VirtualNetworkGatewayname gatewayName

AsPath       :
LocalAddress : 10.1.0.254
Network      : 10.1.0.0/16
NextHop      :
Origin       : Network
SourcePeer   : 10.1.0.254
Weight       : 32768

AsPath       :
LocalAddress : 10.1.0.254
Network      : 10.0.0.254/32
NextHop      :
Origin       : Network
SourcePeer   : 10.1.0.254
Weight       : 32768

AsPath       : 65515
LocalAddress : 10.1.0.254
Network      : 10.0.0.0/16
NextHop      : 10.0.0.254
Origin       : EBgp
SourcePeer   : 10.0.0.254
Weight       : 32768
```

Ruft für das virtuelle Azure-Netzwerkgateway namens "Gatewayname" in der Ressourcengruppe "resourceGroup" Routen ab, die dem Azure-Gateway bekannt sind. In diesem Fall verfügt das virtuelle Azure-Netzwerkgateway über zwei statische Routen (10.1.0.0/16 und 10.0.0.254/32) sowie eine über BGP (10.0.0.0/16) erlernte Route.
