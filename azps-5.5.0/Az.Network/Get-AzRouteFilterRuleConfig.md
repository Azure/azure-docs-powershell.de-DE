# Get-AzRouteFilterRuleConfig

## SYNOPSIS
Ruft eine Routenfilterregel in einem Routenfilter ab.

## SYNTAX

```
Get-AzRouteFilterRuleConfig [-Name <String>] -RouteFilter <PSRouteFilter>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzRouteFilterRuleConfig"** ruft eine Routenfilterregel oder eine Liste der Routenfilterregeln in einem Routenfilter ab.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $rf = Get-AzRouteFilter -Name "MyRouteFilter" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzRouteFilterRuleConfig -RouteFilter $rf -Name "Rule01"
PS C:\> Get-AzRouteFilterRuleConfig -RouteFilter $rf
```

Der erste Befehl ruft den Routenfilter namens "MyRouteFilter" ab und speichert ihn dann in der variablen $rf.
Der zweite Befehl ruft die diesem Routenfilter zugeordnete Routenfilterregel mit dem Namen "Rule01" ab.
Der dritte Befehl ruft eine Liste der Routenfilterregeln ab, die diesem Routenfilter zugeordnet sind.
