# New-AzApplicationGatewayAutoscaleConfiguration

## SYNOPSIS
Erstellt eine Konfiguration der automatischen Skala für das Anwendungsgateway.

## SYNTAX

```
New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity <Int32> [-MaxCapacity <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "New-AzApplicationGatewayAutoscaleConfiguration"** erstellt die Autoskalenkonfiguration für ein Azure-Anwendungsgateway.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $autoscaleConfig = New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity 3
PS C:\> $gw = New-AzApplicationGateway -Name $appgwName -ResourceGroupName $rgname ..  -AutoscaleConfiguration $autoscaleConfig
```

Der erste Befehl erstellt eine Konfiguration der automatischen Skala mit einer Mindestkapazität von 3.
Der zweite Befehl erstellt ein Anwendungsgateway mit der Autoskalenkonfiguration.
