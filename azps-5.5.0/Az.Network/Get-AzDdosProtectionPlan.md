# Get-AzDdosProtectionPlan

## SYNOPSIS
Ruft einen DDoS-Schutzplan ab.

## SYNTAX

```
Get-AzDdosProtectionPlan [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das Get-AzDdosProtectionPlan erhält einen DDoS-Schutzplan.

## BEISPIELE

### Beispiel 1: Erhalten eines bestimmten DDoS-Schutzplans
```
D:\> Get-AzDdosProtectionPlan -ResourceGroupName ResourceGroupName -Name DdosProtectionPlanName

Etag              : W/"a20e5592-9b51-423b-9758-b00cd322f744"
ProvisioningState : Succeeded
VirtualNetworks   : [
                      {
                        "Id": "/subscriptions/d1dbd366-9871-45ac-84b7-fb318152a9e0/resourceGroups/ResourceGroupName/providers/Microsoft.Network/virtualNetworks/VnetName"
                      }
                    ]
```

In diesem Fall müssen wir sowohl die Attribute **"ResourceGroupName"** als auch **"Name"** angeben, die der Ressourcengruppe bzw. dem Namen des DDoS-Schutzplans entsprechen.

### Beispiel 2: Alle DDoS-Schutzpläne in einer Ressourcengruppe erhalten
```
D:\> Get-AzDdosProtectionPlan -ResourceGroupName ResourceGroupName

Etag              : W/"a20e5592-9b51-423b-9758-b00cd322f744"
ProvisioningState : Succeeded

                      {
                        "Id": "/subscriptions/d1dbd366-9871-45ac-84b7-fb318152a9e0/resourceGroups/ResourceGroupName/providers/Microsoft.Network/virtualNetworks/VnetName"
                      }
                    ]
```

In diesem Szenario geben wir nur den Parameter **"ResourceGroupName"** an, um eine Liste der DDoS-Schutzpläne zu erhalten.

Hier geben wir keine Parameter an, um eine Liste aller DDoS-Schutzpläne in einem Abonnement zu erhalten.
