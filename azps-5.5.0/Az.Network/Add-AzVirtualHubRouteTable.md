# Add-AzVirtualHubRouteTable

## SYNOPSIS
Erstellt eine Virtual Hub Route Table-Ressource, die ein untergeordnetes Teil von VirtualHub ist.

## SYNTAX

```
Add-AzVirtualHubRouteTable -Name <String> -Route <PSVirtualHubRoute[]> -Connection <String[]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Die Ressource für die Tabelle für virtuelle Hubrouten enthält die Liste der Routen und eine Liste der Verbindungen, an die sie angefügt werden kann und die zum Routen von Datenverkehr in einem virtuellen Hub verwendet wird.
