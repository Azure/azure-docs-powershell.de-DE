# Get-AzLoadBalancerFrontendIpConfig

## SYNOPSIS
Ruft eine Front-End-IP-Konfiguration in einem Lastenausgleich ab.

## SYNTAX

```
Get-AzLoadBalancerFrontendIpConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzLoadBalancerFrontendIpConfig"** ruft eine Front-End-IP-Konfiguration oder eine Liste der Front-End-IP-Konfigurationen in einem Lastenausgleich ab.

## BEISPIELE

### Beispiel 1: Erhalten einer Front-End-IP-Konfiguration in einem Lastenausgleich
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerFrontendIpConfig -Name "MyFrontEnd" -LoadBalancer $slb
```

Der erste Befehl ruft den Lastenausgleich namens "MyLoadBalancer" ab und speichert ihn dann im Variablenspeicher $slb.
Der zweite Befehl ruft die Front-End-IP-Konfiguration ab, die diesem Lastenausgleich zugeordnet ist.
