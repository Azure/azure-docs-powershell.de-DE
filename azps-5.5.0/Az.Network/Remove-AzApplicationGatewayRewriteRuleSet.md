# Remove-AzApplicationGatewayRewriteRuleSet

## SYNOPSIS
Entfernt einen Neuschreibregelsatz von einem Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewayRewriteRuleSet -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzApplicationGatewayRewriteRuleSet"** entfernt einen Neuschreibregelsatz von einem Azure-Anwendungsgateway.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayRewriteRuleSet -ApplicationGateway $AppGw -Name "RuleSet02"
```

Der erste Befehl ruft ein Anwendungsgateway ab und speichert es in der $AppGw Variable.
Mit dem zweiten Befehl wird der Neuschreibregelsatz mit dem Namen "RuleSet02" aus dem Anwendungsgateway entfernt, das in $AppGw.
