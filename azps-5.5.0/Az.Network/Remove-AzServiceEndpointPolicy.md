# Remove-AzServiceEndpointPolicy

## SYNOPSIS
Entfernt eine Dienstendpunktrichtlinie.

## SYNTAX

### RemoveByNameParameterSet (Standard)
```
Remove-AzServiceEndpointPolicy -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByResourceIdParameterSet
```
Remove-AzServiceEndpointPolicy -ResourceId <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByInputObjectParameterSet
```
Remove-AzServiceEndpointPolicy -InputObject <PSServiceEndpointPolicy> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzServiceEndpointPolicy"** entfernt eine Dienstendpunktrichtlinie.

## BEISPIELE

### Beispiel 1: Entfernt eine Dienstendpunktrichtlinie unter Verwendung des Namens
```
Remove-AzServiceEndpointPolicy -Name "Policy1" -ResourceGroup "resourcegroup1"
```

Mit diesem Befehl wird eine Dienstendpunktrichtlinie mit dem Namen "Richtlinie1" entfernt, die zu einer Ressourcengruppe mit dem Namen "resourcegroup1" gehört.

### Beispiel 2: Entfernen einer Dienstendpunktrichtlinie mithilfe eines Eingabeobjekts
```
Remove-AzServiceEndpointPolicy -InputObject $Policy1 -ResourceGroup "resourcegroup1"
```

Mit diesem Befehl wird ein Dienstendpunktrichtlinienobjekt "Policy1" entfernt, das zu einer Ressourcengruppe mit dem Namen "resourcegroup1" gehört.
