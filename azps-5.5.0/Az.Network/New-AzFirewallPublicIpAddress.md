# New-AzFirewallPublicIpAddress

## SYNOPSIS
Dies ist der Platzhalter für die Ip-Adresse, die für die Azure Firewall mit mehreren Pips verwendet werden kann.

## SYNTAX

```
New-AzFirewallPublicIpAddress [-Address <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Dies ist der Platzhalter für die Ip-Adresse, die für die Azure Firewall mit mehreren Pips verwendet werden kann.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $publicIp = New-AzFirewallPublicIpAddress -Address 20.2.3.4
```

$publicIp als Platzhalter für die ip-Adresse 20.2.3.4
