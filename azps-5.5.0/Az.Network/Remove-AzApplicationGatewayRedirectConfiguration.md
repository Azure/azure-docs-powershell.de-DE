# Remove-AzApplicationGatewayRedirectConfiguration

## SYNOPSIS
Entfernt eine Umleitungskonfiguration von einem vorhandenen Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewayRedirectConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzApplicationGatewayRedirectConfiguration"** entfernt eine Umleitungskonfiguration von einem vorhandenen Anwendungsgateway.

## BEISPIELE

### Beispiel 1
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\>$AppGw = Remove-AzApplicationGatewayRedirectConfiguration -ApplicationGateway $AppGw -Name "Redirect01"
```

Der erste Befehl ruft ein Anwendungsgateway ab und speichert es in der $AppGw Variable.
Mit dem zweiten Befehl wird die Umleitungskonfiguration "Redirect01" aus dem Anwendungsgateway entfernt, das in $AppGw.
