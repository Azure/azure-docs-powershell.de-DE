# Remove-AzApplicationGatewaySslProfile

## SYNOPSIS
Entfernt das Ssl-Profil von einem Anwendungsgateway.

## SYNTAX

```
Remove-AzApplicationGatewaySslProfile -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzApplicationGatewaySslProfile"** entfernt das Ssl-Profil aus einem Anwendungsgateway.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewaySslProfile -ApplicationGateway $AppGw -Name "SslProfile01"
```

Der erste Befehl ruft ein Anwendungsgateway namens ApplicationGateway01 ab, das zur Ressourcengruppe "ResourceGroup01" gehört, und speichert es in der $AppGw Variable. Mit dem zweiten Befehl wird das ssl-Profil mit dem Namen "SslProfile01" aus dem Anwendungsgateway entfernt, das im $AppGw.
