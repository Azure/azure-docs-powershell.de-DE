# Set-AzVirtualNetworkTap

## SYNOPSIS
Aktualisiert das Anzapfen eines virtuellen Netzwerks.

## SYNTAX

```
Set-AzVirtualNetworkTap -VirtualNetworkTap <PSVirtualNetworkTap> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Set-AzVirtualNetworkTap"** aktualisiert das Anzapfen eines virtuellen Netzwerks.

## BEISPIELE

### Beispiel 1: Konfigurieren der Anzapfung eines virtuellen Netzwerks
```
PS C:\>$vTap = Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
PS C:\>$vTap.DestinationNetworkInterfaceIPConfiguration = $newDestinationNic.IpConfigurations[0]
PS C:\>Set-AzVirtualNetworkTap -VirtualNetworkTap $vTap
```

Der Befehl aktualisiert die Ziel-IpConfiguration und aktualisiert das Anzapfen des virtuellen Netzwerks.
Wenn Tippkonfigurationen auf sie verweisen, wird nicht der sämtliche Quelldatenverkehr nach dem Update auf die neue Ziel-IP-Konfiguration gespiegelt.
