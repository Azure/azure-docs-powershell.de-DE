# Remove-AzExpressRoutePortIdentity

## SYNOPSIS
Entfernt eine Identität aus einem ExpressRoutePort.

## SYNTAX

```
Remove-AzExpressRoutePortIdentity -ExpressRoutePort <PSExpressRoutePort>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzExpressRoutePortIdentity"** entfernt die Identität eines lokalen Azure ExpressRoutePort-Objekts. Verwenden **Sie "Remove-AzExpressRoutePort",** um ihn aus ExpressRoutePort zu entfernen.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> $expressroutePort = Remove-AzExpressRoutePortIdentity -ExpressRoutePort $expressroutePort
```
