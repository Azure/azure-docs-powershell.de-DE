# Add-AzKeyVaultCertificate

## SYNOPSIS
Fügt einem Schlüsseltresor ein Zertifikat hinzu.

## SYNTAX

```
Add-AzKeyVaultCertificate [-VaultName] <String> [-Name] <String>
 [-CertificatePolicy] <PSKeyVaultCertificatePolicy> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Add-AzKeyVaultCertificate"** beginnt mit der Registrierung für ein Zertifikat in einem Schlüsseltresor im Azure Key Vault.
