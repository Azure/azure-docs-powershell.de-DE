# Remove-AzKeyVaultCertificate

## SYNOPSIS
Entfernt ein Zertifikat aus einem Schlüsseltresor.

## SYNTAX

### ByVaultNameAndName (Standard)
```
Remove-AzKeyVaultCertificate [-VaultName] <String> [-Name] <String> [-InRemovedState] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByObject
```
Remove-AzKeyVaultCertificate [-InputObject] <PSKeyVaultCertificateIdentityItem> [-InRemovedState] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzKeyVaultCertificate"** entfernt ein Zertifikat aus einem Schlüsseltresor.

## BEISPIELE

### Beispiel 1: Entfernen eines Zertifikats
```powershell
PS C:\> Remove-AzKeyVaultCertificate -VaultName "ContosoKV01" -Name "SelfSigned01" -PassThru -Force

                     [Not After]
                       10/11/2018 4:38:39 PM

                     [Thumbprint]
                       XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX

SecretId           : https://contosokv01.vault.azure.net:443/secrets/selfsigned01/968c3920884a435abf8faea11f565456
Thumbprint         : XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
RecoveryLevel      : Purgeable

```

Mit diesem Befehl wird das Zertifikat mit dem Namen "SelfSigned01" aus dem Schlüsseltresor "ContosoKV01" entfernt.
Dieser Befehl gibt den Parameter *"Force"* an.
Daher werden Sie vom Cmdlet nicht zur Bestätigung aufgefordert.

### Beispiel 2: Endgültiges Löschen des gelöschten Zertifikats aus dem Schlüsseltresor
```powershell
PS C:\> Remove-AzKeyVaultCertificate -VaultName 'Contoso' -Name 'MyCert' -InRemovedState
```

Mit diesem Befehl wird das Zertifikat mit dem Namen "MyCert" dauerhaft aus dem Schlüsseltresor namens "Contoso" entfernt.
Zum Ausführen dieses Cmdlets ist die Berechtigung zum Löschen erforderlich, die dem Benutzer in diesem Schlüsseltresor zuvor explizit erteilt worden sein muss.
