# Unregister-AzResourceProvider

## SYNOPSIS
Registrierung eines Ressourcenanbieters wird aufgehoben.

## SYNTAX

```
Unregister-AzResourceProvider -ProviderNamespace <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Mit **dem Cmdlet "Unregister-AzResourceProvider"** wird die Registrierung eines Azure-Ressourcenanbieters aufgehoben.

## BEISPIELE

### Beispiel 1: Aufheben der Registrierung des Ressourcenanbieters mit ProviderNamespace

```powershell
PS C:\>Unregister-AzResourceProvider -ProviderNamespace "Microsoft.support"
```

Mit diesem Befehl wird die Registrierung des Ressourcenanbieters "Microsoft.support" aufgehoben.
