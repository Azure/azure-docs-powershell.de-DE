# Get-AzRedisEnterpriseCacheDatabaseKey

## SYNOPSIS
Ruft die Zugriffstasten für die Datenbank "RedisEnterprise" ab.

## SYNTAX

```
Get-AzRedisEnterpriseCacheDatabaseKey -ClusterName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## BESCHREIBUNG
Ruft die Zugriffstasten für die Datenbank "RedisEnterprise" ab.

## BEISPIELE

### Beispiel 1: Erhalten von Datenbankzugriffsschlüsseln
```powershell
PS C:\> Get-AzRedisEnterpriseCacheDatabaseKey -Name "MyCache" -ResourceGroupName "MyGroup"

PrimaryKey                                   SecondaryKey
----------                                   ------------
primary-key                                  secondary-key

```

Dieser Befehl ruft die geheimen Zugriffstasten ab, die zum Authentifizieren von Verbindungen mit der Datenbank des Redis Enterprise Cache mit dem Namen "MyCache" verwendet werden.
