# Get-AzMediaService

## SYNOPSIS
Ruft Informationen zu einem Mediendienst ab.

## SYNTAX

### ResourceGroupParameterSet
```
Get-AzMediaService [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### AccountNameParameterSet
```
Get-AzMediaService [-ResourceGroupName] <String> [-AccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzMediaService"** ruft Informationen zu einem Mediendienst ab.

## BEISPIELE

### Beispiel 1: Alle Mediendienste in einer Ressourcengruppe erhalten
```
PS C:\>Get-AzMediaService -ResourceGroupName "ResourceGroup001"
```

Dieser Befehl ruft Eigenschaften für alle Mediendienste in der Ressourcengruppe mit dem Namen "ResourceGroup001" ab.

### Beispiel 2: Erhalten von Mediendiensteigenschaften
```
PS C:\>Get-AzMediaService -ResourceGroupName "ResourceGroup002" -AccountName "MediaService1"
```

Dieser Befehl ruft die Eigenschaften des Mediendiensts "MediaService1" ab, der zur Ressourcengruppe "ResourceGroup002" gehört.
