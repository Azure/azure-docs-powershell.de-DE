# Remove-AzMediaService

## SYNOPSIS
Entfernt einen Mediendienst.

## SYNTAX

```
Remove-AzMediaService [-ResourceGroupName] <String> [-AccountName] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzMediaService"** entfernt einen Mediendienst.

## BEISPIELE

### Beispiel 1: Entfernen eines Mediendiensts
```
PS C:\>Remove-AzMediaService -ResourceGroupName "ResourceGroup001" -AccountName "MediaService0011"
```

Mit diesem Befehl wird der Mediendienst mit dem Namen "MediaService0011" in der Ressourcengruppe "ResourceGroup001" entfernt.
