# Get-AzPostgreSqlConfiguration

## SYNOPSIS
Ruft Informationen zu einer Serverkonfiguration ab.

## SYNTAX

### Liste (Standard)
```
Get-AzPostgreSqlConfiguration -ResourceGroupName <String> -ServerName <String> [-SubscriptionId <String[]>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### Erhalten
```
Get-AzPostgreSqlConfiguration -Name <String> -ResourceGroupName <String> -ServerName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-AzPostgreSqlConfiguration -InputObject <IPostgreSqlIdentity> [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Ruft Informationen zu einer Serverkonfiguration ab.

## BEISPIELE

### Beispiel 1: Auflisten aller Konfigurationen in PostgreSql Server
```powershell
PS C:\> Get-AzPostgreSqlConfiguration -ResourceGroupName PostgreSqlTestRG -ServerName PostgreSqlTestServer

Name                                  Value
----                                  -----
array_nulls                           on
backslash_quote                       safe_encoding
bytea_output                          hex
check_function_bodies                 on
client_encoding                       sql_ascii
...
azure.replication_support             REPLICA
max_wal_senders                       10
max_replication_slots                 10
hot_standby_feedback                  off
logging_collector                     on
```

Dieses Cmdlet listet alle Konfigurationen im angegebenen Server von PostgreSql auf.

### Beispiel 2: Angegebene Konfiguration von PostgreSql nach Name erhalten
```powershell
PS C:\> Get-AzPostgreSqlConfiguration -Name timezone -ResourceGroupName PostgreSqlTestRG -ServerName PostgreSqlTestServer

Name     Value
----     -----
timezone UTC
```

Dieses Cmdlet wird nach Name als "PostgreSql-Konfiguration" angegeben.
