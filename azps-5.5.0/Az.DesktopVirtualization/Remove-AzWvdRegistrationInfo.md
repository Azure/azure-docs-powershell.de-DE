# Remove-AzWvdRegistrationInfo

## SYNOPSIS
Entfernen Sie die Registrierungsinformationen für den virtuellen Windows-Desktop.

## SYNTAX

```
Remove-AzWvdRegistrationInfo -HostPoolName <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## BESCHREIBUNG
Entfernen Sie die Registrierungsinformationen für den virtuellen Windows-Desktop.

## BEISPIELE

### Beispiel 1: Löschen eines Tokens für die Registrierung des virtuellen Windows-Desktops
```powershell
PS C:\> Remove-AzWvdRegistrationInfo -ResourceGroupName ResourceGroupName -HostPoolName HostPoolName
```

Mit diesem Befehl wird ein Windows Virtual Desktop Registration Token in einem Hostpool gelöscht.
