# Get-AzStorageBlobContent

## SYNOPSIS
Lädt ein Speicher-BLOB herunter.

## SYNTAX

### ReceiveManual (Standard)
```
Get-AzStorageBlobContent [-Blob] <String> [-Container] <String> [-Destination <String>] [-CheckMd5] [-Force]
 [-AsJob] [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### BlobPipeline
```
Get-AzStorageBlobContent -CloudBlob <CloudBlob> [-BlobBaseClient <BlobBaseClient>] [-Destination <String>]
 [-CheckMd5] [-Force] [-AsJob] [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>]
 [-ClientTimeoutPerRequest <Int32>] [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ContainerPipeline
```
Get-AzStorageBlobContent -CloudBlobContainer <CloudBlobContainer> [-Blob] <String> [-Destination <String>]
 [-CheckMd5] [-Force] [-AsJob] [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>]
 [-ClientTimeoutPerRequest <Int32>] [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzStorageBlobContent"** lädt das angegebene Speicher-BLOB herunter.
Wenn der BLOB-Name für den lokalen Computer nicht gültig ist, wird er von diesem Cmdlet nach Möglichkeit automatisch aufgelöst.

## BEISPIELE

### Beispiel 1: Herunterladen von BLOB-Inhalten nach Name
```
PS C:\>Get-AzStorageBlobContent -Container "ContainerName" -Blob "Blob" -Destination "C:\test\"
```

Mit diesem Befehl wird ein BLOB nach Name heruntergeladen.

### Beispiel 2: Herunterladen von BLOB-Inhalten mithilfe der Pipeline
```
PS C:\>Get-AzStorageBlob -Container containername -Blob blobname | Get-AzStorageBlobContent
```

Dieser Befehl verwendet die Pipeline zum Suchen und Herunterladen von BLOB-Inhalten.

### Beispiel 3: Herunterladen von BLOB-Inhalten mithilfe der Pipeline und eines Platzhalterzeichens
```
PS C:\>Get-AzStorageContainer container* | Get-AzStorageBlobContent -Blob "cbox.exe" -Destination "C:\test"
```

In diesem Beispiel werden das Sternchen und die Pipeline zum Suchen und Herunterladen von BLOB-Inhalten verwendet.

### Beispiel 4: Herunterladen eines BLOB-Objekts, Speichern in einer Variablen und anschließendes Herunterladen von BLOB-Inhalt mit dem BLOB-Objekt
```
PS C:\>$blob = Get-AzStorageBlob -Container containername -Blob blobname 
PS C:\>Get-AzStorageBlobContent -CloudBlob $blob.ICloudBlob -Destination "C:\test"
```

In diesem Beispiel wird zuerst ein BLOB-Objekt heruntergeladen und in einer Variablen gespeichert, und dann wird der BLOB-Inhalt mit dem BLOB-Objekt heruntergeladen.
