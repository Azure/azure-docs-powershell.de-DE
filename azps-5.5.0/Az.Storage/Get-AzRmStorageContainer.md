# Get-AzRmStorageContainer

## SYNOPSIS
Ruft Speicher-BLOB-Container ab oder listet diese auf.

## SYNTAX

### AccountName (Standard)
```
Get-AzRmStorageContainer [-ResourceGroupName] <String> [-StorageAccountName] <String> [-Name <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### AccountObject
```
Get-AzRmStorageContainer -StorageAccount <PSStorageAccount> [-Name <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzRmStorageContainer"** ruft Speicher-BLOB-Container ab oder listet diese auf.

## BEISPIELE

### Beispiel 1: Erhalten eines Speicher-BLOB-Containers mit dem Namen des Speicherkontos und dem Containernamen
```
PS C:\>Get-AzRmStorageContainer -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount" -ContainerName "myContainer"
```

Dieser Befehl ruft einen Speicher-BLOB-Container mit dem Namen des Speicherkontos und dem Containernamen ab.

### Beispiel 2: Auflisten aller Speicher-BLOB-Container eines Speicherkontos
```
PS C:\>Get-AzRmStorageContainer -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount"
```

Dieser Befehl listet alle Speicher-BLOB-Container eines Speicherkontos mit dem Namen des Speicherkontos auf.

### Beispiel 3: Einen Speicher-BLOB-Container mit Speicherkontoobjekt und Containername erhalten.
```
PS C:\>$accountObject = Get-AzStorageAccount -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount"
PS C:\>Get-AzRmStorageContainer -StorageAccount $accountObject -ContainerName "myContainer"
```

Dieser Befehl ruft einen Speicher-BLOB-Container mit Speicherkontoobjekt und Containername ab.
