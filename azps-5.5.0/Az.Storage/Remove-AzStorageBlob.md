# Remove-AzStorageBlob

## SYNOPSIS
Entfernt das angegebene Speicher-BLOB.

## SYNTAX

### NamePipeline (Standard)
```
Remove-AzStorageBlob [-Blob] <String> [-Container] <String> [-DeleteSnapshot] [-SnapshotTime <DateTimeOffset>]
 [-VersionId <String>] [-Force] [-PassThru] [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>]
 [-ClientTimeoutPerRequest <Int32>] [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### BlobPipeline
```
Remove-AzStorageBlob -CloudBlob <CloudBlob> [-BlobBaseClient <BlobBaseClient>] [-DeleteSnapshot] [-Force]
 [-PassThru] [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ContainerPipeline
```
Remove-AzStorageBlob -CloudBlobContainer <CloudBlobContainer> [-Blob] <String> [-DeleteSnapshot]
 [-SnapshotTime <DateTimeOffset>] [-VersionId <String>] [-Force] [-PassThru] [-Context <IStorageContext>]
 [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzStorageBlob"** entfernt das angegebene BLOB aus einem Speicherkonto in Azure.

## BEISPIELE

### Beispiel 1: Entfernen eines Speicher-BLOB nach Name
```
PS C:\>Remove-AzStorageBlob -Container "ContainerName" -Blob "BlobName"
```

Mit diesem Befehl wird ein blob entfernt, das durch seinen Namen identifiziert wird.

### Beispiel 2: Entfernen eines Speicher-BLOB mithilfe der Pipeline
```
PS C:\>Get-AzStorageBlob -Container "ContainerName" -Blob "BlobName" | Remove-AzStorageBlob
```

Dieser Befehl verwendet die Pipeline.

### Beispiel 3: Entfernen von Speicher-BLOBs mithilfe der Pipeline
```
PS C:\>Get-AzStorageContainer -Container container* | Remove-AzStorageBlob -Blob "BlobName"
```

Dieser Befehl verwendet das Platzhalterzeichen Sternchen (*) und die Pipeline, um das BLOB oder die BLOB abzurufen, und entfernt diese dann.

### Beispiel 4: Entfernen einer einzelnen BLOB-Version
```
PS C:\> Remove-AzStorageBlob -Container "containername" -Blob blob2 -VersionId "2020-07-03T16:19:16.2883167Z"
```

Mit diesem Befehl wird ein einzelner BLOBS-Verion mit VersionId entfernt.

### Beispiel 5: Entfernen einer einzelnen BLOB-Momentaufnahme
```
PS C:\> Remove-AzStorageBlob -Container "containername" -Blob blob1 -SnapshotTime "2020-07-06T06:56:06.8588431Z"
```

Mit diesem Befehl wird eine einzelne Blobs-Momentaufnahme mit SnapshotTime entfernt.
