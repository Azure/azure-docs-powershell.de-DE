# Remove-AzRmStorageContainerLegalHold

## SYNOPSIS
Entfernt juristische Haltetags aus einem Speicher-BLOB-Container.

## SYNTAX

### AccountName (Standard)
```
Remove-AzRmStorageContainerLegalHold [-ResourceGroupName] <String> [-StorageAccountName] <String>
 -Name <String> -Tag <String[]> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### AccountObject
```
Remove-AzRmStorageContainerLegalHold -Name <String> -StorageAccount <PSStorageAccount> -Tag <String[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ContainerObject
```
Remove-AzRmStorageContainerLegalHold -Container <PSContainer> -Tag <String[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Mit **dem Cmdlet "Remove-AzRmStorageContainerLegalHold"** werden gesetzliche Aufbewahrungstags aus einem Speicher-BLOB-Container entfernt.

## BEISPIELE

### Beispiel 1: Entfernen von Markierungen für den gesetzlichen Halteraum aus einem Speicher-BLOB-Container mit dem Namen des Speicherkontos und dem Containernamen
```
PS C:\>Remove-AzRmStorageContainerLegalHold -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount" -ContainerName "myContainer" -Tag  tag1
```

Mit diesem Befehl werden gesetzliche Aufbewahrungstags aus einem Speicher-BLOB-Container mit dem Namen des Speicherkontos und dem Containernamen entfernt.

### Beispiel 2: Entfernen gesetzlicher Haltetags aus einem Speicher-BLOB-Container mit Speicherkontoobjekt und Containername
```
PS C:\>$accountObject = Get-AzStorageAccount -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount"
PS C:\>Remove-AzRmStorageContainerLegalHold -StorageAccount $accountObject -ContainerName "myContainer"  -Tag  tag1,tag2
```

Mit diesem Befehl werden gesetzliche Aufbewahrungstags aus einem Speicher-BLOB-Container mit Speicherkontoobjekt und Containername entfernt.

### Beispiel 3: Entfernen gesetzlicher Haltetags aus allen Speicher-BLOB-Containern in einem Speicherkonto mit Pipeline
```
PS C:\>Get-AzStorageContainer -ResourceGroupName "myResourceGroup" -AccountName "myStorageAccount" | Remove-AzRmStorageContainerLegalHold -Tag  tag1
```

Dieser Befehl entfernt Markierungen für den gesetzlichen Halteraum aus allen Speicher-BLOB-Containern in einem Speicherkonto mit Pipeline.
