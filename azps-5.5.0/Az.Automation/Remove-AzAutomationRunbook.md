# Remove-AzAutomationRunbook

## SYNOPSIS
Entfernt ein Runbook.

## SYNTAX

```
Remove-AzAutomationRunbook [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzAutomationRunbook"** entfernt ein Runbook aus der Azure-Automatisierung.

## BEISPIELE

### Beispiel 1: Entfernen eines Runbooks
```
PS C:\>Remove-AzAutomationRunbook -AutomationAccountName "Contoso17" -Name "Runbook03" -ResourceGroupName "ResourceGroup01"
```

Mit diesem Befehl wird das Runbook "Runbook03" im Azure -Automatisierungskonto namens "Contoso17" entfernt.
