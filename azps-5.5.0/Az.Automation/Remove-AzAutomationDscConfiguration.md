# Remove-AzAutomationDscConfiguration

## SYNOPSIS
Entfernt die DSC-Konfigurationen aus der Automatisierung.

## SYNTAX

```
Remove-AzAutomationDscConfiguration [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzAutomationDscConfiguration"** entfernt konfigurationen von APS Desired State Configuration (DSC) aus der Azure-Automatisierung.

## BEISPIELE

### Beispiel 1

Entfernt die DSC-Konfigurationen aus der Automatisierung. (automatisch generiert)

<!-- Aladdin Generated Example -->
```powershell
Remove-AzAutomationDscConfiguration -AutomationAccountName 'AutomationAccount01' -Name 'Configuration01' -ResourceGroupName myresourcegroup
```
