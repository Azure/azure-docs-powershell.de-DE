# Get-AzStreamAnalyticsTransformation

## SYNOPSIS
Ruft Informationen zu einer Stream Analytics-Auftragstransformation ab.

## SYNTAX

```
Get-AzStreamAnalyticsTransformation [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzStreamAnalyticsTransformation"** ruft Informationen zu einer Transformation ab, die in einem Stream Analytics-Auftrag definiert wurde.

## BEISPIELE

### BEISPIEL 1: Informationen zu einer Datenstromanalysetransformation
```
PS C:\>Get-AzStreamAnalyticsTransformation -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "StreamingJob"
```

Dieser Befehl gibt Informationen zur Transformation "StreamingJob" für den Auftrag "StreamingJob" zurück.
