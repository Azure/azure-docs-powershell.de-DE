# Get-AzStreamAnalyticsOutput

## SYNOPSIS
Ruft die in einem angegebenen Stream Analytics-Auftrag oder einer bestimmten Ausgabe definierten Ausgaben ab.

## SYNTAX

```
Get-AzStreamAnalyticsOutput [-JobName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzStreamAnalyticsOutput"** listet alle Ausgaben auf, die in einem angegebenen Stream Analytics-Auftrag definiert sind oder Informationen zu einer bestimmten Ausgabe erhalten.

## BEISPIELE

### Beispiel 1: Informationen zu Auftragsausgabedaten
```powershell
PS C:\>Get-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob"
```

Dieser Befehl gibt Informationen zu den Im Auftrag von StreamingJob definierten Ausgaben zurück.

### Beispiel 2: Erhalten von Informationen zu einer bestimmten Auftragsausgabe
```powershell
PS C:\>Get-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "Output"
```

Dieser Befehl gibt Informationen über die Ausgabe mit dem Namen "Output" zurück, die im Auftrag von StreamingJob definiert ist.
