# Resume-AzAnalysisServicesServer

## SYNOPSIS
Setzt eine Instanz des Analysis Services Server fort.

## SYNTAX

```
Resume-AzAnalysisServicesServer [[-ResourceGroupName] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das Resume-AzAnalysisServicesServer cmdlet setzt eine Instanz des Analysis Services Server fort.

## BEISPIELE

### Beispiel 1
```
PS C:\> Resume-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Dieser Befehl setzt einen angehaltenen Server namens "testserver" in der Testgruppe der Ressourcengruppe fort.
