# Start-AzVmssRollingOSUpgrade

## SYNOPSIS
Startet ein rollübergreifendes Upgrade, um alle Skalierungssatzinstanzen für virtuelle Computer auf die neueste verfügbare Version des Betriebssystems für Plattformimages zu verschieben.

## SYNTAX

```
Start-AzVmssRollingOSUpgrade [-ResourceGroupName] <String> [-VMScaleSetName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Startet ein rollübergreifendes Upgrade, um alle Skalierungssatzinstanzen für virtuelle Computer auf die neueste verfügbare Version des Betriebssystems für Plattformimages zu verschieben.
Instanzen, auf denen bereits die neueste verfügbare Betriebssystemversion ausgeführt wird, sind nicht betroffen.

## BEISPIELE

### Beispiel 1
```
PS C:\> Start-AzVmssRollingOSUpgrade -ResourceGroupName "Group001" -VMScaleSetName "VMSS001"
```

Dieser Befehl startet ein rolliertes Upgrade aller vm-Instanzen des VM-Skalierungs-Set "VMSS001" in der Ressourcengruppe "Group001".
