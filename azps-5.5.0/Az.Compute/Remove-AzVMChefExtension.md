# Remove-AzVMChefExtension

## SYNOPSIS
Entfernt die "Chef"-Erweiterung von einem virtuellen Computer.

## SYNTAX

### Linux
```
Remove-AzVMChefExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>] [-Linux] [-NoWait]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Windows
```
Remove-AzVMChefExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>] [-Windows]
 [-NoWait] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzVM VirtualExtension"** entfernt die Erweiterung "Chef" von einem virtuellen Computer.

## BEISPIELE

### Beispiel 1: Entfernen einer Erweiterung "Chef" von einem virtuellen Windows-Computer
```
PS C:\> Remove-AzVMChefExtension -ResourceGroupName "ResourceGroup001" -VMName "WindowsVM001" -Windows
```

Mit diesem Befehl wird eine "Chef"-Erweiterung von einem windows-basierten virtuellen Computer namens "WindowsVM001" entfernt, der zur Ressourcengruppe "ResourceGroup001" gehört.

### Beispiel 2: Entfernen einer Erweiterung "Chef" von einem virtuellen Linux-Computer
```
PS C:\> Remove-AzVMChefExtension -ResourceGroupName "ResourceGroup002" -VMName "LinuxVM001" -Linux
```

Mit diesem Befehl wird eine "Chef"-Erweiterung von einem Linux-basierten virtuellen Computer namens LinuxVM001 entfernt, der zur Ressourcengruppe "ResourceGroup002" gehört.
