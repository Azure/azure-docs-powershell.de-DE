# Get-AzVmss

## SYNOPSIS
Ruft die Eigenschaften eines VMSS ab.

## SYNTAX

### Standardparameter (Standard)
```
Get-AzVmss [[-ResourceGroupName] <String>] [[-VMScaleSetName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### FriendMethod
```
Get-AzVmss [[-ResourceGroupName] <String>] [[-VMScaleSetName] <String>] [-InstanceView]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### OSUpgradeHistoryMethodParameter
```
Get-AzVmss [[-ResourceGroupName] <String>] [[-VMScaleSetName] <String>] [-OSUpgradeHistory]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzVmss-Cmdlet** ruft die Modell- und Instanzansicht eines Virtual Machine Scale Set (VMSS) ab.
Die Modellansicht ist die vom Benutzer angegebenen Eigenschaften des Skalierungssets für den virtuellen Computer.
Die Instanzansicht ist der Status der Instanzebene des Skalierungssets für den virtuellen Computer.
Geben Sie den *Parameter "InstanceView"* an, um nur die Instanzansicht eines Skalierungssets für einen virtuellen Computer zu erhalten.

### Beispiel 2: Alle Vms in einer Ressourcengruppe erhalten
```
PS C:\> Get-AzVmss -ResourceGroupName "Group001"

ResourceGroupName                               Name       Location             Sku Capacity ProvisioningState
-----------------                               ----       --------             --- -------- -----------------
Group001                                       VMSS001      eastus Standard_DS1_v2        2         Succeeded
Group001                                       VMSS002      eastus     Standard_A1        2         Succeeded
```

Get all Vmss in resource group "Group001"

### Beispiel 3: Alle Vms in einem Abonnement erhalten
```
PS C:\> Get-AzVmss

ResourceGroupName                               Name       Location             Sku Capacity ProvisioningState
-----------------                               ----       --------             --- -------- -----------------
Group001                                       VMSS001      eastus Standard_DS1_v2        2         Succeeded
Group001                                       VMSS002      eastus     Standard_A1        2         Succeeded
Group002                                       VMSS003      eastus     Standard_A1        1         Succeeded
Group002                                       VMSS004      eastus Standard_DS1_v2        2         Succeeded
```

Holen Sie sich alle Vmss im Abonnement.

### Beispiel 4: "Get all Vmss using filtering" (Alle Vmss mithilfe der Filterung erhalten)
```
PS C:\> Get-AzVmss -Name VMSS00*

ResourceGroupName                               Name       Location             Sku Capacity ProvisioningState
-----------------                               ----       --------             --- -------- -----------------
Group001                                       VMSS001      eastus Standard_DS1_v2        2         Succeeded

Group002                                       VMSS003      eastus     Standard_A1        1         Succeeded
Group002                                       VMSS004      eastus Standard_DS1_v2        2         Succeeded
```

Holen Sie sich alle Vmss im Abonnement, die mit "VMSS00" beginnen.
