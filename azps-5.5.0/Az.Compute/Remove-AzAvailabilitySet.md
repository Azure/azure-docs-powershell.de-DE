# Remove-AzAvailabilitySet

## SYNOPSIS
Entfernt einen Verfügbarkeitssatz aus Azure.

## SYNTAX

```
Remove-AzAvailabilitySet [-ResourceGroupName] <String> [[-Name] <String>] [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzAvailabilitySet"** entfernt einen Verfügbarkeitssatz aus Azure.

## BEISPIELE

### Beispiel 1: Entfernen eines Verfügbarkeitssets
```
PS C:\> Remove-AzAvailabilitySet -Name "AvailabilitySet03" -ResourceGroupName "ResourceGroup11"
```

Mit diesem Befehl wird ein Verfügbarkeitssatz namens "AvailabilitySet03" in der Ressourcengruppe "ResourceGroup11" entfernt.
