# Remove-AzSnapshot

## SYNOPSIS
Entfernt eine Momentaufnahme.

## SYNTAX

```
Remove-AzSnapshot [-ResourceGroupName] <String> [-SnapshotName] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzSnapshot"** entfernt eine Momentaufnahme.

## BEISPIELE

### Beispiel 1
```
PS C:\> Remove-AzSnapshot -ResourceGroupName 'ResourceGroup01' -SnapshotName 'Snapshot01' -Force;
```

Mit diesem Befehl wird die Momentaufnahme mit dem Namen "Snapshot01" in der Ressourcengruppe "ResourceGroup01" entfernt.
