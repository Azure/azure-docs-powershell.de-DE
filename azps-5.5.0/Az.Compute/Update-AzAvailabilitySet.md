# Update-AzAvailabilitySet

## SYNOPSIS
Aktualisiert einen Verfügbarkeitssatz.

## SYNTAX

```
Update-AzAvailabilitySet [-AvailabilitySet] <PSAvailabilitySet> [[-Sku] <String>]
 [-ProximityPlacementGroupId <String>] [-Tag <Hashtable>] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Update-AzAvailabilitySet"** aktualisiert einen Verfügbarkeitssatz.

## BEISPIELE

### Beispiel 1
```
PS C:\> Get-AzAvailabilitySet -ResourceGroupName 'ResourceGroup01' -Name 'AvSet01' | Update-AzAvailabilitySet -Managed;
```

Mit diesem Befehl wird der Verfügbarkeitssatz "AvSet01" in der Ressourcengruppe mit dem Namen "ResourceGroup01" auf einen verwalteten Verfügbarkeitssatz aktualisiert.
