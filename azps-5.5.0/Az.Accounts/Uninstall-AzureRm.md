# Uninstall-AzureRm

## SYNOPSIS
Entfernt alle AzureRm-Module von einem Computer.

## SYNTAX

```
Uninstall-AzureRm [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Entfernt alle AzureRm-Module von einem Computer.

## BEISPIELE

### Beispiel 1
```
PS C:\> Uninstall-AzureRm
```

Durch Ausführen dieses Befehls werden alle AzureRm-Module vom Computer für die Version von PowerShell entfernt, in der das Cmdlet ausgeführt wird.
