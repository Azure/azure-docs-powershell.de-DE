# Get-AzDataBoxJob

## SYNOPSIS
Ruft Informationen zu Databox-Aufträgen ab.

## SYNTAX

### ListParameterSet (Standard)
```
Get-AzDataBoxJob [-ResourceGroupName <String>] [-Completed] [-CompletedWithError] [-Cancelled] [-Aborted]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByNameParameterSet
```
Get-AzDataBoxJob -ResourceGroupName <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzDataBoxJob -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzDataBoxJobs"** ruft Informationen zu Databoxaufträgen in einem Azure-Abonnement ab.
Wenn Sie die Ressourcengruppe angeben, ruft dieses Cmdlet alle Databoxaufträge unter dieser Ressourcengruppe ab. Wenn Sie den Namen des Auftrags zusammen mit dem Namen der Ressourcengruppe angeben, ruft dieses Cmdlet Informationen zu diesem bestimmten Datenfeldauftrag ab.
Wenn Sie nichts anderes als die Abonnement-ID angeben, ruft dieses Cmdlet Informationen zu allen Databoxaufträgen unter diesem Abonnement ab.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Get-AzDataBoxJob

jobResource.Name        jobResource.Sku.Name jobResource.Status  jobResource.StartTime jobResource.Location ResourceGroup
----------------        -------------------- ------------------  --------------------- -------------------- -------------
testtip2                DataBox              Cancelled           10-09-2018 06:34:53   westus               TestRg1
cleanbox                DataBox              Aborted             04-12-2018 16:07:41   westus               TestRg2
cleanbox-Clone          DataBox              Cancelled           25-04-2019 11:31:36   westus               TestRg2
.
.
.
```

Get-AzDataBoxJob ohne Parameter ruft alle Databoxaufträge unter dem Abonnement ab.

### Beispiel 2
```powershell
PS C:\> Get-AzDataBoxJob -ResourceGroupName TestRg1

jobResource.Name        jobResource.Sku.Name jobResource.Status  jobResource.StartTime jobResource.Location ResourceGroup
----------------        -------------------- ------------------  --------------------- -------------------- -------------
testtip2                DataBox              Cancelled           10-09-2018 06:34:53   westus               TestRg1
.
.
.
```

Get-AzDataBoxJob mit dem Parameter "ResourceGroupName" ruft alle Databoxaufträge unter der angegebenen Ressourcengruppe ab.

### Beispiel 3
```powershell
PS C:\> Get-AzDataBoxJob -ResourceGroupName TestRg1 -Name testtip2

jobResource.Name        jobResource.Sku.Name jobResource.Status  jobResource.StartTime jobResource.Location ResourceGroup
----------------        -------------------- ------------------  --------------------- -------------------- -------------
testtip2                DataBox              Cancelled           10-09-2018 06:34:53   westus               TestRg1

```

Get-AzDataBoxJob mit "ResourceGroupName" und "Name" angegeben werden, wird dieser spezifische Datenfeldauftrag abgerufen.

### Beispiel 4
```powershell
PS C:\> Get-AzDataBoxJob -ResourceId "/subscriptions/05b5dd1c-793d-41de-be9f-6f9ed142f695/resourceGroups/TestRg1/providers/Microsoft.DataBox/jobs/testtip2"

jobResource.Name        jobResource.Sku.Name jobResource.Status  jobResource.StartTime jobResource.Location ResourceGroup
----------------        -------------------- ------------------  --------------------- -------------------- -------------
testtip2                DataBox              Cancelled           10-09-2018 06:34:53   westus               TestRg1

```

Get-AzDataBoxJob angegebenen Wert "ResourceId" den jeweiligen Datenfeldauftrag abruft
