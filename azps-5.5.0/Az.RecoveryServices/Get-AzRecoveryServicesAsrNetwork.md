# Get-AzRecoveryServicesAsrNetwork

## SYNOPSIS
Ruft Informationen zu den Netzwerken ab, die von der Websitewiederherstellung f端r den aktuellen Tresor verwaltet werden.

## SYNTAX

### ByFabricObject (Standard)
```
Get-AzRecoveryServicesAsrNetwork -Fabric <ASRFabric> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByName
```
Get-AzRecoveryServicesAsrNetwork -Fabric <ASRFabric> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByFriendlyName
```
Get-AzRecoveryServicesAsrNetwork -Fabric <ASRFabric> -FriendlyName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzRecoveryServicesAsrNetwork"** ruft Informationen zu Azure Site Recovery Networks f端r den aktuellen Azure Site Recovery Vault ab.

## BEISPIELE

### Beispiel 1
```
PS C:\> $Networks = Get-AzRecoveryServicesAsrNetwork -Fabric $Fabric
```

Gets all known networks in the specified fabric.
