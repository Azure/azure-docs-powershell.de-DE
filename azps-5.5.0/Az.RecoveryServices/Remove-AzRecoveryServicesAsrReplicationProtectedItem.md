# Remove-AzRecoveryServicesAsrReplicationProtectedItem

## SYNOPSIS
Beendet/deaktiviert die Replikation für ein geschütztes Element der Azure-Websitewiederherstellungsreplikation.

## SYNTAX

```
Remove-AzRecoveryServicesAsrReplicationProtectedItem -InputObject <ASRReplicationProtectedItem>
 [-WaitForCompletion] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzRecoveryServicesAsrReplicationProtectedItem"** deaktiviert die Replikation des angegebenen geschützten Azure Site Recovery-Replikationselements.
Dieser Vorgang bewirkt, dass die Replikation für das geschützte Element beendet wird.

## BEISPIELE

### Beispiel 1
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrReplicationProtectedItem -ReplicationProtectedItem $ReplicationProtectedItem
```

Startet den Replikationsvorgang zum Deaktivieren für das angegebene replikationsgeschützte Element und gibt den ASR-Auftrag zurück, mit dem der Vorgang nachverfolgt wird.
