# Get-AzRedisCacheKey

## SYNOPSIS
Ruft die Zugriffstasten für einen Cache von Redis ab.

## SYNTAX

```
Get-AzRedisCacheKey [-ResourceGroupName <String>] -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzRedisCacheKey"** ruft die Zugriffsschlüssel für einen Azure Redis Cache ab.

## BEISPIELE

### Beispiel 1: Erhalten der Tastenkombinationen für einen Redis-Cache
```
PS C:\>Get-AzRedisCacheKey -ResourceGroupName "MyResourceGroup" -Name "MyCacheKey"
PrimaryKey        : pJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=
SecondaryKey      : sJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=
```

Dieser Befehl ruft die Zugriffstasten namens "MyCacheKey" ab.
