# Stop-AzHDInsightJob

## SYNOPSIS
Beendet einen angegebenen ausgeführten Auftrag für einen Cluster.

## SYNTAX

```
Stop-AzHDInsightJob [-ClusterName] <String> [-JobId] <String> [-HttpCredential] <PSCredential>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Stop-AzHDInsightJob"** beendet einen angegebenen laufenden Auftrag in einem Azure HDInsight-Cluster.

## BEISPIELE

### Beispiel 1: Beenden eines Auftrags für den angegebenen Cluster
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential
PS C:\> Stop-AzHDInsightJob `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds `
            -JobId $jobId
```

Dieser Befehl beendet einen Auftrag für den Cluster namens "Ihr-Hadoop-001".
