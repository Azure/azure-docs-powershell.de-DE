# Remove-AzHDInsightPersistedScriptAction

## SYNOPSIS
Entfernt eine Aktion für beibehaltene Skripts aus einem HDInsight-Cluster.

## SYNTAX

```
Remove-AzHDInsightPersistedScriptAction [-ClusterName] <String> [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzHDInsightPersistedScriptAction"** entfernt eine Aktion für beibehaltenes Skript aus der Liste der beibehaltenen Skriptaktionen des angegebenen Azure HDInsight-Cluster.
Das entfernte Skript wird nicht mehr ausgeführt, wenn der Cluster skaliert wird.

## BEISPIELE

### Beispiel 1: Entfernen einer Skriptaktion aus der Liste der beibehaltenen Skriptaktionen in einem Cluster
```
PS C:\>Remove-AzHDInsightPersistedScriptAction `
            -ClusterName "your-hadoop-001" `
            -Name "Scriptaction"
```

Mit diesem Befehl wird die Skriptaktion "Scriptaction" aus der Liste der beibehaltenen Skriptaktionen für den angegebenen Cluster entfernt.
