# Get-AzDtlVMsPerLabPolicy

## SYNOPSIS
Ruft die Richtlinie für virtuelle Computer pro Übungseinheit einer Übungseinheit in DevTest Labs ab.

## SYNTAX

```
Get-AzDtlVMsPerLabPolicy [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzDtlVMsPerLabPolicy"** ruft die Richtlinie für virtuelle Computer pro Übungseinheit ab, mit der Sie die Gesamtzahl der in einem Labor zulässigen virtuellen Computer festlegen können.
Das Cmdlet gibt den Status "Aktiviert" oder "Deaktiviert" der Richtlinie sowie die Gesamtzahl der in der Übungseinheit zulässigen virtuellen Computer zurück, die Sie in der Richtlinie festgelegt haben.
