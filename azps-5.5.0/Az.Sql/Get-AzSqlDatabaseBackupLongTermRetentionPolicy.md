# Get-AzSqlDatabaseBackupLongTermRetentionPolicy

## SYNOPSIS
Ruft eine langfristige Datenbankaufbewahrungsrichtlinie ab.

## SYNTAX

```
Get-AzSqlDatabaseBackupLongTermRetentionPolicy [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzSqlDatabaseBackupLongTermRetentionPolicy"** ruft die langfristige Aufbewahrungsrichtlinie ab, die in dieser Datenbank registriert ist.
Die Richtlinie ist eine Azure-Sicherungsressource, die zum Definieren der Sicherungsspeicherrichtlinie verwendet wird.

## BEISPIELE

### Beispiel 1: Erhalten der aktuellen Version der langfristigen Aufbewahrungsrichtlinie
```powershell
PS C:\> Get-AzSqlDatabaseBackupLongTermRetentionPolicy -ResourceGroupName resourcegroup01 -ServerName server01 -DatabaseName database01


ResourceGroupName                      : resourcegroup01
ServerName                             : server01
DatabaseName                           : database01
WeeklyRetention                        : P2W
MonthlyRetention                       : PT0S
YearlyRetention                        : PT0S
WeekOfYear                             : 0
State                                  :
RecoveryServicesBackupPolicyResourceId :
Location                               :
```

Dieser Befehl ruft die aktuelle Version der langfristigen Aufbewahrungsrichtlinie für Datenbank01 ab.
