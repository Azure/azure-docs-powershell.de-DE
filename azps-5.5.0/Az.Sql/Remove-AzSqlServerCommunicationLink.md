# Remove-AzSqlServerCommunicationLink

## SYNOPSIS
Löscht eine Kommunikationsverknüpfung für Datenbanktransaktionen zwischen zwei Servern.

## SYNTAX

```
Remove-AzSqlServerCommunicationLink [-LinkName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzSqlServerCommunicationLink"** löscht eine Server-zu-Server-Kommunikationsverknüpfung für Datenbanktransaktionen in Azure SQL Database.

## BEISPIELE

### Beispiel 1: Löschen einer Kommunikationsverknüpfung
```
PS C:\>Remove-AzSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17" -LinkName "Link01"
```

Mit diesem Befehl wird ein Server-zu-Server-Kommunikationslink namens Link01 auf ContosoServer17 gelöscht.
