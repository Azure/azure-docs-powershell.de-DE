# Get-AzSqlElasticPoolRecommendedAction

## SYNOPSIS
Ruft mindestens eine empfohlene Aktion für einen Azure SQL Poolratgeber ab.

## SYNTAX

```
Get-AzSqlElasticPoolRecommendedAction [-RecommendedActionName <String>] -ServerName <String>
 -ElasticPoolName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzSqlElasticPoolRecommendedAction"** ruft eine oder mehrere empfohlene Aktionen für einen Azure SQL Poolratgeber ab.
