# New-AzSqlServerDnsAlias

## SYNOPSIS
Mit diesem Befehl wird ein neuer Azure SQL Server-DNS-Alias erstellt.

## SYNTAX

```
New-AzSqlServerDnsAlias -Name <String> [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Erstellt einen neuen Azure SQL Server-DNS-Alias, der auf den angegebenen Server verweisen soll.

## BEISPIELE

### Beispiel 1
```
PS C:\> $serverDNSAlias = New-AzSqlServerDnsAlias -ResourceGroupName rg -ServerName serverName -DnsAliasName aliasName

ResourceGroupName  ServerName   DnsAliasName
-----------------  ----------   ------------------
rgname             servername   dnsaliasname
```

Mit diesem Befehl wird ein Azure SQL Server-DNS-Alias mit dem Namen "aliasName" erstellt, der auf den Serverservernamen verweisen soll.
