# Enable-AzSqlInstanceAdvancedDataSecurity

## SYNOPSIS
Aktiviert Advanced Data Security für eine verwaltete Instanz.

## SYNTAX

```
Enable-AzSqlInstanceAdvancedDataSecurity [-DoNotConfigureVulnerabilityAssessment] [-AsJob]
 [-DeploymentName <String>] [-InputObject <AzureSqlManagedInstanceModel>] -InstanceName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet Enable-AzSqlInstanceAdvancedDataSecurity** aktiviert Advanced Data Security für eine verwaltete Instanz. Advanced Data Security ist ein einheitliches Sicherheitspaket mit Datenklassifizierung, Sicherheitsrisikobewertung und Advanced Threat Protection für Ihre verwaltete Instanz. (Es wird automatisch ein neues Speicherkonto erstellt, um Sicherheitsrisikobewertungen zu sparen. Wenn zu diesem Zweck zuvor ein Speicherkonto erstellt wurde, wird es stattdessen verwendet.)

## BEISPIELE

### Beispiel 1: Aktivieren der verwalteten Instanz Advanced Data Security
```powershell
PS C:\>  Enable-AzSqlInstanceAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" 

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### Beispiel 2: Aktivieren der verwalteten Instanz Advanced Data Security von einer Serverressource
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Enable-AzSqlInstanceAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### Beispiel 3

Aktiviert Advanced Data Security für eine verwaltete Instanz. (automatisch generiert)

```powershell
<!-- Aladdin Generated Example --> 
Enable-AzSqlInstanceAdvancedDataSecurity -DoNotConfigureVulnerabilityAssessment -InstanceName 'ContosoManagedInstanceName' -ResourceGroupName MyResourceGroup
```
