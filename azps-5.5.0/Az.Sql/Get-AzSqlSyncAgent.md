# Get-AzSqlSyncAgent

## SYNOPSIS
Gibt Informationen zu Azure SQL Sync Agents zur端ck.

## SYNTAX

```
Get-AzSqlSyncAgent [[-Name] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzSqlSyncAgent"** gibt Informationen zu einem oder mehreren Azure SQL Sync Agents zur端ck.
Geben Sie den Namen eines Synchronisierungs-Agents an, um nur Informationen zu diesem Synchronisierungs-Agent zu sehen.
