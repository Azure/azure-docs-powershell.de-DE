# Set-AzSqlDatabaseGeoBackupPolicy

## SYNOPSIS
Legt eine Datenbankgedatensicherungsrichtlinie fest.

## SYNTAX

```
Set-AzSqlDatabaseGeoBackupPolicy -State <GeoBackupPolicyState> [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Set-AzSqlDatabaseGeoBackupPolicy"** legt die Geosicherungsrichtlinie fest, die für eine Datenbank registriert ist.
Dies ist eine Azure Backup-Ressource, die zum Definieren der Sicherungsspeicherrichtlinie verwendet wird.
