# Add-AzDataLakeAnalyticsFirewallRule

## SYNOPSIS
Fügt einem Data Lake Analytics-Konto eine Firewallregel hinzu.

## SYNTAX

```
Add-AzDataLakeAnalyticsFirewallRule [-Account] <String> [-Name] <String> [-StartIpAddress] <String>
 [-EndIpAddress] <String> [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Add-AzDataLakeAnalyticsFirewallRule"** fügt einem Azure Data Lake Analytics-Konto eine Firewallregel hinzu.

## BEISPIELE

### Beispiel 1: Hinzufügen einer Firewallregel
```
PS C:\>Add-AzDataLakeAnalyticsFirewallRule -Account "ContosoAdlAcct" -Name "My firewall rule" -StartIpAddress 127.0.0.1 -EndIpAddress 127.0.0.10
```

Mit diesem Befehl wird die Firewallregel namens "Meine Firewallregel" aus dem Konto "ContosoAdlAcct" dem IP-Bereich hinzufügt: 127.0.0.1 bis 127.0.0.10
