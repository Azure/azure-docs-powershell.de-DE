# Add-AzDataLakeStoreItemContent

## SYNOPSIS
Fügt einem Element in einem Data Lake Store Inhalte hinzu.

## SYNTAX

```
Add-AzDataLakeStoreItemContent [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Value] <Object>
 [[-Encoding] <Encoding>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Add-AzDataLakeStoreItemContent"** fügt einem Element in einem Azure Data Lake Store Inhalte hinzu.

## BEISPIELE

### Beispiel 1: Hinzufügen von Inhalt zu einer Datei
```
PS C:\>Add-AzDataLakeStoreItemContent -AccountName "ContosoADLS" -Path /abc/myFile.txt -Value "My content here"
```

Mit diesem Befehl wird dem Dateiinhalt Inhalt myFile.txt.
