# Test-AzDataLakeStoreAccount

## SYNOPSIS
Überprüft das Vorhandensein eines Data Lake Store-Kontos.

## SYNTAX

```
Test-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Test-AzDataLakeStoreAccount"** überprüft das Vorhandensein eines Data Lake Store-Kontos.

## BEISPIELE

### Beispiel 1: Testen eines Kontos
```
PS C:\>Test-AzDataLakeStoreAccount -Name "ContosoADL"
```

Mit diesem Befehl wird überprüft, ob das Konto "ContosoADL" vorhanden ist.
