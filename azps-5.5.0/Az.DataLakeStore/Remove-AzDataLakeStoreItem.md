# Remove-AzDataLakeStoreItem

## SYNOPSIS
Löscht eine Datei oder einen Ordner im Data Lake Store.

## SYNTAX

```
Remove-AzDataLakeStoreItem [-Account] <String> [-Paths] <DataLakeStorePathInstance[]> [-Recurse] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzDataLakeStoreItem"** löscht eine Datei oder einen Ordner im Data Lake Store.

## BEISPIELE

### Beispiel 1: Entfernen mehrerer Elemente
```
PS C:\>Remove-AzDataLakeStoreItem -AccountName "ContosoADL" -Paths "/File01.txt","/MyFiles/File.csv"
```

Mit diesem Befehl werden die File01.txt und File.csv Dateien aus dem Data Lake Store entfernt.
