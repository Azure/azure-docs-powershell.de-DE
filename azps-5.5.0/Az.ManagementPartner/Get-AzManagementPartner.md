# Get-AzManagementPartner

## SYNOPSIS
Ruft die MICROSOFT Partner Network(MPN)-ID des aktuell authentifizierten Benutzers oder Dienstprinzipal ab.

## SYNTAX

```
Get-AzManagementPartner [[-PartnerId] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Ruft die MICROSOFT Partner Network(MPN)-ID des aktuell authentifizierten Benutzers oder Dienstprinzipal ab.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Get-AzManagementPartner
PartnerId   : 4977985
PartnerName : Test_Test_DPORTest
TenantId    : 1b1121dd-6900-412a-af73-e8d44f81e1c1
ObjectId    : aa67f786-0552-423e-8849-244ed12bf581
State       : Active
```

Holen Sie sich die id des aktuellen Managementpartners

### Beispiel 2
```powershell
PS C:\> Get-AzManagementPartner -PartnerId 4977985
PartnerId   : 4977985
PartnerName : Test_Test_DPORTest
TenantId    : 1b1121dd-6900-412a-af73-e8d44f81e1c1
ObjectId    : aa67f786-0552-423e-8849-244ed12bf581
State       : Active
```

Holen Sie sich die ID des aktuellen Managementpartners mithilfe einer Partner-ID. Wenn sie nicht übereinstimmen, kann dies nicht fehlschlagen.
