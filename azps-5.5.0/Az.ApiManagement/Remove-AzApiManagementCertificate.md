# Remove-AzApiManagementCertificate

## SYNOPSIS
Entfernt ein API-Verwaltungszertifikat.

## SYNTAX

```
Remove-AzApiManagementCertificate -Context <PsApiManagementContext> -CertificateId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Remove-AzApiManagementCertificate"** entfernt ein Azure API Management-Zertifikat.

## BEISPIELE

### Beispiel 1: Entfernen eines Zertifikats
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementCertificate -Context $ApiMgmtContext -CertificateId "0123456789" -Force
```

Mit diesem Befehl wird das angegebene ZERTIFIKAT für die API-Verwaltung entfernt.
Da der *Parameter "Force"* angegeben wird, ist keine Bestätigung erforderlich.
