# Get-AzApiManagementTenantSyncState

## SYNOPSIS
Ruft den Status der letzten Synchronisierung zwischen der Konfigurationsdatenbank und dem Git-Repository ab.

## SYNTAX

```
Get-AzApiManagementTenantSyncState -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Get-AzApiManagementTenantSyncState"** ruft den Status der letzten Synchronisierung zwischen der Konfigurationsdatenbank und dem Git-Repository ab.

## BEISPIELE

### Beispiel 1: Anzeigen des Status der letzten Synchronisierung
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantSyncState -Context $apimContext
```

Dieser Befehl ruft den Status der letzten Synchronisierung zwischen der Konfigurationsdatenbank und dem Git-Repository ab.
