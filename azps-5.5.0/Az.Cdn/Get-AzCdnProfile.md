# Get-AzCdnProfile

## SYNOPSIS
Ruft ein CDN-Profil ab.

## SYNTAX

```
Get-AzCdnProfile [-ProfileName <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Get-AzCdnProfile-Cmdlet** ruft ein Azure Content Delivery Network (CDN)-Profil und die zugehörigen Informationen ab.
