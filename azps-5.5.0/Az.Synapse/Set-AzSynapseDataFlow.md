# Set-AzSynapseDataFlow

## SYNOPSIS
Erstellt oder aktualisiert einen Datenfluss im Arbeitsbereich.

## SYNTAX

### SetByName (Standard)
```
Set-AzSynapseDataFlow -WorkspaceName <String> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByObject
```
Set-AzSynapseDataFlow -WorkspaceObject <PSSynapseWorkspace> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Set-AzSynapseDataFlow"** erstellt einen Datenfluss oder aktualisiert einen vorhandenen Datenfluss im Arbeitsbereich.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Set-AzSynapseDataFlow -WorkspaceName ContosoWorkspace -Name ContosoDataFlow -DefinitionFile "C:\\samples\\DataFlow.json"
```

Dieser Befehl erstellt einen Datenfluss namens "ContosoDataFlow" im Arbeitsbereich "ContosoWorkspace".
Der Befehl basiert auf Den Datenfluss auf Informationen im DataFlow.jsDatei.
