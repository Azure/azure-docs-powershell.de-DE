# Stop-AzSynapseTrigger

## SYNOPSIS
Beendet einen Trigger in einem Arbeitsbereich.

## SYNTAX

### StopByName (Standard)
```
Stop-AzSynapseTrigger -WorkspaceName <String> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByObject
```
Stop-AzSynapseTrigger -WorkspaceObject <PSSynapseWorkspace> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByInputObject
```
Stop-AzSynapseTrigger -InputObject <PSTriggerResource> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Stop-AzSynapseTrigger"** beendet einen Trigger in einem Arbeitsbereich. Wenn sich der Trigger im Zustand "Gestartet" befindet, beendet das Cmdlet den Trigger und ruft keine Pipelines mehr auf. Wenn sich der Trigger bereits im Zustand "Beendet" befindet, hat dieses Cmdlet keine Auswirkung.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Stop-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
```

Beendet einen Trigger namens "ContosoTrigger" im Arbeitsbereich "ContosoWorkspace".

### Beispiel 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Stop-AzSynapseTrigger -Name ContosoTrigger
```

Beendet einen Trigger namens "ContosoTrigger" im Arbeitsbereich "ContosoWorkspace" über die Pipeline.

### Beispiel 3
```powershell
PS C:\> $trigger = Get-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
PS C:\> $trigger | Stop-AzSynapseTrigger
```

Beenden Sie einen Trigger namens "ContosoTrigger" im Arbeitsbereich "ContosoWorkspace" über die Pipeline.
