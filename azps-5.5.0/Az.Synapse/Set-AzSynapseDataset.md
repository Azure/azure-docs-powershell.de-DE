# Set-AzSynapseDataset

## SYNOPSIS
Erstellt oder aktualisiert ein Dataset im Arbeitsbereich.

## SYNTAX

### SetByName (Standard)
```
Set-AzSynapseDataset -WorkspaceName <String> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByObject
```
Set-AzSynapseDataset -WorkspaceObject <PSSynapseWorkspace> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHREIBUNG
Das **Cmdlet "Set-AzSynapseDataset"** erstellt ein Dataset oder aktualisiert ein vorhandenes Dataset im Arbeitsbereich.

## BEISPIELE

### Beispiel 1
```powershell
PS C:\> Set-AzSynapseDataset -WorkspaceName ContosoWorkspace -Name ContosoDataset -DefinitionFile "C:\\samples\\Dataset.json"
```

Mit diesem Befehl wird ein Dataset namens "ContosoDataset" im Arbeitsbereich "ContosoWorkspace" erstellt.
Der Befehl basiert auf Informationen im Dataset, Dataset.jsdateibasiert sind.
