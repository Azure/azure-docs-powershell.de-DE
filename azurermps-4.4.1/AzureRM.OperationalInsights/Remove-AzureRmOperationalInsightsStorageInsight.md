# Remove-AzureRmOperationalInsightsStorageInsight

## Synopsis
Entfernt eine Speicher Einsicht.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

### ByWorkspaceName (Standard)
```
Remove-AzureRmOperationalInsightsStorageInsight [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-Name] <String> [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByWorkspaceObject
```
Remove-AzureRmOperationalInsightsStorageInsight [-Workspace] <PSWorkspace> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureRmOperationalInsightsStorageInsight** löscht einen Speicher Einblick aus einem Arbeitsbereich.

## Beispiele

### Beispiel 1: Entfernen eines Speicher Einblicks nach Namen
```
PS C:\>Remove-AzureRmOperationalInsightsStorageInsight -ResourceGroupName "ContosoResourceGroup" -WorkspaceName "MyWorkspace" -Name "MyStorageInsight"
```

Mit diesem Befehl wird der Speicher Insight mit dem Namen MyStorageInsight aus dem Arbeitsbereich mit dem Namen "mein Workspace" in der angegebenen Ressourcengruppe entfernt.
Der Befehl gibt nicht den Parameter *Force* an, sodass Sie zur Bestätigung aufgefordert werden, bevor Sie die Speicher Einsicht entfernen.

### Beispiel 2: Entfernen eines Speicher Einblicks ohne Bestätigung
```
PS C:\>$Workspace = Get-AzureRmOperationalInsightsWorkspace -ResourceGroupName "ContosoResourceGroup" -Name "MyWorkspace"

PS C:\>Remove-AzureRmOperationalInsightsStorageInsight -Workspace $Workspace -Name "MyStorageInsight" -Force
```

Der erste Befehl verwendet das Get-AzureRmOperationalInsightsWorkspace-Cmdlet, um den Arbeitsbereich mit dem Namen myworkspace abzurufen, und speichert ihn dann in der Variablen $Workspace. Mit dem zweiten Befehl wird der Speicher Insight mit dem Namen MyStorageInsight aus $Workspace entfernt, ohne dass Sie zur Bestätigung aufgefordert werden.
