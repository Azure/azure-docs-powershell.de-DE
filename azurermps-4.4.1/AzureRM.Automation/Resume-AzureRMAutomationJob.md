# Resume-AzureRmAutomationJob

## Synopsis
Setzt einen angehalten Automatisierungs Auftrag fort.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Resume-AzureRmAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Resume-AzureRmAutomationJob** setzt einen angehenden Azure-Automatisierungs Auftrag fort.
Geben Sie den angehenden Auftrag an.

Verwenden Sie das Suspend-AzureRmAutomationJob-Cmdlet, um einen Auftrag zu unterbrechen.

## Beispiele

### Beispiel 1: Fortsetzen eines angehenden Auftrags
```
PS C:\>Resume-AzureRmAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Mit diesem Befehl wird der Auftrag mit der angegebenen ID fortgesetzt.
