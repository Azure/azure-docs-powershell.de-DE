# Stop-AzureKeyVaultCertificateOperation

## Synopsis
Bricht eine Zertifikat Operation in Key Vault ab.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Stop-AzureKeyVaultCertificateOperation [-VaultName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet " **Stop-AzureKeyVaultCertificateOperation** " wird ein Zertifikat Vorgang im Azure Key Vault-Dienst abgebrochen.
