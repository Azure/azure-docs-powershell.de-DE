# Get-AzureRmMetric

## Synopsis
Ruft die metrischen Werte einer Ressource ab.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

### Parameter für Get-AzureRmMetric-Cmdlet im Standardmodus
```
Get-AzureRmMetric [-ResourceId] <String> [-MetricNames <String[]>] [-DetailedOutput]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Parameter für Get-AzureRmMetric-Cmdlet im vollständigen Parame-Satzmodus
```
Get-AzureRmMetric [-ResourceId] <String> [[-TimeGrain] <TimeSpan>] [-AggregationType <AggregationType>]
 [-StartTime <DateTime>] [-EndTime <DateTime>] -MetricNames <String[]> [-DetailedOutput]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureRmMetric** " Ruft die metrischen Werte für eine angegebene Ressource ab.
