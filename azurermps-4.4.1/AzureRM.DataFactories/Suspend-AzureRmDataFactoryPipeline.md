# Suspend-AzureRmDataFactoryPipeline

## Synopsis
Unterbricht eine Pipeline in Azure Data Factory.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

### ByFactoryName (Standard)
```
Suspend-AzureRmDataFactoryPipeline [-Name] <String> [-DataFactoryName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Suspend-AzureRmDataFactoryPipeline [-Name] <String> [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Suspend-AzureRmDataFactoryPipeline** unterbricht eine Pipeline in Azure Data Factory.
Sie können die Pipeline mithilfe des Resume-AzureRmDataFactoryPipeline-Cmdlets fortsetzen.

## Beispiele

### Beispiel 1: Anhalten einer Pipeline
```
PS C:\>Suspend-AzureRmDataFactoryPipeline -ResourceGroupName "ADF" -Name "DPWikiSample" -DataFactoryName "WikiADF"
Confirm
Are you sure you want to suspend pipeline 'DPWikisample' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

Dieser Befehl unterbricht die Pipeline mit dem Namen DPWikiSample in der Data Factory mit dem Namen WikiADF.
Der Befehl gibt den Wert $true zurück.
