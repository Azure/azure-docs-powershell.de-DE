# Publish-AzureRmCdnEndpointContent

## Synopsis
Lädt Inhalt zu einem Endpunkt.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

### Parametersatz für fields-Parameter (Standard)
```
Publish-AzureRmCdnEndpointContent -EndpointName <String> -ProfileName <String> -ResourceGroupName <String>
 -LoadContent <String[]> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Parametersatz für Objektparameter
```
Publish-AzureRmCdnEndpointContent -CdnEndpoint <PSEndpoint> -LoadContent <String[]> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Publish-AzureRmCdnEndpointContent** " lädt Inhalte von einem Ursprungsserver für den Azure Content Delivery Network (CDN)-Endpunkt.
