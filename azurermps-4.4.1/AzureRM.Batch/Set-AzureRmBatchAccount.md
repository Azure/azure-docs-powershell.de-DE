# Set-AzureRmBatchAccount

## Synopsis
Aktualisiert ein Stapelverarbeitungs Konto.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Set-AzureRmBatchAccount [-AccountName] <String> [-Tag] <Hashtable> [-ResourceGroupName <String>]
 [-AutoStorageAccountId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Satz-AzureRmBatchAccount** " aktualisiert ein Azure-Batch Konto.
Derzeit kann dieses Cmdlet nur Tags aktualisieren.

## Beispiele

### Beispiel 1: Aktualisieren der Tags in einem Stapelverarbeitungs Konto
```
PS C:\>Set-AzureRmBatchAccount -AccountName "cmdletexample" -Tag @{key0="value0";key1=$null;key2="value2"}
AccountName                  : cmdletexample
Location                     : westus
ResourceGroupName            : CmdletExampleRG
CoreQuota                    : 20
PoolQuota                    : 20
ActiveJobAndJobScheduleQuota : 20
Tags                         :
                               Name  Value
                               ====  ======
                               key0  value0
                               key1
                               key2  value2
TaskTenantUrl                : https://cmdletexample.westus.batch.azure.com
```

Mit diesem Befehl werden die Tags für das Konto mit dem Namen pfuller aktualisiert.
