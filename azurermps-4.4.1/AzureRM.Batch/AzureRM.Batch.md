# AzureRM.Batch-Modul
## Beschreibung
Mit den Azure Batch-Cmdlets im Azure-Modul können Sie Microsoft Azure-Batch Dienste in Azure PowerShell verwalten.

## AzureRM.Batch-Cmdlets
### [Deaktivieren-AzureBatchAutoScale](Disable-AzureBatchAutoScale.md)
Deaktiviert die automatische Skalierung eines Pools.

### [Deaktivieren-AzureBatchComputeNodeScheduling](Disable-AzureBatchComputeNodeScheduling.md)
Deaktiviert die Vorgangsplanung auf dem angegebenen Compute-Knoten.

### [Deaktivieren-AzureBatchJob](Disable-AzureBatchJob.md)
Deaktiviert eine Stapelverarbeitung.

### [Deaktivieren-AzureBatchJobSchedule](Disable-AzureBatchJobSchedule.md)

### [Enable-AzureBatchAutoScale](Enable-AzureBatchAutoScale.md)
Aktiviert die automatische Skalierung eines Pools.

### [Enable-AzureBatchJobSchedule](Enable-AzureBatchJobSchedule.md)
Aktiviert einen Stapel Auftragszeitplan.

### [Enable-AzureBatchTask](Enable-AzureBatchTask.md)
