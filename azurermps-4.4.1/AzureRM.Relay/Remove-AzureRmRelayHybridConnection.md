# Remove-AzureRmRelayHybridConnection

## Synopsis
Entfernt die HybridConnection aus dem angegebenen HybridConnection-Namespace.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Remove-AzureRmRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureRmRelayHybridConnection** entfernt die HybridConnection aus dem angegebenen Relay-Namespace.

## Beispiele

### Beispiel 1
```
PS C:\> Remove-AzureRmRelayHybridConnection -ResourceGroupName Default-ServiceBus-WestUS -Namespace TestNameSpace-Relay1 -Name TestHybridConnection
```

Entfernt die HybridConnection `TestHybridConnection` aus dem Namespace `TestNameSpace-Relay1` .
