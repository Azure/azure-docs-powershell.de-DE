# Remove-AzureRmDataLakeAnalyticsAccount

## Synopsis
Löscht ein Data Lake Analytics-Konto.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Remove-AzureRmDataLakeAnalyticsAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **Remove-AzureRmDataLakeAnalyticsAccount** wird ein Azure Data Lake-Analyse Konto endgültig gelöscht.

## Beispiele

### Beispiel 1: Entfernen eines Kontos
```
PS C:\>Remove-AzureRmDataLakeAnalyticsAccount -Name "ContosoAdlAccount"
```

Mit diesem Befehl wird das angegebene Data Lake Analytics-Konto entfernt.
