# Remove-AzureRmDataLakeAnalyticsFirewallRule

## Synopsis
Entfernt eine Firewall-Regel aus einem Data Lake Analytics-Konto.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Remove-AzureRmDataLakeAnalyticsFirewallRule [-Account] <String> [[-Name] <String>] [-PassThru]
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureRmDataLakeAnalyticsFirewallRule** entfernt eine Firewallregel aus einem Azure Data Lake Analytics-Konto.

## Beispiele

### Beispiel 1: Entfernen einer Firewall-Regel
```
PS C:\>Remove-AzureRmDataLakeAnalyticsFirewallRule -Account "ContosoAdlAcct" -Name "My firewall rule"
```

Dieser Befehl entfernt die Firewallregel mit dem Namen "meine Firewall-Regel" aus dem Konto "ContosoAdlAcct"
