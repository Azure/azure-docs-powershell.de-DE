# Remove-AzureRmStorageAccount

## Synopsis
Entfernt ein Speicherkonto aus Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Remove-AzureRmStorageAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureRmStorageAccount** entfernt ein Speicherkonto aus Azure.

## Beispiele

### Beispiel 1: Entfernen eines speicherkontos
```
PS C:\>Remove-AzureRmStorageAccount -ResourceGroupName "RG01" -AccountName "MyStorageAccount"
```

Mit diesem Befehl wird das angegebene Speicherkonto entfernt.
