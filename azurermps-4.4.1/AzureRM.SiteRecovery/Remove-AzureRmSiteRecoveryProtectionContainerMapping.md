# Remove-AzureRmSiteRecoveryProtectionContainerMapping

## Synopsis
Entfernt eine Azure Site Recovery Protection-Container Zuordnung.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Remove-AzureRmSiteRecoveryProtectionContainerMapping
 -ProtectionContainerMapping <ASRProtectionContainerMapping> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureRmSiteRecoveryProtectionContainerMapping** entfernt eine Azure Site Recovery Protection-Container Zuordnung.
