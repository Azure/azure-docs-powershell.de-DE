# Remove-AzureRmSiteRecoveryServer

## Synopsis
Entfernt einen Standort Wiederherstellungsserver aus dem aktuellen Tresor.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Remove-AzureRmSiteRecoveryServer -Server <ASRServer> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzureRmSiteRecoveryServer** hebt die Registrierung eines Azure Site Recovery-Servers auf, der ihn aus dem aktuellen Tresor entfernt.
