# New-AzureRmServiceBusQueueAuthorizationRule

## Synopsis
Erstellt eine neue Autorisierungsregel für die angegebene ServiceBus-Warteschlange.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
New-AzureRmServiceBusQueueAuthorizationRule [-ResourceGroup] <String> -Namespace <String> -Queue <String>
 -Name <String> [-Rights] <String[]> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **New-AzureRmServiceBusQueueAuthorizationRule** erstellt eine neue Autorisierungsregel für die angegebene ServiceBus-Warteschlange.

## Beispiele

### Beispiel 1
```
PS C:\> New-AzureRmServiceBusQueueAuthorizationRule -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -QueueName SB-Queue_exampl1 -AuthorizationRuleName SBAuthoRule1 -Rights @("Listen","Send")
```

Erstellt eine Autorisierungsregel `SBAuthoRule1` mit **Listen-und sende** rechten für die Warteschlange `SB-Queue_exampl1` .
