# Get-AzureRmServiceBusNamespaceAuthorizationRule

## Synopsis
Ruft eine Beschreibung der angegebenen Autorisierungsregel für einen angegebenen Namespace ab. 

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Get-AzureRmServiceBusNamespaceAuthorizationRule [-ResourceGroup] <String> -Namespace <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureRmServiceBusNamespaceAuthorizationRule** " Ruft die Beschreibung der angegebenen Autorisierungsregel im angegebenen Namespace ab.

## Beispiele

### Beispiel 1
```
PS C:\> Get-AzureRmServiceBusNamespaceAuthorizationRule -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -AuthorizationRuleName AuthoRule1
```

Gibt die angegebene Autorisierungsregel Beschreibung für einen angegebenen Namespace zurück.
