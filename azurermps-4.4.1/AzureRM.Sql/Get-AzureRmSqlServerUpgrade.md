# Get-AzureRmSqlServerUpgrade

## Synopsis
Ruft den Status eines Azure SQL-Datenbankserver-Upgrades ab.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Get-AzureRmSqlServerUpgrade -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzureRmSqlServerUpgrade** " Ruft den Status eines Azure SQL-Datenbankserver-Upgrades ab.

## Beispiele

### Beispiel 1: Abrufen des Status eines Upgrades
```
PS C:\>Get-AzureRmSqlServerUpgrade -ResourceGroupName "ResourceGroup01" -ServerName "Server01" | Format-List
ResourceGroupName               : resourcegroup01
ServerName                      : server01
Status                          : Queued
```

Dieser Befehl ruft den Status eines Upgrades vom Server mit dem Namen Server01 in der Ressourcengruppe mit dem Namen ResourceGroup01 ab.
