# Start-AzureRmSqlSyncGroupSync

## Synopsis
Startet eine Synchronisierungsgruppen Synchronisierung.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Start-AzureRmSqlSyncGroupSync [-SyncGroupName] <String> [-PassThru] [-ServerName] <String>
 [-DatabaseName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Start-AzureRmSqlSyncGroupSync** startet eine Synchronisierungsgruppen Synchronisierung.

## Beispiele

### Beispiel 1
```
PS C:\> Start-AzureRmSqlSyncGroupSync -SyncGroupName mysg [-ServerName] mysrv [-DatabaseName] mydb [-ResourceGroupName] myrg
```

Mit diesem Befehl wird eine Synchronisierungs Runde für die Synchronisierungsgruppe mysg.
