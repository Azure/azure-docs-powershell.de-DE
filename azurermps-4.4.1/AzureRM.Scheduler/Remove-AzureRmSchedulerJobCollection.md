# Remove-AzureRmSchedulerJobCollection

## Synopsis
Entfernt eine Auftrags Sammlung.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## Syntax

```
Remove-AzureRmSchedulerJobCollection -ResourceGroupName <String> -JobCollectionName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Remove-AzureRmSchedulerJobCollection** " entfernt eine Auftrags Sammlung im Azure Scheduler.
