# Set-AzDtlAutoStartPolicy

## Synopsis
Legt die Richtlinie für den automatischen Start einer Übungseinheit in DevTest Labs fest.

## Syntax

### Aktivieren (Standard)
```
Set-AzDtlAutoStartPolicy [[-Time] <DateTime>] [[-Days] <DayOfWeek[]>] [-Enable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Deaktivieren
```
Set-AzDtlAutoStartPolicy [[-Time] <DateTime>] [[-Days] <DayOfWeek[]>] [-Disable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Set-AzDtlAutoStartPolicy** " legt die Richtlinie für den automatischen Start einer Übungseinheit fest, mit der virtuelle Lab-Computer für den automatischen Start geplant werden können.
Das Cmdlet verwendet die angegebene Ressourcengruppe und den Namen der Übungseinheit, um die Richtlinie festzulegen.
