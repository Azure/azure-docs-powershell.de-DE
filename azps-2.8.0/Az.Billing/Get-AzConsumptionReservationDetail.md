# Get-AzConsumptionReservationDetail

## Synopsis
Hier erhalten Sie Reservierungsdetails für den bereitgestellten Datumsbereich.

## Syntax

```
Get-AzConsumptionReservationDetail -StartDate <DateTime> -EndDate <DateTime> -ReservationOrderId <String>
 [-ReservationId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzConsumptionReservationDetail** " ruft Reservierungsdetails für den bereitgestellten Datumsbereich ab.
