# Remove-AzMediaService

## Synopsis
Entfernt einen Mediendienst.

## Syntax

```
Remove-AzMediaService [-ResourceGroupName] <String> [-AccountName] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **Remove-AzMediaService** wird ein Mediendienst entfernt.

## Beispiele

### Beispiel 1: Entfernen eines Medien Diensts
```
PS C:\>Remove-AzMediaService -ResourceGroupName "ResourceGroup001" -AccountName "MediaService0011"
```

Dieser Befehl entfernt den Mediendienst mit dem Namen MediaService0011 in der Ressourcengruppe mit dem Namen ResourceGroup001.
