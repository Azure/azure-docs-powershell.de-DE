# Update-AzRecoveryServicesAsrMobilityService

## Synopsis
Push-Updates für Mobilitätsdienst-Agents auf geschützten Computern

## Syntax

```
Update-AzRecoveryServicesAsrMobilityService [-Account <ASRRunAsAccount>]
 -ReplicationProtectedItem <ASRReplicationProtectedItem> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Update-AzRecoveryServicesAsrMobilityService** versucht, Mobilitätsdienst-Agent-Updates auf geschützte Computer zu übertragen (sofern ein Update verfügbar ist).

## Beispiele

### Beispiel 1
```
PS C:\> Update-AzRecoveryServicesAsrMobilityService -ReplicationProtectedItem $rpi -Account $fabric.fabricSpecificDetails.RunAsAccounts[0]
```

Job zum Nachvollziehen des Mobilitäts Dienstanbieters für das Update-Replikations-Element
