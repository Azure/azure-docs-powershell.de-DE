# Get-AzLegacyPeering

## Synopsis
Wird verwendet, um Legacy-Peering Ressourcen in Azure Resource Management (Arm)-Ressourcen umzuwandeln. 

## Syntax

```
Get-AzLegacyPeering [-PeeringLocation] <String> [-Kind] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Der Befehl wird verwendet, um Legacy-Peering Ressourcen anzuzeigen, die Sie alle in Azure Resource Management (Arm)-Ressourcen konvertieren können.
