# Remove-AzDataFactoryV2TriggerSubscription

## Synopsis
Kündigen Sie den Ereignisauslöser für externe Dienstereignisse.

## Syntax

### ByFactoryName (Standard)
```
Remove-AzDataFactoryV2TriggerSubscription [-Name] <String> [-PassThru] [-Force] [-ResourceGroupName] <String>
 [-DataFactoryName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByInputObject
```
Remove-AzDataFactoryV2TriggerSubscription [-InputObject] <PSTrigger> [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Remove-AzDataFactoryV2TriggerSubscription [-PassThru] [-Force] [-ResourceId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit diesem Befehl wird der Event-Trigger für die angegebenen externen Dienstereignisse aus dem Trigger-Definition.

## Beispiele

### Beispiel 1
```
PS C:\> Remove-AzDataFactoryV2TriggerSubscription -ResourceGroupName ADF -DataFactoryName WikiADF -Name Trigger1
```

Mit diesem Befehl wird der BlobEnetTrigger1-Trigger für die angegebenen Ereignisse aus dem Trigger-Definition abgemeldet.
