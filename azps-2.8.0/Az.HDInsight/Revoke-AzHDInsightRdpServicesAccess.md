# Revoke-AzHDInsightRdpServicesAccess

## Synopsis
Deaktiviert den RDP-Zugriff auf einen Windows-Cluster.

## Syntax

```
Revoke-AzHDInsightRdpServicesAccess [-ClusterName] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das **REVOKE-AzHDInsightRdpServicesAccess-** Cmdlet deaktiviert den Zugriff auf einen Windows-basierten Azure HDInsight-Cluster (Remote Desktop Protocol).

## Beispiele

### Beispiel 1: Deaktivieren des RDP-Zugriffs auf einen angegebenen Cluster
```
PS C:\>Revoke-AzHDInsightRdpServicesAccess -ClusterName "your-hadoop-001"
```

Mit diesem Befehl wird der RDP-Zugriff auf den Cluster "Your-Hadoop-001" aufgehoben.
