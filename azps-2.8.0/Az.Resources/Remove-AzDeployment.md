# Remove-AzDeployment

## Synopsis
Entfernt eine Bereitstellung und alle zugeordneten Vorgänge

## Syntax

### RemoveByDeploymentName (Standard)
```
Remove-AzDeployment [-Name] <String> [-AsJob] [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByDeploymentId
```
Remove-AzDeployment -Id <String> [-AsJob] [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByInputObject
```
Remove-AzDeployment -InputObject <PSDeployment> [-AsJob] [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzDeployment** entfernt eine Azure-Bereitstellung im Abonnement Bereich und alle zugeordneten Vorgänge.

## Beispiele

### Beispiel 1: Entfernen einer Bereitstellung mit einem angegebenen Namen
```
PS C:\>Remove-AzDeployment -Name "RolesDeployment"
```

Mit diesem Befehl wird die Bereitstellung "RolesDeployment" im aktuellen Abonnement Bereich entfernt.

### Beispiel 2: Abrufen einer Bereitstellung und entfernen
```
PS C:\>Get-AzDeployment -Name "RolesDeployment" | Remove-AzDeployment
```

Dieser Befehl ruft die Bereitstellung "RolesDeployment" im aktuellen Abonnement Bereich ab und entfernt Sie.
