# Remove-AzKustoDatabase

## Synopsis
Löscht eine vorhandene Kusto-Datenbank.

## Syntax

### ByNameAndResourceGroup (Standard)
```
Remove-AzKustoDatabase [-ResourceGroupName] <String> [-ClusterName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Remove-AzKustoDatabase [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzKustoDatabase [-InputObject] <PSKustoDatabase> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Löscht eine vorhandene Kusto-Datenbank.

## Beispiele

### Beispiel 1: Löschen einer vorhandenen Kusto-Datenbank nach Namen

```
PS C:\> Remove-AzKustoDatabase -ResourceGroupName testrg -ClusterName mykustocluster -Name mykustodatabase
```

Der obige Befehl löscht die Kusto-Datenbank mit dem Namen "mykustodatabase" im Cluster "mykustocluster", der in der Ressourcengruppe "testrg" gefunden wurde.

### Beispiel 2 – Löschen einer vorhandenen Kusto-Datenbank durch Piping

```
PS C:\> Get-AzKustoDatabase -ResourceGroupName testrg -ClusterName mykustocluster -Name mykustodatabase | Remove-AzKustoDatabase
```

Der obige Befehl ruft die Kusto-Datenbank mit dem Namen "mykustodatabase" im Cluster "mykustocluster" ab, die in der Ressourcengruppe "testrg" mithilfe des `Get-AzKustoDatabase` Cmdlets gefunden wird, und führt dann das Ergebnis aus, um `Remove-AzKustoDatabase` es zu löschen.
