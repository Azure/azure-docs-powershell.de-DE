# Remove-AzApiManagementProperty

## Synopsis
Entfernt eine API-Verwaltungseigenschaft.

## Syntax

```
Remove-AzApiManagementProperty -Context <PsApiManagementContext> -PropertyId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzApiManagementProperty** entfernt eine Azure API-Verwaltungs **Eigenschaft**.

## Beispiele

### Beispiel 1: Entfernen einer Eigenschaft
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementProperty -Context $apimContext -PropertyId "Property11" -PassThru
```

Dieser Befehl entfernt die Eigenschaft mit der ID Property11.
