# AZ. DataLakeStore-Modul
## Beschreibung
In den Themen in diesem Abschnitt werden die Azure PowerShell-Cmdlets für Azure Data Lake Store im Azure Resource Manager (Arm)-Framework dokumentiert. Die Cmdlets sind im Microsoft. Azure. Commands. DataLakeStore-Namespace vorhanden. Diese Cmdlets funktionieren nur mit Azure Data Lake-Speicher Gen1-Konten.

## AZ. DataLakeStore-Cmdlets
### [Add-AzDataLakeStoreFirewallRule](Add-AzDataLakeStoreFirewallRule.md)
Fügt dem angegebenen Data Lake Store-Konto eine Firewallregel hinzu.

### [Add-AzDataLakeStoreItemContent](Add-AzDataLakeStoreItemContent.md)
Fügt Inhalt zu einem Element in einem Data Lake Store hinzu.

### [Add-AzDataLakeStoreTrustedIdProvider](Add-AzDataLakeStoreTrustedIdProvider.md)
Fügt einen vertrauenswürdigen Identitätsanbieter zum angegebenen Data Lake Store-Konto hinzu.

### [Add-AzDataLakeStoreVirtualNetworkRule](Add-AzDataLakeStoreVirtualNetworkRule.md)
Fügt dem angegebenen Data Lake Store-Konto eine virtuelle Netzwerkregel hinzu.

### [Enable-AzDataLakeStoreKeyVault](Enable-AzDataLakeStoreKeyVault.md)
Versucht, einen Benutzer verwalteten schlüsseltresor für die Verschlüsselung des angegebenen Data Lake Store-Kontos zu aktivieren.

### [Export-AzDataLakeStoreChildItemProperty](Export-AzDataLakeStoreChildItemProperty.md)
Exportiert die Eigenschaften (Datenträgerverwendung und ACL) für die gesamte Struktur aus dem angegebenen Pfad in einen Ausgabepfad.

### [Export-AzDataLakeStoreItem](Export-AzDataLakeStoreItem.md)
Lädt eine Datei aus dem Data Lake Store herunter.

### [Get-AzDataLakeStoreAccount](Get-AzDataLakeStoreAccount.md)
Ruft Details zu einem Data Lake Store-Konto ab.

### [Get-AzDataLakeStoreChildItem](Get-AzDataLakeStoreChildItem.md)
Ruft die Liste der Elemente in einem Ordner im Data Lake Store ab.

### [Get-AzDataLakeStoreChildItemSummary](Get-AzDataLakeStoreChildItemSummary.md)
Ruft die Zusammenfassung der Gesamtgröße, Dateien und Verzeichnisse ab, die in dem angegebenen Pfad enthalten sind.

### [Get-AzDataLakeStoreFirewallRule](Get-AzDataLakeStoreFirewallRule.md)
Ruft die angegebenen Firewallregeln im angegebenen Data Lake Store ab.
Wenn keine Firewall-Regel angegeben ist, werden alle Firewallregeln für das Konto aufgelistet.

### [Get-AzDataLakeStoreItem](Get-AzDataLakeStoreItem.md)
Ruft die Details einer Datei oder eines Ordners im Data Lake Store ab.

### [Get-AzDataLakeStoreItemAclEntry](Get-AzDataLakeStoreItemAclEntry.md)
Ruft einen Eintrag in der ACL einer Datei oder eines Ordners im Data Lake Store ab.

### [Get-AzDataLakeStoreItemContent](Get-AzDataLakeStoreItemContent.md)
Ruft den Inhalt einer Datei im Data Lake Store ab.

### [Get-AzDataLakeStoreItemOwner](Get-AzDataLakeStoreItemOwner.md)
Ruft den Besitzer einer Datei oder eines Ordners im Data Lake Store ab.

### [Get-AzDataLakeStoreItemPermission](Get-AzDataLakeStoreItemPermission.md)
Ruft die oktale Berechtigung einer Datei oder eines Ordners im Data Lake Store ab.

### [Get-AzDataLakeStoreTrustedIdProvider](Get-AzDataLakeStoreTrustedIdProvider.md)
Ruft den angegebenen vertrauenswürdigen Identitätsanbieter im angegebenen Data Lake Store ab.
Wenn kein Anbieter angegeben ist, werden alle Anbieter für das Konto aufgelistet.

### [Get-AzDataLakeStoreVirtualNetworkRule](Get-AzDataLakeStoreVirtualNetworkRule.md)
Ruft die angegebenen virtuellen Netzwerkregeln im angegebenen Data Lake Store ab.
Wenn keine Regel für virtuelles Netzwerk angegeben ist, werden alle virtuellen Netzwerkregeln für das Konto aufgelistet.

### [Importieren-AzDataLakeStoreItem](Import-AzDataLakeStoreItem.md)

### [Teilnehmen – AzDataLakeStoreItem](Join-AzDataLakeStoreItem.md)
