# Invoke-AzVMReimage

## Synopsis
Umbilden einer virtuellen Azure-Maschine

## Syntax

```
Invoke-AzVMReimage [-ResourceGroupName] <String> [-VMName] <String> [-TempDisk] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **Invoke-AzVMReimage** wird eine Azure Virtual Machine erneut abbildet.

## Beispiele

### Beispiel 1
```powershell
PS C:\> Invoke-AzVMReimage -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Mit diesem Befehl wird der virtuelle Computer mit dem Namen VirtualMachine07 in ResourceGroup11 erneut abbildet.
