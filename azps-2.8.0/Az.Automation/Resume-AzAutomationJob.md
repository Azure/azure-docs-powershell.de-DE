# Resume-AzAutomationJob

## Synopsis
Setzt einen angehalten Automatisierungs Auftrag fort.

## Syntax

```
Resume-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Resume-AzAutomationJob** setzt einen angehenden Azure-Automatisierungs Auftrag fort.
Geben Sie den angehenden Auftrag an.
Verwenden Sie das Suspend-AzAutomationJob-Cmdlet, um einen Auftrag zu unterbrechen.

## Beispiele

### Beispiel 1: Fortsetzen eines angehenden Auftrags
```
PS C:\>Resume-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Mit diesem Befehl wird der Auftrag mit der angegebenen ID fortgesetzt.
