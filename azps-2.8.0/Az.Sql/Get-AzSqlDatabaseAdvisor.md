# Get-AzSqlDatabaseAdvisor

## Synopsis
Ruft mindestens einen Ratgeber für eine Azure SQL-Datenbank ab.

## Syntax

```
Get-AzSqlDatabaseAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -DatabaseName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzSqlDatabaseAdvisor** " Ruft einen oder mehrere Azure SQL-Daten Bank Ratgeber für eine Azure SQL-Datenbank ab.
