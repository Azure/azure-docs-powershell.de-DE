# Reset-AzRedisCache

## Synopsis
Startet Knoten eines Caches neu.

## Syntax

```
Reset-AzRedisCache [-ResourceGroupName <String>] -Name <String> -RebootType <String> [-ShardId <Int32>]
 [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **Reset-AzRedisCache** werden die Knoten einer Azure-a-Cache-Instanz neu gestartet.

## Beispiele

### Beispiel 1: beide Knoten neu starten
```
PS C:\>Reset-AzRedisCache -ResourceGroupName "ResourceGroup13" -Name "RedisCache06" -RebootType "AllNodes" -Force
```

Dieser Befehl startet beide Knoten für den Cache mit dem Namen "RedisCache06" neu.
