# New-AzApplicationGatewayAutoscaleConfiguration

## Synopsis
Erstellt eine AutoScale-Konfiguration für das Anwendungs Gateway.

## Syntax

```
New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity <Int32> [-MaxCapacity <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **New-AzApplicationGatewayAutoscaleConfiguration** erstellt die Konfiguration von AutoScale für ein Azure Application Gateway.

## Beispiele

### Beispiel 1
```powershell
PS C:\> $autoscaleConfig = New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity 3
PS C:\> $gw = New-AzApplicationGateway -Name $appgwName -ResourceGroupName $rgname ..  -AutoscaleConfiguration $autoscaleConfig
```

Der erste Befehl erstellt eine AutoScale-Konfiguration mit minimaler Kapazität 3.
Mit dem zweiten Befehl wird ein Anwendungsgateway mit der Konfiguration für das AutoSkalieren erstellt.
