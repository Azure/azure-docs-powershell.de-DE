# Remove-AzApplicationGatewayIdentity

## Synopsis
Entfernt eine Identität aus einem Anwendungsgateway.

## Syntax

```
Remove-AzApplicationGatewayIdentity -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
**Remove-AzApplicationGatewayIdentity-** Cmdlet entfernt die Identität aus einem Anwendungsgateway.

## Beispiele

### Beispiel 1
```powershell
PS C:\> $appgw = Remove-AzApplicationGatewayIdentity -ApplicationGateway $appgw
PS C:\> $updatedgateway = Set-AzApplicationGateway -ApplicationGateway $appgw
```

In diesem Beispiel wird die Identität eines vorhandenen Anwendungs Gateways entfernt.
Hinweis: Wenn das Gateway auf ein keyvault-Geheimnis verweist, ist es auch wichtig, diese SSL-Zertifikat Verweise entlang dieses Vorgangs zu entfernen.
