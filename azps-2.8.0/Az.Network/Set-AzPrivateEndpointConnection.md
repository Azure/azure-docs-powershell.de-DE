# Set-AzPrivateEndpointConnection

## Synopsis
Aktualisiert einen privaten Endpunkt Verbindungsstatus für den privaten Link Dienst.

## Syntax

```
Set-AzPrivateEndpointConnection -Name <String> -ServiceName <String> -ResourceGroupName <String>
 -PrivateLinkServiceConnectionState <String> [-Description <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Satz-AzPrivateEndpointConnection** " aktualisiert einen privaten Endpunkt Verbindungsstatus für einen privaten Link Dienst

## Beispiele

### Beispiel 1
```
Set-AzPrivateEndpointConnection -Name TestPrivateEndpointConnection -ResourceGroupName TestResourceGroup -ServiceName TestPrivateLinkService -PrivateLinkServiceConnectionState "Approved"
```

In diesem Beispiel wird ein privater Endpunkt Verbindungsstatus auf genehmigt aktualisiert.
