# Remove-AzPrivateEndpoint

## Synopsis
Entfernt einen privaten Endpunkt.

## Syntax

```
Remove-AzPrivateEndpoint -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **Remove-AzPrivateEndpoint** wird ein privater Endpunkt entfernt. 

## Beispiele

### Beispiel 1
```
Remove-AzPrivateEndpoint -Name MyPrivateEndpoint1 -ResourceGroupName TestResourceGroup
```

In diesem Beispiel wird ein privater Endpunkt mit dem Namen MyPrivateEndpoint1 entfernt.
