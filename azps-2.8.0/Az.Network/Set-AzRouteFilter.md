# Set-AzRouteFilter

## Synopsis
Aktualisiert einen Routenfilter.

## Syntax

```
Set-AzRouteFilter -RouteFilter <PSRouteFilter> [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Satz-AzApplicationGateway** " aktualisiert einen Routenfilter.

## Beispiele

### Beispiel 1
```powershell
PS C:\> Set-AzRouteFilter -RouteFilter $rf
```

Dieser Befehl aktualisiert den Routenfilter mit den Einstellungen in der $RF Variablen.
