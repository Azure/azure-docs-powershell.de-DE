# Remove-AzApiManagementAuthorizationServer

## Synopsis
Entfernt einen autorisierungsserver.

## Syntax

```
Remove-AzApiManagementAuthorizationServer -Context <PsApiManagementContext> -ServerId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzApiManagementAuthorizationServer** entfernt einen Azure API-Verwaltungs autorisierungsserver.

## Beispiele

### Beispiel 1: Entfernen eines autorisierungsservers
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementAuthorizationServer -Context $ApiMgmtContext -ServerId "authserverid" -Force
```

Mit diesem Befehl wird der angegebene API-Verwaltungs Autorisierungs Server entfernt.
Da der Parameter *Force* angegeben ist, ist keine Bestätigung erforderlich.
