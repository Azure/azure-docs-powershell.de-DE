# Remove-AzApiManagementApiRevision

## Synopsis
Eine bestimmte API-Revision entfernt

## Syntax

### ByApiId (Standard)
```
Remove-AzApiManagementApiRevision -Context <PsApiManagementContext> -ApiId <String> -ApiRevision <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzApiManagementApiRevision -InputObject <PsApiManagementApi> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzApiManagementApiRevision** entfernt eine bestimmte API-Revision.

## Beispiele

### Beispiel 1: Entfernen einer API-Revision
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzAzureRmApiManagementApiRevision -Context $apimContext -ApiId "echo-api" -ApiRevision "2"
```

Mit diesem Befehl wird die `2` Überarbeitung der API `echo-api` vom API-Verwaltungsdienst entfernt.
