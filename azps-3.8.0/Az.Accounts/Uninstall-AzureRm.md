# Uninstall-AzureRm

## Synopsis
Entfernt alle AzureRm-Module von einem Computer.

## Syntax

```
Uninstall-AzureRm [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Entfernt alle AzureRm-Module von einem Computer.

## Beispiele

### Beispiel 1
```
PS C:\> Uninstall-AzureRm
```

Wenn Sie diesen Befehl ausführen, werden alle AzureRm-Module vom Computer für die Version von PowerShell entfernt, in der das Cmdlet ausgeführt wird.
