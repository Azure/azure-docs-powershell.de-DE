# Clear-AzDefault

## Synopsis
Löscht die vom Benutzer im aktuellen Kontext festgelegten Standardeinstellungen.

## Syntax

```
Clear-AzDefault [-ResourceGroup] [-PassThru] [-Force] [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Clear-AzDefault-Cmdlet entfernt die vom Benutzer festgelegten Standardeinstellungen, abhängig von den vom Benutzer angegebenen Schalterparametern.

## Beispiele

### Beispiel 1
```
PS C:\> Clear-AzDefault
```

Mit diesem Befehl werden alle Standardeinstellungen entfernt, die vom Benutzer im aktuellen Kontext gesetzt wurden.

### Beispiel 1
```
PS C:\> Clear-AzDefault -ResourceGroup
```

Dieser Befehl entfernt die vom Benutzer im aktuellen Kontext gesetzte Standardressourcen Gruppe.
