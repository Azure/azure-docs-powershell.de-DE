# Remove-AzRelayHybridConnection

## Synopsis
Entfernt die HybridConnection aus dem angegebenen HybridConnection-Namespace.

## Syntax

```
Remove-AzRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzRelayHybridConnection** entfernt die HybridConnection aus dem angegebenen Relay-Namespace.

## Beispiele

### Beispiel 1
```
PS C:\> Remove-AzRelayHybridConnection -ResourceGroupName Default-ServiceBus-WestUS -Namespace TestNameSpace-Relay1 -Name TestHybridConnection
```

Entfernt die HybridConnection `TestHybridConnection` aus dem Namespace `TestNameSpace-Relay1` .
