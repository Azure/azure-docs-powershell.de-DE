# Remove-AzRmStorageShare

## Synopsis
Entfernt eine Speicherdatei Freigabe.

## Syntax

### Kontoname (Standard)
```
Remove-AzRmStorageShare [-ResourceGroupName] <String> [-StorageAccountName] <String> -Name <String> [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Kontoobject
```
Remove-AzRmStorageShare -Name <String> -StorageAccount <PSStorageAccount> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ShareResourceId
```
Remove-AzRmStorageShare [-ResourceId] <String> [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Freigebenobjekt
```
Remove-AzRmStorageShare -InputObject <PSShare> [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **New-AzRmStorageShare** " entfernt eine Speicherdatei Freigabe.

## Beispiele

### Beispiel 1: Entfernen einer Speicherdatei Freigabe mit dem Namen des speicherkontos und dem Freigabenamen
```
PS C:\>Remove-AzRmStorageShare -ResourceGroupName "myResourceGroup" -StorageAccountName "myStorageAccount" -Name "myshare"
```

Dieser Befehl entfernt eine Speicherdatei Freigabe mit dem Namen des speicherkontos und dem Freigabenamen.

### Beispiel 2: Entfernen einer Speicherdatei Freigabe mit dem Speicherkonto Objekt und dem Freigabenamen
```
PS C:\>$accountObject = Get-AzStorageAccount -ResourceGroupName "myResourceGroup" -StorageAccountName "myStorageAccount"
PS C:\>Remove-AzRmStorageShare -StorageAccount $accountObject -Name "myshare"
```

Dieser Befehl entfernt eine Speicherdatei Freigabe mit dem Speicherkonto Objekt und dem Freigabenamen.

### Beispiel 3: Entfernen aller Speicherdatei Freigaben in einem Speicherkonto mit Pipeline
```
PS C:\>Get-AzStorageShare -ResourceGroupName "myResourceGroup" -StorageAccountName "myStorageAccount" | Remove-AzRmStorageShare -Force
```

Dieser Befehl entfernt alle Speicherdatei Freigaben in einem Speicherkonto mit Pipeline.
