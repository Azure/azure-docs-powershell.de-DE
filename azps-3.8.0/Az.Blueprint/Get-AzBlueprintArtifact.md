# Get-AzBlueprintArtifact

## Synopsis
Abrufen von Artefakten aus einer Blueprint-Definition

## Syntax

```
Get-AzBlueprintArtifact [-Name <String>] -Blueprint <PSBlueprintBase> [-BlueprintVersion <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Abrufen von Artefakten aus einer Blueprint-Definition Wenn keine Blueprint-Definitionsversion angegeben ist, wird die Entwurfsversion abgerufen. In dem Fall, in dem es keine Entwurfsversion gibt, wird die neueste veröffentlichte Blueprint-Definition zurückgegeben.
