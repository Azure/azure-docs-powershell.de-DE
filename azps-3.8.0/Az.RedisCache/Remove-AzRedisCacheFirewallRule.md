# Remove-AzRedisCacheFirewallRule

## Synopsis
Entfernen einer Firewall-Regel aus einem Zwischenspeicher mit einem Zwischenspeicher.

## Syntax

### NormalParameterSet (Standard)
```
Remove-AzRedisCacheFirewallRule [-ResourceGroupName <String>] -Name <String> -RuleName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### PSRedisFirewallRuleObject
```
Remove-AzRedisCacheFirewallRule -InputObject <PSRedisFirewallRule> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Entfernen einer Firewall-Regel aus einem Zwischenspeicher mit einem Zwischenspeicher.

## Beispiele

### Beispiel 1: Entfernen einer einzelnen Firewall-Regel
```
PS C:\>Remove-AzRedisCacheFirewallRule -Name "mycache" -RuleName "ruleone" -PassThru
True
```

Mit diesem Befehl wird eine Firewallregel mit dem Namen "ruleone" aus dem a/a-Cache "myCache" entfernt.
