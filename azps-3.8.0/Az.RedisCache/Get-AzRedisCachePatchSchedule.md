# Get-AzRedisCachePatchSchedule

## Synopsis
Ruft einen Patch-Zeitplan ab.

## Syntax

```
Get-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzRedisCachePatchSchedule** " erhält einen Patch-Terminplan für einen Cache im Azure-a/a-Cache.

## Beispiele

### Beispiel 1: Abrufen des Patch-Zeitplans
```
PS C:\>Get-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06"
```

Dieser Befehl ruft den Patch-Zeitplan aus dem Cache mit dem Namen RedisCache06 ab.
