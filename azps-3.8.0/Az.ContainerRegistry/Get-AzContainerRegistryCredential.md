# Get-AzContainerRegistryCredential

## Synopsis
Ruft die Anmeldeinformationen für eine Container Registrierung ab.

## Syntax

### NameResourceGroupParameterSet (Standard)
```
Get-AzContainerRegistryCredential [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### RegistryObjectParameterSet
```
Get-AzContainerRegistryCredential -Registry <PSContainerRegistry> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzContainerRegistryCredential -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Das Get-AzContainerRegistryCredential-Cmdlet ruft die Anmeldeinformationen für eine Container Registrierung ab.

## Beispiele

### Beispiel 1: Abrufen der Anmeldeinformationen für eine Container Registrierung
```powershell
PS C:\>Get-AzContainerRegistryCredential -ResourceGroupName "MyResourceGroup" -Name "MyRegistry"

Username   Password                         Password2
--------   --------                         ---------
MyRegistry +Y+==B==KdT=YV=ZgH=p/zQ/e1sNQq/d //JRPkgxx+r+z/ztU=R//E==vum=pRKL
```

Dieser Befehl ruft die Anmeldeinformationen für die angegebene Container Registrierung ab.
Administrator Benutzer muss für die Container Registrierung \` myregistry aktiviert sein \` , um Anmeldeinformationen abzurufen.
