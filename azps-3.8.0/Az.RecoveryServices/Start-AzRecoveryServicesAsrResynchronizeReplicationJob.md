# Start-AzRecoveryServicesAsrResynchronizeReplicationJob

## Synopsis
Startet die erneute Synchronisierung der Replikation.

## Syntax

### Standard (Standard)
```
Start-AzRecoveryServicesAsrResynchronizeReplicationJob -ReplicationProtectedItem <ASRReplicationProtectedItem>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Start-AzRecoveryServicesAsrResynchronizeReplicationJob -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das **Start-AzRecoveryServicesAsrResynchronizeReplicationJob-** Cmdlet startet die erneute Synchronisierung der Replikation für das angegebene geschützte Element, wenn der geschützte Zustand für die erneute Synchronisierung erforderlich ist.

## Beispiele

### Beispiel 1
```
PS C:\> Start-AzRecoveryServicesAsrResynchronizeReplicationJob -ReplicationProtectedItem $rpi
```

Startet den Auftrag, um die Replikation beim übergebenen Replikations geschützten Element erneut zu synchronisieren.
