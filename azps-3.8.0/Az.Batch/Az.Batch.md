# Az.Batch-Modul
## Beschreibung
Mit den Azure Batch-Cmdlets im Azure-Modul können Sie Microsoft Azure-Batch Dienste in Azure PowerShell verwalten.

## Az.Batch-Cmdlets
### [Deaktivieren-AzBatchAutoScale](Disable-AzBatchAutoScale.md)
Deaktiviert die automatische Skalierung eines Pools.

### [Deaktivieren-AzBatchComputeNodeScheduling](Disable-AzBatchComputeNodeScheduling.md)
Deaktiviert die Vorgangsplanung auf dem angegebenen Compute-Knoten.

### [Deaktivieren-AzBatchJob](Disable-AzBatchJob.md)
Deaktiviert eine Stapelverarbeitung.

### [Deaktivieren-AzBatchJobSchedule](Disable-AzBatchJobSchedule.md)
Deaktiviert einen stapelverarbeitungsauftrags Zeitplan.

### [Enable-AzBatchAutoScale](Enable-AzBatchAutoScale.md)
Aktiviert die automatische Skalierung eines Pools.

### [Enable-AzBatchComputeNodeScheduling](Enable-AzBatchComputeNodeScheduling.md)

Aktiviert eine Stapelverarbeitung.

### [Enable-AzBatchJobSchedule](Enable-AzBatchJobSchedule.md)
Aktiviert einen Stapel Auftragszeitplan.

### [Enable-AzBatchTask](Enable-AzBatchTask.md)
