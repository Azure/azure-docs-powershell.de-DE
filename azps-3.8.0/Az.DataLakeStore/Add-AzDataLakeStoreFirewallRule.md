# Add-AzDataLakeStoreFirewallRule

## Synopsis
Fügt dem angegebenen Data Lake Store-Konto eine Firewallregel hinzu.

## Syntax

```
Add-AzDataLakeStoreFirewallRule [-Account] <String> [-Name] <String> [-StartIpAddress] <String>
 [-EndIpAddress] <String> [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **Add-AzDataLakeStoreFirewallRule** wird dem angegebenen Data Lake Store-Konto eine Firewallregel hinzugefügt.

## Beispiele

### Beispiel 1: Hinzufügen einer neuen Firewallregel zu einem Data Lake Store-Konto
```
PS C:\> Add-AzDataLakeStoreFirewallRule -AccountName "ContosoADL" -Name MyRule -StartIpAddress "127.0.0.1" -EndIpAddress "127.0.0.2"
```

Dadurch wird eine neue Firewall-Regel mit dem Namen "myrule" im Konto "ContosoADL" mit einem IP-Bereich von 127.0.0.1-127.0.0.2 erstellt.
