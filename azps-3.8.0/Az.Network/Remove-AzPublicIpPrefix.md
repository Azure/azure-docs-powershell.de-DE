# Remove-AzPublicIpPrefix

## Synopsis
Entfernt ein öffentliches IP-Präfix

## Syntax

### RemoveByNameParameterSet (Standard)
```
Remove-AzPublicIpPrefix -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByResourceIdParameterSet
```
Remove-AzPublicIpPrefix -ResourceId <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByInputObjectParameterSet
```
Remove-AzPublicIpPrefix -InputObject <PSPublicIpPrefix> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzPublicIpPrefix** entfernt ein Azure Public IP-Präfix, solange keine öffentlichen IP-Adressen zugewiesen sind.

## Beispiele

### Beispiel 1
```powershell
PS C:\> Remove-AzPublicIpPrefix -Name $prefixName -ResourceGroupName $rgName
```

Entfernt das öffentliche IP-Präfix mit dem Namen $prefixName aus der Ressourcengruppe $rgName
