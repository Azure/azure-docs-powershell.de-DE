# Add-AzVirtualHubRouteTable

## Synopsis
Erstellt eine virtuelle Hub-Route-Tabellen Ressource, die ein untergeordnetes Element von VirtualHub ist.

## Syntax

```
Add-AzVirtualHubRouteTable -Name <String> -Route <PSVirtualHubRoute[]> -Connection <String[]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Die Ressource für virtuelle Hub-Routingtabelle enthält die Liste der Routen sowie eine Liste der Verbindungen, an die Sie angefügt werden kann, und wird verwendet, um den Datenverkehr in einem virtuellen Hub weiterzuleiten.
