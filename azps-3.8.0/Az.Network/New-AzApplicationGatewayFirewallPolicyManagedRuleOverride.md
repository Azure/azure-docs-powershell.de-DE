# New-AzApplicationGatewayFirewallPolicyManagedRuleOverride

## Synopsis
Erstellt einen managedRuleOverride-Eintrag für den RuleGroupOverrideGroup-Eintrag.

## Syntax

```
New-AzApplicationGatewayFirewallPolicyManagedRuleOverride -RuleId <String> [-State <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Der **New-AzApplicationGatewayFirewallPolicyManagedRuleOverride** erstellt einen ruleOverride-Eintrag.

## Beispiele

### Beispiel 1
```powershell
PS C:\> ruleOverrideEntry = New-AzApplicationGatewayFirewallPolicyManagedRuleOverride -RuleId $ruleId -State Disabled
```

Erstellt einen ruleOverride-Eintrag mit der Option "Lineal" als $ruleId und Zustand als deaktiviert.
