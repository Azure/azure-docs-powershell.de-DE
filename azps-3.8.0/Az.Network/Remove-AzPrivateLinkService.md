# Remove-AzPrivateLinkService

## Synopsis
Entfernt einen privaten Link Dienst

## Syntax

```
Remove-AzPrivateLinkService -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Remove-AzPrivateLinkService** " entfernt einen privaten Link Dienst

## Beispiele

### Beispiel 1
```
Remove-AzPrivateLinkService -ResourceGroupName TestResourceGroup -Name TestPrivateLinkService
```

In diesem Beispiel wird ein privater Link Dienst mit dem Namen TestPrivateLinkService entfernt.
