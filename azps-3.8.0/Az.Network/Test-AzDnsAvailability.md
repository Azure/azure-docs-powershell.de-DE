# Test-AzDnsAvailability

## Synopsis
Überprüft, ob ein Domänenname in der cloudapp.Azure.com-Zone für die Verwendung verfügbar ist.

## Syntax

```
Test-AzDnsAvailability -DomainNameLabel <String> -Location <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Überprüft, ob ein Domänenname in der cloudapp.Azure.com-Zone für die Verwendung verfügbar ist.

## Beispiele

### Beispiel 1: Überprüfen Sie, ob contoso.westus.cloudapp.Azure.com zur Verwendung verfügbar ist.
```
Test-AzDnsAvailability -DomainNameLabel contoso -Location westus
```
