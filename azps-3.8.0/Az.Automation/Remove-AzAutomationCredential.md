# Remove-AzAutomationCredential

## Synopsis
Entfernt eine Automatisierungs Anmeldeinformationen.

## Syntax

```
Remove-AzAutomationCredential [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **Remove-AzAutomationCredential** werden Anmeldeinformationen aus Azure Automation entfernt.

## Beispiele

### Beispiel 1: Entfernen von Anmeldeinformationen
```
PS C:\>Remove-AzAutomationCredential -AutomationAccountName "Contoso17" -Name "ContosoCredential" -ResourceGroupName "ResourceGroup01"
```

Dieser Befehl entfernt die Anmeldeinformationen mit dem Namen ContosoCredential im Azure-Automatisierungs Konto mit dem Namen Contoso17.
