# Set-AzAutomationSchedule

## Synopsis
Ändert einen Automatisierungs Zeitplan.

## Syntax

```
Set-AzAutomationSchedule [-Name] <String> [-IsEnabled <Boolean>] [-Description <String>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **festlegen-AzAutomationSchedule** " ändert einen Zeitplan in Azure-Automatisierung.

## Beispiele

### Beispiel 1: Ändern eines Zeitplans
```
PS C:\>Set-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule01" -Description "Automation Schedule" -ResourceGroupName "ResourceGroup01"
```

Mit diesem Befehl wird die Beschreibung des Zeitplans mit dem Namen Schedule01 geändert.
