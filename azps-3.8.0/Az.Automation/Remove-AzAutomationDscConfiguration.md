# Remove-AzAutomationDscConfiguration

## Synopsis
Entfernt DSC-Konfigurationen aus der Automatisierung.

## Syntax

```
Remove-AzAutomationDscConfiguration [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzAutomationDscConfiguration** entfernt APS-Konfigurationen (Desired State Configuration, DSC) aus Azure Automation.
