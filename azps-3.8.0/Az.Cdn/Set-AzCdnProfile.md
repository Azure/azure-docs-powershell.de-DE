# Set-AzCdnProfile

## Synopsis
Aktualisiert ein CDN-Profil.

## Syntax

```
Set-AzCdnProfile -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Satz-AzCdnProfile** " aktualisiert ein Azure Content Delivery Network (CDN)-Profil.
