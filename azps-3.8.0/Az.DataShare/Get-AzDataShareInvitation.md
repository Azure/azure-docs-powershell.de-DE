# Get-AzDataShareInvitation

## Synopsis
Ruft Informations Einladung zur Datenfreigabe ab.

## Syntax

### ByFieldsParameterSet (Standard)
```
Get-AzDataShareInvitation -ResourceGroupName <String> -AccountName <String> -ShareName <String>
 [-Name <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Get-AzDataShareInvitation -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzDataShareInvitation** " Ruft Informationen zu Einladungen auf, die in der Datenfreigabe hinzugef체gt wurden. Wenn Sie den Namen der Einladung angeben, ruft dieses Cmdlet Informationen zur Einladung ab. Wenn Sie keinen Namen angeben, ruft dieses Cmdlet Informationen zu allen Einladungen in einer Freigabe ab.
