# Remove-AzDataLakeAnalyticsCatalogCredential

## Synopsis
Löscht eine Azure Data Lake-Analyse Anmeldeinformationen.

## Syntax

```
Remove-AzDataLakeAnalyticsCatalogCredential [-Account] <String> [-DatabaseName] <String> [-Name] <String>
 [[-Password] <PSCredential>] [-Recurse] [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Remove-AzDataLakeAnalyticsCatalogCredential-Cmdlet löscht eine Azure Data Lake Analytics-Katalog Anmeldeinformationen.

## Beispiele

### Beispiel 1: Entfernen von Anmeldeinformationen
```
PS C:\> Remove-AzDataLakeAnalyticsCatalogCredential -AccountName "ContosoAdla" `
                      -DatabaseName "DatabaseName" `
                      -Name "CredName"
```

Mit diesem Befehl werden die Anmeldeinformationen für den angegebenen Data Lake Analytics-Katalog entfernt.
