## Beschreibung
Mit dem **Submit-AzDataLakeAnalyticsJob-** Cmdlet wird ein Azure Data Lake-Analyse Auftrag übermittelt.

## Beispiele

### Beispiel 1: Einreichen eines Auftrags
```
PS C:\>Submit-AzDataLakeAnalyticsJob -Account "ContosoAdlAccount" -Name "New Job" -ScriptPath $LocalScriptPath -AnalyticsUnits 32
```

Mit diesem Befehl wird ein Data Lake-Analyse Auftrag übermittelt.

### Beispiel 2: Übermitteln eines Auftrags mit Skriptparametern
```
PS C:\>$parameters = [ordered]@{}
$parameters["Department"] = "Sales"
$parameters["NumRecords"] = 1000
$parameters["StartDateTime"] = (Get-Date).AddDays(-14)
Submit-AzDataLakeAnalyticsJob -Account "ContosoAdlAccount" -Name "New Job" -ScriptPath $LocalScriptPath -AnalyticsUnits 32 -ScriptParameter $parameters
```

U-SQL-Skriptparameter werden über den Hauptskript Inhalten vorangestellt, beispielsweise: Declare @Department String = "Sales"; DECLARE @NumRecords int = 1000; DECLARE @StartDateTime DateTime = New DateTime (2017; 12; 6; 0; 0; 0; 0);
