# AZ. OperationalInsights-Modul
## Beschreibung
In diesem Thema werden Hilfethemen zu den Azure-Cmdlets für operationelle Einblicke angezeigt.

## AZ. OperationalInsights-Cmdlets
### [Deaktivieren-AzOperationalInsightsIISLogCollection](Disable-AzOperationalInsightsIISLogCollection.md)
Beendet die Sammlung von IIS-Protokollen von Computern.

### [Deaktivieren-AzOperationalInsightsLinuxCustomLogCollection](Disable-AzOperationalInsightsLinuxCustomLogCollection.md)
Beendet die Sammlung benutzerdefinierter Protokolle von Linux-Computern.

### [Deaktivieren-AzOperationalInsightsLinuxPerformanceCollection](Disable-AzOperationalInsightsLinuxPerformanceCollection.md)
Beendet die Sammlung von Leistungsindikatoren auf Linux-Computern.

### [Deaktivieren-AzOperationalInsightsLinuxSyslogCollection](Disable-AzOperationalInsightsLinuxSyslogCollection.md)
Beendet die Sammlung von syslog-Daten von Linux-Computern.

### [Enable-AzOperationalInsightsIISLogCollection](Enable-AzOperationalInsightsIISLogCollection.md)
Startet die Sammlung von IIS-Protokollen von Computern in einem Arbeitsbereich.

### [Enable-AzOperationalInsightsLinuxCustomLogCollection](Enable-AzOperationalInsightsLinuxCustomLogCollection.md)
Startet eine Sammlung von benutzerdefinierten Protokollen von Linux-Computern.

### [Enable-AzOperationalInsightsLinuxPerformanceCollection](Enable-AzOperationalInsightsLinuxPerformanceCollection.md)
Startet die Sammlung von Leistungsindikatoren auf Linux-Computern.

### [Enable-AzOperationalInsightsLinuxSyslogCollection](Enable-AzOperationalInsightsLinuxSyslogCollection.md)
Startet die Sammlung von syslog-Daten von Linux-Computern.

### [Get-AzOperationalInsightsDataSource](Get-AzOperationalInsightsDataSource.md)
Abrufen von DataSources unter Azure Log Analytics-Arbeitsbereich

### [Get-AzOperationalInsightsIntelligencePack](Get-AzOperationalInsightsIntelligencePack.md)
Ruft die verfügbaren Intelligence Packs ab.

### [Get-AzOperationalInsightsLinkTarget](Get-AzOperationalInsightsLinkTarget.md)
Ruft Konten ab, die keinem Abonnement zugeordnet sind.

### [Get-AzOperationalInsightsSavedSearch](Get-AzOperationalInsightsSavedSearch.md)
Gibt alle gespeicherten Suchvorgänge für einen angegebenen Arbeitsbereich zurück.

### [Get-AzOperationalInsightsSavedSearchResult](Get-AzOperationalInsightsSavedSearchResult.md)
Gibt die Ergebnisse einer Abfrage zurück.

### [Get-AzOperationalInsightsSchema](Get-AzOperationalInsightsSchema.md)
Gibt das einem Arbeitsbereich zugeordnete Schema zurück.

### [Get-AzOperationalInsightsSearchResult](Get-AzOperationalInsightsSearchResult.md)
Gibt Suchergebnisse basierend auf den angegebenen Parametern zurück.

### [Get-AzOperationalInsightsStorageInsight](Get-AzOperationalInsightsStorageInsight.md)
Ruft Informationen zu einem Speicher Einblick ab.

### [Get-AzOperationalInsightsWorkspace](Get-AzOperationalInsightsWorkspace.md)
Ruft Informationen zu einem Arbeitsbereich ab.

### [Get-AzOperationalInsightsWorkspaceManagementGroup](Get-AzOperationalInsightsWorkspaceManagementGroup.md)
Ruft Details zu Verwaltungsgruppen ab, die mit einem Arbeitsbereich verbunden sind.

### [Get-AzOperationalInsightsWorkspaceSharedKey](Get-AzOperationalInsightsWorkspaceSharedKey.md)
Ruft die freigegebenen Schlüssel für einen Arbeitsbereich ab.

### [Get-AzOperationalInsightsWorkspaceUsage](Get-AzOperationalInsightsWorkspaceUsage.md)
Ruft die Verwendungsdaten für einen Arbeitsbereich ab.

### [Invoke-AzOperationalInsightsQuery](Invoke-AzOperationalInsightsQuery.md)
Gibt Suchergebnisse basierend auf den angegebenen Parametern zurück.

### [Neu – AzOperationalInsightsApplicationInsightsDataSource](New-AzOperationalInsightsApplicationInsightsDataSource.md)
Sammeln von Protokollen von angegebenen Application-Insights Anwendung.

### [Neu – AzOperationalInsightsAzureActivityLogDataSource](New-AzOperationalInsightsAzureActivityLogDataSource.md)
Das Azure-Aktivitätsprotokoll wird von einem bestimmten Abonnement erfasst.

### [Neu – AzOperationalInsightsComputerGroup](New-AzOperationalInsightsComputerGroup.md)
Erstellt eine Computergruppe.

### [Neu – AzOperationalInsightsCustomLogDataSource](New-AzOperationalInsightsCustomLogDataSource.md)
Definiert eine benutzerdefinierte Protokoll Sammlungs Richtlinie.

### [Neu – AzOperationalInsightsLinuxPerformanceObjectDataSource](New-AzOperationalInsightsLinuxPerformanceObjectDataSource.md)
Fügt allen Linux-Computern in einem Arbeitsbereich Leistungsindikatoren hinzu.

### [Neu – AzOperationalInsightsLinuxSyslogDataSource](New-AzOperationalInsightsLinuxSyslogDataSource.md)
Fügt eine Datenquelle zu Linux-Computern hinzu.

### [Neu – AzOperationalInsightsSavedSearch](New-AzOperationalInsightsSavedSearch.md)
Erstellt eine neue gespeicherte Suche mit den angegebenen Parametern.

### [Neu – AzOperationalInsightsStorageInsight](New-AzOperationalInsightsStorageInsight.md)
Erstellt einen Speicher Einblick in einen Arbeitsbereich.

### [Neu – AzOperationalInsightsWindowsEventDataSource](New-AzOperationalInsightsWindowsEventDataSource.md)
Sammelt Ereignisprotokolle von Computern, auf denen das Windows-Betriebssystem ausgeführt wird.

### [Neu – AzOperationalInsightsWindowsPerformanceCounterDataSource](New-AzOperationalInsightsWindowsPerformanceCounterDataSource.md)
Fügt eine Windows-Leistungsindikator-Datenquelle für verbundene Computer hinzu, auf denen das Windows-Betriebssystem ausgeführt wird.

### [Neu – AzOperationalInsightsWorkspace](New-AzOperationalInsightsWorkspace.md)
Erstellt einen Arbeitsbereich.

### [Remove-AzOperationalInsightsDataSource](Remove-AzOperationalInsightsDataSource.md)
Löscht eine Datenquelle.

### [Remove-AzOperationalInsightsSavedSearch](Remove-AzOperationalInsightsSavedSearch.md)

### [Remove-AzOperationalInsightsStorageInsight](Remove-AzOperationalInsightsStorageInsight.md)
Entfernt eine Speicher Einsicht.

### [Remove-AzOperationalInsightsWorkspace](Remove-AzOperationalInsightsWorkspace.md)
Entfernt einen Arbeitsbereich.

### [Satz-AzOperationalInsightsDataSource](Set-AzOperationalInsightsDataSource.md)
Aktualisiert eine Datenquelle.

### [Satz-AzOperationalInsightsIntelligencePack](Set-AzOperationalInsightsIntelligencePack.md)
Aktiviert oder deaktiviert das angegebene Intelligence Pack.

### [Satz-AzOperationalInsightsSavedSearch](Set-AzOperationalInsightsSavedSearch.md)
Aktualisiert eine gespeicherte Suche, die bereits vorhanden ist.

### [Satz-AzOperationalInsightsStorageInsight](Set-AzOperationalInsightsStorageInsight.md)
Aktualisiert eine Speicher-Insight.

### [Satz-AzOperationalInsightsWorkspace](Set-AzOperationalInsightsWorkspace.md)
Aktualisiert einen Arbeitsbereich.
