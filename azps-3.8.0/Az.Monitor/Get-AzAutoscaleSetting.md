# Get-AzAutoscaleSetting

## Synopsis
Ruft AutoScale-Einstellungen ab.

## Syntax

```
Get-AzAutoscaleSetting -ResourceGroupName <String> [-Name <String>] [-DetailedOutput]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzAutoscaleSetting** " ruft alle AutoScale-Einstellungen ab, die einer Ressourcengruppe oder einer angegebenen AutoScale-Einstellung zugeordnet sind.
