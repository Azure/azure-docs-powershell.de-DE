# Remove-AzSqlServerDisasterRecoveryConfiguration

## Synopsis
Entfernt eine SQL Database Server-System Wiederherstellungskonfiguration.

## Syntax

```
Remove-AzSqlServerDisasterRecoveryConfiguration [-VirtualEndpointName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzSqlServerDisasterRecoveryConfiguration** entfernt eine SQL Database Server-System Wiederherstellungskonfiguration.
