# Set-AzDtlAutoShutdownPolicy

## Synopsis
Legt die Richtlinie für das automatische Herunterfahren einer Lab DevTest Labs fest.

## Syntax

### Aktivieren (Standard)
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Enable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Deaktivieren
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Disable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Set-AzDtlAutoShutdownPolicy** " legt die Richtlinie für das automatische Herunterfahren einer Übungseinheit fest, die alle virtuellen Computer in der Übungseinheit zu einer bestimmten Tageszeit automatisch herunterfährt.
Das Cmdlet verwendet die angegebene Ressourcengruppe und den Namen der Übungseinheit, um die Richtlinie festzulegen.
