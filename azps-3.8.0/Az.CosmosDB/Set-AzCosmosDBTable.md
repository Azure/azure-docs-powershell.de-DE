# Set-AzCosmosDBTable

## Synopsis
Legt die CosmosDB-Tabelle fest.

## Syntax

### ByNameParameterSet (Standard)
```
Set-AzCosmosDBTable -ResourceGroupName <String> -AccountName <String> -Name <String> [-Throughput <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Set-AzCosmosDBTable -Name <String> [-Throughput <Int32>] -InputObject <PSDatabaseAccount>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Satz-AzCosmosDBTable** " erstellt eine neue Tabelle oder aktualisiert eine vorhandene Tabelle, wobei die vorhandene Tabelle vollständig ersetzt wird.

## Beispiele

### Beispiel 1
```powershell
PS C:\> Set-AzCosmosDBTable -AccountName {account} -Name {tableName} -ResourceGroupName {rgName}

Name    Id    Resource
{name}  {id}  Microsoft.Azure.Commands.CosmosDB.Models.PSTableGetPropertiesResource
```

Das Ressourcenobjekt enthält _rid, _ts, _ ETag-Eigenschaften der Tabelle.
