# New-AzCosmosDBSqlSpatialSpec

## Synopsis
Erstellt ein neues Objekt vom Typ PSSpatialSpec. Sie kann als Parameterwert für "AzCosmosDBSqlIndexingPolicy" übergeben werden.

## Syntax

```
New-AzCosmosDBSqlSpatialSpec -Path <String> -Type <String[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Erstellt ein Objekt, das dem SpatialSpec der SQL-API entspricht.

## Beispiele

### Beispiel 1
```powershell
PS C:\> New-AzCosmosDBSqlSpatialSpec -Path "/abc" -Type String
Path Types
---- -----
/abc {String}
```
