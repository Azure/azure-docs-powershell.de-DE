# Get-AzCosmosDBCassandraTable

## Synopsis
Ruft eine CosmosDB Cassandra-Tabelle ab.

## Syntax

### ByNameParameterSet (Standard)
```
Get-AzCosmosDBCassandraTable -AccountName <String> -KeyspaceName <String> -ResourceGroupName <String>
 [-Name <String>] [-Detailed] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Get-AzCosmosDBCassandraTable [-Name <String>] -InputObject <PSCassandraKeyspaceGetResults> [-Detailed]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet " **Get-AzCosmosDBCassandraTable** " erstellt ein neues oder aktualisiert einen vorhandenen CosmosDB Cassandra-Keyspace.

## Beispiele

### Beispiel 1
```powershell
PS C:\> $table = Get-AzCosmosDBCassandraTable -ResourceGroupName {resourceGroupName} -AccountName {accountName} -Keyspace {keyspaceName} -Name {name}

Name    Id   Resource
{name}  {id} {resourceObject}
```

{{Get-AzCosmosDBCassandraTable ruft die Eigenschaften eines vorhandenen CassandraKeyspace ab. Sie können die Ressource erweitern, um die DefaultTtl-, Schema-, _etag-, _ts-_rid-Eigenschaften abzurufen.}}
