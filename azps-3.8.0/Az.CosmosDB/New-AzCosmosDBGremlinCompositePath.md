# New-AzCosmosDBGremlinCompositePath

## Synopsis
Erstellt ein neues Objekt vom Typ PSCompositePath. Sie kann als Parameterwert für "AzCosmosDBGremlinGraph" übergeben werden.

## Syntax

```
New-AzCosmosDBGremlinCompositePath [-Path <String>] [-Order <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Beschreibung
Das Objekt, das dem CompositePath der Gremlin-API entspricht.

## Beispiele

### Beispiel 1
```powershell
PS C:\> New-AzCosmosDBGremlinCompositePath -Path "/abc" -Order Ascending

Path Order
---- -----
/abc Ascending
```
