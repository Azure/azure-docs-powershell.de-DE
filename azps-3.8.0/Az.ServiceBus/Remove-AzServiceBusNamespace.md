# Remove-AzServiceBusNamespace

## Synopsis
Entfernt den Namespace aus der angegebenen Ressourcengruppe. 

## Syntax

### NamespacePropertiesSet (Standard)
```
Remove-AzServiceBusNamespace [-ResourceGroupName] <String> [-Name] <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### NamespaceInputObjectSet
```
Remove-AzServiceBusNamespace [-InputObject] <PSNamespaceAttributes> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### NamespaceResourceIdParameterSet
```
Remove-AzServiceBusNamespace [-ResourceId] <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzServiceBusNamespace** entfernt den Namespace aus der angegebenen Ressourcengruppe.

## Beispiele

### Beispiel 1
```
PS C:\> Remove-AzServiceBusNamespace -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1
```

Entfernt den ServiceBus-Namespace `SB-Example1` aus der angegebenen Ressourcengruppe `Default-ServiceBus-WestUS` .

### Beispiel 2,1-Inputobject-using-Variable:
```
PS C:\> $inputobject = Get-AzServiceBusNamespace <params>
PS C:\> Remove-AzServiceBusNamespace -InputObject $inputobject
```

Entfernt den ServiceBus-Namespace, der über die $Inputobject bereitgestellt wird.

### Beispiel 2,2-Inputobject-using Piping:
```
PS C:\> Get-AzServiceBusNamespace <params> | Remove-AzServiceBusNamespace
```

Entfernt den ServiceBus-Namespace mithilfe von Piping.

### Beispiel 3-Resourcen-Nr
```
PS c:\> $ResourceId = (Get-AzResource -ResourceType Microsoft.ServiceBus/namespaces).ResourceId
PS C:\> Remove-AzServiceBusNamespace -ResourceId $resourceid
```

Entfernt den ServiceBus-Namespace, der über die Arm-ID bereitgestellt wird, in $ResourceId für-Resourcen-Parameter oder durch Rohrleitungen.
