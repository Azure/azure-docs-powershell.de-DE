# AZ. DataBox-Modul
## Beschreibung
Modul zum Erstellen, Abbrechen, löschen und Abrufen von DataBox-Aufträgen

## AZ. DataBox-Cmdlets
### [Get-AzDataBoxCredential](Get-AzDataBoxCredential.md)
Ruft die Datenbox-Anmeldeinformationen eines bestimmten Auftrags ab.

### [Get-AzDataBoxJob](Get-AzDataBoxJob.md)
Ruft Informationen zu DataBox-Aufträgen ab.

### [Neu – AzDataBoxJob](New-AzDataBoxJob.md)
Erstellt einen neuen DataBox-Auftrag unter Verwendung der angegebenen Parameter

### [Remove-AzDataBoxJob](Remove-AzDataBoxJob.md)
Löscht den DataBox-Auftrag

### [Stopp-AzDataBoxJob](Stop-AzDataBoxJob.md)
Bricht einen laufenden DataBox-Auftrag ab, wenn sich der Auftrag im Status "Abbrechen" befindet.
