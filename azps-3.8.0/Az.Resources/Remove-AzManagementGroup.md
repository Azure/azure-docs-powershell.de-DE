# Remove-AzManagementGroup

## Synopsis
Entfernt eine Verwaltungsgruppe

## Syntax

### GroupOperations (Standard)
```
Remove-AzManagementGroup [-GroupName] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ManagementGroupObject
```
Remove-AzManagementGroup -InputObject <PSManagementGroup> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Beschreibung
Das Cmdlet **Remove-AzManagementGroup** löscht eine Verwaltungsgruppe.

## Beispiele

### Beispiel 1 – Entfernen einer Verwaltungsgruppe
```
PS C:\> Remove-AzManagementGroup -GroupName "TestGroup"
```

### Beispiel 2 – Entfernen einer Verwaltungsgruppe durch Piping PSManagementGroup-Objekt
```
PS C:\> Get-Remove-AzManagementGroup -GroupName "TestGroup" | Remove-AzManagementGroup
```
