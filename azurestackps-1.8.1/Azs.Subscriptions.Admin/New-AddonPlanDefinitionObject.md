# New-AddonPlanDefinitionObject

## Synopsis
Enthält den Namen des gewünschten Plans, mit dem ein Angebot verknüpft oder dessen Verknüpfung aufgehoben werden soll.

## Syntax

```
New-AddonPlanDefinitionObject [[-PlanId] <String>] [[-MaxAcquisitionCount] <Int64>] [<CommonParameters>]
```

## Beschreibung
Enthält den Namen des gewünschten Plans, mit dem ein Angebot verknüpft oder dessen Verknüpfung aufgehoben werden soll.
