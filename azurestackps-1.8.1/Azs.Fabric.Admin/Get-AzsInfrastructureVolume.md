# Get-AzsInfrastructureVolume

## Synopsis
Gibt eine Liste aller Speicher-Volumes an einem Speicherort zurück.

## Syntax

### Liste (Standard)
```
Get-AzsInfrastructureVolume -StoragePool <String> -StorageSystem <String> [-Location <String>]
 [-ResourceGroupName <String>] [-Filter <String>] [-Skip <Int32>] [-Top <Int32>] [<CommonParameters>]
```

### Erhalten
```
Get-AzsInfrastructureVolume -Name <String> -StoragePool <String> -StorageSystem <String> [-Location <String>]
 [-ResourceGroupName <String>] [<CommonParameters>]
```

### ResourceId
```
Get-AzsInfrastructureVolume -ResourceId <String> [<CommonParameters>]
```

## Beschreibung
Gibt eine Liste aller Speicher-Volumes an einem Speicherort zurück.

## Beispiele

### Beispiel 1
```
Get-AzsInfrastructureVolume -StoragePool SU1_Pool -StorageSystem S-Cluster.azurestack.local
```

Sie erhalten eine Liste aller Speicher-Volumes an einem bestimmten Ort.

### Beispiel 2
```
Get-AzsInfrastructureVolume -StoragePool SU1_Pool -StorageSystem S-Cluster.azurestack.local -Name a42d219b
```

Besorgen Sie sich ein Speichervolumen nach Name an einem bestimmten Ort.
